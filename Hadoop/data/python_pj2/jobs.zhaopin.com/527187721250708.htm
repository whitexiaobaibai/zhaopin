岗位职责：

1、负责BI系统搭建、BI模型设计及开发

3、完成报表开发（自助式报表+传统报表）

任职要求：

1、扎实的Python编程基础，热爱编程、具有良好的代码风格，并熟悉Linux下Python开发

2、一年以上Python Web框架(Flask/Django)开发经验，掌握RESTful API的开发思想

3、熟悉 SQL

4、良好的自学能力、独立解决问题的能力及沟通与团队协作能力

5、使用过Tableau、SuperSet、Cognos、BIEE、Qlikview等BI工具者优先，有相关BI工具开发经验更佳

6、熟悉 Javascript/CSS/HTML5，JQuery、D3.js、Echarts等优先


工作地址：

实力：高思教育是中国领先的教育产品创新公司、总部位于北京市海淀区北四环中路，旗下拥有1000余名员工、700余名全职教师与产品研发人员（其中一半以上为北大、清华的顶级教育产品经理），在北京拥有31个产品创新实验教学点，在北四环五道口拥有11层的现代化写字楼“高思教育大厦“，在过去6年中，北京市有50万的孩子在这里学习，全国有100万的孩子目前在使用其研发的教育产品，其连续6年增长率超过100%，这个增长率是行业增长率的3倍、其2011年与2014年先后2次获得著名投资机构”晨兴创投“逾千万美元的产品研发资金投资,2015年8月份，高思获得了来自昆吾九鼎投资的4亿元融资，帮助高思进一步拓展和完善现有的K12培训业务，2017年2月份高思教育已登录新三板。

 业务：高思教育为3-18岁年龄层的孩子提供个性化的教育产品方案及所有与教育相关的服务，旗下拥有“思泉语文”、“高思数学”、“高思VIP一对一”、“中学理科”“一路向北”、互联网教育产品：“爱学习”、“爱上古诗”“爱尖子”、 “爱提分” 等多个全国著名的学习子品牌，这些产品目前已经被大量的中小学辅导机构及公立学校引进，其经验证普遍提升了孩子的学习体验和学习效果。
