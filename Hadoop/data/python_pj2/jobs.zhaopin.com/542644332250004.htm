举报收藏

岗位职责：

1.  负责数据平台的设计、开发、维护与优化，不断创新，满足上层数据运营体系各项需求；

2.  审核数据平台项目总体技术方案，对各项目进行质量评估；

3.  参与应用分析系统的系统分析、设计以及实施工作;

4.  负责分析数据采集需求，全局设计数据处理的技术流程和规范；

5.  负责设计改良数据质量, 根据实际情况解决核心数据质量问题；

6.  研究与跟踪大数据新技术发展方向，主持制定大数据平台技术发展战略规划；

7.  负责大数据部门的技术研究、技术知识培训；

岗位要求：

1.  本科以上学历，5年以上软件和分布式系统研发经验，2年以上大数据系统架构经验；

2.  熟悉Hadoop/Hive/HBase/Spark等的工作原理、系统构建和优化；

3.  熟悉主流数据库技术，精通SQL，有较强的SQL编码及调优经验；

4.  具备Java, Python等开发经验，熟悉数据挖掘和分析的算法；

5.  需要有较强的学习能力、自我管理能力、能够在压力环境下工作；

6.  需要有责任心, 工作积极主动, 思路清晰，擅长沟通、有团队合作精神。


工作地址：

北京市亦庄经济技术开发区荣昌东街甲5号隆盛大厦C座10层 查看职位地图

同源数智(北京)科技有限公司 该公司其他职位

同源数智公司是一家国内领先的大数据技术咨询公司，专注于提供数据技术和模型应用解决方案。目前面向金融、互联网、电商、零售、汽车、交通、制造、医疗、服务等行业客户，主要提供深度商业场景数据模型构建和应用、商业数据挖掘与分析、金融产品量化设计、金融风险控制、BI系统搭建等服务。

公司拥有大量行业成功经验，凝聚行业最先进理念和技术，已与多家数据及互联网金融拥有合作关系。现承接金融及各行业的数据分析和处理项目。
