举报收藏

职位描述：

1、从海量数据中，利用数据挖掘技术，挖出用户/资产的偏好特征、异常行为，用户画像等

2、使用数据挖掘和机器学习识别已知或者未知安全威胁；

3、探索业界和学术界前沿的安全分析。


工作职责：分析业务逻辑，对数据进行数据挖掘和分析，提取关键信息。


招聘要求：

1. 本科以上学历，计算机相关专业，数学专业和研究生优先

2. 掌握常见的机器学习相关算法，并深入理解应用场景；

3. 必须有一年以上代码编写经验，熟悉一种大数据语言R/SCALA/PYTHON

4. 有一个成熟的数据挖掘项目建模经验，特征的提取和算法的选择有方法论

5. 有深入研究画像、图分析和语义分析等领域的优先

6. 踏实勤奋，自我驱动，善于沟通


工作地址：

广州市天河区黄埔大道中路336号御发商务中心6楼（研发中心）

蓝盾信息安全技术股份有限公司 该公司其他职位

蓝盾信息安全技术股份有限公司（蓝盾股份，股票代码：300297）成立于1999年10月，总部在广州。公司是国家信息安全工程研究中心，国家火炬计划重点企业、广东省高新技术企业、广东省双软企业、广东省网络安全技术研究工程中心、国际云安全联盟（CSA）企业成员之一等，是国内最早从事信息安全产品研发、生产、销售，拥有自主知识产权的专业网络安全企业。
十年以来公司致力于网络安全产品的研发，已自主研发出蓝盾防火墙、多功能安全网关(UTM)、账号集中管理、安全审计、入侵检测、漏洞扫描等十个系列，近50个型号的产品，各项产品通过国家公安、安全、保密、军队等权威主管部门检测认证；并以安全产品为基础，建立信息安全保障体系为目标，以等级保护为主线，并提供等级保护咨询、风险评估、安全体系设计、安全规划、策略制定、安全建设与集成、安全运维外包服务、安全托管监控服务、安全培训等全方位的专业安全服务。是以安全产品、安全服务、安全集成的综合性网络安全公司。
“蓝盾”已成为国内知名的网络安全品牌，公司拥有覆盖全国，涉及政府、医疗、金融、军队、能源、交通、教育、流通、邮政、制造等行业的客户群体。公司现已在北京、上海、重庆、深圳、武汉、合肥、石家庄、南宁、哈尔滨、西安等主要大中城市设有分支机构。
