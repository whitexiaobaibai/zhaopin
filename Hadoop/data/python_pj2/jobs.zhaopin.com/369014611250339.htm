举报收藏

岗位职责：

1. 规划咨询：对水利行业信息化建设进行规划咨询，对行业趋势、客户业务、架构、咨询方法论等方面进行跟踪研究。

2. 售前支持：协助销售拜访客户，进行必要的技术支持，制定相应的解决方案并进行讲解；负责销售资料的撰写，帮助销售人员熟悉和理解解决方案和产品的竞争优势；

3. 解决方案编制：根据水利行业客户的特点及业务要求，启发并整理客户需求，引导客户的项目规划思路，形成具有竞争力的方案；包括可行性报告、项目建议书、信息化预算、规划实施方案等等；

4. 投标支持：组织项目整体投标方案的制作，包括必要的技术方案、实施方案、报价方案等；同时具备较强的方案落地能力；

5. 产品规划：针对水利行业市场需求，了解公司现有软硬件产品并应用到解决方案中，协助规划公司软硬件产品。

任职要求：

1. 5年以上从事水利行业信息化工作经验，计算机或水利相关专业本科以上学历；

2. 丰富的山洪预警、防汛抗旱、水资源、农田水利等方向的售前经验，参与过相关项目整体解决方案的设计、编写、应标工作，有相应的工作成果；

3. 热爱方案设计工作，PPT编写与演讲能力强；有创意和执行力，善于学习、思考、分析与总结；

4. 有较好的软件开发背景，熟悉包括信息采集设备、网络、安全、软件、大数据分析等相关IT技术；

5. 身体健康，能够适应经常出差的需要；

6. 工作积极主动，有良好的团队合作精神和敬业精神，能够承受较强的工作压力；

7. 有项目管理经验者优先考虑。


工作地址：

北京市河北省廊坊市燕郊经济技术开发区燕灵路69号中兴科技园1

中兴长天信息技术(南昌)有限公司 该公司其他职位

   中兴长天信息技术有限公司成立于2008年8月，是中兴通讯集团旗下专注智慧水利（水务）业务的高科技公司。

  中兴长天致力于以客户为中心，用“物联·感知·云”的创新理念，为行业用户提供系统级智能感知硬件产品、大数据云平台软件以及符合行业信息化特点的软硬件一体化解决方案，是水利等行业应用的系统提供商。

  在水利信息化方面，中兴长天于2007年在全国率先完成了“江西山洪灾害”系统的省、市、县三级部署与运营。2008年水利部以此技术方案为蓝本，开展了全国试点工作，中兴长天山洪灾害系统市场占有率全国领先。

   目前已形成以水利应用为主，兼顾其他行业的具有RTU 无线通讯和自组网技术为核心的感知硬件、数据网络侧和应用系统等多个层级的系列产品。用户及项目覆盖全国26个省市自治区的各级水文、防汛抗旱减灾、水资源监测与管理、农田节水灌溉、中小河流、灌渠信息化、城市水务、海绵城市、采砂管理、水库管理、水利资产管理与运维、电站、农场、林场等。

  公司研发团队具有十多年信息化建设及项目开发经验。部分硬件产品获得了水利部科技推广中心优秀产品称号，软件产品曾获得国际软件博览会“创新奖”、“金奖”，是国家火炬计划重点企业，获得了国家重点新产品三项，省级科学技术进步奖，省级优秀新产品奖等。参与国家六项标准制定，已有软件著作权四十四项、申请专利十八项。
