  • 职位描述

举报收藏

职责：

1. 学习、分析和掌握统计分析、数据挖掘及机器学习算法；

2. 数据分析和面向应用的数据建模；

3. 基于统计分析、数据挖掘及机器学习算法开发应用算法（一般以开源算法为基础开发）

4. 应用算法的原型实现、评估和优化（可以使用R、Matlab等工具语言，具有分布式程序设计和实现能力最好）

5. 应用算法分布式设计（主要是基于MapReduce模型进行并行化设计）或辅助设计；

6. 应用算法分布式实现支撑（支持平台工程师实现算法），具有实现能力最好；

任职资格：

1. 扎实的数学基础和功底，（熟练）掌握软件开发知识或技能，在相关领域工作3年以上，学历为本科（包含）以上；

2. 精通统计分析、数据挖掘及机器学习算法（包括各种开源和典型算法），尤其是基于MapReduce模型的相关算法（例如Mahout、基于Spark的机器学习和图算法等等）；

3. 数据建模和基于基础算法的应用开发能力强，具备快速原型实现能力；

4. 具有基于MapReduce模型的并行计算的设计能力；

5. 熟悉Hadoop, Spark的体系架构和运行原理，具有基于MapReduce模型，设计和开发经验尤佳；

6. 熟练使用Spark开发者优先；
