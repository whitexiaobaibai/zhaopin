举报收藏

岗位职责：

1、参与项目的需求分析、概要设计和详细设计，撰写相关技术文档； 

2、负责系统框架的设计，性能优化、架构扩展及技术创新；

3、具有良好的协调、沟通能力，能够根据客户需求制定相关技术解决方案。


任职要求：

1、三年以上.NET、C#开发和设计经验，熟悉.Net Framework、CLR和面向对象理念，熟练使用 ASP.NET 进行 Web 开发； 
2、熟悉SQL SERVER，能编写复杂业务的存储过程、函数等；
3、对大数据处理和优化有经验者优先； 
4、熟练使用JavaScript、JQuery进行脚本编程； 
5、熟悉ASP.NET MVC、Entity Framework、微信接口开发；
6、熟悉OR-Mapping，MVC,IOC,AOP等技术,对相关开源框架有一定了解； 
7、具有良好的代码编写风格，熟悉Web程序代码安全； 
8、工作认真负责，能自觉主动合理安排好自身的工作，并具有良好的沟通能力及团队协作思维，强烈的进取心、求知欲。 


工作时间与假期管理：

1、公司实行五天8小时工作制，周末双休； 

2、公司设有活动室、休闲吧，提供下午茶、节日福利，团队旅游等； 

公司地址：深圳市龙华新区淘金地大厦E座810 

乘车路线：港铁4号线（上塘地铁站A出口 直行150米）
