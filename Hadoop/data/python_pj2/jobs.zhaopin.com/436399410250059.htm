举报收藏

岗位职责：

1. 互联网产品销售：云主机销售、服务器托管，服务器租用，IDC增值业务,主要通过面对面的跟踪客户，开发潜在客户完成销售指标建立和维护良好的客户关系。 

2. 负责公司ISP、IDC机房的服务器托管及增值业务的洽谈及合同签订，管理团队并完成公司下达的销售任务，提高销售业绩； 

3. 积极开拓新市场和新客户，维护客户关系，扩大公司ISP、IDC产品的市场占有率； 

4. 收集和分析市场数据，并定期反馈最新信息; 

5. 根据客户需求和市场变化，对公司的产品和服务提出改进建议； 

6. 负责开拓ISP、IDC业务，扩展现有客户的业务范围； 

7. 开拓并实施公司IT服务新产品及销售计划; 


任职要求：

学力不限，敢于挑战销售高薪。

1. 营销、市场或计算机等相关专业,有互联网行业产品知识者优先； 

2. 有IDC行业销售经验优先 

3. 较强的统筹协调能力、数据统计分析能力、团队协作能力，思维敏捷，较强观察能力和应变能力； 

4. 积极热情、敬业爱岗、心理素质佳、较强的抗压能力。
