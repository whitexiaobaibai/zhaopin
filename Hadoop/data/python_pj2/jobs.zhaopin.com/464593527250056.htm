举报收藏

岗位职责： 1.负责公司APP产品线平台软件架构与产品研发，以及APP产品的规划设计，撰写详细的产品需求文档及原型设计文档；
2.负责公司APP产品的日常运营维护，根据各类分析、测试结果和用户反馈，持续改进已上线产品，安排产品优先级；
3.负责及时、准确的研究、监控、分析APP产品市场，并提出相应的对策；
4.负责产品线的生命周期管理，提升用户留存率和粘度，养成良好的用户使用习惯；
5.带领APP技术研发团队，完成公司Android/ios APP产品的架构设计、开发与优化工作；
6.遵循软件开发流程，部署技术研发工作，与其他部门协调进度，督导工作的进展和质量；7.参与重点、难点的攻坚，参与或指导下属团队做出技术革新，鼓励创新，并指导创新工作的可实现化；
8.跟进工作领域新技术的发展，定期编写技术手册，定期组织培训，对下属的工作进行指导和解惑。
9.有校招服务和校企合作服务运营平台的设计开发经验优先考虑。


任职要求：精通Android和ios开发平台及框架：spring 框架 或者 spring boot 框架 、spring security jwt、Junit 、rabbit 消息队列（临时队列）、同步异步、mysql数据库、JAVA IO流、工作流、大数据处理、java系统日志管理、熟悉 linux系统 、熟悉java基础知识、maven框架、熟悉前端框架；                                  2、4年以上APP工作经验，有APP应用程序开发经验，有团队管理经验者优先；        3.有较强的沟通协调能力，具备团队意识，有组织能力，工作积极主动，具备高度责任心，工作效率高。

广州市德慷电子有限公司 该公司其他职位

广州市德慷电子有限公司（以下简称“德慷电子”）成立于2003年，是专业从事高校教务服务及教务优化解决方案的科技企业。

 

公司的主要产品包括：行业解决方案、系统工程解决方案及相关软件产品、平台及服务等，可满足教育教学等行业用户的全部业务需求，广泛应用于基础平台、教学管理与服务、学生管理与服务、电子校务、数字资源等高校业务，并能根据客户需求来定制的服务模式，结合自主研发的平台软件，尤其以智能教务解决方案为重点，提供完整的企业级系统解决方案。

 

公司的业务范围已遍布全国各重点高校，积累了丰富的教学教务系统建设、学生管理服务、项目实施、运行维护的经验，为高校信息化建设提供个性化的解决方案并取得了良好的效果。
