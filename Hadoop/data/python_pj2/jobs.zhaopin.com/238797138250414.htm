岗位职责：
1、行业研究：在互联网广告投放方向，针对行业、产品做定向专题研究，输出研究报告。
2、运营分析：针对于行业组、广告主、产品线进行多维度运营分析，模型搭建，对业务提出改善意见。
3、竞争分析：竞争对手信息收集与研究、进行竞争情况分析并输出建议。
4、战略输出：结合行业数据、业界观点和市场趋势，为长期业务发展方向提供战略层面的意见和建议。
5、经营分析：对企业盈利模式，经营细节做推敲，输出改善意见。
任职要求:
1、数学、统计学、计算机相关专业本科及以上学历；
2、有IT大数据分析经验、大型互联网公司数据分析经验、互联网数据建模分析经验优先；
3、对数据有极强的敏感性，善于分析公司业务数据及用户行为数据，发现问题，找出解决方案，给公司运营提供运行监控及建议报告。
