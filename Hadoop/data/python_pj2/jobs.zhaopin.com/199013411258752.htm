举报收藏

岗位职责：

1.负责数据云平台算法架构应用工作，利用机器学习/深度学习等技术挖掘海量数据，构建业务级的用户画像体系

2.和业务团队深入合作，解决在业务发展中遇到的算法模型和数据服务问题，具备一定的前瞻性，影响和指导业务发展

岗位要求：

1. 本科以上学历，计算机/数学相关专业，4年以上数据算法相关工作经验

2. 熟悉大规模数据挖掘、机器学习、自然语言处理等领域技术，深入理解常用的机器学习算法及平台并熟练应用

3. 主导开发过用户画像建模、用户画像系统或DMP系统的开发工作，对基于海量数据分析挖掘用户特征有丰富经验

4. 熟练使用一种以上工程开发语言（C/C++/Java/Go/Python等），掌握基于hadoop等大数据平台工具的开发与设计，熟悉Hive，map/reduce开发

5. 具有良好的沟通能力和团队协作精神，对数据敏感和敏锐的业务直觉，学习能力强

工作地址：

北京市海淀区北四环丹棱街3号中国电子大厦B座10层 查看职位地图

汽车之家 该公司其他职位

 汽车之家（NYSE：ATHM）成立于2005年，是中国领先的汽车互联网平台——为汽车消费者提供选车、买车、用车、换车等所有环节的全面、准确、快捷的一站式服务。


  我们致力于通过产品服务、数据技术、生态规则和资源为用户和客户赋能，建设“车媒体、车电商、车金融、车生活”4个圈、从“基于内容的垂直领域公司”转型升级为“基于数据技术的‘汽车’公司”。

  2016年7月中国汽车互联网用户通过PC端浏览汽车信息时，有45%的时间花费在汽车之家（根据iResearch数据）；2016年9月，每天通过移动端访问汽车之家的用户数量超过1800万（根据汽车之家2016年Q3财报）。
