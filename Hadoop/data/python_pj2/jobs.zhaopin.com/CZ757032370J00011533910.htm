举报收藏

职位诱惑：

高绩效,狼性团队,高成长性,强大资源

职位描述：

A、B轮AI大数据、消费、文娱项目FA；

岗位职责：

1. 负责金融机构、VC、上市公司、企业客户的开发、拓展和维护，提升客户的认同度；

2. 根据业务能力，逻辑分析能力，优秀者可转岗投资人关系及项目经理，负责指定项目的融资管理全过程；

3. 根据一线工作了解到机构客户反馈，向公司提出产品及流程优化建议；

4. 公司发展安排的其他工作。

任职要求：

1. 从事过商务、市场、金融领域销售工作优先考虑，具有拓展市场能力，具有解决问题的思路和思考能力，具有良好的沟通能力，从业过投资圈、券商具有相关人脉关系的优先考虑；

2. 全日制985或者211院校、名校、名企经历优先；

3. 具有较强的研究和分析能力，乐于学习，能够以创新精神思考问题；

4. 具有良好的客户沟通、人际交往及维系客户关系的能力；

5.  应届毕业生必须为金融专业、工科专业、计算机专业相关专业，公司为应届毕业生提供优质的投融资平台，欢迎有志应届生同学来挑战；

6.  待遇：6-8K底薪＋业绩收入＝年薪10-30W；

7.  公司提供全方位的培养培训，采用内部培养提升制。
