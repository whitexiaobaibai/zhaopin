举报收藏

岗位职责：

1、规划设计，架构，部署涉及系统集成、网络、安全、虚拟化、IaaS、政务应用、政务大数据等方面大型项目演进规划和整体方案设计；
2.负责云平台技术方案、分布式系统架构设计和核心模块设计；
3.负责云计算行业应用项目的需求分析、云计算行业应用方案设计、传统解决方案云化设计等工作；
4. 负责云管理平台研发/云计算产品的性能、稳定性优化；
5.负责云平台系统整体性能指标；
6.负责系统关键模块的技术攻关以及新技术探索；
7.研发人员技术培训、指导；
8.负责售前阶段的技术交流和技术支持工作；
9.负责客户上云项目的技术支持；
10.带领团队，引入和推广云平台相关新技术和工具的落地实施；

任职要求：

1、全日制本科及以上学历，电子信息、通信、计算机等专业，5年以上虚拟化、云平台相关工作经验。

2.熟悉主流云平台（Ali、Azure、AWS等）机构及技术特性，熟悉主流虚拟化技术（KVM,XEN,VMvare等），对IaaS / PaaS/SaaS平台构架有深层次的了解；
3. 熟悉虚拟化技术、云计算技术、分布式平台、主流开源云计算平台、容器技术、Linux内核等，如OpenStack、CloudStack、Hadoop、Cloudfoundry、Docker等；
4. 擅长云转型相关的需求分析、架构设计，并形成能够落地的技术方案，擅长规划和设计高扩展性、高性能、安全、稳定、可靠的云平台能力；
5.对云平台的技术发展趋势有较深的理解和研究，了解当前和未来的主要技术发展方向；
6.良好的沟通能力、热爱技术，对技术问题有清晰的思维和全局逻辑。

山西云时代技术有限公司 该公司其他职位

山西云时代技术有限公司（简称云时代公司）是山西省国资委直属的一级国有独资大数据企业。做为山西省IT行业的领军企业，以实现“云聚山西、云惠山西、云殖山西、云安山西”四大工程为目标，倾力构建大数据产业集群， 为智慧城市的建设提供高标准的应用云服务。

公司拥有CMMI、信息系统集成与服务和两化融合管理体系贯标咨询服务机构等资质，建立了规范的软件开发、测试体系，制定了完备的项目管理制度，健全的客户服务体系，有效的人才激励、培养和后勤保障机制，具有良好的工作、学习环境，为公司发展奠定了扎实的基础。

公司采用人性化管理模式，对科研技术人员进行多元化、多层次的培训和学习实践，辅以有竞争力的薪酬福利体系，激励员工与企业共同发展成长。

衷心希望与业内精英精诚合作，共享发展成果，共创新的辉煌！
