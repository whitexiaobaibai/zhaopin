举报收藏

岗位职责：

1.    负责完成数据规则的定义、数据模型的建立、数据清洗、数据迁移等工作；

2.    负责AI大数据采集、存储、挖掘分析、集成、管理；

3.    负责大数据平台搭建，离线计算，流式计算，机器学习，图计算等领域;

4.    负责大数据平台持续集成相关技术组件的设计和开发工作；

5.    负责实时计算技术体系包括数据采集、消息队列、计算引擎等；

6.    协助管理Hadoop集群，参与新技术选型和调研，解决不断增长的海量数据带来的存储和计算挑战。


任职要求：

1.    对实时计算所涉及的事务、容错、可靠性有深入理解；

2.    精通java/python/scala等一种或多种语言，掌握基本数据结构与算法；熟悉 linux 平台，掌握 shell 的日常使用，掌握 clojure 的加分；

3.    熟悉 Hadoop 生态圈：有MapReduce、Hive、Pig、Kafka等项目开发经验；

4.    熟悉Hadoop、Spark、Impala、HBase、Flink、Storm、Kafka、Kylin、Tensorflow、MXNet、Caffe等开源框架；

5.    熟悉NoSQL、MQ、Cache、TCPIP等原理；对数据库的基本理论和内部实现机制有比较深刻的理解，熟悉主流数据库（Oracle、MySQL等），熟练掌握SQL、存储过程等，拥有较好的数据库设计能力；

6.    具有大型实时计算系统构建者，大数据引用经验者优先。


工作地址：

建邺区奥体大街68号新城科技园研发总部园二幢22-24层
