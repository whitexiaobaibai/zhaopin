举报收藏

岗位职责：
1. 根据产品和部门的年度目标，制定并执行业务拓展计划。
2. 负责产品财务绩效分析、行业和客户分析、市场占有率分析、竞争对手SWOT分析和pipeline跟踪管理。
3. 跨业务线的内部协调。
4. 合作伙伴、渠道发展及管理。
5. 协同区域销售，跟进外部销售机会。
6. 商务关系管理。
7. 对业务目标的结果负责。
任职要求：
1. 至少为本科学历，具备MBA最佳。
2. 3年销售或BD经验（最好能提供）。
3. 具备较强的业务敏锐度、问题解决能力、资源管理和分析能力。
4. 具备较强的沟通能力，能用英文流利交流能力。
5. 具备云计算、大数据相关销售经验或技术背景为佳。

北京世纪互联宽带数据中心有限公司 该公司其他职位

世纪互联数据中心有限公司（21ViaNet Group, Inc，简称世纪互联）是中国目前规模最大的电信中立互联网基础设施服务提供商。为客户提供业界领先的服务器及网络设备托管服务，并通过优质的网络互联服务，帮助企业全面提升其互联网基础设施的性能、可用性和安全性
该公司创立于1996年，是中国最早的 ISP/IDC 服务商之一，也是目前中国规模最大的电信中立互联网基础设施服务提供商及全程、增值服务提供商。总部设在北京，在上海、广州、成都、东京、美国硅谷等地设有分支机构。主营业务包括互联网数据中心服务（IDC）、互联网内容分发/加速服务（CDN）、企业数据中心服务（EDC）以及全方位的增值服务和完整的行业解决方案。
