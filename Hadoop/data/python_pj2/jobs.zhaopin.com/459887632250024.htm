举报收藏

岗位职责：

1、基于hadoop生态的大数据存储平台搭建和部署; 

2、承担数据抽取、清洗、转化等数据处理程序开发； 


任职要求：

1、计算机或者相关专业本科及以上学历, 2年以上相关工作经验;

2、CET-4，能熟练用英语阅读技术文档 ;

3、精通java语言开发，同时熟悉groovy/scala语言开发优先考虑； 

4、熟悉Hadoop工作流程，熟悉MR执行流程，能够编写MapReduce程序，同时具有storm/spark开发经验优先考虑； 

5、熟练使用SQL，熟悉数据库原理，熟练使用至少一种主流关系型数据库； 

6、熟悉Linux操作系统，熟练使用常用命令，熟练使用shell脚本。

7、熟练掌握java，具有hadoop或者spark的开发经验。

8、熟悉Linux系统，能在Linux系统环境下完成开发

9、强烈的责任意识，有进取心，有较强的学习能力，理解能力，沟通能力，表达能力。

10、有金融及相关行业从业经验者优先


工作地址：

北京海淀区上地

上海洞识信息科技有限公司 该公司其他职位

洞识信息科技有限公司（英文名Paradise Insight，以下简称“洞识科技”）是深度学习大数据分析和智能IT运维领域的厂商。洞识科技的技术团队由来自深度学习大数据算法，IT运维专家，系统架构专家三个领域的顶尖专家组成，以“通过对接目前最尖端的大数据+深度学习的人工智能技术来帮助企业实现业务场景智能化为己任即Sciences to Client(S2C)”，通过开创性的智能S2C平台引擎产品和企业运维解决方案，快速成为中国最具潜力的智能运维领域的软件产品和解决方案提供商。洞识科技研发了目前世界上首款智能S2C平台引擎产品。该引擎实现自定义PI-L的数据湖多数据源的统一访问接口协议统一访问各种结构化数据和非结构化数据。该引擎产品可以调用目前主流的各种深度学习平台（TensorFlow，Spark等），通过引擎的服务编排功能发挥出各种深度学习平台优势能力。该引擎可以通过结合PI-L数据湖接口辅以动态菜单，可视化配置操作方式对数据进行各种大数据图形展现满足客户不同业务场景。该引擎还能实现集团层面，行业层面，区域层面乃至SaaS云化部署要求，通过业务场景申请流程真正实现大数据平台Online的用户体验。洞识科技结合目前近10年的企业金融客户的运维经验，研发了运维算法知识库，汇聚了目前企业运维经验标签。提供从故障定位，多指标预警，关联分析，知识挖掘，资源规划，性能容量趋势分析等多种深度学习智能运维算法。为企业在系统运维提供跨时代的智能运维算法。目前已经形成了维密Operation Secret系列家族的运维产品系列即LogInsight，DBInsight，OperationInsight。洞识科技目前已经和国际国内主流的运维产品形成生态联盟关系。兼容IBM Tivoli，BMC监控，Zabbix监控，IBM Qradar安全，Symantec Security Information Management安全，RiverBed，天旦网络分析等多种运维产品闭环快速融合。可以在不影响企业运维体系之下快速融入实际运维工作中，为企业提供更高效，更快速，更效率的主动预测性运维方式。洞识科技目前在北京、上海设有分支机构，构建了完善的服务体系，为国内客户提供从咨询、设计、产品实施、客户化定制到售后支持的全方位的S2C服务。
