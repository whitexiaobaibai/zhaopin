举报收藏

岗位职责:

1.  根据公司的发展规划,与全国几百万制造业企业或工厂达成合作；

2.  通过电话、拜访、会议营销等方式，很好地完成个人的销售目标；

3.  除了市场部新的工厂资源补充外，有计划地寻求新的客户线索；

4.  和买家部做好配合，定期举办线上线下的各种活动；

5.  领导交付的其他相关工作。

 

 

 

任职资格：

1.   1 年以上销售工作经验，大专以上学历，专业不限；

2.  有良好的沟通和表达能力，有强烈的销售欲望；

3.  有良好的职业素养，具备一定的抗压能力，勇于接受挑战；

4.  有过对接企业端客户经验者尤佳

海智在线（HAIZOL），是一家由海尔集团投资建立并专注于工业零部件领域的产业链服务平台，平台结合“工业4.0”和“互联网+”的特性，在工业零部件领域拥有超过万亿级别的市场份额。从原材料的采购，到信息的对接交换，从设备的购买、融资、租赁，到产品的检测、发货、验收，每一个环节都可以整合在网络之上。海智在线致力于让所有的工业零部件生产企业都能成为互联网和大数据的终端企业。
