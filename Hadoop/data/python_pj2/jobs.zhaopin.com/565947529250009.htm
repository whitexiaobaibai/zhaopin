举报收藏

岗位职责：
1、 按系统设计完成功能模块的编写；
2、 根据开发进度和任务分配，完成相应模块软件的设计、开发、编程任务；
3、 编写模块详细设计文档、解决方案；
4、 参与软件系统的设计和分析；
任职要求：
1、本科以上学历，2-3年相关工作经验，熟悉ASP.NET / C# ；
2、 熟练掌握XML、Web Service技术、WCF、Windows服务、多线程、HTML, Javascript, CSS等相关技术。
3.、熟悉掌握SQL Server 数据库设计及SQL脚本、存储过程、视图、索引等技术，熟悉大数据量处理，了解SQL语句优化方案。
4、 良好的工作态度、学习习惯及团队合作能力；
5、有良好的团队合作精神，能够独立承担工作压力，要有较强的分析和解决问题的能力。
6、熟悉工作流开发者优先，有银行从业经验者优先。

 


工作地址：

北京东路689号（浙江路牛庄路口）东银大厦2楼

 上海天正简介

        上海天正信息科技有限公司是浙江天正信息科技有限公司（前身为浙江省计算技术研究所，省内最早从事计算机技术及应用开发的专业研究所之一）在上海设立的，面向上海及周边地区用户的I T公司。公司凭借雄厚的技术实力和优质的客户服务，赢得了大量客户的广泛信任与支持。公司正处于一个稳健、高速发展的势态，公司先后设立了浙江办事处，山东办事处，江苏办事处、成都办事处等机构。
        上海天正软件公司是上海天正信息科技有限公司投资的专注于提供金融行业软件开发及解决方案的公司。上海天正通过与金融企业长期的合作，能够真正了解金融业务的需要，提供各种客户化的开发及顾问服务，为金融企业裁剪出一套适合自身业务需要的解决方案。
        上海天正智能数据服务有限公司是上海天正信息科技有限公司投资的专注于提供商业智能解决方案的公司，公司专注于提供商业智能及数据仓库领域的产品及解决方案，拥有一支专业化的服务队伍和国际先进的方法论，能够为客户提供领域内从咨询规划、系统实施到运营支撑的专业服务。
        上海天正以一流IT企业为目标，本着“为客户创造价值”、“为社会创造进步”的经营理念，致力于为广大客户提供及时、周到、专业、优质、一流的产品、技术和服务。公司重视提供完整的解决方案，重视和客户建立长久的合作，始终把满足和超越满足客户要求作为每个员工竭力实现的目标。
        公司于2006年10月通过BSI公司的ISO9001：2000质量管理体系认证。公司设立7×24小时800免费服务热线，第一时间响应客户的需求。

上海天正五个业务方向简介

金融软件  Finance  Software

        上海天正为客户提供如下金融软件产品与服务：金融中间业务系列产品，包括证券理财系统、银基通系统、记账式国债交易系统、银保通系统、代缴费平台；平台系列产品，包括 ISDP服务平台、企业影像平台、企业工作流平台；金融客户服务系列产品，包括天正个人理财系统；金融辅助管理系列产品，包括数据仓库、银行报表分析中心系统、绩效考核系统、客户信息管理系统、辅助决策系统；金融渠道服务产品，包括各种非现金类自助终端、自助跨平台系统、无人银行系统、手机银行系统等。

商业智能  Business Intelligence

        天正商业智能致力于为客户提供基于数据分析来支持运营和决策的端到端的解决方案，我们拥有国际先进的数据仓库解决方案及方法论、专业的服务团队和丰富的行业产品。通过 帮助客户对企业海量数据进行深入分析，实现数据到信息和知识的转化，提升客户洞察先机的能力进而帮助客户取得竞争优势。具体服务包括：企业技术及业务路线图（Roadmap）规划、模型设计、架构设计、实施、培训及知识转移，以及数据管理服务。同时我们还提供银行综合管理报表、绩效考核、客户信息服务（客户单一视图）等系列解决方案产品，还提包括数据仓库平台运营支持的整套服务、流程产品及知识库。

酒店软件   Hotel Software

        浙江天正率先于1986年成功开发了酒店管理软件，凭借着总公司丰富的研发与市场经验，上海天正于2002年推出功能更强大的TopHis系列产品，新版本有各类系统模块可根据实际需求供选择，让你更有效地利用信息技术来提高经济效益。上海天正宾馆管理软件包括前台管理模块、电话计费模块、餐厅管理模块、娱乐管理模块、洗浴管理模块、酒吧管理模块、工程管理模块、库存管理模块等。采用B/S与C/S技术相结合，设计技术先进，功能强大，操作方便，使宾馆管理科学化。

I T 服务  IT Service

        上海天正作为专业IT服务解决提供商，为企业提供如下服务：桌面及网络技术支持，包括客户端工作站维护服务、局域广域网络维护、服务器管理维护服务、多厂商硬件维护支持等；常驻工程师服务，天正为各行业企业提供派驻工程师服务；办公IT系统支持，包括各种操作系统、数据库的维护技术支持；IT企业信息化服务，包括客户响应中心服务、IT咨询及规范服务。上海天正还为你提供软件外包服务，既可承接软件项目开发，也可提供软件人员外包服务。

系统集成   System Integration

        上海天正的系统集成项目包括智能化布线、网络工程、主机和数据中心建设、网络及网络安全系统建设、一卡通、Voip等IT系统集成业务和应用软件、增值代理、技术及业务咨询服务等，应用于银行、保险、政府及广大企事业单位。从专业的解决方案到提供优质服务，都体现出对信息技术非凡的驾驭能力。
