烟台欣和企业食品有限公司

举报收藏

岗位职责：

1、利用移动大数据制定零售和餐饮行业解决方案，负责数据产品规划、业务流程设计、功能设计、产品优化等工作；

2、制定数据产品需求计划，进行可行性分析和概要设计, 撰写用户需求撰写PRD,负责需求的跟踪和控制；

3、联络、协调与支持产品相关的内部与外部人员与项目团队配合，阐述产品的设计思路，提供产品实施方案，促进项目成功实施；

4、汇总分析产品的全方面数据，对产品改进提供数据支持，持续提升用户体验；关注行业信息和用户对产品的反馈，分析并提供产品持续提高和改进方案。

任职要求：

1、本科及以上学历，理工科背景优先，3年以上互联网产品经验，2年以上大数据工作经验；移动互联网产品背景者优先。

2、熟悉数据产品实施过程，包括需求分析、产品功能设计、业务流程设计、界面设计、用户研究和可用性测试等；

3、熟练使用产品设计类软件，对研发有了解者优先；

4、具有较强的沟通能力、逻辑能力和产品设计能力，对数据敏感，具备较强分析加工能力；

工作地址：

长宁区天山西路789号中山国际广场B楼2层

烟台欣和企业食品有限公司 该公司其他职位

欣和是专业调味品酿造公司，历经二十余年的发展创新，欣和已成为调味品行业知名企业，旗下有六月鲜、味达美、葱伴侣、6月香、黄飞红、欣和、有所思、禾然、小康、遵循自然和醯官醋等知名品牌。

欣和成立的初衷是让所有家庭吃得放心，我们坚信食物对于家庭和生活，不仅仅是物质的基础，更是维系情感、提升生活品质的根本。因此欣和立志从更广泛的饮食生态入手，改善食物从源头到餐桌过程中的每一个环节，包括食物的来源、生产加工的方式，以及所有与饮食相关的体验。欣和致力打造的饮食生态系统，包括可持续的农业种植、可靠的饮食供应链、创新的产品研发、高标准的生产制造、便捷的零售渠道、极致的饮食体验、全方位的食物教育。

欣和立志成为能代表中国的领先食品企业，为整个中国食品行业建立新的标准，并将可信赖的中国美食带给世界。欣和上下都努力践行这份愿景，为了打造中国饮食新生态，为了给所有家庭一个更健康的未来。
