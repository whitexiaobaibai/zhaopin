岗位职责：

1、负责新研所内部项目的研发及相关技术的研究和应用；

1、本科以上学历，计算机相关专业；

2、精通java,c++,python 后端语言的一种，并掌握主流框架；

3、熟悉web开发技术，能够独立完成小型web平台开发；

4、熟练掌握mysql,sqlserver等数据库服务器的使用及维护具有扎实sql功底；

5、熟悉linux系统，能够在Linux环境下工作；

6、有数字出版系统相关开发经验者优先，熟悉虚拟化、共有云、大数据等优先。

机械工业信息研究院（机械工业出版社） 该公司其他职位

    机械工业信息研究院（机械工业出版社）成立于1952年，是国内最具规模的工业信息研究服务机构和最大的综合性科技出版社之一,主营业务主要涵盖图书出版、期刊出版、信息咨询及图书分销四个产业。

      图书出版：机工图书产品覆盖科技出版、教育出版、大众出版三大板块，其中，店面书产品覆盖机械、电气、汽车、建筑、计算机、经管、外语、文教、少儿、大众、生活等领域，教材产品覆盖技校、中职、高职、本科、研究生等五大层次。目前我单位年出版图书8000余种，其中新书3700余种，年销售规模已突破10亿码洋。

      期刊出版：机工期刊产业以“机工传媒”为品牌，成为引领中国专业媒体市场的主导力量之一。它以国际化视角和专业化服务为特色，通过纸质媒体、视听媒体、网络媒体，借助多种形式的展会、论坛，为西门子、ABB、Mazak、丰田等数以万计的企业提供全方位、立体化的展示平台，持续带动装备制造业的技术进步和市场繁荣。目前，机工传媒由工业和财经两大板块组成，涉及机械、电气、年鉴、设计、制造、财经、汽车生活等七大领域，正式出版期刊达21种，有8种进入“国家核心期刊”方阵。

      信息咨询：机工咨询年承接课题30余项，涉及情报研究、战略研究、技术管理研究等领域，是国资委、发改委、工信部、科技部等国家部委重要的信息咨询支撑单位。研究成果为国家中长期科技规划、产业技术升级、装备制造业振兴、科技成果转化等相关政策的出台，提供了广泛而重要的依据。机械工业信息研究院情报研究所是国家图书文献中心的组成单位，是国家工程技术图书馆第一分馆，现有馆藏30多万册，是国家定点收藏国外机械、汽车、动力、工业自动化等工程文献及相关经济与管理类文献的文献资源中心。

      图书分销：分销产业是机工社延伸产业链的巨大创新成果，它开创了出版社办书城的先河，首创店面销售、大客户销售和网络销售“三位一体，错位经营”的运营模式。其中，北京百万庄图书大厦经营面积8000平方米，汇集全国科技、教育、经管、文学、生活类图书近20万种，已跻身书店业全国50强；大客户销售在全国优秀馆配商评选活动中进入综合实力5强；中国科技金书网坚持大科技特色稳步发展。

      机械工业出版社是全国优秀出版社，全国百佳出版单位，教育部教材出版基地，全国版权贸易先进单位。近年来，我单位先后荣获中国500最具价值品牌、中央国家机关五一劳动奖状先进集体、中央国家机关先进基层党组织、中央国家机关文明单位标兵、首都文明单位标兵等荣誉。
