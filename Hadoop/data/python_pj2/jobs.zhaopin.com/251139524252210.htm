举报收藏

 公司简介：

       绿瘦品牌创立于2007年，总部设于广州，是目前互联网新经济环境下，中国领先的体重管理综合服务提供商之一。公司集电子商务、大数据营销、线上一对一体重管理服务、线下纤体店直营与加盟体系的优势于一体，正成长为我国体重管理的标杆性企业。

       目前，绿瘦以“做中国最专业的体重管理公司”为企业愿景，并基于此展开了多层次战略布局，构建了包括广东绿瘦健康信息咨询有限公司、广东绿瘦电子商务有限公司、广东绿瘦美容纤体有限公司、广东绿瘦生物科技有限公司、广东省绿瘦慈善基金会等在内的战略集群。公司业务涉及保健品及食品销售、美容纤体、基因检测、体重管理咨询指导服务等多个领域，已成为互联网新经济环境下，中国体重管理行业的领先品牌。

工作职责：

B 1、根据公司提供的资源，通过公司的电话平台对已有客户的需求进行挖掘和分析，促成销售；

 

2、对客户进行跟踪回访，解答客户的疑问，熟练运用公司减重营养产品，帮助客户解决减重中遇到的各种营养健康问题； 

 

3、维护客户关系，进行有效跟踪，做好售前、售后指导和服务工作。 


岗位要求

1 .年满18周岁，高中（或以上）学历；

2. 普通话标准流利，表达与沟通能力较强；

3. 思维敏捷，有良好的电话沟通技巧，具有相关销售经验；

4. 有一定抗压能力与强烈的事业进取心。


薪资结构：

无责底薪+全勤+绩效奖金+提成+工龄工资

   1)、试用期：工资总额3000-4000 （试用期1-3个月）

   2)、转正：工资总额4500-17000+高额提成


员工福利:

1. 入职即签订劳动合同，属绿瘦正式编制员工，享受绿瘦正式待遇；
2. 健全的福利制度（五险一金、节日礼品等）；
3. 享受带薪年假；
4. 为员工提供公寓式的住宿环境；

5. 提供员工饭堂；
6. 上下班期间有班车接送；
7.丰富多彩的员工活动：员工聚餐、节日晚会、旅游活动、趣味运动会、优秀员工表彰活动等；

8.提供带薪岗前培训、衔接培训、管理能力培训等专业培训和广阔的职业发展空间：

公司关注员工的发展，为每个层级的员工制定了发展规划：
专业路线： 电销专员职级的晋升管理路线： 电销专员-> 主管->经理->高级经理->总监
此外，电销专员也可向行政管理、人员管理、培训讲师、业务支持管理、市场分析管理等多方向发展。透明，公正，公开的考核制度，可以让您的职业规划的更加清晰。
 

公司地址：广州市荔湾区花地大道南渔尾桥绿瘦健康产业园B栋1楼。

乘车指引： 乘1号线地铁到西朗地铁站A出口， 转乘公交车410到海南南丫村村口站下车即到。
