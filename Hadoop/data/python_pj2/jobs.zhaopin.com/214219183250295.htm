举报收藏

岗位职责：

1. 对公司电商业务在大数据量条件下涉及的新功能、新技术和新架构进行预研和分析；

2. 对系统架构进行优化/重构，对系统进行分布式的业务拆分，以适应业务规模的增长；

3. 对系统性能进行持续优化，对系统进行比较全面的性能测试，并提出优化方案；

4. 对前后端分离进行方案预研、测试和部署；

5. 按时保质的完成公司业务涉及的相关功能的研究、开发、测试和优化；

6. 基础框架代码编写和核心代码编写，以及疑难杂症处理和线上问题处理。


岗位要求：

1. 具备3年（包含）以上大型网站开发/设计/架构经验，熟悉大规模高并发访问的架构设计，多年岗位工作经验者工资可面议；

2. 在面向对象设计理念和面向服务架构方面的深刻理解与实务经验；

3. 熟练掌握至少一种服务器后端语言（PHP），同时具备其他开发语言经验者优先，如Python，Linux Shell，JAVA等；

4. 熟练使用, Redis, MongoDB, MQ, Dubbo, Zookeeper等主流框架，精通原理；

5. 深刻理解数据库设计思想，能够对数据库操作进行性能分析及优化，具备数据库分库/分表等实战经验；

6. 具有系统需求分析和设计能力，能快速理解业务需求并进行相关逻辑设计，精通系统分层/分割原则；

7.对技术有浓厚的兴趣和敏锐洞察力，喜欢写代码，能快速学习掌握新技术；

8. 有大规模分布式云服务实际设计/架构经验或基于阿里云设计/架构经验者优先；

9. 具备大数据开源框架开发经验或有大数据系统相关经验者优先。


工作地点：山东临沂（可安排住宿）
