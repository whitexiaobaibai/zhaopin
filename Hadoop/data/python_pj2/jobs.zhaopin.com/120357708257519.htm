举报收藏

岗位职责:

1. 负责业务数据指标的抽取、转换、加载，数据维护；
2. 数据质量控制和元数据整合；
3. 分析业务需求、数据建模以及数据仓库应用产品的设计、开发；
4. 负责数据应用服务开发。
加入我们之后：
1、用大数据为我们的产品挖掘更多的用户价值；
2、深刻理解业务和数据的关系，通过数据驱动业务的方式让产品运营更加高效化；
3、除设计外，你还可以学到更多互联网产品、运营、以及培训学习的相关知识和资源，从各个方面提升自己的能力，爆发自己的小宇宙！


任职资格:

1. 本科及以上学历， 计算机相关专业有限；
2. 熟练掌握开源大数据处理技术栈，有Hadoop/storm/hive/spark等计算框架两年以上实际项目使用经验，深入阅读或修改过源码者优先；
3. 熟练掌握HBase及数据仓库建模技术；
4. 熟悉java应用开发及常用脚本语言(shell,perl,python)；
5. 熟悉阿里云、亚马逊等云计算数据框架和第三方优秀插件等优先。
6. 有过大数据系统架构经验或相关开源社区活跃者优先。

工作地址：
