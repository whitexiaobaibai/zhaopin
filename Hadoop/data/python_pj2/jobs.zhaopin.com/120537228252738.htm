举报收藏

岗位职责：

1. 对接沟通风控业务条线团队和大客户，提供数据驱动风控策略方面的专业咨询方案，解决业务的疑难点，并对业务工作提出建设性意见；

2. 协助部门建立风险数据库，了解同行业最新模型及分析技术，将数据分析成果应用于产品实践，创造用户价值，并结合业务现状持续进行数据模型和算法的分析与优化；

3. 从大数据中挖掘客户高维度行为特征属性，识别异常并分析与风险的关联度，据此产生创新应用。

4. 了解结构化及非结构化数据挖掘方法，熟悉统计建模、机器学习等量化建模方法。

5. 参与或负责模型技术设计、数据提取清洗、数据衍生变换、模型开发、模型验证评估到最终模型实施的项目全生命周期，解决不同场景下的风控业务问题，包含但不限于审批、贷中管理、催收和反欺诈等。


 

任职要求:

1. 本科及以上学历，计算机科学、统计学、数学、计量经济学、运筹学等相关专业，硕士博士优先。

2. 3-5年相关工作经验，具有银行、P2P、消费金融、信贷、零售信用风险咨询（如FICO、Experian、四大）等金融行业量化风险管理和建模经验者优先，具备BAT、FLG等国内外知名互联网或高科技公司数据挖掘和风控模型经验者优先。

3. 掌握扎实的统计学，数据挖掘，机器学习理论基础，具有分类、聚类、神经网络、NLP最优化等挖掘项目实战经验。

4. 具有扎实的技术基础，熟练掌握主流数据处理、分析及算法开发工具(SAS/Python/R等），具有大数据平台上项目开发经验者优先。
