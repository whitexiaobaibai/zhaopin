举报收藏

岗位职责：

1、 协助制定餐饮云因管理大数据相关产品的发展规划和商业策略，探索并构建大数据商业模式 

2、 对产品的竞争力负责，对产品的最终市场表现负责 

3、 负责相关产品的各个生命周期，协调并驱动技术、运营、销售、品牌、客服等多个团队共同完成产品业务目标。 

4、 协助打造基于公司大数据平台的生态系统

5、 关注用户反馈，分析用户行为，进行产品用户体验的持续优化。对用户转化、留存、客户价值及客户满意度负责

任职要求：

1、 熟悉移动互联网及大数据相关行业知识，熟练掌握主流产品设计工具，能熟练使用Axuer、office等软件工具，完成详细PD文档撰写；

2、 善于沟通，能独立与外部合作伙伴商务及项目协作沟通交流

3、 3年以上互联网产品经理工作经验，具有成功的iOS/Android产品设计案例；

4、 熟悉 C端 和 B 端用户需求和心理，对用户体验有深入理解；

5、 较强的用户需求判断、引导、控制能力；

6、 具备项目管理经验，有很强的执行力，保证项目按时推进上线；

7、 思维活跃，逻辑条理清晰，能明确的表达自己的想法和见解，善于与其他部门同事沟通，善于倾听他人的意见建议但不盲从；

8、 有餐饮、市场运营经验者优先

公司简介

   上海汉源企业管理咨询有限公司是一家专门服务于餐饮行业的教育培训与管理咨询机构。


        汉源东方，是专注于餐饮管理咨询和教育培训的专业咨询集团，总部位于上海，在北京、深圳和成都设有分公司。曾服务于王品、西贝莜面、望湘园、绿茶、爱茜茜里、釜山料理、黄记煌等众多知名品牌，被誉为“餐饮业的西点军校”。2007年创立至今，旗下拥有四家子公司：上海汉源企业管理咨询有限公司、上海汉一品牌设计咨询有限公司、上海汉悦投资管理有限公司、上海酷师智能科技有限公司。


       汉源东方餐饮商学院倡导“集团化发展、系统化运作”的行业发展思路，通过培养卓越的餐饮经营领袖和优秀运营管理团队，帮助学员企业完成以“品牌模式系统、运营管理系统、文化理念系统、人才发展系统”为核心的四大系统建设，从而实现企业运作的“规范化、标准化、系统化”，建立一个让伙伴＂开心工作，幸福生活＂的幸福餐饮企业，最终实现提升餐饮行业业格，提升餐饮行业幸福感。汉源东方餐饮商学院已被众多学员广泛推崇为业内最权威、最专业、最实战的餐饮咨询培训机构。


       汉源人始终秉持“助推中国餐饮，提升行业幸福”的神圣使命，立志“让餐饮业成为最幸福的行业”。
