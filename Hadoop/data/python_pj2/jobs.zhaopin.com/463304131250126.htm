举报收藏

岗位职责：

1. 负责总体规划并设计公司内部数据可视化产品。

2. 能对数据进行分类与分析，挖掘数据潜力，图形化表达数据的结构与内容。

3. 配合公司数据挖掘同事完成相应的数据可视化模块。

4. 把握数据可视化设计趋势，并有效结合到实际设计中。

任职要求：

1. 有web／app视觉或交互设计工作经验者、有媒体从业经验者优先。

2. 熟练 Html、DIV/CSS、Javascript，Ajax 前端开发语言，有独立的网站开发经验。

3. 本科及以上学历，设计类专业毕业，具有深厚的美术功底和良好的创意构思能力，对色彩有深刻的把握力、独特的设计风格、独到的创意视点与创新意识。

优先条件：

1. 有参与过大型数据项目或数据平台建设的经验。

2. 有大数据可视化系统建设或前端经验。


工作地址：

上海市徐汇区宜州路180号华鑫天地二期C601栋9楼

国信数道信息科技有限公司 该公司其他职位
