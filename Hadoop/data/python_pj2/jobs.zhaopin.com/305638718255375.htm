举报收藏

岗位职责：

1. 负责数据分析算法开发
2. 负责基于R/python的大数据算法的优化
3. 负责企业内工业大数据收集，提出数据结构优化
4. 负责基于智能决策需求的数据挖掘
5. 负责制程控制的模型开发

 

任职要求：

1. 统计学、数学、应用数学本科教育背景
2. 有FPD、半导体产业2年及以上制程/整合/IT/品质经验
3. 英文读写流利，能熟练阅读英文文献
4. 对数据敏感，较强的学习力，对数据分析和挖掘有浓厚兴趣
5. 会Python/R语言编写建模脚本，会JAVA编程优先
6. 通过六西格玛黑带（绿带）认证优先
7. 具备良好的规划能力和跨部门沟通协调能力

● 中华心世界屏
华星光电成立于2009年11月，作为我国自主建设的，迄今国内最高时代的TFT-LCD生产线之一，华星一直致力打造成为极具价值创新能力的全球领先显示技术企业。华星光电一期，二期项目主要着力于电视面板市场，正在武汉快速建设的三期项目将着力于中小尺寸、中高端面板市场，二期和三期项目更加注重技术提升，实现先进技术的综合利用和产品线从大尺寸到小尺寸的拓展，同时进行4K、8K、3D和OLED等新型显示技术的迭代开发。

● 深圳华星
华星光电一期项目与2012年8月8日正式量产，用时17个月建成投产，实现当年投产当年赢利，2013-2014年持续满产满销，专利数量超过三千件，海外申请专利与华为中兴同为全国前三并创造多项世界第一。主要经营指标连续8个季度领跑全球同行。成立至今，累计实现销售收入逾400亿，利润超过50亿，上缴税收近20亿，成为全球单一产能最大，最有效率的8.5代液晶面板生产线。
华星光电二期项目于2015年4月24日正式量产，仅用时15.9个月就完成了从打桩建设到产品点亮的过程，再次刷新全球业界记录，二期项目正式投产后60万平米的8.5代线厂区也成为亚洲最大单体厂房，预估年均销售额202亿，将带动上下游产业链年均工业增加值1100亿元。二期项目结合广东省战略新性新兴产业发展“十二五”规划，自主创新，研究铜制程，COA，氧化物半导体，WOLED工艺等四大关键新型显示技术产品，助力显示领域迅速向大尺寸化，高解析度，低能耗、高色饱以及轻薄化升级。

● 武汉华星
华星三期项目，总投资160亿元。建厂武汉光谷，创武汉市单体项目投资之最。LTPS面板属高端面板，目前国内仅两条生产线，作为武汉市战略性新兴产业和传统支柱产业的领军代表，华星进一步乘胜追击后将实现LTPS、AFFS、OLED、Touch panel等新技术的应用。
