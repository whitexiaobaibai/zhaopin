举报收藏

职位描述：

1、负责数据产品及市场的研究与分析，了解数据产品的发展方向；

2、与客户交流与沟通，收集与分析客户的需求，确定产品研发方向，对数据产品进行持续优化；

3、与数据规划和产品规划人员、客户沟通，进行指标、数据业务逻辑整理，规划数据产品的发展方向；

4、与开发人员有效协作，配合开发人员和测试人员完成指标代码开发与测试，高质量完成开发项目；

5、根据市场的基本情况，制定所负责的产品线的商业化战略以及商业战略落地。

 

招聘要求：

1. 本科以上学历，具备统计、数学、计算机等理工科和经济、金融、财务复合背景者优先，有在证券、基金等金融机构工作经验者优先；

2. 两年以上相关岗位工作经验，有相关数据研究与产品经验优先；

3. 熟悉掌握股票、债券、基金、期货等相关金融领域业务知识，通过证券从业资格考试、CPA、CFA优先；

4. 熟悉数据库知识，熟练使用SQL Server，可以进行复杂SQL查询操作；

5. 至少熟练使用如下一种工具：Matlab、python、R、SAS、SPSS；

6. 责任心强，执行力强，做事严谨，具备团队合作精神、良好的沟通能力。

我们的股东背景: 03年成立至今14年,创始人+阿里系蚂蚁金服+国金+嘉实等业内多家知名公司;

我们的企业愿景:成为金融数据化,工具化,平台化最安全的线上服务商;

我们的平台构建: 是业内专业的上市公司盈利预测数据、私募数据服务提供商、高端财富管理者的服务平台，也是中国金融人脉圈的卓越聚合者;

我们的朝阳业务:金融大数据业务,金融智能终端及增值服务业务,朝阳财富基金销售业务,资产管理业务;

我们的客户群体:覆盖基金、保险、信托、券商、私募等众多机构;

    上海朝阳永续信息技术股份有限公司（简称“朝阳永续”）成立于2003年，公司股东包括蚂蚁金服、国金、嘉实等业内多家知名公司。朝阳永续致力于成为中国领先的金融科技服务商及资管人服务平台，是业内专业的上市公司盈利预测数据、私募数据服务提供商、高端财富管理者的服务平台，也是中国金融人脉圈的卓越聚合者。

    朝阳永续成立十余年来，本着专业性、前瞻性、实务性的研究宗旨，以及对产品品质的苛求，成功的推出了Go-Goal数据业务的系列产品，包括：受到专业证券研究机构追捧与好评的上市公司盈利预测系统（一致预期数据）、私募数据库等产品，并获得了超高的主流市场占有率。基于上市公司盈利预测数据库，朝阳永续推出了适合投资机构和专业投资者使用的集证券资讯、报告、数据于一体的研究分析工具——Go-Goal新一代金融终端，辅助投资决策，让更多的用户能够在实战中分享研究的价值。而基于私募数据库推出的私募基金评测与配置云服务，在业内也获得广泛认可，多家大型金融机构在发行FOF/MOM产品时均采用朝阳永续私募数据及评测体系进行投资顾问筛选等工作。
    基于在金融服务行业多年的数据积累和研究以及良好的口碑，朝阳永续推出财富管理业务，成立全资子公司上海朝阳永续基金销售有限公司(简称“朝阳永续财富”)，为专业机构及高净值投资人提供最有投资价值的资产配置方案，专注于私募研究咨询、私募数据、私募会议论坛、私募产品发行、母基金管理及财富管理等私募基金相关的增值服务业务。    同时，朝阳永续也是中国金融人脉圈的卓越聚合者。公司通过业务合作、成立俱乐部和组织丰富的交流活动，与券商、银行、信托公司、期货公司、基金公司、私募投资顾问等保持长期良好的紧密合作关系；同高端投资机构、个人投资者、专业财经媒体等保持长期良好的交流沟通。多年来，朝阳永续还成功策划和举办了金融行业内数个领域的系列评选活动，如中国私募基金风云榜、证券最佳伯乐奖、中国最佳分析师、最佳量化产品评选等大型活动，树立了该类评选活动的权威标准，并在业内形成朝阳会务的独特影响力。 更多信息，请访问：www.suntime2003.com ; www.go-goal.com
