举报收藏

岗位职责：
1.对客户的需求进行结构化处理，构建客户所需的系统监测任务，有针对性规划数据研究方向；
2.根据已有网络舆情分析系统提供的原始数据可以发现数据特征、数据趋势，并独立完成舆情监测分析工作；
3.针对数据分析可以对客户的媒体投放、网络活动做出策略指导和建议；
4. 和内部部门协调，讨论并解决项目中遇到的问题，确保项目质量。 
任职资格：
1. 专科以上学历，新闻传播、网络传播、市场营销、经济学等相关专业优先；
2. 熟悉网络信息传播特性及发展趋势，对舆情和媒体传播有一定了解；
3. 具有基本的逻辑思维能力、客户沟通能力，熟练使用EXCEL等统计分析软件，熟练使用PPT撰写分析报告；
4. 具有基本的互联网传播研究经验，能够独立完成或带领团队成员完成常规或专题监测分析项目；
5. 具有探索精神，热爱舆情大数据工作，具有强烈的自我提升愿景，具备较为成熟的工作思路；
6. 具备互联网或舆情监测研究行业经验者优先考虑。



工作地址：

北京市海淀区北四环西路68号左岸工社510室

云鸽在线(北京)信息技术有限公司 该公司其他职位

云鸽在线（北京）信息技术有限公司（以下简称云鸽）www.yunger.net，致力于用大数据分析技术推动社会与商业进步，以先进的技术与专业的服务为客户创造价值。

云鸽成立于2011年，是互联网数据分析的技术与服务新兴厂商，也是领先的云服务提供商，向政府、企业和个人提供互联网在线数据分析服务。目前主要提供的产品有舆情监测分析服务、数据服务、大企业私有云解决方案等。

云鸽舆情监测分析服务

基于云架构以及大数据处理技术，实时抓取新闻、论坛、微博、博客、视频等互联网数据，为企业、政府、个人提供信息采集、分析、敏感事件预测和预警、企业竞争情报监测分析、品牌口碑监测、用户网络行为分析等在线服务。

云鸽定向数据分析服务

凭借云鸽信息在数据分析领域长期积累的技术和经验，旨在通过对特定类别数据全面、精准、实时的分析，为企业和个人用户提供新鲜的行业资讯和行业分析报告等服务，同时提供专业的数据分析工具及客户端，便于客户更好的进行特定领域内的商业决策，为企业和个人用户发现和创造更大的价值。

云鸽开放平台服务

云鸽开放平台是基于云鸽数据服务系统的开放数据对接平台，不仅提供开放的API接口供企业、政府井数据服务接入自有系统中，同时有能力的企业或开发者可基于云鸽开放平台开发个性化的服务，旨在满足不同用户对数据的不同要求，形成由用户、伙伴、营销机构共赢的生态系统。

云鸽企业私有云产品解决方案

针对大型集团企业提供全面的信息监控和分析服务的解决方案，包括互联网资源爬取、海量数据存储、非结构化数据解析、分布式计算、可定制的数据挖掘模型，在线信息监控和分析平台，同时帮助企业、政府用户将内部数据和外部互联网数据较好的整合起来，在企业内部建立一整套数据挖掘和分析平台，并且可以基于企业、政府用户内部特定的需求进行功能定制。

云鸽将长期致力于互联网大数据分析和服务，满足不同用户对数据的不同需求，同时形成由用户、技术伙伴、集成伙伴、经营伙伴共赢的生态系统。

云鸽的理念：将互联网数据转化为企业资产。

云鸽发展目标是成为领先互联网数据分析的技术与服务提供商！
