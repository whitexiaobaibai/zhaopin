举报收藏

岗位职责：

1 负责技术核心竞争力构建，关键技术难题攻关、专项技术、工具设计

2 根据产品和项目需求，完成所承担的业务模块的设计与开发

3  根据产品和项目需求，完成相应的技术文档编写

4 按要求完成相应数据仓库核心功能开发， 关键算法的开发

5  按要求完成产品和项目的交付、实施及相应的技术支持工作

 

 

任职要求：

1  本科（含）以上学历，具备2年以上的数据分析经验，具备1年（含）以上的大数据相关开发经验

2  技术方面：

    1.      熟悉linux开发环境；熟练掌握java、scala、python、shell中的一种或多种

2.      熟悉或对开源项目有兴趣：如Spark、MapReduce、Hive、Hbase、kafka、flume、mahout、Pig等

3.     精通SQL编程开发，对性能调优有一定程度了解。熟悉Oracle 、DB2、Teradata、Greenplum，SQL Server 等行业主流数据库的一种或多种。

4.     熟练掌握使用R，SPSS、SAS等至少一种数据挖掘分析工具, 具有互联网公司大数据数据挖掘与分析相关经验优先；

5.     有数据仓库开发经验，以及数据产品开发者优先

6.     具有互联网大数据平台开发经验优先

 

工作地址：

上海市徐汇区漕溪北路333号中金国际广场B座7楼 查看职位地图

艾瑞咨询集团 该公司其他职位

                                 艾瑞咨询集团介绍

 

    艾瑞咨询集团成立于2002年，一直致力于成为中国大数据时代下最佳互联网收视率及消费者洞察公司。艾瑞咨询集团以“洞察互联网的力量”为理念，期望为客户提供中国市场最专业的互联网相关领域的数据产品、研究咨询等专业服务，帮助客户提高对互联网产业的认知水平、盈利能力和综合竞争力，让互联网的力量点燃中国各个行业。

 

公司员工数量：超过400名

公司办公总部：北京·上海

区域办公地址：广州·深圳·成都·杭州·硅谷·纽约·香港

样本数据基础：互联网收视率数据基于百万级的电脑、手机、平板等用户行为监测样本。

公司数据产品：包括用户行为研究、广告营销监测、网络零售研究、企业数据分析等多条数据产品线，为客户提供互联网多个连续性数据指标，帮助客户动态决策。

行业研究基础：艾瑞研究体系自2003年开始研究中国互联网产业，已经累计出版超过1500份互联网研究报告，涵盖互联网、移动互联网、电子商务、互联网金融、网络营销、网络服务等各个领域。

专项研究咨询：艾瑞近百名研究及咨询人员，服务于不同客户的针对性需求，包含市场竞争环境、战略规划、投融资机会、大数据管理、数字营销及媒介、产品用户体验、用户满意度、网络渠道建设、广告效果评估、品牌管理等课题。

媒体营销服务：艾瑞旗下拥有艾瑞网、艾瑞广告先锋等网络媒体，并每年主办多次自主品牌艾瑞峰会和行业活动，在行业内具有广泛的影响力。

专业服务体系：艾瑞已经开展多项围绕客户的专业服务体系建设，包括专业培训服务、数据挖掘服务、人力资源顾问与猎头服务、企业会务外包服务等多项专业服务业务。

公司品牌影响：艾瑞具有极高品牌认知度，在网络营销、电子商务、移动互联网、大数据和互联网金融领域研究具有领先的市场地位，艾瑞发布的互联网产业及用户数据被各大媒体广泛引用，在多个领域已经树立数据标准。在多个互联网公司IPO上市报告中，艾瑞是主要的第三方数据服务提供方。

公司服务客户：艾瑞累计服务超过1000家客户，涵盖多个行业领域，包括互联网、移动互联网、媒体及娱乐、广告及公关、零售及电商、通信、游戏、金融服务、投资研究、消费品、医疗健康、汽车、房产、电子与科技、交通及旅行、政府及公共事业等。艾瑞的客户几乎覆盖中国所有主要的互联网公司、90%的互联网广告代理公司、主要的电子商务企业、主流的投资银行及互联网对冲基金等。

公司创始团队：艾瑞是由杨伟庆发起创立，公司早期创始团队包括邹蕾、郝欣诚。

公司高管团队：艾瑞具有一支稳定并具有深厚行业服务经验的管理团队。目前的高管为杨伟庆、阮京文、邹蕾、郝欣诚、黄嘉骊、曹军波、陈超、霍禹、吴畏。
