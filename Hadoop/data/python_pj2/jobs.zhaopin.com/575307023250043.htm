举报收藏

Web开发方向

岗位职责：

1. 参与公司产品或项目的Web开发。

2. 参与公司产品或项目的数据库开发。

3. 参与公司产品或项目的需求评估和构架设计。

任职要求：

1. 本科或硕士，计算机、理工科等相关专业。

2. 精通MVC框架，有使用Spring, Struct2, Hibernate, Servlet的相关经验。

3. 熟悉MySQL，有相关数据库设计和使用经验。

4. 能够相关系统需求文档、开发文档、接口文档等。

 

C++开发方向

岗位职责：

1. 参与核心系统代码编写

2. 参与算法研究，进行相关算法实现、测试与优化

3. 参与系统需求分析、构架设计以及相关测试工作

4. 撰写相关系统开发文档、接口文档

任职要求：

1. 本科或硕士，计算机、理工科等相关专业

2. 熟练掌握C/C++，有相关项目开发经验；精通数据结构并具有相关开发经验，能够进行算法设计；了解基本的程序优化方法

3.有多线程编程、线程/进程间通信、网络编程、信号处理程序开发、嵌入式开发、驱动开发、大数据分析经验者优先

4. 具有高效的阅读英文文献资料的能力；具有较好的代码编写习惯，能够撰写相关技术文档；具有较强的自学能力和独立分析、解决问题的能力；为人踏实，具有良好的科学素质、创新意识和合作精神

 

C#开发方向

岗位职责：

1. 参与C/S系统的设计与开发；

2. 根据项目具体要求及任务划分，完成相关模块的代码开发、测试及相关文档撰写；

3. 参与软件项目的需求分析、设计工作；

4. 参与软件项目的测试、优化。

任职要求：

1. 本科或硕士，计算机、理工科等相关专业；

2. 精通.NET、C#或MFC，精通C/S架构开发；

3. 精通windows绘图方法，了解OpenGL或DirectX，有具体开发经验者优先；

4. 具有高效的阅读英文文献资料的能力；具有较好的代码编写习惯，能够撰写相关技术文档；具有较强的自学能力和独立分析、解决问题的能力；为人踏实，具有良好的科学素质、创新意识和合作精神。

    询康科技于 2013 年注册，总部位于上海市杨浦区创智天地园区。是专业从事移动健康、智能化医疗、区域化健康管理及社会化服务整体解决方案能力的服务供应商。公司立足互联网+领域，利用自我技术（大数据智能分析、远程识别等）与知识产权优势，业务已覆盖全国多个省市，引领智能化、自动化管理新趋势的高端技术与产品。

    公司技术背景雄厚，同时拥有强大的技术顾问团队（院士团队、国内、外知名专家及研究机构），与国内外知名科学研究单位多有合作，主要的研发人员也是由国内外著名高校毕业的博士、硕士构成。

    公司目前正处在快速发展时期，员工数量不断增长，与科研单位、政府、企业的合作也逐渐增多。因此，现面向国内外应届、往届的优秀毕业生，或者有相关工作经验者，进行2017年度的招聘计划。
