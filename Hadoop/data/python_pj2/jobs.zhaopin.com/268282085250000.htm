举报收藏

工作职责：

1. 负责业务数据分析、主题分析和数据可视化方案制定；

2. 针对能源行业特定领域、不同业务主题开展探索性分析建模和数据挖掘；

2. 负责数据整合与分析模型的建立、优化以及跟进；

3. 持续推动和跟踪成果的产品化落地；

4. 跟踪最新的数据分析、数据挖掘领域的技术和成果。

 

招聘要求：

1. 硕士及以上学历，计算机、应用数学、统计及相关专业，1年以上数据采集、清洗整理、分析建模相关工作经验；

2. 有优秀的洞察力，能从纷繁复杂的数据中发掘规律并构建模型；掌握各类算法，具备优秀的大数据建模功底；

3. 具有较深厚的数学、统计学和计算机相关知识，熟悉数据仓库和数据挖掘的相关技术及方法，熟悉数据可视化相关理论；

4. 熟悉软件开发流程，懂java；

5. 熟练使用Python、R、SPSS或SAS等至少一种工具进行分析和建模（Python优先），熟悉MySQL/Oracle, 会使用Hadoop等产品；

6. 具有良好的分析报告撰写能力，能制作专业分析报告。

深圳市数聚能源科技有限公司 该公司其他职位

深圳市数聚能源科技有限公司是一家以能源大数据为核心，基于移动互联网技术面向企业及大众提供节能服务的科技公司。公司创始人均曾在能源及能源软件领域从业十年以上，并曾担任某大型软件公司核心事业部总经理和首席架构师，运作过多个超亿元的软件项目，管理过数百人的团队规模，在能源行业的内部应用系统中进行过多次技术和业务管理方面的创新颠覆，主导的相关研究与成果曾获得过多个行业奖项，拥有深厚的行业背景。

随着国家对电力体制改革的逐步深入，电力必将被一步步市场化，电网企业垄断的局面将被改变；新型技术，例如大数据、移动互联网、物联网等技术会很快应用在电力领域，创造更多的用户价值。更多的民营资本将被准许进入电力市场，让消费者得到更好的服务、同时加强国家电力建设、提高电力的使用效率，也将使社会整体的电能利用更加合理。我们从事的节能服务工作是一份对社会非常有益的事业，是国家支持领域，具有广阔的市场前景！

    公司在移动互联网技术架构、hadoop、并行计算等方面有深入的研究与实操经验。如果你对Java EE大型企业应用架构、云计算架构、SOA服务、PaaS平台、分布式系统架构、移动应用平台、开发自动化、前端技术框架(jQuery、AngularJS、RequireJS、HTML5)/R语言/数学建模……等领域感兴趣，欢迎加入我们！

    我们的使命：基于对能源数据分析对居民及企业提供节能服务、对供电企业（或其他公共资源服务商）提供精确有效的需求侧管理及服务，对第三方的能源服务企业提供节能和售电技术服务。

    我们的愿景：成为国内基于能源消耗大数据分析提供节能服务企业的领导者和标杆。

    公司尊崇正直、勇敢，开放、透明、平等、互助的价值观；崇尚轻松、和谐、快乐的工作环境所有员工只有分工不同，没有高低贵贱之分，公司尊重每位一员工。人才是公司最为宝贵的财富，公司期望从事对社会有价值的事业，并为员工提供体面的生活。

    我们期待你的加入，一起实现事业与生活的梦想！
