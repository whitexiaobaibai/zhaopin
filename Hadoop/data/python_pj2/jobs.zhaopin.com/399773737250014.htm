举报收藏

岗位职责：

1. 平台市场分析调研，大数据分析，了解消费者需求，评估市场容量；

2. 负责产品开发，能独立开发新产品线，规划公司产品线及产品策略；

3. 追踪、搜集、分析、汇总及考察评估新产品的信息，提炼新产品卖点，配合新品上架推广；

4. 全面对接销售部门，保持与销售部门沟通并配合各销售部门开展工作；

 

任职资格：

1.本科及以上学历，英语四级以上，有5年及以上产品开发团队管理经验，电商工作经验者优先；

2.热爱互联网、电子商务行业，善于发现、分析并解决问题，创新能力强；

3.具有良好的沟通能力及团队合作精神，工作积极主动,责任心强,敢于挑战自我，能承受工作压力；

4. 愿意与公司共同发展，具备在公司的平台上实现个人价值的积极心态；


关于我们：

1.薪酬结构：底薪+全勤奖+补贴+绩效奖金；

2.严格按照国家法定节假日放假，节日福利；

3.人性化的管理：手机打卡、健身娱乐、下午茶、节日PARTY、月活动组织；公费旅游等；

4.购买五险一金；带薪休假、年终双薪等；


工作时间：周一到周五朝9晚6，周末单双休制，单休休一天半

工作地址：

广州市白云区万达广场 查看职位地图

广州云鹭庭贸易有限公司 该公司其他职位

广州云鹭庭贸易有限公司是一家朝气蓬勃的外贸电商公司，平均年龄90后，我们的核心团队已经在外贸行业耕耘超过10年。主营各类流行饰品，时尚新品，面向市场为欧美日本，拉美等国家。团队福利超过同行业水平，平均薪水在5K－15K。我们始终相信Work hard play hard的信条，张弛有度才能更有效率地工作。

我们的优势：
完善的薪酬结构：底薪+绩效奖金+全勤奖+补贴；
人性化的管理：手机打卡、健身娱乐、动不动就下午茶、节日生日PARTY
优厚的福利机制：说走就走的公费旅游、敢说敢做的带薪休假、别人有的五险一金、补贴、全勤奖等我们一件                都不落！国家规定放的假，放！时不时还送点节日礼物！
