举报收藏

商情数据于2016年8月9日成功在新三板挂牌上市，股票简称：商情数据，股票代码：838497，网址：www.askci.com 。
商情数据是一家通过大数据技术重构传统产经数据行业成为“互联网+数据”的新型数据公司，目前已经构建完成了大数据平台的基础设施，随着公司产品线的持续完善，商情数据将向业界提供各种大数据服务，构建开放平台，探索全新的商业模式。
公司创立于2003年，旗下中商研究院主营数据服务产品及各类咨询服务。

1、研究类业务（市场调研、行业报告、行业顾问）
2、投资类业务（可行性研究、商业计划书、IPO募投可研及新三板、四板挂牌咨询）
3、规划类业务（产业规划、产业园区规划、招商规划、重大项目包装策划）
4、金融类业务（民营银行、保险公司筹建咨询、企业融资财务顾问）
5、企业类业务（企业营销咨询、企业战略咨询、企业管理咨询）
6、大数据业务（千数堂大数据交易平台(T.askci.com)、八方查征信大数据(8.askci.com)、各行业大数据解决方案及实施、政府大数据解决方案及实施）

 

岗位职责：

1. 负责公司业务开拓渠道，挖掘新的业务增长模式并执行实施；

2. 根据公司年度销售任务，设置部门销售目标、销售模式、销售战略；

3. 管理销售队伍，组织各种销售技能培训；

4. 参与重点客户售前谈判，制作建议书及报价方案，合同签订等工作；

5. 维护大客户关系，挖掘市场需求信息，拓展新客户；

 

任职要求：

1. 本科以上学历，经济、金融、市场营销等相关专业优先；

2. 2年咨询行业工作经验，3年以上销售、市场营销管理工作经验

3. 具有较强的判断和决策能力、人际沟通协调能力、计划与执行能力；

4. 优秀的市场拓展、项目协调、谈判能力；

5. 性格开朗，善于交际。


薪酬待遇：

公司提供丰富客户资源，无责任底薪+高提成+奖金+福利，转正后年薪达15-50万，享受五险一金、5天工作制（周末双休）、法定节假日，带薪年假，良好的培训，丰富的娱乐活动。


工作地址：

深圳中商产业研究院有限公司 该公司其他职位

    中商产业研究院是中国产经大数据第一股商情数据（股票代码：838497）旗下的研究机构。

 

    公司核心团队具备良好的教育背景，拥有大数据处理、掌握大数据算法和行业洞察的专业人才，建立了多种数据分析模型，通过大数据技术，输出标准化以及定制化数据产品。在重要产业如智能装备制造、新能源、新材料、新金融、大健康、文化旅游、“互联网+”等领域均能提供高价值的数据产品。致力于为国内外企业、各级政府部门、高等院校、科研单位以及投资机构提供强各类数据服务、行业研究报告、细分市场报告等，同时为客户提供高价值的咨询服务，如产业规划、园区规划、定制市场调查、新三板挂牌服务、、细分产业行业顾问、IPO上市咨询（募投可研）、PPP项目咨询、特色小镇咨询、民营银行、保险公司筹建咨询等。

 

   自2003年以来，公司专注产业研究10余年，为优质客户长期提供产经大数据产品服务，服务世界500强企业、国内500强企业，政府机构、科研单位、高等院校及咨询机构等。目前，中商产业研究院已成为中国领先的产业研究咨询服务机构。
