举报收藏

岗位职责：

1.   负责公司的云计算、大数据平台的设计和开发工作；

2.   负责平台的架构设计；

3.   负责分布式计算/存储开发，调试等工作；

4.   负责详细设计，操作手册等的编写工作；

5.   负责分布式计算/存储新领域的技术调研工作。


任职要求：

1.   本科及以上学历；

2.   3年以上工作经历，有云计算经验尤佳；

3.   熟悉Java、node.js、Python等开发语言以及相关架构模式；

4.   熟悉开源的Spark或Hadoop平台；

5.   熟练使用Oracle、MySQL、MongDB等数据库，具备数据库调优相关经验；

6.   有大型分布式计算/存储开发，部署等相关经验者优先；

7.   良好的语言表达,沟通能力,和团队合作精神。

北京国泰北斗科技有限公司 该公司其他职位

北京国泰北斗科技有限公司（简称：国泰北斗）是一家具有独立法人资质的信息技术高科技企业。公司成立于2013年8月，主要从事信息技术开发、信息系统集成以及行业产品销售等，拥有软件企业资质和自主知识产权的硬件产品、行业应用软件系统。

　北京国泰北斗科技有限公司是一家以航空领域军用和民用导航、测试、监控及通讯等市场需求为牵引，以自主研发平台及产品为核心，秉承“信誉第一、客户至上、以人为本、合作共赢”的理念，为用户提供优质产品和全方位解决方案的高新技术企业。

随着我国私人飞机和公务飞机的不断兴起，民间资本对通用航空投入不断提高，低空空域改革不断深入，我国的通用航空正面临着前所未有的发展机遇。    目前，全社会都在积极探讨“互联网+”与自身行业的结合。谷歌、亚马逊、Facebook、腾讯、小米等互联网企业纷纷高调开展无人机业务。“通航+”、“无人机+”与“互联网＋”类似，都可通过与传统行业跨界融合开辟发展新“蓝海”，具有巨大的市场发展潜力，成为目前市场投资的热点。

国泰北斗近2年来，积极投身于通航和无人机领域的互联网应用研发，在“通航+”、“无人机+”市场上取得丰硕成果，获得了相关用户的一致好评。
