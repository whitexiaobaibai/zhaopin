举报收藏

岗位职责：

1、 负责公司内部系统和网站维护，以及相关新功能开发；

2、 与公司内部其他部门或系统协作进行数据接口对接功能开发；
3、 参与研发公司其他管理系统；
4、 对系统功能进行测试，保证代码质量；

任职要求：

1、2-3年.Net或者Java开发工作经验；
2、正规院校计算机或相关专业本科毕业；
3、具备扎实的开发技能，具有面向对象的编程思想，熟悉常见软件系统开发架构， 能熟练运用.Net技术进行开发；
4、熟悉数据库存储原理，并熟练应用(oracle/ms sql server/mysql任意一种)数据库，有数据库调优或应对大数据需求经验者优先；
5、对业务有较强的理解能力，可以和需求部门一起梳理需求，提出建议；
6、有责任心，良好的团队合作意识，乐于交流，工作积极主动，能够主动获取新知识；
7、能承受开发工作压力；

北京益生康健商贸有限公司 该公司其他职位

益生康健公司总部位于北京，是一家多元化的上市集团企业（证券代码：837667），产业涉及电子商务、有机养殖、房地产、文化传媒、生物科技、IT等多个领域。多年来，益生康健始终专注于百姓健康事业，现已发展成为覆盖全国的营养品、有机食品、健康家用电器、高端资源性食品等综合性健康服务的提供商，是目前国内最早、最大、最专业的营养健康品销售和服务平台。公司拥有员工近千人，在北京和青岛两地设有3个大型办公区，在上海等全国重点城市设有5家直营店。

十多年来，益生康健秉承“让人人都拥有一份健康”的企业理念，始终以“优质平价的良心工程，合作多赢的市场理念”领跑国内营养健康品产业，并成为营养健康品行业的龙头企业。公司曾多次荣获“全国质量、服务、信誉AAA级企业”、“中国行业十大领先品牌”、“全国消费者信赖首选品牌”、“3.15放心单位”等众多奖项。

益生康健遵循“有机、自然、生态”的品牌理念，为广大消费者提供有认证的稀缺健康资源，打造可控资源的垂直型链条，构建全裸可视的生产、供应链体系。公司销售的从岭藏鸡、从岭藏鸡蛋、珍品海参、美国威斯康辛州西洋参、北美纯自然熟蜜及来自美国、加拿大、澳大利亚、新西兰等众多健康保健食品在消费者中形成了良好的口碑。益生康健集团以最佳原产地、原生态、无添加、无污染、高品质臻品为广大消费者的健康保驾护航。
