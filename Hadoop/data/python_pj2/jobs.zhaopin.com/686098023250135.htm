一、团队简介

无讼AI团队由一群具有国内顶尖学府计算机、法律专业背景的工程师和法律人组成，团队成员年轻   

有活力，致力于通过技术变革法律行业，目前在法律人工智能及大数据领域已取得行业瞩目的成就。

二、招聘岗位

人工智能团队—法律数据标注实习生

三、岗位描述

标注裁判文书数据，校验数据问题

四、实习地点

北京

五、实习时间

每周至少实习四天，最好保证全周实习。实习周期至少两个月。

六、岗位要求

1. 法律专业本科以上学历，大三、研一和研二学生优先；

2. 头脑清晰，工作认真细致，有责任心，有耐心；

3. 具有较强的学习能力和沟通能力。

七、薪酬和福利

1.100元／天，每日免费提供三餐和精美下午茶（可为外地实习生解决住宿问题），节假日还会举办各种好玩有趣的活动哦～；

2. 办公地点高大上，临近故宫和王府井，环境优美，交通便利；

3.公司创业和学习氛围浓厚，集结了众多行业优秀人才，有很多机会接触法律＋互联网行业最前沿的资讯和成果，帮助你拓宽眼界，快速成长。

无讼网络科技(北京)有限公司 该公司其他职位

无讼力图通过完善行业内部协作和效率工具创新，推进法律服务过程的标准化与律师评价客观化。在面向企业客户的具体业务上，无讼提供诉讼全过程的顾问服务，包括帮助客户寻找合适的专业律师，协助客户完成服务过程管理，以及在全程对服务质量和风险进行监控。无讼网络科技(北京)有限公司致力于打造【中国强大的互联网法律诉讼服务平台】。无讼希望通过前沿的技术和巧妙的产品设计，引领中国法律行业的互联网创新革命。

公司发展历程：

2014年8月 公司诞生，开启征程

2014年12月 公司推出【无讼阅读】，开启国内法律人的专业阅读社区

2015年4月 公司推出【无讼案例】，免费、易用、高效的智能案例检索工具

2015年4月 公司获得IDG资本2700万元人民币A轮融资

2015年6月 公司推出【无讼学院】，提供针对全国法律人的实务技能培训

2015年9月 公司推出【无讼名片】，通过大数据将律师与案例数据关联匹配，展现律师的专业领域、执业经验

2016年3月 公司推出【无讼法规】，检索功能和数据能力得到进一步扩充

2016年4月 公司推出【无讼办法】，以崭新的方式为中小企业提供统包的日常法律服务

2016年5月 公司推出【无讼合作】，从异地查档切入律师合作

2016年9月 公司推出无讼智能法务，为企业提供一体化的案件管理服务

2016年9月 公司正式推出人工智能产品“法小淘”，实现基于特定案情的智能化律师精准匹配

2016年12月 公司完成1.2亿元人民币B轮融资，本轮由华创资本领投，IDG资本旗下管理的人民币基金跟投
