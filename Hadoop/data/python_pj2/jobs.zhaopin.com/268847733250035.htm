举报收藏

1.深入了解与挖掘学校、教师及学生信息，分析教育产品的需求，规划教研资源方向，并提出建设性改进建议，把控教学产品方向；


2. 研发信息安全、人工智能、大数据等领域教学课程体系、课程内容，含应用于高校专业共建和短期培训两大类。


3. 负责教师的日常管理工作。


4. 完成公司领导交办的其他工作。


任职要求：


1. 网络安全、人工智能、大数据等相关专业背景。


2. 至少2年以上在院校或培训机构进行同类课程研发经验。


3. 能承担短期出差任务.


工作地址：

北京神州泰岳教育科技有限公司 该公司其他职位

北京神州泰岳教育科技有限公司是神州泰岳（股票代码：300002）控股的专业从事“互联网+”教育的企业。泰岳教育结合高校创新创业人才培养的需求，通过创新创业实训基地、高校教师创业指导课程以及校企合作产业学院等多种模式与全国高校共同打造中国创新创业人才培养的全新模式，通过人才培养服务当地产业。基于集团公司技术领域优势，泰岳教育整合了网络安全攻防、大数据、物联网等多领域整合，形成具有泰岳特色的校企合作模式。

泰岳教育，立足创新创业教育，培养专业技术人才。
