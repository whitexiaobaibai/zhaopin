举报收藏

岗位职责：

1、参与大数据分布式应用系统服务器端或客户端软件开发工作（需求、开发、故障解决和性能优化等）

2、承担数据抽取、清洗、转化等数据处理程序开发

完成数据分析应用开发，对数据进行挖掘、优化及统计


任职要求：

1、计算机或相关专业本科以上学历/有大数据相关认证优先；

2、3年以上大数据平台产品或应用开发经验；

3、熟悉Java语言，3年以上开发经验；

4、了解 Linux 操作系统使用，熟悉分布式系统的架构与算法；

5、熟悉HBase、Hadoop、Spark、Kafka、Storm 等开源大数据软件；

6、至少熟悉一种数据库，例如 Oracle/MySQL/PostgreSQL/MongoDB/Redis/Cassandra 等；

7、优秀的分析和解决问题的能力，对挑战性问题充满激情；

8、具备良好的团队合作精神，较强的沟通能力，具备良好的技术领导能力。


工作地址：
