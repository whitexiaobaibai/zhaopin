举报收藏

岗位职责
1. 承担车联网大数据与大数据分析平台功能需求分析,组织完成各平台功能的制定
2. 承担各平台系统架构设计、软件架构设计及评审
3. 组织硬件部署、系统集成、系统调试、系统测试与发布以及日常管理维护
4. 承担各平台系统运营的技术支持工作
5. 实施大数据分析算法建模、算法实施
6. 实施移动应用软件开发与测试

任职要求：
1. 本科以上学历，车辆工程、电子或软件类相关专业
2. 三年以上工作经验，熟悉整车开发流程、汽车电子电器零部件开发流程以及汽车电子V型开发模式
3. 精通机器学习算法、数据挖掘等（大数据分析方向），熟悉大数据分析方向编程及相关工具
4. 熟悉计算机系统的数据库管理、数据安全、数据恢复与备份、IT设备运维管理等专业知识
5. 熟悉Android或ios操作系统开发语言、开发流程
6. 具备运用IT系统运维自动化工具进行系统监控、维护与事件记录的能力
7. 良好的英语听说读写能力

工作地址：

苏州博沃创新能源科技有限公司 该公司其他职位

 博沃创新能源科技有限公司专注于新能源汽车电池管理系统、整车控制器系统、ADAS系统等产品的研发、生产和制造销售；同时公司围绕AUTOSAR/FUSA(ISO26262等）/软件算法等核心技术为国内外汽车整车企业及零部件企业提供软件系统开发、测试验证服务和整车电驱系统集成等领域的开发、测试服务，目前已经和国内外众多客户建立良好的合作关系。
    生产制造基地位于江苏南通高新技术经济开发区，公司一期工厂占地25亩，总投资1.2亿元，已于2016年底竣工投产。
    技术研发中心位于上海漕河泾开发区，核心技术团队来自于奔驰、PSA、德尔福等国际一流整车或零部件企业，技术中心拥有完整、标准化、符合国际开发流程的技术开发能力和丰富项目经验一流人才团队。
2016年，公司通过ISO/TS16949质量体系认证，产品从设计开发，到生产制造、产品检验、售后服务，全过程严格执行高标准的汽车行业质量体系流程，以确保产品从开发到量产的全生命周期的产品质量可靠性盒稳定性。在技术开发服务和产品领域获得了广大客户的高度认可。
    由于目前业务的快速发展需要，诚招优秀人才加入我们这个国际化、多元化且年轻而优秀的团队，我们相信蓬勃的汽车行业将带给我们无限的机会和发展前景。同时公司也会为每位员工提供良好的发展前景和个人待遇。
我们期待优秀的您加入！
