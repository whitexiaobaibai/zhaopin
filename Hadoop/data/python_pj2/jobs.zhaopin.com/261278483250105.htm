举报收藏

职位描述
1、负责互联网产品的框架的设计及开发，并负责公共核心模块的代码编写；
2、业务核心模块的开发，能够独立进行模块级需求分析、流程设计并按照编码规范进行代码编写、自测试
3、熟悉整个软件过程，能够沟通需求、控制项目进度，有良好的文档能力，可独立编写详细设计文档
4、具备良好的沟通表达能力，协同他人并组织跨团队协作，保证项目质量与进度；
5、针对新人、普通开发人员进行有效辅导，帮助其快速成长；

任职条件
1. 精通Web编程，5年及以上使用Java语言进行web开发的经验，或者微服务框架开发经验3年以上
2、视野开阔，熟悉主流的java web开发相关的开源框架，如消息队列，数据库中间件等，并对他们的优缺点和适用场景有自己的认识。
3. 熟悉MySQL/PostgreSQL/Oracle数据库中的一种或多种，有数据库调优经验
4. 有良好的组件级建模能力，熟悉MQ、Cache、TCP/IP原理，能够设计复杂业务、高并发、大数据量的系统
5、熟悉操作系统，精通linux，熟练使用nginx, apache , tomat等web服务器； 
6、对于分布式缓存以及nosql有一定了解，熟练使用memcahed, redis、Dubbo等服务

北京海淀区
