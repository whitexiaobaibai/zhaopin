举报收藏

岗位描述

1、负责企业级大数据平台的整体规划与设计，以及风控数据集市的设计开发；

2、负责数据仓库建设对应的技术框架选型、核心模块搭建和技术难点攻关；

3、带领团队完成ETL过程的设计、开发和优化，将采集来的源数据按照业务需求进行清洗校验、加工计算；

4、跟踪数据仓库平台的运行维护及BI报表的开发和维护。

任职要求

1、5年及以上相关工作经验，有互联网金融从业经历和风控数据仓库搭建经验者优先；

2、熟悉MySQL等关系型数据库，熟练SQL开发；

3、熟悉Hadoop, HBase, Spark, Storm, Redis, Mongodb, Elastic Search等大数据相关技术，对Hadoop生态圈有较全面了解；

4、具备数据集市建模，有大数据ETL实施经验；

5、具备数据挖掘/机器学习项目经验者优先；

6、具有职业化的自我管理能力，优秀的协调沟通与团队协作能力，责任心强。

北京星河金服科技有限公司 该公司其他职位

星河金服集团是中国领先的企业金融科技集团，是领先的产业互联网集团——星河世界集团旗下A股上市公司步森股份（002569.sz）的全资子公司。星河金服致力于打造开放的金融生态系统，通过大数据连接中小企业和中小银行，实现精准匹配、产品设计、风险管理等智能化服务，用科技手段优化和重构中小企业与中小银行之间的传统金融服务模式，让中小银行为中小企业创造价值，让金融服务更快、更准、更稳、更优。

星河金服集团旗下及相关业务包括星河金融管家、企业金融中心、大数据征信中心和星河金融资产交易中心等四大平台，正在为50万家中小企业提供一站式金融服务。

与此同时，星河金服集团与投资控股的金融科技公司及关联公司一起产生业务协同，意在打造“金融大脑”，连接中小企业和中小银行，构建起一个人工智能化的金融服务生态圈，以期更好地服务广大中小企业，助力实体经济发展。

  星河世界是中国领先的产业互联网集团，连接互联网与产业，致力于用互联网技术实现传统产业的升级改造，重构产业链及业务链，优化产业效率。

  星河世界成立于2005年，目前已控股三家上市公司，并拥有四大集团、16个事业群、以及200多家互联网成员公司，已形成了创业、大数据、金融三轮驱动且相生相融的产业互联网矩阵。

星河世界2015年净利润26.7亿，成员公司总市值约1000亿人民币，已为20万创业者、50万企业及3亿用户提供了服务，正在向中国各大城市及美国等海外市场拓展。

星河世界控股的三家上市公司，分别是A股天马股份（SZ002122）、A股步森股份（SZ002569）、美股WOWO Limited（NASDAQ: JMU）。
