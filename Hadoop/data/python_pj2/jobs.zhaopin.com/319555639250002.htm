举报收藏

岗位职责
1. 根据公司发展蓝图，统筹、拟定公司技术发展方向，确保技术发展规划与业务相符；
2. 负责组织、制定和实施重大技术决策和技术方案；
3. 领导技术和产品团队实现系统架构设计、系统分析、产品开发、平台搭建和性能优化等各项任务；
4. 建立和完善公司技术支持体系，不断推动产品技术创新和流程优化，持续提升公司整体技术能力；
5. 负责技术团队的组织规划、日常运营和管理，领导部门实施年度工作计划，完成年度目标；
6. 推动并实施技术管理、技术开发等相关制度体系的建设，制订符合业务发展需要的规范化制度、流程并监督落地实施。

任职资格
1. 计算机相关专业本科以上学历，8年以上工作经验；
2. 熟悉医疗信息化、互联网医疗领域业务，深刻理解行业发展方向，具备敏锐准确的洞察力和缜密的逻辑思维，能够把握行业业务发展动向和关键技术发展趋势，并有独立见解；
3. 具有产品规划、设计的成功经验和具体实践，参与过多层应用程序、用户面向网站开发；
4. 精通大型系统架构，具备宏观的大数据应用经验对高负荷超大数据量的系统架构及性能调优有深入的研究；
5. 精通.net/JAVA/C++等主流技术体系的架构和设计模式，有大数据、大并发的项目经验；
6. 精通软件研发（开发平台、WEB框架、数据库、操作系统、软件设计）信息安全及项目管理；
7. 具备优秀的综合分析能力、商业谈判能力、部门管理技能。

工作地址：

上海市静安区灵石路697号健康智谷8号楼

目前已经开展了CT、核磁和胶囊胃镜的远程服务，服务的客户近百家；
大象医生远程诊室，从远程会诊到治疗再到转诊，构建全国最大的连锁专科远程医疗平台。
