举报收藏

岗位职责：

1.      负责进行产品线规划,制定产品发展路标；

2.      负责完成产品立项、产品需求生成、产品实现策划的具体工作,并负责组建团队进行产品实现；

3.      负责与客户的技术交流、技术方案宣讲、产品演示等工作；

4.      行业/竞争/客户的跟踪和分析，识别重大机会和风险，并输出相关的分析报告，形成对相关业务的建议、推动决策，支撑中长期业务发展。

 

任职要求：

1.      通信、计算机或电子类相关专业本科以上学历，两年以上工作经验；

2.      具有扎实的无线通信基础和一定的研究分析能力；

3.      良好的写作和表达能力；

4.      对行业新技术有较强的洞察力和敏感度，学习能力强；

5.      领悟能力强，对客户需求有敏捷的感知能力，能准确理解需求；

6.      具有良好的团队意识和合作精神，责任感强、性格开朗、善于沟通和交流。

 

工作地址：

海淀区学院路29号大唐移动大厦

大唐移动通信设备有限公司 该公司其他职位

　　大唐移动通信设备有限公司（以下简称“大唐移动”）是国务院国资委下属的大型高科技央企——大唐电信科技产业集团旗下的核心企业，是我国拥有自主知识产权的第三代移动通信国际标准 TD-SCDMA 的提出者、核心技术的开发者以及产业化的推动者，始终主导 TD-SCDMA 后续技术标准 TD-LTE 的技术演进和产业化进程，并拥有 TD-SCDMA 和 TD-LTE 的核心知识产权。　　

　　大唐移动依托企业品牌与大唐集团的央企特质，针对我国政府、企业、事业单位的信息化需求，依靠“自研 + 集成”的解决方案、策划与实施能力，拥有自主核心产品，并在智能安防、信息安全领域打造一定行业知名度，力争成为市场多元、产品提供丰富，具备核心竞争力的政企信息化解决方案与服务提供商。凭借自身 TD-SCDMA 、 TD-LTE 移动通信领域的核心优势，致力于行业信息化建设的研究和推广工作，秉承“行业信息化专家”的发展理念，以前瞻性的信息技术，率先开发出满足行业信息化应用需求的小型化、专业化移动通信专用系统产品，为煤炭、石油、海油、轨道交通、电力、核电、港口、航空、水利等行业用户提供全方位、个性化的整体信息化解决方案，并已成功应用于神华集团、中石油、中海油、陕煤集团、中煤集团、山西潞安集团等大型企业。在众多行业领域奠定了优势地位，打下了坚实的基础。　

　　大唐移动现有员工中，研究生以上学历比例为 60% ，拥有占总人数 45% 以上的专业技术团队，汇聚众多国内外知名的信息通信技术专家，拥有大量专利和世界领先的科技创新成果。通过积极探索行业龙头企业与高校联合培养人才的新模式，大唐移动加快高端创新型人才的培养，与国内 20 多所高校、院所全面建立了战略合作，持续打造国际化、职业化、成熟化的人才队伍。　　

　　大唐移动，与您携手共赢未来！
