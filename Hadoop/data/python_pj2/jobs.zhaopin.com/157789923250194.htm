举报收藏

岗位职责： 
1. 参与互联网大数据内容分析应用（如舆情系统等）的设计，核心代码开发，系统优化，内容分析等工作；

2. 参与公司互联网大数据内容分析产品应用需求，规划方案，并能根据方案展开研发工作； 


任职要求： 

1.     计算机相关专业本科以上学历，3 年以上开发工作经验

2.     有2年以上hadoop使用、开发经验；

3.     了解网络爬虫系统和搜索引擎系统，并有成熟的设计理念。 

4.     熟悉MySQL ，Nosql，mongodb数据库技术；

5.     精通相关语言和框架，       

       如 Java,WebService,Hibernate,Struts,Spring,XML,Json,nutch ；

6.     熟练使用软件设计和开发的相关软件如svn、cvs或vss,PowerDesigner,和其他        相关项目管理软件,遵守编程规范。

企业简介

   北京顶尖时代科技发展有限公司,成立于2005年，自成立之日起，致力于企业信息搜索、竞争情报、舆情检索领域商业应用软件的自主开发，专注于将最新的专业技术与最先进的管理理念相结合，为制造业、流通业及服务业的诸多企业提供专业化的服务，以协助企业完成生产效率、管理水平及服务质量的提升。 公司核心技术团队人员在IT行业有着多年开发与实施经验，在几年的发展中，公司密切关注互联网时代的软件技术的发展与变迁，掌握了全文搜索应用软件在有效支持企业应用提升上的核心方法，同时根据客户行业特点与对不同的业务需求，设计开发与实施了许多不同行业、不同形式的大中型企业软件应用系统，积累了丰富的行业应用知识与经验。
    公司目前是HP-Autonomy公司的核心合作伙伴，负责对其产品的销售、应用开发与实施维护，提供最佳的企业级全文检索与数据整合服务解决方案。


企业网站: http://www.toptimetech.com
