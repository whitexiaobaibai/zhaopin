举报收藏

岗位职责

1.负责凤凰文学BI、大数据平台的搭建和开发，负责产品主要部分代码编写；

2.参与BI业务需求分析、架构设计、开发；

3.负责数据收集、数据模型设计、报表开发，仪表盘开发，系统集成；

4.支持运营，根据业务需求，测试问题，持续修改完善产品,提升性能；

5.负责BI技术团队管理，根据公司发展需要制定目标，带领团队执行工作计划，保证目标实现。

任职要求

1. 三年以上大数据项目经验，一年以上团队管理经验；

2. 具备大型数据仓库架构设计、模型设计、ETL设计研发的相关经验；

3. 具有良好的编程能力，熟悉Java/Python/shell等编程语言；

4. 熟悉SQL/HQL，熟悉MapReduce及Hadoop/Hive的使用，有Spark平台经验更加；

5. 对数据敏感，学习能力强，优秀的逻辑思维能力和业务理解能力；

6. 良好的执行力和沟通协作能力；

7. 有用户行为研究、数据驱动运营、互联网产品相关数据分析等方面工作经验者优先考虑。
