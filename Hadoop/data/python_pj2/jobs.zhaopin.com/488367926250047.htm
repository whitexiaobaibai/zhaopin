举报收藏

岗位职责：

1.  协助公司业务团队为企业类客户提供专业的咨询服务和解决方案；

2.  深度挖掘客户需求，捕捉潜在商机，与业务团队一起承担销售任务；

3.  熟悉行业上下游生态体系；

4.  不断学习新的IT技能与热点，持续给企业用户布道；

5.  承担用户需要的技术解决方案的书面文档的编写与传递；

 

岗位要求：

1.  性格开朗大方，表达流利，热爱沟通；

2.  具有熟练的英文阅读&表达能力（各类英语考级认证不纳入考量标准）；

3.  大专以上学历，计算机类相关专业，或热爱IT&云计算的理工类专业；

4.  1-2年工作经验，或优秀应届毕业生；

5.  对云计算、大数据、BI、AI、数据库等领域有项目或实践经验者；

6.  有编程能力者，优先考虑；

 

公司介绍：

1. 精鲲科技由一群原IT顶尖厂商Very Niubility的职业经理人站在人文与科技的路口创立而成，立足IDC业务，致力于企业云计算与大数据业务；

2. 目前通过顺利大逃杀，成功完成双软认证、高新企业认证，准挂牌企业；

3. 公司暂时蜗居杨浦某工业园一隅，但不久的将来会给大拿们准备自由驰骋的五百平米办公室；

4、     公司没有高逼格的意大利咖啡机,但仍然时常有飘香的咖啡点心；

4. 公司没有女神，但有一群男女比例平衡的团结一心的年轻小伙伴，我们的口号是：“不做备胎”；

5. 免费享受一年两次国内、国外纯玩Outing福利（让购物团去死吧）；

6. 公司关心你的健康和形象，转正之后享受正装置装费，免费体检和额外保险；

7. 公司关注你的眼界和视野，一有机会就会公费派你到世界尽头参加IT各类行业party；

8. 还有其他可以想可以提的五花八门的各种良心补贴；

9.      只要你创造价值，你就会得到精鲲暖心的回报，你成长，精鲲成长，精鲲成功，你成功。

上海精鲲计算机科技有限公司 该公司其他职位

上海精鲲计算机科技有限公司成立于2009年，是专业的IT系统集成服务商，业务领域涵括服务器虚拟化，桌面虚拟化，终端虚拟化，网络安全，数据安全，云计算，大数据以及配套的技术支持服务。精鲲有一支专业能力较强的技术工程师队伍和充满活力的销售团队，90%的技术人员拥有计算机相关学士学位，且具备优粮良沟通与分析能力。成立以来，精鲲已成功服务上百家行业领先客户，为行业客户的IT资产价值最大化提供了有力的支持和保障。快速发展的精鲲十分渴求优秀人才的加入，公司内部有健全的培训体系、科学合理的人才培养计划，远超行业平均水平的奖金提成体系，力求精鲲的每一个成员都能得到个人价值的最大化实现，和公司一起共同分享成功的果实。 
  对于客户，精鲲是您最专业的IT外包服务商. 精鲲 专业、高效、优质的产品和解决方案助力您更好的服务您的商业客户，取得业绩和成功.精鲲专注于云计算架构下的企业级IT整体解决方案研发和实施，多年来致力于此，积累了丰富的产品实施经验，是您值得托付的IT服务商.
