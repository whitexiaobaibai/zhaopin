举报收藏

工作职责：

1.       独立完成互联网大数据项目的数据分析及消费洞察报告撰写；

2.       根据销售线索，撰写研究项目计划书；

3.       独立完成周、月、季度连续型分析报告的撰写和客户服务；

4.       对目标行业进行研究与分析，提出有意义的观点和解决方案；

5.       参与完成数据咨询的解决方案框架，确定数据分析目标，完成方案报告的撰写及汇报；

6.       负责数据咨询项目的售前支持工作，与目标客户进行方案的讲解与沟通；

7.       负责数据咨询知识模块的提炼，研究新的数据分析思路。

 

任职要求：

1.       本科以上学历，市场营销、企业管理、社会学、新闻学、心理学等相关专业；

2.       2年以上数据分析经验，具有数据敏感度及钻研精神；

3.       精通定性与定量分析；

4.       学习领悟能力强，能快速进行信息收集、分析、归纳、总结，能够撰写逻辑清晰、结构紧凑、观点突出的报告；

5.       良好的沟通表达能力，抗压能力强，积极主动，良好的团队协作能力；

6.       精通PowerPoint、Excel等办公工具；

7.   英语读写能力良好。


Benefits:

股票期权、热门领域、大数据核心产品及技术、发展空间巨大、上班超方便、国际化团队、绩效奖金、五险一金……


工作地址：

上海市黄浦区山东中路311号6楼

信励(上海)信息科技有限公司 该公司其他职位

simplyBrand（信励(上海)信息科技有限公司）成立于2013年，专注于互联网大数据技术服务，提供基于数据驱动的商业洞察解决方案。通过采集分析海量互联网UGC(用户贡献内容)数据，帮助企业挖掘数据的商业价值，更轻松、简单、智能地掌握价值信息，提升在研发、生产、营销等方面的科学决策。

总部上海，设有北京、香港、日本、台北、新加坡业务联络处。核心团队长期致力于互联网消费者行为洞察研究、行业竞争分析咨询及媒体情报监测工作，在软件技术、数字营销、数据分析方面积累了丰富的实战经验。数据产品服务线包括消费市场分析工具、微信公众账号分析平台、数据API接口，及消费者行为洞察、互联网市场研究等解决方案，已为快消、金融、零售等多个行业知名品牌提供服务。

simplyBrand通过服务行业领先用户及特定痛点需求用户，积累大数据算法模型，加强对行业知识及领先品牌主用户的服务经验，提升行业的专业分析能力；同时，也开发标准数据分析平台如simplyKOL及API接口产品，满足一般企业用户的数据使用及分析需求，培育目标用户群的同时，增加企业的营业收入。
