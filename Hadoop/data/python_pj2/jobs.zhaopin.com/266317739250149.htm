举报收藏

岗位职责：

1、定期组织收集有关人事招聘、培训、考核、薪酬等方面的信息，为公司重大人事决策提供信息支持

2、负责集团总部公司人力资源战略的执行 。

3、根据公司的情况，组织制订公司人事管理制度、人事档案管理制度、员工手册等规章制度、实施细则和人力资源部工作程序，并组织实施。 

4、根据公司的发展规划，提出机构设置和岗位职责设计方案，对公司组织结构设计提出改进方案 。

5、负责开发、引进外部人才，组织实施招聘工作，并参与对应聘人员的面试筛选。

6、负责制定公司员工的培训和发展计划，组织安排对员工的培训 。

7、负责组织公司员工的考核，并对考核结果做出评价。

8、组织公司薪酬管理工作，审核公司员工每月的工资。

9、受理员工投诉，并组织相关部门妥善解决 。

10、负责员工五险一金的管理工作 。

11、与员工签订各种劳动合同，以及处理各种与合同相关的事宜。

12、进行各类人事档案的归档保管工作。

任职要求：

1、人力资源管理相关专业毕业；

2、统招本科以上学历；

3、5年以上人力资源全模块实操及管理经验；

4、具有国企背景经验者有先考虑。

工作地址：

北京市东城区歌华大厦B座422 查看职位地图

中云文化大数据科技有限公司 该公司其他职位

中云文化大数据科技有限公司注册资本5.2亿元，由贵州省最大的文化传媒集团广电传媒集团及多家上市公司发起设立，是贵州省实施出版广电行业互联网转型升级重要试点单位。是“国家十三五时期文化发展改革纲要重点项目”和贵州省文化建设重点工程。公司承担中国文化出版广电产业项目的开发实施，负责中国文化出版大数据产业园的开发建设，获得“贵州省十三五重点文化产业项目”、“贵州省十三五重点战略性新兴产业项目”、“贵州省文化产业示范基地”。

公司依托领先的互联网、大数据、云计算、人工智能等领先的技术能力，建设国家级出版广电行业唯一的数据中心，建设国家级版权数据库，建设国家级数字内容传播监测平台、分发平台、交易平台，打造涵盖视频、音乐、文学、游戏、动漫、图片等领域的全产业链大文化电商平台。

中云公司拥有贵州省文化科技企业最一流的办公环境、最优厚的福利待遇，为文化传媒、投资、技术等专业人员提供最大的发展平台。
