举报收藏

岗位职责：

1.对公司业务的合规风控进行把控（私募股权投资基金方向），制订风控政策、流程和制度，完善风险控制和内控制度体系；

2. 搜集、整理与公司经营有关的法律、法规、行业政策等信息，评估公司相关项目管理制度、流程、方案及计划的法律合规性；

3.参与金融产品交易结构的搭建，提示存在的风险点；在产品/项目上线前，坚持独立、客观、公正的工作原则，及时、独立出具风险评估意见；

4. 负责监督和跟踪产品的实际运行情况，与其他部门配合，对风控规则、流程进行持续优化。


任职要求：

1、有私募股权投资基金项目经验；

2、全日制统招本科及以上学历，财务、法律、金融类等相关专业，具备基金从业资格证或已经通过基金从业人员资格考试；

3、具有金融行业风险管理工作经验，对大数据风控有一定的了解和研究；

4、具有优秀的风险控制意识和风险管理能力。9

中国森田企业集团有限公司 该公司其他职位

       中国森田企业集团有限公司（简称：中国森田）是伴随中国改革开放而快速成长的综合性企业集团，成立于1985年，总部位于北京。全资持有森田投资集团、森田资产管理、森田信息科技股份、森田国际贸易、森田文化产业集团等多家专业集团和子公司，产业分布于华东、华中、华北、西南、香港、蒙古等地区和国家。

       中国森田企业集团经历数十年风雨历程和资源整合，紧密结合市场与自身实际，勇于进取、力求创新，实现了跨越式增长。

       森田投资集团有限公司是中国森田企业集团有限公司的全资二级专业集团公司，1989年成立，总部设在厦门，是一家致力于互利、共赢、创新、发展的专业投资集团公司。集团由森田环保、森田信科、京森国贸、森田农林、河南森田、森博测绘、励和咨询等公司组成，是中国森田的对外投资平台。

       森田资产管理有限公司是中国森田企业集团有限公司旗下的二级子公司，2010年成立于北京，是整合集团内金融资源而成立的综合服务平台，专业从事委托资产经营管理、投资管理、投资咨询、股权投资、PE基金管理及投资、并购及结构性融资、信托、证券基金管理及投资等业务的管理与服务机构，是中国森田的资产管理平台。

        森田信息科技股份有限公司是中国森田企业集团有限公司旗下的三级子公司，致力于互联网+金融创新，2015年成立于北京，是中国森田金融版块重要组成部分。作为行业内首家在不良资产领域完成全行业链条布局的科技公司，森田信科结合集团资源的先发优势，以精准的定位切入点和完整的商业逻辑领跑于同行，发展出综合了技术服务、平台资源整合、专业运营处置、资金资产灵活周转的强项业务，是中国森田的互联网金融平台。

        森田国际贸易有限公司是中国森田企业集团有限公司的全资二级公司，1996年成立于北京。主要是以燃料油、矿产品、金属材料、化工产品、农副产品、纺织原料、橡胶、木材等大宗贸易业务为主的进出口贸易专业公司，是中国森田企业集团通过贸易纽带紧密联结国内与国际市场的贸易流通平台。

        森田文化产业集团有限公司是中国森田企业集团有限公司的全资二级集团公司，2010年成立于北京。业务主要以电影、电视剧、舞台剧的投资、制作、发行和演艺经纪为主，以导演、编剧、儿童电影、青年戏剧等工作室、中国传统文化传播及经典华服定制等为辅，是中国森田的文化传播平台。

        森田能源集团2010年成立于北京，是以石油、天然气等资源的勘探、开采、储运、销售为主，以稀土、贵金属、有色金属等矿产的勘探、开采、销售为辅的专业集团公司，获得了蒙古国史上颁发的首张天然气开采许可，拥有蒙古国阿拉坦面积为6566平方公里的石油天然气开采权以及蒙古国南戈壁省占地面积约4万平方公里的天然气勘探许可权，是中国森田的实业平台。

        森田法律服务主要提供处理不良资产和金融非诉讼法律两个方向的“全方位、全流程”的专业化法律支持。无论是传统优势领域，如银行融资、外商投资、公司事务、国际贸易、争议解决以及证券业务，还是新兴的业务领域，如知识产权保护、劳动法、破产、不良资产、金融衍生产品、资产证券化和杠杆收购业务，森田法律服务一直处于行业前沿，以创造性思维和富有实效的法律服务方案为客户提供优质、高效的法律服务，建立起客户与森田法律服务之间的持久信任。

        中投律师事务所是中国森田唯一对外提供法律服务的专业合作机构，其精英团队拥有娴熟的专业技能、广泛的社会资源、独特的办案能力和敬业精神，建立了一套专业化的服务模式和标准，同时拥有资产调查和资产处置的上下游渠道，结合客户的实际需求，提供优质、精品的专业法律服务，是中国森田的法律保障和服务平台。

        森田四海、创新共赢；天地森宇、物泽田兴。在全体森田人的共同努力下，中国森田企业集团逐步成长为实力雄厚、影响广泛的综合性企业集团。中国森田企业集团将秉持“人和智信，勤德精细”的企业核心价值观，实现企业与员工、合作伙伴的共赢发展。
