举报收藏

岗位职责：

1.      负责公司对外的项目、技术、产品、理念和思路交流相关文档的编写；

2.      负责对合作伙伴、用户和公司内部进行产品＆业务的交流；

3.      负责公司解决方案、产品资料和政府项目申报文件技术文稿的编写；

4.      参与公司投标项目，负责表述技术部分的编写及答疑、应标等；

5.      与市场人员共同协作推广公司产品和业务。


任职要求：
 

1.      本科及以上学历，计算机类相关专业；

2.      5年以上工作经验（有政府项目、大数据应用、人工智能行业经验优先）；

3.      熟悉项目和售前工作流程，能明确用户需求，具备良好的建议书、解决方案、产品资料和宣传资料的写作技能；

4.      具有出色的方案策划和演讲能力；

5.      有责任心，为人真诚、热心。

中科点击(北京)科技有限公司 该公司其他职位


　　中科点击（北京）科技有限公司成立于2007年，公司战略布局人工智能与区块链，专注于大数据分行业应用。公司在大数据领域拥有30项软件产品知识产权，精心打造“军犬”与“慧数”两大系列大数据产品矩阵，迅速成长为中国行业大数据应用专家。2016年，完成3500万Pre A轮融资；2017年，中科点击获得1亿A轮融资。
公司推出“军犬系列”专业大数据平台——军犬舆情系统、军犬舆情管家、军犬网评管家、企业内参、军犬标王、大数据一体机等产品；
　　公司重磅上线“慧数系列”行业大数据平台——慧数汽车、慧数教育、慧数金融、慧数医美、慧数招商等多个分行业大数据应用平台。
　　以中科点击北京总部为中心，推进全国网格化落地布局，目前已设立贵州、广西、河北等三个省公司。全国各地大数据精英200余人，其中80%以上为技术研发人员，核心团队成员具有多家上市企业管理经验。
　　10年来，公司服务的各类客户涵盖党、政、军、企业、个人等高端客户，分别来自公安、国安、宣传、网信办、招商局、质监、科技、教育等几十个政府机构与委办局，覆盖汽车、互联网、金融、文化、通信、旅游、健康等数十个行业领域。典型代表客户空中网、新时代国珍、汽车之家、奥迪、宝马、韩国现代、林肯汽车等国内外知名客户。
