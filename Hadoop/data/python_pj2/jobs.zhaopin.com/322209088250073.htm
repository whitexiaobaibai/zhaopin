关于享骑About xiangqi

享骑电单车是全国成立最早、规模最大的共享电单车运营商。运营一年半以来，享骑始终坚持为用户提供安全可靠、绿色环保、轻松出行的产品，致力于解决城市中10公里以内的中短途出行问题。

作为国内首家共享电单平台运营商，享骑从2015年10月运营伊始就自主研发了“电子围栏”技术，成为了全国首个全面推行该技术的共享电单车平台。该特有的模式为市政部门对市容市貌的管理，提供了良好的帮助和支持。

2015年10月，享骑共享电单车率先在上海展开服务，为安全的绿色出行开启新征途。

享骑电单车

享骑完成了传统电动车的技术攻关，成就独特出行品类；从“缓启动”技术进一步创新推出“非零启动”，使车辆可以平稳启动。

 

我们的标准Our standards

平稳启动、电助力、符合国标、非零启动技术、缓慢加速、最减负最轻松、安全保险

单车搭载电助力后，解决了骑行中脚踏带来的体力消耗，可以轻松骑行更远的距离。除了锂电助力外，还配有篮筐、车灯等设备，使用户骑行时可达到“最减负、最轻松”状态，搭载的“非零启动”技术，可使车辆可以平稳启动、缓慢加速，让使用者有最舒适的体验。

 

01- 合规国标车

时速不超过20公里、车身重量24kg、脚踏骑行功能

享骑电单车是全国首家符合“国标”的共享电单车平台，完全有别于传统的“电动车”。享骑严格按照“国标车”标准生产，时速不超过20公里，车身重量24kg，在国标要求的40kg范围之内，带脚踏骑行功能。在速度把控、整车重量及技术配置等方面，都朝着更为安全、有序、健康的方向发展。

 

02- 锂电助力，续航能力更胜一筹

防水、低温性、SOC计算电量等自身保护功能

采用便于后台监管的智能化电池管理系统(BMS)

新增SOC(剩余电量显示)、SOH(故障诊断)、故障预警等

 

享骑全线配备知名锂电池品牌“天能”，在应对防水、低温性、SOC计算电量等自身保护功能方面，已非常安全。另外，对电池外壳进行结构改进，可采用便于后台监管的智能化电池管理系统(BMS)，新增SOC(剩余电量显示)、SOH(故障诊断)、故障预警等和电池监管切身相关的一些参数显示，一旦出现参数异常，就能被后台监控到，及时派出享骑运维和充电团队进行处理。

 

03- GPS精准定位，360度主动监控车辆

装有独立GPS系统的享骑电单车，可以随时监控车辆的位置；内置的后发电机，即使在车辆无电的状态下，也能7x24h向后台持续上报位置，通过GPS定位来全面防御车辆的丢失。

 

04- 首创“电子围栏”技术，破解共享单车“乱停放”问题

享骑的“电子围栏”技术具有无桩无线、定位精准的优点，很大程度上解决了当下无序停放的诸多管理问题，这一技术获得了政府认可，并推而广之，被诸多的共享单车企业所借鉴。

 

 

 

享骑业务支撑系统

业务支撑系统为生产制造、线上运营、线下运控、数据采集、实时监测等十余个业务版块，

提供强有力的支持。

 

 

业务支撑系统business support system

享骑的业务支撑系统包含，物料管理、财务管理、客服管理、质量管理、实验室管理、设备维修管理、充电管理、仓库管理、运输管理。

同时具备具有强大的数据储存、分析和可视化监控功能，可全天候监控车辆分布、站点分布、站点热度、电池电量等信息。目前，系统已在数据分析、供需分析、站点分析等领域发挥巨大作用，进一步巩固了享骑电单车在智能共享方面的领先优势。

 

01- 运控保障

系统的车辆信息检测功能，能为线下运控的车辆查找、续航、调度、维修等工作提供支撑，并做到360度可视化直观展示。

02- 运营保障

系统完善的数据分析功能，可对全平台用户注册、充值、借车、用车、还车等多个维度进行的全方位监测，根据数据反馈可制定相应的运营策略，实现精准营销。

 

03- 实时监测

系统的实时监测功能，根据时间日期多轮采集筛选。对车辆欠压、故障、散落等车辆状况进行实时监测，并自动分辨异常情况。

 

04- 数据采集

系统的数据采集功能，可对站点的车辆流出、流入、超容；站点热度、供给需求、用车习惯等数据进行采集分析，根据分析情况，进一步优化的平台战略布局。

 

 

享骑智能APP

享骑app以使用场景及用户便捷操为导向，设置智能方便的一键借还车操作，实际使用过程中设有“临时锁车”的人性化设计，全面提升用户体验

 

共享

指定停车点式集中借还车，使用户可以第一时间知道哪里有车可用，养成良好的用车习惯，同时也能为用户带来更便捷、更高效的出行体验。

 

智能

享骑智能APP可为用户提供借车、还车、临时锁车、站点查找、站点路线导航等全面的用车操作。

 

 

适用场景

住宅区、地铁公交、写字楼、公共服务区、商圈

 

产品使用方法

1.智能找车

打开APP即可查看周边享骑停车点及附近车辆

2.智能解锁

点击我要用车，扫描车身二维码或输入车辆编码即可对车辆进行解锁使用

3.临时停靠

骑车途中如需临时停靠，点击APP中的“临时锁车”按钮，即可临时锁定车辆，满足用在用车过程中临时办事的需求

4.一键还车

在APP上查看离您最近的停车点，把车停在规定停车点白线内，在APP中点击“我要还车”即可成功结束订单
