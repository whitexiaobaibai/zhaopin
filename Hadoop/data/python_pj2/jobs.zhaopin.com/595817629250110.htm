五险一金绩效奖金加班补助带薪年假弹性工作节日福利
  • 职位月薪：8001-10000元/月 
  • 工作地点：北京
  • 发布日期：2017-10-13 09:27:40
  • 工作性质：全职
  • 工作经验：不限
  • 最低学历：大专

  • 职位描述
  • 公司介绍

举报收藏

岗位职责：

1、负责集团产品的文案撰写，了解大数据、AI智能；

2、协同品牌推广和营销活动策划相关推广文案的撰写；

3、参与各种营销推广策划，并撰写策划方案和推广方案，配合营销团队完成产品描述；

4、负责公司品牌和产品的宣传广告与新闻稿、软文等撰写。

任职资格：

1、市场营销、中文、广告、新闻等相关专业毕业；

2、有广告公司、公关、媒体从业及策划文案经验优先；

3、有新媒体运营等相关经验优先；

4、能写各类软文的文案高手；

5、出色的沟通和语言表达能力，具有良好的团队合作能力；

6、熟悉微博、微信账号的使用，是公司推文吸引眼球，粉丝数量大增，有过国内成功微博微信案例者优先。


工作地址：

北京市西城区北三环中路甲29号院华尊大厦B座605
