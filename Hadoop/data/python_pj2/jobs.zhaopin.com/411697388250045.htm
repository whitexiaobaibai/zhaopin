举报收藏

岗位职责：

1. 负责公司相关数据业务规划和落地工作，搭建基础数据收集、数据仓库和大数据应用平台；

2. 熟悉常用数据分析模型，跟踪和分析运营数据，与分析师、数据工程师、算法工程师、业务方紧密合作，发现产品业务潜在的缺陷与机会，为产品业务提供决策支持；

3. 结合业务进行数据监控和分析统计，定期输出用户、市场数据专题报告，挖掘数据的商业价值；

4. 监控重点数据指标，能从数据异动中主动发现问题，结合对业务的深入理解，推动产品、算法团队迭代优化；

5. 负责数据产品的需求整理，原型设计，上线和迭代；

任职要求：

1. 计算机及相关专业或管理类专业本科及以上学历

2. 2年以上互联网产品/数据分析经验，可独立进行业务分析和产品规划，熟悉数据仓库及海量数据挖掘者优先；

3. 具备良好的商业意识和逻辑思维能力、跨部门沟通和问题处理能力；

4. 较强的逻辑分析，对数据敏感，执行力强；

5. 思路清晰，工作细致、主动、责任心强，具有良好的组织协调能力、团队精神及较强的学习能力。



工作地址：
