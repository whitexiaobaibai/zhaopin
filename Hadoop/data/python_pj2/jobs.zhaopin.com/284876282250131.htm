举报收藏

岗位职责
1. 负责构建公司征信大数据体系和评分模型体系的顶层设计；、
2. 负责数据源接入管理体系搭建、运营，负责多数据源的管理；
3. 负责数据模型管理，包括数据模型运营、数据模型修正、数据模型能力封装；
4. 负责项目管理：负责与运营商体系内各部门的对接，确保模型量化策略的准确实施和上线后的监控优化；

任职资格
1. 硕士以上学历，数学、应用数学、物理、金融数学、统计学相关专业优先；
2. 8年以上工作经验，4年以上数据分析、数据建模工作经验；
3. 作为项目负责人搭建过数据分析平台系统，负责建立多套数据模型；
4. 熟练掌握python，有使用pandas，numpy，sklearn包经验，熟悉GBDT，xgboost等机器学习模型；可以提交模型编程代码优先考虑；
5. 熟悉数大数据分析行业动态，熟悉数据处理的各项技术知识，熟悉数据分析行业规范等相关信息。
6. 曾在FICO、德勤等知名信用风险评估公司具有项目管理和咨询工作经验者优先。
