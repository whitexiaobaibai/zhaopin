举报收藏


职位描述：

公司是获得了重量级投资人投资的金融大数据初创公司。


公司专注于与金融相关多元数据的采集，存储，建模，分析，挖掘与可视化。公司的愿景是希望我们的产品可以提升金融市场投融资和定价效率，让金融市场能更好的起到资源优化配置的基础作用。



我们需要您：

1. 充分理解产品短中长期战略，收集整理市场团队反馈和用户述求，设计和维护系统化的产品设计方案；

2. 与开发、测试、数据部门紧密配合，细化每个产品迭代的详细需求，制订并推动迭代计划的实施；

3. 领导产品团队，负责产品内部设计文档、设计图，产品帮助文档、教程、宣传册的编写和维护工作；


我们期待这样的您：

1. 具备产品设计系统方法论，有丰富经验，主导过偏数据分析类产品的设计工作；

2. 有良好的逻辑能力和一定的数学基础，能理解复杂的数据分析流程，能充分理解各种繁杂的用户场景，并抽象、转化到产品的设计中；

3. 有良好的沟通能力和抗压能力，与各个部门沟通协调，推动产品的开发过程；

4. 在金融领域或数据分析领域有相关经验者优先。
