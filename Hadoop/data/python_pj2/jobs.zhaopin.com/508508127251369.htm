举报收藏

岗位职责：

    1. 负责云平台日志采集和存储；

    2. 参与公司BI平台建设和数据仓库的优化；

    3. 负责集群的维护和优化，保证集群稳定；

    4. 参与公司业务日志的分析和挖掘；

    任职要求：

    1. 具有大数据开发3年以上工作经验；

    2. 具有熟练使用java，scala或python其中至少两门语言的编程能力；

    3. 熟悉Linux开发环境，具有熟练的编写shell脚本、独立进行系统搭建的能力；

    3. 具有Hadoop/Hive/Spark/Storm/HBase等分布式系统开发、维护经验，有故障处理能力；

    4. 具有良好的沟通协作能力、独立任务完成能力，具有较强的分享精神；

    5. 具有数据仓库系统建设和优化、流式计算经验，熟悉日志挖掘常用算法者优先；

供专业的智能语音服务。
