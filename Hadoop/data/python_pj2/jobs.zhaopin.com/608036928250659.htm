举报收藏

岗位职责：

1. 分析海量数据，对用户基础数据/行为数据进行算法建模，生成用户标签并落地应用

2. 利用机器学习/深度学习技术构建用户标签库，专家知识库并落地应用

岗位要求：

1. 本科以上学历，计算机/数学相关专业，2年以上数据算法相关工作经验

2. 熟悉大规模数据挖掘、机器学习、自然语言处理等领域技术，熟悉掌握常见的分析挖掘算法如决策树，SVM，聚类算法等

3. 参与过用户画像建模、用户画像系统或DMP系统的开发工作，对基于海量数据分析挖掘用户特征有丰富经验

4. 熟练使用一种以上工程开发语言（C/C++/Java/Go/Python等），掌握基于hadoop等大数据平台工具的开发与设计，熟悉Hive，map/reduce开发

5. 对数据敏感和敏锐的业务直觉，学习能力强
