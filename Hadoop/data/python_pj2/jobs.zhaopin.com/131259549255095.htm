举报收藏

岗位职责：
1. 负责官方微博、微信、今日头条、一点资讯等平台的日常内容的撰写和运营；
2. 收集、研究网络热点话题，结合新媒体特性，对新媒体内容的实时调整和更新；
3. 负责新媒体活动方案的策划、创意、执行、运营以及汇报和总结；
4. 分析同行业新媒体内容结构及话题热点，调研目标用户群体喜好，在此基础上定位客户需求及喜好，增长粉丝量；
5. 负责分析新媒体相关运营数据，通过数据分析对推送内容及日常运营方向进行调整。

任职要求：
1. 有2年以上新媒体相关工作经验；
2. 具有较强的新闻、热点敏敢性，有较强的文案功底；了解新媒体运作方式和特点，熟悉内容传播的相关技巧；
3. 具备一定数据分析能力，可以通过数据分析来知道日常工作；
4. 网感好，创意优，一定程度上掌握图片处理软件；
5. 知识面广，思维活跃，工作主动，有责任感，能承受较大的工作压力；
6. 对微博和微信运营成功案例者优先；
7. 良好团队合作精神，较强的执行力。

居然设计家（http://www.shejijia.com/）是居然之家打造的融线上线下为一体的家居智能服务平台，包括设计云平台、家具材料采购平台、商品销售平台、施工管理平台、物流配送平台和智能家居服务平台六大部分。居然设计家的使命是以互联网、大数据和云计算为手段，改善家居消费者的服务体验、提升家居行业运行效率，从而让家居服务变得快乐简单，并以此为基础构建以家居为中心的家庭消费生态圈。
通过居然设计家，设计师可以免费获取效果图、施工图、预算一体化的3D家装设计工具，实现设计独立和个人的自主创业；通过居然设计家，消费者可以随时随地发布装修需求，找到自己满意的装修设计方案和心仪的设计师，也可以自己DIY设计自己的家；通过居然设计家，家具材料生产商可以将数字化的产品库与设计家的家具材料采购平台无缝对接，延伸销售渠道，扩大销售网络，并实现产品的跨境出海；通过居然设计家，家具建材经销商可以获得平台为其免费搭建线上销售平台，实现线上线下营销一体化；通过居然设计家，施工队可以凭借信誉口碑获取派单，实现家装过程的自动化管理和监控；通过居然设计家，可以将家装的物流配送服务缩减至1-2次，并实现后家装服务的智能化管理，等等。

北京居然之家投资控股集团有限公司 该公司其他职位

 北京居然之家投资控股集团有限公司(www.juran.com.cn)成立于1999年，是以中高端为经营定位，为顾客提供设计、材料、家具、家居用品及饰品等“一站式”服务，融家具建材品牌专卖店、家装设计中心（乐屋）、建材超市（丽屋）、灯饰广场（靓屋）、酒店用品城、家居产业园、商务写字楼租赁（居然大厦）、电子商务（居然在线）、安康进口超市等多种业态为一体的大型家居建材主题购物中心。
 
 
 
截止到2015年底，居然之家发展成为足迹遍布全国28个省（市）、连锁店面达125家、总经营面积超过600万平米、营业额接近400亿的大型家居连锁集团。连续十一年蝉联“北京市十大商业品牌”、八年荣获“北京十大商业品牌金奖”称号，成为中国家居建材流通领域管理和服务最好、口碑最好的行业龙头。
 
居然之家“家居建材主题购物中心”模式的特点
 名副其实的“一站式”
经过几年的发展，居然之家的产品经营链条逐步完善，“一站式”购物的特征逐步凸显，从设计到装修，从购买材料到家具配套，从家居用品到家居饰品，也就是说从消费者拿到房子钥匙到正式入住，所有消费环节都可以在居然之家实现。
 

 博采众长的全业态
居然之家“家居建材主题购物中心”的特点是家具建材市场、家具建材品牌专卖店、建材超市、家居商场各种业态相结合，对中高档品牌和非标准化产品实行招商制市场管理，并向专卖店转变；对标准化、大众化的产品实行超市化管理；对“后家装”阶段的家居用品和饰品则实行商场化管理。居然之家取各种家居业态之长，实现了家装设计中心、家具建材市场、家具建材品牌专卖店、建材超市、家居商场的有机结合。
 

 清晰明确的中高档定位
居然之家明确定位为服务中高档收入人群，经营品种以中高档品牌为主，引领家居品质和时尚。
 

 诚信为本的服务意识
服务为本、诚信为本既是企业的经营理念，也是企业应该恪守的职业道德，是居然之家创业、立业的根本。“先行赔付”、“向消费者倾斜”、“绿色环保”、“统一退换货”、“一个月无理由退换”、“同一品牌同一价”、装修“零增项”、送货安装“零延迟”，都是居然之家服务为本、诚信为本的体现。
