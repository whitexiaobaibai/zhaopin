举报收藏

岗位职责：

1. 负责移动端应用产品运营工作，建立有效运营手段提升APP活跃度，增加粉丝数，提高关注度； 

2. 快速响应社会、行业热点话题，对移动端APP、微信公众号的关注度及内容效果负责；

3. 通过线上线下营销活动策划、对外合作等多种方式获取新用户并维系老用户；

4. 通过数据统计、分析等方法，关注平台的市场表现，不断提出改进意见。

任职要求：

1. 大专以上学历，2年以上互联网平台、活动运营经验； 

2. 优秀的运营能力，扎实的策划和文档功底，能高效地输出运营策划方案；

3. 熟练掌握互联网及移动互联网产品运营方法体系，擅长大数据分析，用数据驱动产品及运营；

4. 很强的执行力，工作积极主动，能承受一定压力并高效工作；

5. 熟悉线上推广平台及APP应用市场分发，对APP推广有一定的思路和创意；

6. 有教育行业从业经验或熟悉教育行业者优先。



工作地址：

罗湖区莲塘鹏兴路中兴花园《中国教师报》华南培训基地4-5F 查看职位地图

深圳市精华文化发展有限公司 该公司其他职位

   深圳精华教育机构是一家致力于培养中小学生自主学习能力，改变学生不良学习习惯，增强学生学习兴趣，并采用灵活的分层次授课方法与心理辅导相结合的教学模式，以全面提高学生的各科学习成绩和综合素质为目的的专业性教育培训辅导机构。自1999年成立以来，深圳精华教育依托深圳市各中小学庞大的教育资源，获得了极大的发展壮大，目前已经形成了包括一个培训基地、一个总校、五个分校在内的发展规模，教学点遍布深圳各个区域，教学软件、硬件软件日趋得到完善，教学场地及教学设备齐全，现有教学场地14000多平米，教学培训辅导科目覆盖了从小学到高中各学科各层次的所有课程，多种学习方式可供学生和家长选择，并经常开设各类特长班、兴趣班。在面授班的基础上，深圳精华于2004年开通了远程教育网络系统，实现了面授教育与网络教育相结合的全方位的课程辅导体系，2006年推出了成人职业技能培训，2009年开办了学困生转型班、高效课堂潜力开发班、周末班，这将精华教育的办学规模和水平推向一个新的台阶。我们本着“一切为了孩子”的教育教学宗旨，以高度负责的态度，将不断改进教学方法，探索新型教学方式，提高教学质量，注重教学效果，用我们优秀的师资队伍，高效的学习方法和先进的教育技术，逐步发展成为深圳市中小学生教育培训领域规模最大、科目最齐全、师资力量最雄厚的专业教育机构，最终打造成为深圳教育培训辅导领域第一品牌。
