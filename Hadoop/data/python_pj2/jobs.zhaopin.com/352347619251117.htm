举报收藏

岗位职责：

产品设计

1.  深入了解公司数字营销（类 DMP）平台目标，制定产品规划，推动产品落地；

2.  推动用户画像、兴趣挖掘、推荐引擎、数据接入与分析等具体事务，为精准化广告投放提供依据；

3.  关注大数据数字化营销的市场动向，进行行业/竞品分析，探索新的表现方式和方法；

产品实施

编写产品PRD文档，协同内部团队保证需求落地，推动产品按时高质量上线，且持续迭代优化


任职要求：

1.  本科及以上学历，3年以上数字媒体广告或RTB、DMP、SSP、DSP、AdExchage领域相关产品经验，有 DMP 平台架构经验优先；

2.  熟悉广告网络、广告联盟、广告系统、广告统计等相关广告系统的需求及功能点；

3.  可熟练使用Visio、Axure、MindManager等应用软件，具备产品原型设计、PRD编写能力；

4.  有移动网盟、个性化推荐引擎、移动DSP公司经验的人优先


关于ideabinder
作为一家立足于医疗行业(healthcare)及企业服务市场(B2B)交集的科技企业，ideabinder 希望构建透明化医疗环境，提高每个人的医学认知，通过技术手段帮助医药行业企业完成营销环节的数字化转型。
ideabinder通过利用大数据分析、机器学习、人工智能等技术手段，帮助医药企业精准定位目标市场的医生，利用数字化手段联结线上及线下医生触达渠道，高效准确传递信息、实时有效追踪医生学术行为、量化衡量学术影响效果及产出、动态分析调整营销策略及资源营销投入模型。
通过六年的行业深耕，我们构建了诸如医药行业垂直CRM及一系列移动化SFA管理工具等完善卓越的产品线；积累了诸如西安杨森（Janssen）、辉瑞（Pfizer）、葛兰素史克（GSK）、诺和诺德（Novo Nordisk）等灯塔级标杆客户；获得了诸如红杉资本（Sequoia Capital）的青睐；此外我们拥有一支久经历练的技术队伍，拥有开放及扁平化的工作风格，同时清晰稳健的商业模式和强健的现金流为创业激情提供了有力且稳定的保障，让理想变得脚踏实地。

专注

ideabinder 是一家专注于生命科学行业的移动云计算公司，通过一系列移动应用软件，及云计算平台，帮助生命科学行业企业（医药、医疗器材等企业）的销售团队从关系型向专业型升级。ideabinder 的产品始终坚持“移动为先”的设计理念，以最贴切生命科学行业一线业务需求、良好的用户体验、灵活的扩展及伸缩性、对销量提升极为有效的直接作用，逐渐得到众多生命科学行业企业的认可。 

 

创新

作为一家企业级移动云计算公司，创新是 ideabinder 最核心的构成基因，ideabinder 将技术发展的创新与生命科学行业的业务流程有机融合，探寻针对生命科学行业市场及销售团队的业务流程优化及创新，让技术的创新助力业务的增长。

 

发展

作为一家以创新驱动的公司，ideabinder 历经近5年的发展，已成为国内专注生命科学行业最具代表性的企业级移动云计算公司。ideabinder的产品及解决方案不仅被众多跨国企业、中国大陆地区大型企业广泛应用，同时也被欧洲及亚太地区的客户所采用。2014年 ideabinder 进入高速发展期，全面推进技术创新及业务发展，以迎接移动及云计算技术在传统行业的爆发浪潮
