举报收藏

工作职能：

          1.根据公司战略目标，完成SAGA智能科研服务销售目标。

          2.对接医院、科室以及信息科，协助实施团队完成SAGA系统部署。

          3. 随时关注所负责医院最新信息化项目信息，进行资料收集和准备；

          4. 主动开拓新的市场和项目，与医院客户进行有效谈判与沟通；

岗位要求：

          1. 医学、IT等相关专业本科学历；

          2. 有良好的产品演示宣讲、机会跟踪、订单管理能力；

          3. 具有较强的市场拓展能力，熟悉医院科研项目运作模式；

          4. 有临床医疗、科研项目、或技术开发背景优先；

          5. 有大数据项目经验、KA经验优先；

          6. 有创新、实干、团队协作、追求卓越精神优先。

公司介绍：

Hitales（上海翼依信息技术有限公司）成立于2014年9月12日，打造中国最领先的医疗大数据平台。Hitales取名自Hippocrates（希波克拉底，古希腊医学家，被誉为医学之父）和Thales（泰勒斯，古希腊哲学家和数学家，被誉为科学之父），希望结合医学与数学，用数据服务医疗，开创更美好的未来。Hitales致力于用大数据技术协助医生看好一个患者，看懂一种疾病、为人们提供更高质量的医疗服务，推动医学进步。

Hitales相信拥有快乐的工作是人生的趣事，成就有意义的事业是完美生活的一部分。

Hitales认为和优秀的同事一起，可以激发潜力，超越自我。

Hitales立足创新，追求卓越。
