中科智道（北京）科技股份有限公司

  • 职位月薪：8001-10000元/月

举报收藏

岗位职责：
1. 负责企业客户部的销售工作，包括目标客户的开发与维护，达成公司制定的销售指标，主要产品为各种广告业务、会议展览招商、公关活动、行业报告、企业咨询、市场调研、企业培训；
2. 定期拜访目标客户，充分了解客户需求并积极跟进，制定合理方案，负责方案演示、谈判，了解追踪公司相关部门的工作，保证方案的有效实施，主要目标客户对象为500强的IT厂商及解决方案供应商，中国电信、联通、移动等基础运营商及数据中心企业及IBM，戴尔，英特尔，华为等IT厂商，大型互联网公司，解决方案提供商等；

3. 维持与现有大客户的良好业务关系，及时更新公司产品信息，传达企业及品牌文化；
4. 建立大客户档案，确保各项资料完整、准确，并做好动态管理；
5. 准时提交各项销售报告，遵守公司的各项管理制度；
6. 配合市场部门开展各类与大客户部有关的客户活动。


任职要求：
1. 本科及以上学历，三年以上的销售经验，一年以上管理经验；
2. 具备良好的大客户销售技巧和经验，具备IDC行业/互联网行业，了解电信行业/大数据和云计算背景者优先；
3. 具备广告销售、咨询报告销售等经验者高薪录用；
4. 出众的表达能力与沟通技巧，擅于发掘客户的潜在需求，优秀的谈判技巧和快速反应决策能力。
5. 高度的责任感，对事业充满热情，富有感染力。愿意在压力下工作，心理承受能力强；
6. 熟练应用Microsoft Office软件，尤其擅长Excel与PowerPoint操作。

中国IDC圈—中科智道（北京）科技股份有限公司

      中国IDC圈（Web Site:www.idcquan.com）成立于2006年，是数据中心行业权威的媒体平台和市场研究机构，立足于TMT产业，专注于互联网、数据中心及云计算领域的深度挖掘，形成了以数字传媒、会议展览、市场研究、技术评测、大数据分析、资源交易、培训及人力资源等多元化的第三方服务机构。

在相关部门的指导下，负责《中国IDC产业发展年度报告》数据分析及撰写，发行最具代表性的行业杂志《中国IDC行业资讯大全》，组织和承办业界标志性盛会“中国IDC产业年度大典”，为国内外电信和互联网行业搭建国际交流平台；整合3000家IDC企业资源，掌握中国互联网100强企业的IDC及带宽策略，为行业高效、绿色、健康发展提供“智囊库”。

中国IDC圈的核心受众群体为数据中心及云计算的上下游企业，会员企业集中在国内外民营数据中心、基础电信运营商、IDC及云服务商、政府及行业用户。

会员企业主要是国内外民营数据中心、基础电信运营商、大型企业用户这三类，如太平洋电信、万国数据、中金数据、上海数讯、KDDI、各地产业园区数据中心、各省及地市的电信、联通、移动运营商等，百度、新浪、搜狐、乐视、阿里、360、盛大、工商银行等在大型互联网等企业的运维和采购部门。 

      行业协会

      云计算发展与政策论坛(Web Site:www.3cpp.org)是由工业和信息化部、国家发展和改革委员会、科技部的指导和支持成立的首个云计算政策组织，论坛以规划、政策、标准为切入点，以支撑政府，推动产业发展为主要目标，着力搭建政府主管部门与产业界、标准组织以及科研机构的交流与互动平台。原信息产业部部长吴基传出任名誉理事长，中国IDC圈出任云计算发展与政策论坛的副秘书长，并为云计算发展与政策论坛用户委员会工作办公室，负责用户委员会的日常工作管理。

       数据中心联盟为中国通信标准化协会（CCSA）的二级协会，由业内主要企业共同组建的非营利性第三方组织，涉及数据中心IT设备和基础设施提升、数据中心绿色节能、可信云服务环境建设、政府采购云服务推进、大数据等相关领域。积极推动中国数据中心行业技术及服务模式的创新，推动完善数据中心行业的法律、政策和标准，倡导数据中心行业自律，促进数据中心产业的发展和进步。由工信部通信发展司陈家春副司长任联盟理事长，中国IDC圈任联盟理事并为数据中心联盟用户工作组常务办公室，负责日常的工作管理。

中国IDC圈下设4大职能中心及1个采购者平台：

传媒中心运营着国内数据中心行业最权威的媒体平台，包括中国IDC圈（www.idcquan.com）、《中国IDC行业资讯大全》等杂志、“IDC行业一周圈点”等知名电子周刊、数据中心行业意见领袖公众号（微信号：idc-quan）及APP平台；每年承办IDC行业标志性的产业年会中国IDC产业年度大典（IDCC）及云计算用户大会（CCUC）等大型会展。

研究中心凭借近十年在数据中心、云计算领域最完整的历史数据积累，结合最具有竞争力产业上下游沟通渠道，定期发布行业权威市场数据及深度的市场研究报告。

监测实验室基于行业用户监测、服务商监测等动态数据分析，根据国家相关政策和标准为数据中心基础设施及服务商、云服务商提供独立的第三方监测服务及用户需求监测服务；联合相关行业组织，不定期发布全国IDC机房网络质量报告、全国IDC服务商服务质量报告、云服务相关报告、云计算及IDC用户需求监测报告等。

数据中心商学院联合知名院校及企业为数据中心、云计算行业的从业人员提供商业、管理、市场、技术等多样化的内容。采用总裁研修班、高层沙龙、校友俱乐部、投资基金及私董会、实训基地、工程师训练营等形式，并为企业培养和输送优秀人才。

采购者平台深度挖掘各行业的用户在泛IDC、云计算的使用需求，建立标准化的需求自评估模型、服务比价模型，通过线上线下的交易及交付，为用户提供清晰透明的一站式数据中心及云服务采购。
