举报收藏

深圳前海联动云汽车租赁有限公司简称联动租车（公司投资者为深圳钜盛华股份有限公司），成立于2016年5月，总部位于中国深圳。将在成都、武汉、长沙、三亚、南京、天津等各大城市陆续开设分公司。主要业务涵盖汽车租赁、专车业务、汽车金融、汽车后服务四大板块。是多样化的汽车服务方案的提供商。联动租车将通过产业资本、金融协同、互联网引流，打造国内最大的汽车综合服务出行集团。未来将运用大数据解析、车联网、和不同行业伙伴协作互补，提供中国的汽车出行、汽车金融、汽车后市场的最优汽车服务方案，为消费者提供多样、便捷、安全的出行方案。

【招聘职位】

联动云客服专员

【职位描述】

1、 处理客户呼入电话，替客户办理租车业务。

2、 解答客户关于业务的疑问，受理异常问题和投诉建议等。

【工作时间】

每天8小时工作制，项目安排轮休，月休8天

【工作地点】

深圳市罗湖区宝岗路裕华大厦

【职位要求】

1. 高中及中专以上学历，性别不限；年龄在18岁以上。

2. 普通话标准，具有出色的沟通、表达能力，良好的客户服务意识，有电话营销经验者优先。

3. 口齿清晰，普通话标准，语气亲和，有良好的沟通表达能力。

4. 具有较高的客户服务意识以及服务技巧。

5. 有亲和力，有较强的责任感和团队合作精神。

6. 欢迎应届毕业生选择此岗位实习。

 

【福利待遇】

1. 薪酬：试用期三个月，月薪3500；转正后月薪3800-4600；每四个月调薪一次。

2. 劳动保障：入职签订劳动合同即有五险一金。

3. 休假福利：享受法定节假日等各类假期，法定节假日三倍工资及过节费。

4. 培训机会：完善的培训体系，为员工提供岗前培训、技能培训、职业发展培训等各类培训。

5. 晋升机会：完善的职业发展通道，每年拥有晋升机会。

【丰富多彩的文化生活】

1. 团队年轻：团队年轻化、激情有活力，互帮互助，氛围和谐；

2. 活动丰富：公司不定期举办各种文化活动，丰富员工的业余生活。

 


工作地址：

深圳市罗湖区宝岗路裕华大厦8楼

深圳市前海润杨金融服务有限公司 该公司其他职位

深圳市润迅电话商务有限公司是国内规模最大、率先通过ISO9001国际认证资质的人力资源外包服务商、同时也是最早投入专业呼叫中心及金融服务外包的供应商。为响应国务院建立前海深港金融创新合作区的号召，把握行业发展趋势，润迅公司于2013年10月25日正式成立了深圳市前海润杨金融服务有限公司，作为前海润杨金融公司专属金融服务品牌，专注服务金融客户，致力于打造银行客户服务一站式解决方案！
润迅公司成立于1997年，注册资金2000万元，现拥有员工30000多人。全国拥有超过27个分公司。1998年率先通过ISO9001国际质量体系认证。中国最早的金融业务服务外包供应商之一。中国最早的呼叫中心服务外包供应商之一。中国最早的人力资源服务外包供应商之一。目前是国内最大的综合客户服务提供商。业务涉及电信、金融保险、旅游、IT、家电、制药和汽车等领域。
依托润迅公司在中国市场积累的全方位的服务营运经验，及其在国内服务行业独树一帜的品牌管理、服务管理和技术管理，润杨金融在吸收先进金融服务理念的基础上，积极总结金融服务经验，从团队搭建，课程打造，团队提升，服务固化逐步形成一套独有的适合现有国情银行管理理念和运营体系。
润杨经过多年的不懈努力，润杨公司已发展成为国内最大、最专业的金融服务外包商之一。具有专项的金融业务流程外包等经营范围的资质，并且在1998年率先通过ISO9001国际质量体系认证。公司在北京、上海、昆明、广州和深圳建立了具有国际先进水平的联网空中理财呼叫平台，为国内外金融客户提供全方位金融服务外包。
