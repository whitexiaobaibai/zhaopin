总体教学目标

培养能够满足企业要求的以java web开发技术为主要能力的工程师。
完成学习后的工程师应当胜任java web开发工程师、大数据开发工程师等职位。

第一阶段 教学目标 掌握面向对象的思考方法，掌握java基本语法。
学习内容 Java
1. 掌握世界上使用人数较多的编程语言Java
2. 掌握面向对象的思考方式
3. 掌握线程的管理方式

第二阶段 教学目标 掌握java Web平台的开发技术
学习内容 mysql、servelt、jsp、js
1. 掌握mysql的安装、配置与使用
2. 掌握tomcat的安装配置与使用
3. 掌握servelt、jsp的开发
4. 掌握基础html语言及基本css样式
5. 掌握javascript基本语法
6. 掌握java请求网络的常用方式及框架
7. 熟悉javascrript基本语法
8. 掌握easyUI框架

第三阶段 教学目标 掌握开发必备框架
学习内容 html5，CSS3、bootStrap、AngularJS
1. 掌握SSH框架集
2. 掌握SSM框架集
3. 掌握Hadoop、HBase、Kontagent等工具

第四阶段 教学目标 项目实训
学习内容 团队合作、项目分析
1. 掌握项目分析的方法
2. 掌握团队开发项目的流程
3. 掌握项目管理的方法
4. 掌握项目开发流程规范

第五阶段 教学目标 项目实训与就业指导
学习内容 简历指导、模拟面试
1. 使用hadoop、HBase、spark、Hive、kafka综合技术,实训信息检索、用户行为分析案例。
2. 掌握制作简历的方法和技巧
3. 掌握面试技巧
4. 掌握薪资谈判的技巧
