举报收藏

技能要求：

1、精通J2EE体系架构及B/S三层结构系统的开发，熟悉常用Java开发框架，如Struts、Hibernate、Spring、jQuery、Ajax等，熟练应用eclipse等IDE工具；


2、熟悉使用主流应用服务器：Tomcat、WEBLOGIC、JOSS，能够在linux环境下开发、部署及性能调试；


3、熟悉主流数据库：Mysql、SqlServer、Oracle数据库中的一种或者多种,并具有一定SQL优化能力；能够编写复杂逻辑的存储过程，触发器，调度任务及数据库大数据量优化


4、熟悉软件工程概念和软件开发流程，能使用svn、cvs等项目管理工具； 


5、具备实际 Java 项目3年以上开发经验（不包括实习项目、培训项目）； 


5、良好的代码编写习惯，具有较强的学习、沟通和交流能力；


6、具有良好的表达和交流能力，较强的沟通能力，富有进取心，能承受一定的工作压力；


7、具有阅读相关技术需求文档能力；具有一定的软件设计及文档编写能力。


8、具有GIS地图应用开发经验者优先。


岗位要求：

1、编写相关项目文档

2、带领开发小组进行需求设计

3、负责团队开发代码质量

4、项目核心编码编写

5、协助处理项目中遇到的技术问题


6、制定项目进度计划并跟进项目的进度和质量

元道通信股份有限公司
元道通信股份有限公司为专业从事移动通信设备维护、网络优化等通信技术服务公司。公司成立于2008年，公司所在地为河北省石家庄市，注册资金500万元人民币。公司致力于专注和专业为中国通信运营商提供从通信网络建设到基站、传输、电源、应急保障、核心网维护、网络优化等整体解决方案和全面服务。目前已成功取得了河北移动公司石家庄、唐山分公司；黑龙江移动齐齐哈尔、大庆分公司的通信网络基站代维业务，维护站点近4000个。
深圳元道通信技术有限公司
深圳元道成立于2009年12月，以通信产品、通信服务、增值业务、IT服务和软件开发为核心，服务于通信市场和提供IT综合服务，具有丰富的产品和市场经验，公司与中国电信、移动、联通、网通等通信运营商及中兴、爱立信、港湾、朗讯、阿尔卡特、诺基亚、西门子、NEC等国内、国际知名设备供应商建立了长期合作关系。公司业务区域遍及全国各地，在北京、江苏、广东、四川、江西、湖北、辽宁、河北、浙江、上海、贵州等省市，为客户提供专业的产品及服务。
公司经过多年的发展，积累了丰富的市场资源和运营经验，在中国通信市场建立了良好的客户资源与品牌信誉，加上一直以来的不懈努力、求变创新的原则以迎合市场及客户的需要，成为一家多元化的综合通信服务提供商，由最初的提供通信设备、产品及技术服务为主的企业，现已成功的将业务的重点推展至富创意的高增长服务领域，现今通信增值业务、通信工程技术服务、IT服务已成为公司的三大核心业务。
