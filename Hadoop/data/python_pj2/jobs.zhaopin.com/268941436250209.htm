举报收藏



职位描述：

工作职责：

1. 负责BI商业智能产品技术方案与系统架构的设计、标书的准备、讲解及用户答疑等工作；针对用户不同的业务应用场景，能提出服务器硬件配置建议；

2. 负责与用户的技术交流、技术方案宣讲、系统演示、POC等工作；

3. 行业内信息的收集、分析、总结，并支持市场及产品；

4. 典型客户案例提炼与总结，完善售前知识库；

5. 客户接洽，技术谈判等；

6. 培训合作伙伴进行售前、实施等工作。


任职要求：

1. 本科及以上学历，计算机相关专业优先，3~5年以上售前经验；

2. 熟悉市面上bi商业智能产品及相关技术，熟悉数据相关技术，特别是数据处理、数据分析相关的技术，例如数据清洗etl、数据建模、数据分析等；熟悉软件架构；

3. 优秀的沟通协调能力、文档编写能力、逻辑思维能力、谈判能力和人际关系处理能力；

4. 学习能力强，有责任心和敬业精神，有较好的抗压能力，适应经常性全国范围内出差；

5. 加分项：

a) 掌握大数据、ETL、数据仓库技能；

b) 掌握BI基础知识、数据分析师经验。


此岗位所负责的项目为大数据平台
