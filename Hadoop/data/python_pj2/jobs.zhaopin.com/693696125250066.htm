举报收藏

岗位职责

1、负责公司在线业务系统的研发和维护工作；

2、与业务合作方进行系统 API 的对接； 

3、优化抓取策略和算法，提升爬虫的抓取效果；

4、负责相关运营后台系统研发工作；


任职要求

1、熟悉Linux平台开发，三年以上Python开发经验，有分布式系统开发经验为佳；

2、熟悉Tornado框架，有API的开发经验，能够进行快速开发；

3、熟悉RDB（MySQL，Oracle）和NoSQL DB（Mongo），熟悉Redis，有大数据开发经验者优先；

4、自我驱动的学习和工作习惯，对未知技术和领域能快速掌握并实践；

5、良好的沟通、协调能力，有项目管理、团队管理经验者优先；

6、在开源社区活跃并有积极贡献者优先；

公司有着清晰的商业模式以及稳定的资金供应，顺应国人出境游爆发式增长的趋势，团队处于业务的高速发展期，每月为十余万人提供出行以及住宿的服务。

公司的快速增长依仗高质量的人才助力，希望与旅行行业前景看好，并有梦想能力的伙伴一同耕耘，将公司不断做大做强！
