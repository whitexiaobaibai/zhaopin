举报收藏

 岗位职责：
1. 负责基于Hadoop生态系统的海量数据处理分布式平台总体架构设计及海量数据分析研究方案的设计及研发；
2. 制定项目设计及实现规范，指导设计、开发及部署工作；
3. 带领、协助指导工程师解决关键问题；
4. 设计开发关键功能模块。

任职要求：
1. 计算机、电子信息等相关专业本科或以上学历，具备4年大数据从业经验者，并具有Hadoop生态系统大数据平台建设经验；
2. 熟悉Hadoop家族的多个组件的开发和使用，如Hive、HBase、Spark、Kafka、flume、oozie、ambari、phoenix、Solr等；
3. 精通基于Hadoop平台等分布式开发，熟悉Linux系统；
4. 具有较强的责任心、执行力，有良好的沟通和团队合作能力；
5. 在大数据处理和分析领域技术规划、架构设计经验者优先；
6. 熟悉统计软件（SAS、R），或有数据分析和数据挖掘项目实施经验者优先；
7. 热爱软件开发，有激情，自我驱动力强，能够独立分析、解决复杂问题，能不断跟踪学习新技术；


工作地址：
