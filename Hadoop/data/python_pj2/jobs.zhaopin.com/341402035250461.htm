举报收藏

岗位职责：

1. 参与产品的需求分析、设计、开发； 
2. 参与软件质量审核，提升员工开发质量和效率；
3. 遵守项目组制定的各种研发及项目规范，保证研发过程的持续改进；
4. 能理解产品设计文档以及其他同事所写的技术文档，并有效参与评审；
5. 能编写相关设计文档，并完成高质量的代码实现和单元测试；
6. 对软件品质有着很好的追求，对新技术有着强烈学习与实践的欲望； 

任职要求：

1.五年以上基于Java语言开发经验；
2.精通J2EE架构，精通常用的设计模式，精通数据持久化、缓存、消息、通讯等的相关开源框架，对大容量、高并发系统的设计与调优有丰富经验尤佳；
3.熟悉TCP/IP协议,熟悉java网络编程，熟悉NIO，熟练使用相关框架，如Netty、Mina等；深入理解序列化与反序列化机制；
4.熟练使用redis 、memcache 等缓存中的一种；
5.熟悉至少一种Web开发框架（Spring+Mybatis）；
6.至少熟悉一种关系数据库如mysql，并能根据业务需求设计数据库结构,有大数据相关技术经验、熟悉大流量、高并发、高负载、高可用性的分布式系统的设计开发者优先。
7.积极主动，有较强的责任心，并具备良好的沟通能力、理解能力和较强的学习能力，具备团队合作精神；
8.有汽车金融软件相关经验优先

工作地址：

上海市浦东新区陆家嘴软件园峨山路91弄58号伟泰大厦B区4楼

上海合勋车融资产管理有限公司 该公司其他职位

   上海合勋车融资产管理有限公司成立于2015年12月，注册资本1000万元人民币。公司致力于在汽车金融信贷领域，为各金融机构提供高品质的信贷资产全生命周期管理服务和信贷资产安全保障服务，为汽车消费贷款客户提供便捷的金融咨询服务和专业的汽车全领域增值服务。

合勋车融倡导“团结、包容、诚信、务实”的企业文化，推崇团队职业精神，打造专业化平台，提倡创新，追求卓越，实现共赢。

目前公司已发展成为拥有多家控股子公司、近1000余人的直营客户经理团队，服务网点遍及上海、四川、云南、贵州、湖北、湖南、河北、河南、山东、山西、陕西、广西、江西、江苏、安徽等地区。

公司拥有完善的运营管理及风险防控体系，为客户提供一站式优质服务，助理每位客户实现超乎想象的“车生活”。
