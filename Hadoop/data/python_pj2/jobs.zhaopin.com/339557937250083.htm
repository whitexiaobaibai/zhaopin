举报收藏

 岗位职责： 

a.参与业务的需求调研，了解业务团队的业务数据需求；

b.负责大数据平台产品线的架构设计和研发；

c.根据业务需求，进行系统的整体技术框架的选型，进行相关业务框架和系统架构的搭建及优化；

d.和团队成员一起完成大数据分析工作的流程、规范和方法建立；

e.对系统框架相关技术和业务进行培训，指导开发人员进行开发；

f.参与相关数据分析、评估监控的工具平台的研发，负责数据模型设计、ETL开发等工作；

g.完成数据模型的ETL实施，ETL流程优化以及相关技术问题的解决。

任职资格：

a.计算机相关专业，大学本科以上学历；

b.4年以上工作经验及至少3年大数据开发经验；

c.对各种大数据架构有深入理解，了解不同架构的优缺点；有大型系统架构经验，熟悉分布式系统的设计和应用，熟悉大数据领域的开源产品原理和机制；

d.熟悉大数据相关技术，包括但不限于Hadoop、Spark、Storm、YARN、Impala、Flume、Kafaka、mongodb等

e.熟练使用java或者python；

f.精通SQL，有较好的SQL性能调优经验，理解Hive/Mysql基本原理和调优策略；

g.有互联网产品BI开发经验，有网站数据、用户数据、精准营销等相关经验者优先。


工作地址：

北京市海淀区西北旺东路10号院东区21号楼汇冠大厦四层

北京富邦智慧物联科技有限公司是一家从事消防物联网信息服务的互联网公司。司与公安部沈阳消防研究所战略合作，应用“十二五”国家重点科技课题《基于物联网的火灾征兆探测与消防设施综合信息平台技术研究及应用示范》（课题编2012BAH21B03）成果，依托双方的技术、资金、人才和市场优势，打造消防物联网技术与数据服务平台。
       富邦消防利用“感、传、知、用”等物联网技术手段，综合RFID（射频识别）、无线传感、云计算、大数据等技术，通过互联网、无线通讯网、专网等通信网络，对消防设施、器材、人员等状态进行智能化感知、识别、定位、跟踪、监督和管理，通过信息处理、数据挖掘和态势分析，为防火监督管理和灭火救援提供信息支撑，进一步提升信息主导、准确研判、主动防控水平。 富邦消防物联网涵盖“火灾自动报警”、“电气火灾监测”、“消防水源监控”、“消防设施监控”“消防通道监控”、“维护保养管理”和“APP服务端”等功能，实现24小时全方位监测与管理，是目前国内最先进的消防物联网系统解决方案。 富邦科技站在发展“智能消防，平安生活”的战略高度，怀揣使命与梦想，积极拓展消防物联网应用领域，持续提升研发能力和服务水平，为中国智慧城市的建设贡献能量，为人民安居乐业和经济发展保驾护航。
