举报收藏

职位描述：

1. 负责对药品数据组织与分析，进而对产品运营战略、各类营销活动效果、指定产品改进策略等提供数据支持；

2. 负责数据仓库和会员标签的架构设计、开发、性能调优、完善基础数据及会员标签的建设；

3. 负责海量数据的多渠道整合、处理、分析、统计、挖掘相关工作；

4. 负责推荐系统、会员生命周期系统建设等设计与实现；

 

任职要求：

1. 计算机相关专业本科及以上学历；

2. 熟悉常用开源分布式系统,Hadoop/Spark/Hive/Hbase，至少有1年的Spark(Core/Streaming/SQL)开发经验；

3. 熟悉Unix/Linux及shell；熟悉Java/Scala/Python语言(至少一种)；

4. 熟悉Sql代码编写，熟悉Oracle等关系型数据库；对mogondb/redies等nosql有一定了解;

5. 具备一定的英文阅读能力；

6. 有统计学数学知识，海量数据处理、数据分析和挖掘经验者优先;
