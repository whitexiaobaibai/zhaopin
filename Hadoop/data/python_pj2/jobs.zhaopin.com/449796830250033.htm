举报收藏

岗位职责：

1.负责公司大数据产品可视化呈现的整体架构、方案设计，完成核心代码的编写，并根据情况协调指导外包团队进行相关开发、优化；

2.根据公司大数据不同业务场景，和数据分析师深入探讨，构建、完善数据可视平台；

3.深入了解公司大数据产品在业务中的具体应用，抽象构建相关应用API，并完成相关框架搭建；

 

任职要求：

1. 计算机相关专业本科以上学历、从事大数据、开发和设计工作5年以上，有过相关产品经验，请提供产品样例

2. 了解企业数据总线概念、熟悉企业大数据的在业务中的各种应用场景，对IT领域产品的业务发展和技术发展有前瞻性，

3. 能快速跟进业务诉求和数据特征，抽象公共API。对业界数据展现、API抽象等相关技术和实现方法有深入研究，有成熟的架构和设计实践经验

4. 精通JavaScript、HTML5、CSS3，熟悉W3C标准，对页面性能及浏览器兼容性有丰富的实践经验；熟练掌握ECharts、D3.js、Highcharts等可视化框架；

5. 具有独立解决问题的能力, 工作责任心强，能够和团队保持顺畅沟通。

工作地址：
