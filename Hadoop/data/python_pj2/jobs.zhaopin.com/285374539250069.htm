举报收藏

岗位职责：

1.根据公司确定的产品方向，确定软件开发的技术路线，进行软件框架搭建并编写相应设计文档；

2.负责软件开发项目的实施、管理和协调工作，带领团队进行项目开发，确保项目按期按质完成交付；

3.领导与管理开发团队。

4.具備1年以上移动端开发经验。


任职要求：

1.计算机软件或其它相关专业全日制本科及以上学历,五年以上 PHP或C#.NET开发设计经验；

2.具有独立开发能力，且有两年以上带领团队完成项目的经验；

3.熟悉PHP 或C#编程思想，扎实的PHP 或 C#.Net经验和知识体系，有优秀的编程习惯；

4.熟悉HTML、CSS+DIV、JavaScript，对Jquery有一定的了解；

5.精通数据库设计，熟练掌握: MySQL, SQL Server，能够熟练使用的存储过进行开发；

6.熟悉软件开发流程，设计模式，体系结构；

7.有一定的系统架构能力，具有分布式、高性能、大数据量WEB应用架构经验者优先；

8.有云平台系统架构规划设计、实施经验优先考虑，云计算相关技术(公有云、私有云)及架构，如阿里云。

﻿﻿公司介绍

中辉伟创是一家国际财富管理公司，香港公司于2008年正式成立，总部设在香港。我们拥有多项专业财富管理业务资格，包括香港保险经纪牌照、香港会计师事务所牌照及香港房地产代理等牌照。我们于全国不同省份设有代理渠道及分公司。
中辉伟创专注于为高净值客户提供全方位的环球财富管理服务，我们跟环球实力雄厚金融机构合作，产品全受欧美海外及香港地区政府监管，服务范围涵盖国际财富管理、家族信托传承、境外融资策划、海外移民顾问等财富管理服务。
我们的资深国际财富管理专家，通过国际金融知识及对本地市场的了解，为客户度身定制财富管理方案，以独立专业的顾问角色，协力助客户创造财富，缔造璀璨人生。

专家团队

中辉伟创采用精英团队的服务方式，专属私人客户经理和专家团队竭诚为每位尊贵客户提供全方面的财富管理服务。
我们的专家团队由多名经验丰富的顾问组成，包括法律专家、财富管理顾问、家族信托专家和企业银行等。他们与世界各地顾问团队保持紧密联系，及时获得信托服务，商贸，保险及投资银行等方面的一手资讯。
通过与您的直接交流，资深团队将充分了解您的投资需求和理财目标，为您度身制订个性化的投资方案和策略。定期回顾您的投资组合，以确保财富管理计划与既定目标方向一致且实施有效。
 
业务分布

直至2014年底，中辉伟创拥有十多个渠道销售点营业点营业点，如北京、上海、广州等，分布全国不同省份。
未来集团策略全力支持不同地区分公司及合作渠道发展业务，务求把国际先进的财富管理概念带到全国各地。
