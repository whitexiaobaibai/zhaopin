举报收藏

岗位职责：
1、负责公司贷款产品的销售及推广；
2、根据市场营销计划，完成部门销售指标；
3、负责销售区域内销售活动的策划和执行，完成销售任务；
4、管理维护客户关系以及客户间的长期战略合作计划。
岗位要求：
1、大学专科及以上学历，年龄：20—35岁；
2、标准普通话，口齿清晰，思维敏捷，具有良好的沟通表达能力；
3、自信乐观，正能量，工作认真，个性开朗、热衷销售行业、具有较强的事业心与团队协作精神；
4、公司有专业培训，无金融相关经验也可；
5、有兴趣可直接过来公司面试。

公司薪资待遇：

1.  保障底薪：底薪3000-5500元上不封顶（每季度调整一次）；

2.  业绩提成：最高提成50%（行业内最高、确保员工每一分付出得到最大的回报）；

3.  综合薪资：前两个月综合8000元/月   第三月以后稳保12000元/月；

4.  发展空间：客户经理-团队主管-市场总监-分公司负责人（公平晋升）；

5.  专职培训：公司安排专人一对一培训，让你轻松掌世界级销售大师的技能及技巧；

6.  基本福利：五险一金、生日福利、节日福利、免费培训 、下午茶、出国旅游等。


你的顾虑： 
1、没接触过这个行业，怕做不好。 
公司新进员工，90%都没有做过。公司有专业的培训，愿意帮助你的成长。 
2、压力太大，赚不到钱怎么办？ 
我们有培训老师和你的主管教你每一步， 
无责任底薪，给你提供基本的生活保障，无上限的提成比例让你实现自己的梦想 ！


核心优势： 
1. 入职前系统专业培训、师傅带徒弟手把手教包教包会 ；
2. 职业生涯规划、不定期参加各类型培训学习机会 ；
3. 集团公司下属分公司实力强大深圳各区业务均可互通 ；
4. 大数据客户来源渠道、不用自己到处找客户，自主研发自动化营销系统 ；
5. 团队默契度强、主管经理帮忙谈客户 ；
6. 不管你是新手老手 一视同仁 。

服务介绍：
    针对深圳地区公务员、上班族、房产业主、车主、中小企业融资渠道受限制，我们选择缺乏融资能力的社会弱势群体作为目标市场，帮助他们从我们的合作银行轻松获取企业、个人消费性贷款！为其提供薪金贷、业主贷、车主贷、中小企业贷，全力帮助他们解决各种急用资金周转难题。

公司产品：
1、“工薪贷”无论您有什么金钱上的需要，如现金周转、房产首期、换新车、装修、进修或旅游等，我们都能为您提供简便的个人融资服务；只要您有薪资证明，现金即时到手。

2、“中小企业贷”无论各行各业，企业规模大小，需要扩充营运或周转应急，我们一样能配合您的需要；只要您是生意人，现金即时到手。

3、“业主贷”我们为身为业主的您，提供最灵活方便的融资服务；只要您拥有房产，无论按揭与否，现金即时到手！
 
4、“车主贷”我们为身为车主的您，提供最灵活方便的融资服务；只要您拥有小型汽车，无论按揭与否，现金即时到手！
 
“保单贷”我们为身为投保人的您，提供最灵活方便的融资服务；只要您拥有寿险类保单，无论保费金额大小，现金即时到手！
 
公司使命：帮助企业和个人解决资金难题。
 
公司愿景：
为中国广大的中小微企业和创业者提供便捷高效安全的金融信息服务，让企业和个人得到更多更好的资金，从而使企业更具有发展力。使公司成为一站式综合金融服务平台。
