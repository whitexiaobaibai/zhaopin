举报收藏

岗位职责： 
1、确保项目目标的实现，领导项目团队从质量时间成本三方面控制项目进度。
2、与客户沟通，了解项目的整体需求。并与客户保持一定的联系，即时反馈阶段性的成果，和即时更改客户提出的合理需求。 
3、制定项目开发计划文档，量化任务，并合理分配给相应的人员。 
4、跟踪项目的进度，识别和控制项目风险，协调项目组成员之间的合作。 
5、监督产生项目进展各阶段的文档，并即时沟通，保证文档的完整和规范 

职位要求： 
1．两年以上软件开发经验；一年以上软件开发项目管理经验，对项目管理和软件工程有深刻理解； 
2. 熟悉项目管理流程，掌握规范化软件编程，能编写规范的项目文档，有较强的项目团队管理经验； 
3. 精通Java/JavaEE开发技术及常用的开发框架,熟悉J2EE主流应用服务器及主流数据库；
4. 精通常用的设计模式，具有很强的分析设计能力，精通UML及相关软件设计工具（Viso、ROSE、Together、PowerDesigner、StarUML/ArgoUML、MindManager其中二个）； 
5. 有大型并发网站开发与部署经验,有负载均衡，缓存，集群，安全，分布式，性能优化等实践经验，或精通数据库设计和优化，具有大数据以及高并发的系统数据库设计经验的优先； 
6. 有较强的沟通组织协调能力，良好的口头及书面表达能力；具有较强的责任心、执行力和团队合作能力； 


工作地址：

北京市石景山区中础大厦

北京天拓数信科技有限责任公司 该公司其他职位

北京天拓数信科技有限责任公司成立于2004年3月，注册资金1000万元。是一家以计算机应用软件研发为主，技术服务为辅的专业技术型公司。公司是经过北京市科委批准的国家高新技术企业，是北京市经济和信息化委员会认定的软件企业。
公司成立以来，始终坚持走自主研发产品的道路，注重软件品质，将向客户提供优质软件产品和服务作为企业可持续发展的核心动力。经过多年发展，公司形成了四大产品：线上职业教育平台、共享作业中心、电商服 务平台、智慧组织系统。同时可为客户提供高端定制化服务。公司已经为政府、金融、冶金、教育、互联网、食品等行业的近百家企业提供服务。
作为一个技术型公司，技术是核心竞争力。经过公司多年积累，形成了自己的前端和后端开发平台。前端开发平台Brooder是一个完全自主开发的基于Web技术的人机交互工具库和开发框架，可以快速构建运行在电脑端和移动端的Web应用，具有上手快、开发简单、页面效果酷炫的特点。后端开发平台是一个基于Java的支持分布式和集群的可构建超大规模应用的研发平台，具有易扩展、性能好、运行稳定、安全性高的特点。同时公司紧随互联网发展的趋势，在移动端开发技术上也有专门的团队，具有自主的核心技术。
公司始终秉承“开拓、共赢、笃信、人和”的经营理念。不仅为客户提供优质产品和服务，也为员工创造宽松、和谐、可成长的工作环境。吸引了许多中高端人才加入公司。公司人员平均年龄28岁，是富有激情和创造力的年轻团队。
