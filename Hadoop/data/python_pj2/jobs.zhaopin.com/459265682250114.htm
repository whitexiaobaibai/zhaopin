举报收藏

岗位职责：

1. 根据业务需求，负责数据处理工作（数据采集、清洗、加工、集成等）；

3. 负责日常的跑批调度开发；

4. 负责数据质量检验和监控；

5. 参与公司大数据平台的搭建、设计与开发。

 

岗位要求：

1. 全日制本科以上学历，扎实的计算机基础；

2. 能够熟练使用python语言进行数据清洗；

3. 理解MySql数据库原理，熟悉SQL查询、存储过程、SQL性能调优；

4. 熟悉shell脚本，数据批处理、数据调度、异常处理；

5. 有金融领域数据仓库实施经验优先考虑。


工作地址：

北京市朝阳区酒仙桥兆维工业园

北京坚果金服科技有限公司 该公司其他职位

北京米投网络科技有限公司于2015年1月创立于北京核心团队由金融、互联网、旅游、第三方支付等行业专业人士组成。米投是基于用户多维度的数据的场景支付和数据分析，对客户形成即时、主动、快捷的审核后，帮助客户获得与征信结果相对应的贷款的新型金融服务公司。
米投可以为优质的B端企业客户提供全方位、专业、可定制化的金融服务；同时也可以为C端消费者提供不同场景的消费贷款。
2015年1月，获得梅花创投（2014、2015十佳天使创投）数百万美元级别天使轮融资；2015年7月，获得知名投资人钱峰雷先生（阿里巴巴个人股东）数千万人民币Pre-A轮融资。
