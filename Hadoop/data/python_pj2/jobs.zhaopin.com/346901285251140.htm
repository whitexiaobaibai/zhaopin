举报收藏

岗位职责：

1. 参与产品和项目研发工作，完成产品大数据架构和功能开发；

2. 搭建和调优大数据基础平台，包括Hadoop/HBase/Hive/Sqoop/Kafka/Flume/Spark/Storm等大数据组件的环境搭建；

3. 维护线上系统的稳定性、可靠性，通过分析现象快速定位问题，进而提出优化方案，改进要求；

 

工作经验要求：

1. 计算机相关专业本科及以上学历，2年以上大数据工作经验,有扎实的计算机理论基础；

2. 熟练使用MapReduce、Spark提供的API编程；

3. 熟练使用大数据的工具平台，熟悉Hadoop生态环境，对Hadoop、Spark、Storm、HBase、Hive等至少一个项目有着深入了解；

4. 掌握数据分析的基本流程，具备数据采集、清洗、分析等环节的实战经验；可以进行数据产品ETL需求的调研及开发；

5. 熟悉ETL任务调度及管理；

6. 熟悉Shell命令，进行简单的Shell编程；熟悉Linux文本处理命令，VI、AWK、Sed等命令；


工作地址：

上海市

企业简介

       上海西线教育科技有限公司是一家以互联网电子商务技术为核心的服务商，致力于电子商务在全国各地的普及、应用、运营以及技术咨询服务，为客户提供最具价值的电商运营、电商系统开发、教育行业技术开发、技术咨询以及整体解决方案，以此迎接全民创业的全新电子商务时代的到来，满足客户在电子商务领域内包括系统开发、店铺运营、技术咨询，教育类软件开发等多元化需求。

 

企业团队

       公司在发展过程中，汇聚了一批国内外资深专家与行业精英，他们在电商领域、教育软件开发领域有着多年的项目开发和团队管理经验，对于行业的前景、技术发展有着深刻的理解。上海西线已经通过自身价值与企业理念吸引大量优秀人才加盟，打造一流团队为国内外客户提供超一流服务！我们希望与各方合作伙伴携手，秉承专业、诚信、创新的公司经营理念，不断追求卓越努力实现共赢的局面。

 

企业愿景

       上海西线教育科技有限公司响应国家"十二五"规划中大力发展互联网、电子商务、云计算、大数据等为核心的新一代信息技术等战略性新兴产业及未来传统行业“互联网+的国家政策。励志为中国互联网行业发展做出不懈努力，为壮大中国互联网行业添砖加瓦。
