举报收藏

岗位职责：

1. 负责学校大数据分析与服务工作。

2. 带领团队研发学校大数据汇聚与分析系统

3. 编写并发布年度国开大数据分析报告。

4. 负责体系内大数据应用培训与技术支持工作。

5. 组织申报教育大数据相关课题。

6. 完成领导交办的其他工作。

任职条件：

1. 计算机、统计学、应用数学相关专业；硕士研究生及以上学历。

2. 精通自然语言处理，机器学习和海量数据分析，熟悉Hadoop、Spark技术。精通至少一种编程语言，精通数据分析语言。

3. 五年以上互联网行业数据分析与挖掘工作经验。

3. 具备较强的协调沟通能力和团队管理能力，两年以上软件研发项目管理经验。

4. 年龄40岁以下。

5  条件优秀者可适当放宽年龄、学历限制。


工作地址：

北京市海淀区复兴路75号

中央广播电视大学出版社有限公司 该公司其他职位

   中央广播电视大学出版社成立于1982年10月。它是由中央广播电视大学(现为国家开放大学)主办的一所综合性大学出版社，也是广播电视教育的专业出版社。中央广播电视大学出版社于2011年3月完成企业转制，现名为中央广播电视大学出版社有限公司。中央广播电视大学出版社有限公司依托国开大学，面向全国，依靠一支由著名专家、中青年学者和社会各界人士组成的作者队伍，组织出版电大教材和教学参考书，同时也出版现代远程教育的学术专著、成人教育图书及面向社会的一般图书。现在年出书和电子出版物1 000多种，重版书占80%，教材和学术著作占90%以上。

  为了适应我国高等教育事业和两个文明建设发展需要的，建设成为具有现代远程教育特点、中国特色的一流的出版社，我社大力建设一支具有一流水准的编辑队伍。
