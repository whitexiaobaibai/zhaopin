举报收藏

岗位职责：

1. 负责企业客户用餐现场的管理及客情关系维护，包括客户系统调试更新，现场服务，及时处理现场出现的突发问题；

2. 完成公司制定的年度、季度、月度维护额度计划；

3. 进行客户分析，建立并维护客户关系，挖掘客户深度需求，寻求新的合作模式及业务增长点；

4. 内部沟通协作，与市场部、产品部和客服中心紧密合作，确保客户达到百分之百满意；

5. 现场兼职人员管理。


任职要求：

1. 男女不限；

2. 头脑灵活、应变能力强，具备良好的沟通能力；

3. 熟练使用办公软件；

4. 一至两年酒店、餐饮等服务行业工作经验，有餐饮行业实习经验的优秀应届毕业生也可以接受；

5. 大专及以上学历。


工作地址：

广州市天河区中山大道华景路1号南方通信大厦

盐城派合大数据有限公司 该公司其他职位

盐城派合大数据有限公司，是国内领先的人力资源服务机构派合集团在盐城地区的全资子公司，是一家专注于定向人才深度调研，人才培养、孵化，人才猎挖、人才匹配的人力资源服务机构 ，通过人才大数据系统，从技术人才到高端人才孵化、招聘到人才培养，从单一招聘服务到人力资源整体解决方案，我们都能成功服务不同需求的客户。
