举报收藏

岗位职责：

1.负责9APPS相关算法工作，包括数据化运营、个性化推荐等业务；
    2.结合9APPS业务特点和用户特点，利用真实商业数据、海量用户行为数据，提出有效的产品和算法发展方向；
    3.追踪算法前沿技术，结合阿里业务特点，探索将前沿的算法技术应用于实际业务；
    4.负责数据仓库与集市建模，数据ETL的设计、开发与性能优化，负责业务需求的需求理解、数据探查和分析，进行业务产品的数据研发；
    5.参与数据产品、数据平台的架构设计，负责数据产品、数据平台优化或创新；     

任职要求：

1.计算机，电子信息或相关专业硕士及以上学历；  
    2.在机器学习，数据挖掘，自然语言处理，推荐系统等一个或多个算法领域有扎实的理论基础和丰富的研发经验，对算法原理及应用有较深入的理解；
    3.熟悉python/java，对数据结构和算法知识应用熟练；
    4.良好的逻辑思维能力，善于发现和推理不同事物之间的关系和影响，良好的业务理解力，能够结合业务发展制定技术规划；
    5.较好的主动性，良好的沟通能力和团队管理能力，具备出色的规划、执行力，强烈的责任感；
      6.有互联网产品用户行为研究/日志采集/大数据处理等实战经验,具备有Hadoop/Hive/Hbase/Storm/Flume/NoSQL等一种或多种开发经验    

工作地址：

广州市天河区
