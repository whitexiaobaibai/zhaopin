举报收藏

葡萄需要您

1、 负责研究用户、需求、同类产品，收集、整理各方需求；

2、 负责产品规划、流程设计、功能设计、交互设计等工作；

3、 撰写用PRD、产品业务规范等，利用相关产品设计工具完成原型设计；

4、 负责产品研发周期控制，确保产品开发至最终上线运营，对进度质量负责；

5、 负责跨部门协调和沟通，推动界面、开发、测试、运维、运营等人员紧密合作达成产品目标；


葡萄希望您

1、 5年以上互联网软件产品经验，3年以上产品经理经验；

2、 大学本科以上学历，软件相关专业毕业，有3年以上开发经验；

3、 深度了解互联网或娱乐行业，对云技术、大数据、工作流、业务流有深入理解；

4、 优秀的团队管理能力及沟通能力，能有效与研发部门协同；

5、 具备极强的产品构架能力、用户研究能力、需求分析能力；

6、 必须具有相关产品的全过程经验。


葡萄互动是上市企业星网锐捷（A股002396）旗下的一家专注聚会领域娱乐消费升级服务的公司，现有系统市场占有率达到近70%，在KTV行业中位居第一；葡萄互动致力于打造最具规模和影响力的聚会娱乐服务平台。通过聚会增效的方式为娱乐空间提升娱乐服务品质，为用户提供全新的互动娱乐体验。公司包揽了系统研发、内容设计、投资制作及运营四个方面。凭借丰富的场景设计、尖端的核心技术、专业主持人培训机制，构成了完善的整合体系，2016-2017年成功创立了KPAR（“开趴”的谐音）品牌。

KPAR是综艺式聚会.超级娱乐场景新媒体。KPAR强势开启集综艺、娱乐、派对为一体的“综艺式聚会娱乐”新模式。KPAR不只是开趴，更是爱与快乐的文化传播者，致力于打破时间和地点的约束，把更多的爱和快乐通过 “超级娱乐场景”全新模式传递给周围的人。KPAR的品牌理念是： pass love together, not only for pleasure（传递快乐传递爱）。KPAR提倡的聚会态度是：释放、表达、狂欢。

国内综艺式聚会新体验缔造者的KPAR采用“主题式策划设计+互动游戏+道具+高颜值主持人”的组合，借助综艺节目式的互动娱乐方式呈现给用户，让参与者秒变综艺节目的明星嘉宾，真正沉浸在派对的娱乐体验里。简单来讲KPAR就是综艺节目搬到线下来做的综艺式聚会，组团开趴，分队PK，不仅可以事先选定喜欢的主题，也不用经历硬推一个人上台主持的尴尬，还可亲身参与综艺节目中出现的游戏。KPAR目前为在KTV聚会的用户提供聚会类主题派对：足球主题趴、校园主题趴、复古DISCO趴与春宫主题趴，即将推出脱单趴、生日趴、闺蜜趴、团建趴等。

KPAR适用于非常广泛的场景和场所，包括Live House、Club、剧场、会议中心、家庭、甚至各种户外场所，KPAR被首先植入到KTV中，打破了传统KTV里以唱歌为主的单调娱乐形式，为KTV行业在娱乐形式甚至是娱乐升级方面的创新提供了很好的模式。目前已入驻5万家KTV场所，拥有150万间综合娱乐包厢，300万块投放屏幕，1万名聚会引导师的优质资源，触达优质用户5000万+，场所日均触达消费娱乐消费人群430万+。

 “KPAR是当下最流行的聚会模式，不玩KPAR就OUT了。”
