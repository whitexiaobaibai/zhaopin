易往信息技术（北京）有限公司

举报收藏

岗位职责：

1.  负责一个或多个行业的销售工作，对行业的市场和客户进行整体分析，制定行业销售工作推进策略；

2.  寻找销售线索，推进客户合作，完成公司企业级大数据解决方案和应用的销售工作；

3.  收集营销行业和客户动态信息，制定合理的应对策略；

4.  提供客户端反馈数据和建议；

5.  负责管理团队和自己负责客户的产品竞标、销售跟进、客户维护、收款工作。


任职要求：

1.  有能力且善于做销售预测，Pipeline管理，且具备很强的时间观念；

2.  至少具有三年以上的肩负销售指标的工作经验，且具备良好的销售记录；

3.  具备很强的人际沟通交流技巧，较强的组织能力和独立的工作能力，整合各种资源完成销售任务；

4.  在医药、化工、快消等流程制造行业至少一个行业有深入的了解，有稳定的客户关系；对自己熟悉的行业的客户有全面的了解，有策略且有能力推进行业快速拓展；

5.  拥有较为广泛的品牌营销客户人脉关系，能接触客户制造、安全或主要管理核心负责人；

6.  熟悉B2C企业销售、市场推广、市场研究等部门的工作；有Oracle、IBM、 SAP、 HP等国内外软件公司从事销售和营销相关咨询服务或软件行业销售岗位经验优先；

7.  产品平均客单价在100-300万之间，有能力进行300万客单价的销售工作，个人年度销售额不低于1000万。


工作地址：

上海市徐汇区桂平路391号新漕河泾国际商务中心A座2602室

易往信息技术（北京）有限公司 该公司其他职位

易往信息技术有限公司是一个集技、工、贸为一体的从事高新技术的高科技企业, 是家专业的制造执行系统供应商。随着公司业务日益发展，公司在上海、北京成立分支机构。公司立足于汽车及其关联行业，实现行业管理最佳实践与信息技术应用的最佳结合，提供贯穿大型制造企业研发、采购、生产、销售、服务全过程的信息化咨询、软件开发、系统集成、运维服务、信息增值服务，产品与服务线覆盖了产品研发（R&D）、供应链管理(SCM/SRM)、生产制造执行（MES）、生产制造管理(EAL)、SAP实施、集团资源管理（GRM）、市场与营销(M&S)、信息化基础建设（INFRA）、IT服务（ITS）、外包服务等领域。同时也从事自动化工程、生产线设备安装调试、系统集成等工程的设计及施工。并且与美国IBM公司和罗克韦尔自动化公司保持着长期的友好合作关系。在众多国内外知名汽车企业（丰田、通用、ABB、上汽、李尔、大众、东风集团、长丰、长安、昌河、江铃等）中应用实施，取得了显著的成效。
因公司业务发展迅速，现在北京、上海已成立分公司，武汉、广州成立办事处欢迎有志之士加盟，共创美好未来！
