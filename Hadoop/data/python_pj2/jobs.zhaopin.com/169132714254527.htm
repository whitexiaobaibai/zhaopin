举报收藏

岗位职责：

我们提供多场景数亿级渠道客户行为信息，多资金端接口模式；我们致力于完善不同场景、不同产品的实时决策引擎和大数据风控技术。

1. 根据不同场景和渠道客户的行为数据，以量化分析为基础，融合专家模型，参与/负责线上信贷模型开发工作；

2. 参与/负责跟踪策略实施情况，并对出现的风险事件进行学习挖掘，持续完善风控体系；

3. 参与/负责将策略转化为具体实时或离线的规则并跟进落实，最终部署在决策系统中；

4. 参与/负责完成各类分析及监控报告等。

 

岗位要求：

2018年本科或研究生毕业，每周实习时间3日以上，连续实习4个月以上熟悉linux平台，了解一门脚本语言(shell/python/perl/php等)有一定的计算机基础，具有良好的编程习惯和算法基础 较强的分析问题和解决问题的能力 热衷技术，有责任心和团队合作精神 

 

八客简介：

上海八客信息科技有限公司系起航股份（股票代码：833380）旗下子公司，八客是国内领先的银行小微金融与消费金融解决方案供应商，以“为普惠金融带来美好的改变”为愿景，致力于将国内外先进的微贷技术与本土化实践成功经验、管理运营模式相结合，在产品、人才、技术、效益、品牌、机制、系统等方面，构建符合国内中小银行有竞争力的普惠金融解决方案。持续为合作伙伴创造更大价值。

上海市杨浦区纪念路8号国家财大科技园2号楼

上海起航企业管理咨询股份有限公司 该公司其他职位

    起航股份于2007年1月正式成立，总部位于上海，现有金融教育（培训、咨询、在线学习），业务外包、小微信贷三大板块，与全国 1649家银行机构（包括工行、农行、中行、建行、交行五大国有银行，招商、中信、民生、兴业及浦发银行等大型股份制银行，按核心一级资本净额排序前20家农商行中的19家农商行）有着深入合作，业务覆盖全国30个省1200余市县，培训人次超过 350 万。 公司为上海市高新技术企业，拥有自主知识产权和专利，在银行教育培训领域市场占有率第一。
起航股份于2015年8月25日正式挂牌新三板，股票代码为833380，是第一家登陆新三板的金融综合服务的公司。
    同年12月份，起航股份和深圳市思达优悦企业管理顾问有限公司（简称思达优悦）合并，实现强强联合，优势互补。 思达优悦成立于1993年, 是中国培训行业的先行者之一。
思达优悦始终坚持“以教学为本、以服务取胜”的宗旨, 致力于为各企业提供全方位、个性化、有实效的专业服务礼仪、外拓营销培训和督导服务。 培训客户满意度达到90%以上，在业内首屈一指。 双方合并后，公司团队人员已达680余人，公司在资源调配，产品研发及服务质量又有了大幅提升。
起航股份以银行教育培训为基础, 以为银行提供业务服务为纽带, 全方位协助银行提升竞争力。 起航股份立志成为金融行业综合服务的领袖企业。 公司服务对象将延伸到证券、基金、信托、保险、互联网金融等大金融行业, 服务内容在金融教育及人才外包深耕，同时，整合业内知名公司资源，向小微信贷、消费信贷、财富管理、金融市场等领域发展，并借助大数据及互联网，打造金融综合服务平台，为金融机构及金融人才创造更大的价值。
