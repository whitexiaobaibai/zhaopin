举报收藏

岗位职责：

1. 构建并持续完善相关数据的分析,为机器模型人员提供帮助；

2. 对业务数据、用户行为数据、客户相关数据进行跟踪分析，定时提交相关报告；

3. 根据业务需求，进行数据搜集、整理、分析工作，完成各项专题分析报告；

4. 基于数据分析，及时发现、总结有价值信息，为业务决策、管理等提供数据支持；

 

任职资格：

任职资格：

1. 数据挖掘、数据计量学、统计学、计算机等相关专业大学本科（含）以上学历；

2. 两年（含）以上数据分析相关工作经验，具有互联网数据挖掘、用户行为分析等经验者优先；

3. 精通python等挖掘工具，熟悉数据库操作，能完成大数据量在数据库中的查询与运算；

4. 熟练使用EXCEL、PPT等office工具；

5.具有出色的逻辑分析能力和对数据的敏感度，善于发现问题和独立问题解决能力；

6.具有团队精神。


工作地址：

北京市海淀区丹棱街1号互联网金融中心八层

狐狸金服（北京）信息科技有限公司 该公司其他职位

简介：搜易贷，搜狐集团旗下互联网金融平台。

搜易贷由搜狐畅游（NASDAQ:CYOU）原CFO何捷先生创建，核心管理团队均来自于知名IT企业及金融机构，拥有丰富的互联网及金融行业经验。搜易贷聚焦民间小微借贷，致力于推动中国信贷行业的市场化、平民化及高效化，搭建中国最大、用户体验最好的个人及中小企业的互联网信贷平台。

我们的使命：
以满足员工物质和精神两方面幸福的同时，成为中国最大，用户体验最好的个人及中小企业的互联网信贷平台公司，推动中国信贷行业的市场化、平民化及高效化。

获得由《第一财经》和上海陆家嘴金融贸易区管理委员会联合发起的“2014中国金融创新榜”颁发的“年度创新互联网金融平台”奖项

团队介绍:
1、管理层：
1) 何捷：搜易贷创始人兼CEO。香港大学财务与会计专业学士，清华大学EMBA。美国注册会计师协会及香港会计公会会员。中国领先的互联网公司搜狐畅游（NASDAQ：CYOU）原CFO，中国领先的在线招聘平台智联招聘（NYSE:ZPIN）独立董事，在搜狐（NASDAQ:SOHU）任职高管长达十年。曾长期管理搜狐畅游高达50亿元资金，策划过众多重大资本运作、投资及并购，拥有超过17年的中国互联网行业及财务管理行业实战经验。
2) 周霖：搜易贷CTO兼搜狐集团高级副总裁。毕业于清华大学计算机系，负责搜狐集团技术平台的研究开发工作,在北京奥运会期间曾负责奥运官网的网络架构和安全工作。
3) 范学红：搜易贷CRO。北京大学经济学硕士，近20年金融行业数据分析、风控管理经验。曾在淡马锡旗下新加坡富登银行中小企业部负责其亚太项目的风控与业务，并曾任中银富登项目高管。

2、搜易贷核心部门：
1) 产品技术部：搜易贷拥有一支由来自国内众多互联网知名企业尖端人才组成的资深技术产品团队，他们是软件开发、安全、大数据、用户交互体验等各个方面的专家，拥有网站开发、信息安全、数据管理、用户体验等领域的丰富经验。由他们支持的安全、稳定、高效的搜易贷平台将给用户带来安全、可靠、放心的用户体验。
2) 风险管理部：搜易贷拥有一支专业独立的风险管理团队，团队成员多来自渣打银行、中银富登、捷信、安永等国际知名商业银行、消费信贷机构、专业评级及风控咨询等机构，在金融业务风险控制领域具有丰富的从业经验。管理人员均曾亲自负责所在机构风控体系创建、信审流程及规则制定、风险评级及风控模型研发等工作，熟悉所有关键节点和难点突破口。此外，团队核心成员均具有P2P行业一线审批、分控及反欺诈经验，灵活掌握各类风险识别和控制技术，为搜易贷平台用户建立起一道坚不可摧的金融防火墙。
3) 法律支持部：搜易贷法律支持团队成员均来自国内知名律师事务所，长期从事经济法、民法、合同法等相关领域工作；同时搜易贷聘请国内成立最早、规模最大的合伙制律师事务所之一的大成律师事务所担任长期法律顾问，双方共同对搜易贷平台各项业务的合规要求做严格的论证和审核，为用户解决后顾之忧。
4) 用户满意部：搜易贷用户满意团队秉承“真诚、专业、信赖”的服务理念，依托行业领先的用户服务支持系统，通过科学管理、规范运营，致力于为搜易贷平台打造最优的用户服务体验。
