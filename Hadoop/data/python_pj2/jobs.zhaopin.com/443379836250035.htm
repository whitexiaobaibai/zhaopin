【岗位职责】：

1、负责公司cdn等产品海量的访问日志、带宽、质量等数据的分析处理、展示工作。。

【任职要求】：

1、 计算机或相关专业本科及以上学历；

2、 具备1-2年大数据相关工作经验

3、 熟练掌握Java、scala、python起码任意一种开发语言；

4、熟悉大数据开源架构，常用算法。参与过单天日志数据量TB级别项目。

5、熟练使用大数据从收集、清洗、分析、展示等全流程相关工具，如kafka、hdfs、hbase、yarn、sarpk 、spark sql 、elk等。

6、 对数据分析结果进行可视化设计，直观简洁地展示分析结果。

7、对数据敏感，能快速学习业务并能理解数据分析需求。

8、为人务实上进、具备较强的团队合作意识、善于表达沟通。对大数据技术有浓厚兴趣，有较强的逻辑思维能力，能够在压力环境下工作。

蛮蛮天下(北京)网络科技有限公司 该公司其他职位

蛮蛮天下（北京）网络科技有限公司（以下简称蛮蛮云）是国内领先的云计算领域综合网络内容分发解决方案提供商，蛮蛮云通过国内分布式Cache节点以及自建规模化数据中心，为客户提供定制化解决方案。公司于2016年9月在北京成立总公司，并在成都、河南设立了2家子公司，在广州、上海设立研发中心。

公司秉承卓越、务实、包容、专业的企业经营理念，依托专业的团队和强大的技术优势，以更快的响应速度，更贴心的7*24的服务、更完善的端到端安全保障系统，提供差异化、定制化服务，最终将蛮蛮云打造成为“世界一流的云计算综合服务商”。

公司目前已建成沁阳市计算数据中心，可提供10000+的丰富机柜资源，多运营商接入具备丰富的网络带宽资源，路由可一跳到骨干。通过自主研发核心系统平台，为您提供全面的DNS解析、精准的IP定位、全程监控、最优链路、高效防劫持，通过领先的大文件切片技术、Qos保障技术、链路优化技术和防Ddos能力为不同行业提供专业化的解决方案。
