微诺时代(北京)科技股份有限公司 该公司其他职位

微诺时代（北京）科技股份有限公司（股票代码837287）成立于2011年5月6日。公司成立至今，一直秉承“再微小的事情也要100%承诺”的企业精神，致力于大数据及云计算的研究和应用。

微诺时代一直致力于以市场为导向，将大数据、云计算等领先技术融合到解决方案和服务中，利用这些技术全面改善用户的 IT 系统运营环境，帮助政府、企事业单位实现了安全、高效、低成本的信息化系统架构优化，并提供了更加节能、可靠的业务连续性解决方案。
微诺时代用短短三年时间，其产品和服务得到民航、铁路、教育等行业的高度认可，在竞争激烈的北京软件行业中树立起自己的品牌形象。我们先后与甲骨文、NOVELL、VMware等国际知名企业合作，成为其金牌代理商，并为中国民航信息网络股份有限公司、中国铁路总公司（12306）、北京大学、北京交通大学、中国农业大学、北京外国语大学、北京航空航天大学、北京中医药大学等大型国有企业及高校独立提供虚拟化和云管理、大数据等解决方案。
微诺时代非常注重自主产品的研发，先后开发出了WNCloud微诺云计算平台、WNJobs作业自动化平台、WNPM容量管理软件、WNDATA数据库一体机系统等十余项拥有完整软件著作权的自主产品，以及企业级SSD存储卡等硬件产品。微诺时代拥有北京市和中关村高新技术企业称号、“诚信长城杯企业”称号，是北京信息产业协会、中关村企业信用促进会等商协会的会员单位，并于2013年通过了ISO9001:2008质量管理体系认证。

目前公司凝聚了一批颇具创新意识的高科技人才和市场营销专业人才，奉行“团结、务实、创新、奉献”的企业精神，坚持“成就客户”的经营理念。我们将先进技术与优质服务相结合，努力使公司成为一流的解决方案提供商。
2016年5月17日，刚刚过完5岁生日的微诺时代正式登陆新三板，实现上市，顺利完成公司第一个“五年发展”计划。目前，微诺时代正在大踏步前进，走向更加辉煌的未来。
