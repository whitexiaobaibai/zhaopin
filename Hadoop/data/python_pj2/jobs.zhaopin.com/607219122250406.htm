举报收藏

岗位职责：

1.    负责设计、构建和优化基于云环境的人工智能大数据平台，包括需求分析、架构设计、研发、以及性能分析工作；以及核心技术产品的方案设计及研发；

2.    负责整体提升平台集群的高可用性、高性能、高扩展特性；

3.    负责AI平台建设，稳定性保障，性能优化，资源优化，运维优化；

4.    根据业务需求，提出最优的技术解决方案；分解详细的开发任务，能配合其他项目制定开发计划、开发文档、开发流程图；

5.    负责数据架构的规划，并制定实施标准和规范，确保得到有效的执行，同时保障数据以及文档的质量达到预定的标准。


任职要求：

1.    熟悉主流云平台（Openstack、AWS、Azure等）的架构和功能特性，尤其对PaaS云平台构架有深层次的理解； 熟悉SOA模型，了解微服务架构，对Mesos或K8s+Docker有较深的理解；

2.    具有大规模人工智能数据平，高并发大型系统，大数据等架构设计和开发经验；具有分布式系统架构设计和开发能力，熟悉hadoop、spark、flink等大数据技术；

3.    具备数据管理和存储相关项目和系统的开发经验，具有丰富的数据分析、挖掘和数据仓库建模的项目实践经验；

4.    精通C++、java、python、R等编程语言一种或以上；

5.    熟悉大数据挖掘、机器学习、深度学习等技术；熟悉一种深度学习机器学习框架或库（Caffe, tensorflow, MXNet, Torch, TheanoSparkml, mahout,sklearn等） ；

6.    具有金融行业大数据应用工作经验者优先。有主导完成优秀应用或大型大数据项目开发经验者优先。


工作地址：

建邺区奥体大街68号新城科技园研发总部园二幢22-24层
