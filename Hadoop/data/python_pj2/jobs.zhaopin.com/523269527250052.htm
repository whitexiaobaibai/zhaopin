举报收藏

岗位职责：

1.  医学数据的统计分析、智能分类、大数据处理

2.  数据分析的软件功能实现

3.  基于医学数据的评分诊断系统的模型建立

4.  与医院合作项目的数据分析技术支持


任职要求：


1.  数学、统计学、电子、通信、计算机等专业，具备统计分析、信号处理、机器学习的基本理论知识具备信号处理、运动学数据分析、肌电心电信号分析的经验优先

2.  具备数据建模实践能力，熟悉常见的数据挖掘算法，包括回归、决策树、SVM、神经网络、k-means等常用算法的适用场景、优点、缺点以及弥补办法

3.   熟练掌握一种以上统计分析工具软件，如Spss等能够使用Matlab或者Python等脚本编写工具实现统计分析算法的实现

4.   英文文献阅读能力，工作态度积极和责任感强，结果导向  

5．  好的沟通能力和沟通技巧、书面表达能力、学习能力。

上海逸动医学科技有限公司简介Innomotion Inc.

    上海逸动医学科技有限公司(Innomotion Inc.) 2012年9月注册成立，坐落于G60上海松江科创走廊的核心地带--临港松江科技城。是以数字化、智能化医疗解决方案为核心的高新技术企业，松江区重点扶持企业。公司致力于骨科、运动医学、康复医学、及大健康领域的自主创新产品研发、生产、销售以及科研服务。 
    目前已取得4项发明专利、4项实用新型专利、17项软件著作权、6件商标；另有1项PCT国际专利、8项发明专利已处于公布和实质审查阶段，数十项知识产权正在受理审批中。主要产品有：OptimumTM系列三维关节动态功能评估系统（分别针对临床、科研、健康）、逸云TM数字骨科云服务平台、Fluo_Motion正交双平面透视三维成像及分析系统、KingkongTM（逸能）生物力学机器人、OK_Marquer纳米涂层手术导航反光耗材、Acu_AlignTM 3D打印个体化手术定位耗材等。同时，AllWaysTM手术智能通道机器人、NirvanaTM通用型康复机器人等正进入样机试生产阶段，准备开始临床试验。 
    公司产品现已成功进入国内10余家知名三甲医院，如：上海第九人民医院、上海第六人民医院、复旦大学附属华山医院、北京积水潭医院、北京301解放军总医院、广州军区广州总医院、四川大学华西医院等。并和美国麻省总医院、美国哈佛大学、加拿大艾伯塔大学、香港中文大学，香港威尔士亲王医院、韩国圣心医院等国外医疗科研机构有紧密合作。 
    公司创始人王少白博士为美国麻省理工学院、哈佛大学、麻省总医院三方联合培养的博士，曾担任美国哈佛大学教师，研究生导师（Instructor），美国麻省总医院骨科生物工程中心研究员。怀揣国内运动医学领域革新与突破的坚定信念，毅然放弃美国完善的科研环境和优越的生活，选择回国创业。 
    医疗器械由于其行业特性，存在技术壁垒高，投入成本大，研发周期长等问题，创新创业尤为不易。在创业路上，公司有幸获得政府多次的认可和多个项目资金的支持，这让整个团队更加坚定了科技创新的决心，我们将一如既往秉承"Innovation in Motion"的理念，努力为骨科诊疗提供创新产品，实现从主观经验到客观定量分析的转变，实现从无数据到智能大数据的转变，并为手术规划、康复方案制定提供支持。我们立志在数字骨科产品临床化的道路上不断迈进，创立国产自主品牌，力争成为行业领先企业。
