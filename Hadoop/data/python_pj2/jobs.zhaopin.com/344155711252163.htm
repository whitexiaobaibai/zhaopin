举报收藏

岗位职责：

1.根据项目计划按时完成软件系统代码的实现，编写代码注释和开发文档；

2.与前端、产品人员沟通，保证开发质量和进度；

3.根据设计文档或需求说明完成代码编写，调试，测试和维护；

4.分析并解决软件开发过程中的问题；

5.根据概要设计完成详细设计文档； 

6.完成日常维护工作及领导交付的其他任务。

任职要求：

1.计算机相关专业，有较扎实的数学功底； 
2. 二年以上Java 开发经验，熟练掌握常用设计模式，有算法工作经验或者相应能力； 
3. 熟悉MVC架构及软件开发流程，熟练应用Struts2 、Ibatis、Spring3等框架； 
4. 熟练使用Jquery/JSP/Servlet/Html/JavaScript/CSS，具有Ajax 使用经验者优先； 
5. 熟练使用Oracle、Mysql 等主流数据库，精通SQL 语法和存储过程，并能对SQL 语句进行性能调优； 
6. 具有良好的表达能力和沟通能力，性格开朗乐观，具有良好的团队合作精神，工作责任心强，自主学习能力强；
7. 有文本分析及数据分析能力；
8. 有大项目大数据的处理经验者优先。


工作地址：

北京市西城区金融大街20号航宇大厦B座10层
