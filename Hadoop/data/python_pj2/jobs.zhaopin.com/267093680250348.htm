工作职责：

1.Hadoop数据分析平台建设、开发、测试、部署和优化；

2.基于Hadoop的存储平台架构设计与性能优化；

3.设计和开发海量数据的管理系统；

4.基于现有数据统计平台，开发数据统计报表；

5.结合当前大数据平台技术，设计和实现数据收集模型、数据分析处理模型和数据汇总展现模型；

6.实现各运营组织要求的相关数据的统计报表。


任职要求：

1. 精通计算机算法，及大数据相关算法；

2. 精通SQL语句，可以快速编写复杂查询；

3. 懂得 Hadoop、GFS 一类分布式存储原理；

4. 熟练编写 Java 和 Linux shell 程序；

5. 有大数据工作经验者优先。
