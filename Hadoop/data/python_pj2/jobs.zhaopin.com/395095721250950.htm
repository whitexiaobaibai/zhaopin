举报收藏

岗位职责：

1. 按照部门领导安排制订的工作目标制订本人工作计划。 2. 围绕工作目标开展工作。  3. 及时反馈各平台动态。 4. 收集市场信息，反馈用户问题并做记录。 5.根据营运支持部部门工作规划及要求开展日常工作；  6.对营运支持部相关工作流程制度及规范提出改进建议及意见； 7.根据数据分析汇总情况制作相应的数据分析报表； 8.协调配合其他部门开展与运营相关的工作事项；


任职要求：

1. 学历要求：本科以上学历 2. “大数据”分析能力 3. 熟练掌握现代化数据统计工具 4. 熟练运用各类软件工具良好的数据敏感度,能从海量数据提炼核心结果；有丰富的数据分析、挖掘、清洗和建模的经验；  5、 具备良好的抗压能力、沟通能力、逻辑思维能力，有独立开展分析研究项目经验；

北京微赢互动科技有限公司 该公司其他职位

     微赢互动是一家专注于全球移动互联网广告投放、运营及发行的领先移动信息服务提供商。公司于2011年成立，总部位于北京朝阳区，在上海、香港、重庆、淮安、上饶、常州等地设立了15个分子公司，并于2014在纽约、柏林、首尔和伦敦设立海外商务中心，完成全球市场业务战略布局。

公司目前拥有员工450余人，其中研发团队150余人，建立了完备的设计、研发、运营、维护体系，形成了以移动互联网为主导，集广告投放、APP开发、手游运营为一体的业务体系，公司是目前中国最大的互联网综合服务运营商。2013年12月，公司因为优秀创始人团队、掌握行业的核心技术、稳定的高速增长，获得广发证券的亿元资金注入，从此公司的发展注入快车道，在国内业务保持领先的同时，积极开拓海外市场，目前业务覆盖日本、东南亚、欧洲、北美、南美等地。

目前与我们合作的公司都颇具实力，企业包括：中国邮政、广发银行、中国联通、阿里巴巴、百度、新浪、360等央企及大型机构。随着无线互联网的高速发展，公司将会迎来新一轮的发展高潮，实现公司为致力于全球市场的客户提供极致专业、高效的服务的终极目标，并最终实现成为全球跨屏推广第一平台的理想。

公司以“诚信、责任、分享、创新”为企业文化， 公司坚持“以人为本”的管理理念，设身处地为员工着想，针对员工发展安排岗位培训、发展规划指导，为了维护“给员工创造机会”的经营宗旨，公司给每一位有志在公司成长的员工提供广阔的发展、晋升平台；为了丰富员工的业余生活，公司利用现有的良好企业文化平台，定期开展丰富多彩的文化娱乐活动，如定期组织体育活动、郊游、年会、拓展等；以实现公司发展和员工发展的共赢。
