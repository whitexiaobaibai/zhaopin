举报收藏


职位描述：

1.  负责设计、搭建和应用基于基因检测的过程和结果数据，联合表型数据共同构成的的大数据平台；

2. 负责通过大数据的分析平台实现更优的分析模型的产出，进一步提升产品性能；

3. 负责组织、培训大数据分析和支撑团队；

4. 负责根据关联业务和产品的需求，以及公司发展需要，持续优化大数据平台的数据产出效能；

5. 负责保障大数据平台的稳定可靠。

任职要求：

1. 211/985学校计算机或相关专业本科及以上学历，硕士优先；

2. 3年以上大数据平台工作经验，熟悉主流大数据平台技术框架，并能区分其特点；

3. 参加过3个以上的大型软件项目的模块设计和代码开发，并至少领导过其中的一次开发工作；

4. 熟悉 mysql，mongodb 数据库的使用及优化方法；

5. 熟悉 python/java/scala 其中一种开发语言；

6. 熟悉高性能计算的软硬件基础知识，并能熟练使用命令行工具和技巧，在 linux 系统上工作；

7. 有至少2年以上的团队管理经验；

 

优先考虑

1. 有机器学习领域研究的优先

2. 有基因组学遗传学背景的优先

3. 有 spark 应用经验者优先

4. 有 培训讲师 经历者优先

公司介绍：

北京吉因加科技有限公司（Gene+）以基因科技为核心，从事医学健康领域的科学研究、应用开发和产业服务， 是一家集科研、临床检测和健康服务于一体的基因科技公司。目前主要致力于推动高通量基因检测技术在肿瘤防治领域的应用，包含肿瘤的精准用药指导、疗效监测、风险预测和早期检测，是肿瘤精准医疗领域的先行者。

吉因加在未来的发展过程中将整合全球的科技、医疗智慧，本着“一切为了人类健康”的理念， 依托多组学的大平台，对危害人类健康的重要疾病进行组学层面的大数据研究和转化，通过对人体健康的精准组学评价，实现“基因科技，维护健康”的愿景。

公司网址:www.geneplus.org.cn
