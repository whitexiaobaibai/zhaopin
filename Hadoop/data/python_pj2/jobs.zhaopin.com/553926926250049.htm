举报收藏

岗位描述：

1.  负责构架，开发大数据处理平台，满足互联网数据驱动为主的各种业务；

2.  带领团队开发数据存储、查询和分析的超融合架构云化数据集成平台；

3. 指导并参与核心代码的书写，组织解决项目开发过程中的重大技术问题；

4. 把握大数据技术研究的大方向及技术路线，为业务团队提供技术指导；

5. 对开发流程、产品质量、开发进度的规划、控制、监督和管理。

岗位要求：

1. 5年以上大数据工作经验，熟练掌握C和Java；

2. 代码级熟悉Hadoop、Hive、HBase、Spark等相关开源大数据架构；

3. 有大数据分布式处理、实时计算、机器学习算法应用的工程经验和数据平台架构经验；

4. 深入理解大数据应用系统架构，熟悉数据仓库，数据挖掘技术；

5. 有较强的书面与口头沟通表达能力，独立分析、解决问题的能力。

北京九章云极科技有限公司 该公司其他职位

2013年2月北京九章云极科技有限公司在中关村创业大厦正式成立。公司现有员工八十余人，并在美国设有技术研发中心，致力于企业大数据的智能分析和平台开发。

公司提供的云极智能数据解决方案，是以SaaS模式提供大数据理解、分析和预测的服务，旨在通过机器学习模型实现对数据逻辑的理解，借助有限的人工训练，自动优化系统，提高对逻辑的识别精度，保障数据结果的客观性和准确性，为企业数据的实时理解和预测提供保障。

公司同时也提供大数据开发平台的研发，帮助企业以极低的时间和人力成本，迅速建立自己的大数据工作流，提升企业数据价值，降低数据开发的门槛，在Enterprise 2.0时代将大数据开发，关联变成随时可用，可得并可见的生产力。

公司在机器学习，分布式系统开发等领域拥有丰富的行业经验和技术积累，目标客户主要集中在中美的互联网公司，移动互联网公司和电商等大数据集中的行业。公司已与数家行业客户展开前期合作，提供公有云和私有云的解决方案。

公司获得了国际知名风投的投资，能够为员工提供良好的薪酬待遇以及个人发展空间。技术团队充满朝气并富有激情。我们热忱欢迎对数据分析和开发感兴趣的朋友加盟，共同创业。
