举报收藏

JAVA高级程序员
岗位职责：
1、 负责公司IT部门的管理，IT系统设计的架构设计、业务服务业务推广，对公司业务增长、客户满意度情况负责；
2、 系统分析、软件实现、性能优化，并评估新技术的可行性；
2、根据设计要求实现中后台应用的底层开发，调试与集成，并能提出软件或业务的优化方案；
3、从事核心代码的编写、能够指导程序员工作，并负责Code Review，保证产品运行性能稳定性；
4、承担前后台独立功能模块的需求分析，技术设计和开发任务；
5、具备对运行系统的性能调优，BUG分析及修正等优化工作。

岗位要求：
1、 愿意成为学习型、复合型、包容型且有理想的人才；【公司人员特征】
2、 赞同并愿意以阳光、健康的做事方式作为工作态度的基本准则；【公司价值观】
3、 本科以上学历；
4、 扎实的java基础，5 年以上java实际开发经验；
5、 熟悉互联网常用技术和中间件：消息中间件、分布式框架、搜索引擎等；
6、 熟练使用spring、mybatis、springMvc等开源框架；
7、 熟悉并使用过各种大数据相关框架或组件优先，如Hadoop、Kafka、Spark等，有海量数据处理经验者优先；
8、 对技术有激情，喜欢钻研，能快速接受和掌握新技术，有较强的学习能力；
9、 熟悉常用的数据库技术，如Oracle，mysql及NoSQL等，能利用SQL语句实现各类较复杂的业务处理；
10、 有强烈的责任心及良好的团队合作精神，具备挑战精神来应对技术难题。
薪资待遇：底薪，底薪：15K-20K/月；

    我们深知在处于大数据时代，未来的市场研究、消费需求和体验，将依赖于数据的海量集成和分类处理。我们目前已积累并具备了专业的数据分析和整合能力，我们的核心团队具有国际视野和强烈的民资责任感，他们拥有互联网技术、数据分析、产品管理，进出口贸易，国际物流，国际结算和国际人才管理的各方面优秀人才。
    随着互联网行业地快速发展，我们坚信：只有不断融汇新知识、贯通新理念，积极建立学习型组织，优化人才储备机制，才能确保Viomall平台系统始终与时俱进。
    未来我们期待：坚持正直、阳光积极，具备强烈工作主动性和责任心，具有国际视野并愿意不断创新的人才加入。我们将提供员工职业发展规划支持，并将致力于帮助员工不断提升专业能力和素养，我们认为高素养员工是企业具有更高格局的基础，更是员工个人未来职业发展的重要和决定性因素!
