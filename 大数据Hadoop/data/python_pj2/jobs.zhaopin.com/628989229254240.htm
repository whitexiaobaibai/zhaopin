举报收藏

工作职责：

1. 负责Hadoop及相关组件的部署和维护，保证其稳定运行； 

2. 开发和使用Hadoop大数据自动化运维与监控工具； 

3. 基于大数据应用需求，不断调整和优化Hadoop框架及组件的配置，提升性能； 

4. 为数据仓库、数据挖掘建模等数据应用项目提供运行环境支持；


任职要求：

1. 本科及以上学历，计算机相关专业，2年以上相关工作经验； 

2. 熟悉Linux开发环境和shell脚本； 

3. 熟悉Hadoop及其相关组件部署维护，有实际大数据集群运维经验； 

4. 熟悉IT系统基础架构及网络原理，具备大型集群运维经验；  

5. 熟悉数据仓库。


工作地址：

公司介绍：
       中腾信金融信息服务（上海）有限公司（简称“中腾信”）是中信产业投资基金管理有限公司投资成立的专注于消费信贷服务的创新型、专业化、互联网化金融信息服务平台，注册资本1.83亿元。公司致力于成为行业内具有专业性且值得信赖的消费信贷服务公司，为广大工薪阶层解决资金需求，释放信用价值。
       中腾信基于创新的商业模式、贴心周到的服务体系、卓越的运营管理团队以及雄厚的资金实力，着力打造出无抵押业务、有抵押业务、纯线上业务三大产品线，已在全国建立90家服务网点，并开通互联网服务，倾力打造线上+线下，7*24小时专业消费信贷服务，为客户提供优质的金融服务体验。
       中腾信于2015年进军纯线上消费信贷服务并于2015年底推出首款产品“小花钱包”，“小花钱包”是专为年轻人打造的一款新产品，利用生物识别技术并结合大数据风控方法，实现“随时申请、实时审批、随借随还”的极致客户体验。

 

公司愿景：

成为中国领先的专业互联网消费信贷平台

 

股东介绍：
       中信产业投资基金管理有限公司（以下简称＂中信产业基金＂）是中信集团和中信证券股份有限公司旗下从事投资业务的专业子公司，经国家发展和改革委员会批准于2008年6月成立。目前，中信产业基金已发展为国内最大的私募投资机构之一，注册资本18亿元人民币，管理资产规模约510亿元人民币。
       中信产业基金曾被美国福布斯（Forbes）杂志评为2013年度中国最佳PE投资机构。2015年中信产业基金在全球著名私募股权杂志PEI300（全球最大的300家私募股权投资机构）的最新排名中，位列全球第54名，并被评价为成长最快的亚洲机构。
