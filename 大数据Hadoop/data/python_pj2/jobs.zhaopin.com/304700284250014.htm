举报收藏

岗位职责：1. 针对特定行业需求，从数据模型、数据分布、数据传输、数据存储等方面进行大数据系统架构的设计和数据架构；
2. 分析数据治理现状，结合行业先进经验与实际情况，提出数据治理框架、数据治理框架各领域的关系及各领域所包含的相关内容，为政府企业客户的大数据相关IT建设提供战略指导和战术规划，并推动大数据应用解决方案在具体项目中的落地；
3.  参与大数据平台规划及建设，负责大数据存储系统、分布式计算系统、挖掘算法等设计、研发以及维护、优化工作；
4.  参与项目的系统设计和核心代码开发，指导和培训其他工程师，整理和提交技术文档，负责核心功能模块的代码编写和测试工作。
 任职条件：
1. 大学本科及以上学历，计算机、电子、数学等相关专业；
2. 至少具有一至两年相关工作经验；
3. 熟练使用SQL数据库，掌握Matlab、R、Python、SPSS等分析工具之一；
4. 熟悉Spark、Hadoop、Hbase等大数据开源项目；
5. 工作积极主动，具备较强的责任感且有良好的沟通和团队协作能力。

工作地址：

浦东新区金苏路200号D&F幢

上海产业技术研究院 该公司其他职位

上海产业技术研究院成立于2012年7月，是由上海市委、市政府批准成立的新型应用技术研发和服务机构。其主要任务是承担共性技术研发和服务体系中的战略规划、组织协调、联合研发、成果转化和政府决策咨询，以及重大项目和重大产业推进中的规划、路线、评估和协调服务，促进各类创新主体形成创新联盟，促进技术创新与商业模式创新、管理模式创新结合，有效提升上海协同创新、科技创新的能力。
上海产业技术研究院以“开放创新、服务产业”为理念，按照“政府引导、顶层设计，合同管理、柔性参与，资源联投、利益共享”的运作模式，为政府和企业做好“智囊”，为科研供给和产业需求搭好“桥梁”，为产业共性技术研发组建“平台”，形成知识流动和成果转化的“枢纽”。
当前，上海产业技术研究院围绕国家战略和上海产业转型升级的长远和重大需求，重点聚焦数字服务、生物医学、新材料与先进制造以及新能源环保等领域，以重大科研项目、产业共性技术研发与服务平台为载体，发挥顶层设计、资源配置和集成创新方面的优势，通过机制创新，组织产学研金用等社会各类创新资源，结成创新伙伴，联合开展产业技术创新活动，为上海传统产业升级和战略性新兴产业发展提供支撑和服务，为科研成果转化为产业竞争能力传好“接力棒”。
