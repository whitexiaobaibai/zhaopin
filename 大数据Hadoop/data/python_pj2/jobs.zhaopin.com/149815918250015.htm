举报收藏

岗位职责：

1、面向移动/PC互联网构建产品平台，支持多终端、大数据量、大规模并发访问、分布式部署；

2、负责产品架构分析与设计，提出软件和系统架构的整体设计方案，软件功能模块划分等；

3、参与部门的技术决策、技术选型、技术架构和技术风险评估；

4、参与制定设计及实现的规范流程标准，指导设计、实现及部署工作；

5、负责核心代码的开发工作，核心技术问题的攻关和系统优化工作，解决项目开发过程中的技术难题。高级JAVA开发工程师

任职要求：

1、计算机或相关专业本科以上学历，具有四年以上项目开发工作经验，两年以上大型项目/移动互联网项目的设计、架构经验；

2、精通Java语言，熟悉Linux，熟悉软件开发流程和常用项目工具，如：SVN、git等；

3、熟悉高并发、高可用性、高可扩展性及高可运维性系统运行机制及设计；

4、熟练掌握Oracle数据库的原理和使用；

5、了解Hadoop、HDFS等领域的框架或产品的机制与代码；

6、熟悉java开源社区项目，并有丰富的开源项目应用经验，如：zookeeper，hadoop等；

7、熟悉 TCP/IP、HTTP、SSL、RESTful Web service等WEB应用常用网络协议；

8、了解Html5、CSS、javascript、Jquery等前端技术；

9、要求具有良好的沟通能力和团队精神。

工作地址：

广州市环市中路207号恒生大厦C座506、507室 查看职位地图

广州市保力通讯设备有限公司 该公司其他职位

        广州市保力通讯设备有限公司前身“保力通讯系统有限公司”于一九八零年在香港成立，旨在为国内外用户提供高品质高科技的专业通讯系列产品，成立后深受用户的信赖及支持。于一九九三年在深圳成立公司，为进一步开拓国内市场，于一九九九年公司迁移到广州，成立现在的广州市保力通讯设备有限公司。
        我公司主要致力于服务国内公安通讯市场，一直持有公安部通信中心颁发的经营许可证，并且已在广东省的清远、韶关、揭阳、云浮、湛江等省内十多个地区组建了高速公路350兆大型集群及常规通讯网。在二十几年组网过程中，我公司积累了丰富的实践经验。
       近年来，我司加大研发力量，投身于国内警用设备技术开发、刑侦使用的检测设备的开发，成功自主研发产品：用于现场执法的“汽车行驶记录检测仪”设备、全国唯一一款专门针对干扰车锁的探测设备“电子干扰探测仪”。
