举报收藏

岗位职责：

 1. 独立完成中小型项目的系统分析、设计，并主导完成详细设计和编码的任务，确保项目
的进度和质量；
2. 能够在团队中完成code review的任务，确保相关代码的有效性和正确性，协同团队作
战
3. 灵活支撑业务的同时，对于系统容量、稳定性等有较为深厚的理解。

任职要求：

1. 扎实的java编程基础，熟悉常用的Java开源框架，五年以上经验；
2. 具有基于数据库、缓存、分布式存储开发高性能、高可用数据应用的实际经验；
3. 具备良好的识别和设计通用框架及模块的能力；
4. 热爱技术，工作认真、严谨，对系统质量有近乎苛刻的要求意识，善于沟通与团队协
作；
5. 具备大型电子商务网站或金融行业核心系统开发、设计工作经验者优先；
6. 具备大数据量高并发在线处理工作经验者优先。

北京恒善信诚科技有限公司 该公司其他职位

北京恒善信诚科技有限公司成立于2015年6月，坐落于北京市海淀区中关村科技金融街，为广州恒信集团股份有限公司的全资子公司，注册资金5000万元，定位为迅捷、创新、透明的新型网络金融服务平台，希望通过创新的网络金融科技，实现服务百姓大众的梦想，公司旗下有松鼠理财、松鼠借款、松鼠普惠等子品牌。 


本公司提供面向个人的金融服务、面向企业的金融服务以及面向供应链的金融服务，通过自建的高性能金融云平台，提供资金管理、收付结算、数据处理、企业服务等金融服务，通过与实体经济行业的知名企业合作，为其提供供应链金融解决方案，实现金融服务实体经济的目标。

形象：选择松鼠作为吉祥物，是因为他代表着我们的态度，我们希望通过创新的网络金融科技，实现服务百姓大众的梦想。凌空飞跃而过的松鼠，他乖巧、灵动、健康、友善、积极，象征着一个迅捷、创新、透明的新型网络金融服务平台的诞生。

使命：通过创新技术手段提升金融服务效率

愿景：成为技术和数据驱动的迅捷网络银行

价值观：顺应大势、坚持方向、量化方法、创新技术、深度学习
        系统分析、胆大心细、全心全意、满足客户、发挥价值
