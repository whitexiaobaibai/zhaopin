举报收藏

岗位职责：

1. 负责数据平台的ETL流程搭建, 优化和维护管理, 保证平台稳定可靠运行；

2. 负责商业数据平台的数据仓库的设计和开发工作；

3. 负责各个业务系统数据分析处理及建模开发；

4. 负责数据平台的分析产出,提供接口/可视化/业务多维度的结果支持；

5. 日常数据仓库管理、维护、安全控制。


任职要求：

1. 熟悉一种以上关系型数据库（Postgresql/MySQL等），具有大规模数据集处理经验,了解ETL相关流程；

2. 了解Hadoop、HBase、Phoenix、Hive、Spark、Kafka等数据平台软件和框架；

3. 具有Perl/Python至少一种语言的编程经验, 精通SQL语言和优化；

4. 熟悉Scala/PHP及Linux/Shell优先；

5. 具备良好的需求理解能力, 沟通能力以及工作责任感。
