举报收藏

岗位职责：

1、负责开发公司线上、线下 Docker 集群、监控、业务运行支撑系统

2、基于云平台阿里云的基础系统运维、应用运维

3、负责Linux服务器和其上开源应用的自动部署脚本开发

4、需要在不同的运维场景和业务场景中快速完成具有挑战性的项目

5、对新技术和方案进行调研评估和引进

6、负责运维相关平台设计和开发工作

7、负责hadoop大数据平台的搭建

8、实时日志查看系统及大数据日志分析平台的搭建及维护

9、运维自动化系统的搭建


任职要求：

1.全日制本科及以上学历，计算机相关专业，相关DevOps管理工作经验 2+ 年以上；

2.熟练使用linux操作系统，熟悉常见基础服务的搭建和配置,熟悉shell，python，go等脚本语言；

3.熟悉(nginx/haproxy)的配置及调优,了解运维高容量、大流量、大并发Web系统的业务知识以及解决方案；

4.熟悉zabbix监控管理工具,熟悉puppet/saltstack/ansible中的一种；

5.熟悉jenkins,git等开发工具；

6.悉docker容器技术，有kubernetes集群管理经验者优先；

7.熟悉MySQL、MongoDB等常用数据库的运维及优化；

8.有虚拟化kvm经验优先,有阿里云等云平台使用经验者优先；

9.强烈的责任心和主动性，良好的沟通能力，团队意识强；

10.Python开发者优先考虑。

11.风险安全意识要高，具备一定开发能力

12.对于各种报警能做出及时的相应并处理，周末的报警处理实行轮班处理。

13.熟悉hadoop大数据平台的搭建及维护

14.能独立开发运维自动化平台并能根据公司的需求做各种定制化

15.能独立开发实时日志查看系统及大数据日志分析平台

上海风创信息咨询有限公司 该公司其他职位

海风教育由郑文丞等几位复旦大学在校学生创立于2010年。海风教育现有学科培优、自主招生辅导两大课程服务体系，全面满足了中小学生在小升初、中考、高考等升学过程中的各类需求。公司拥有超过2,000名的精英教师团队，独创海风教育“知识切片”体系，囊括了全国范围、十二个年级、全学科、所有考纲、各种难度层次的教学内容。海风教育中央厨房式的教研后台数据库，能够对教学过程进行全程记录并通过大数据分析对整体和个体间教学工作中的各个环节提供全面支持。这些使得海风教育在教研和教学方面成为行业领军者。
2015年7月，海风自主研发的在线教学平台正式上线。这个由海风教育独立研发的智能学习中心平台，在强大技术支持下，通过在线授课教研题库、知识点矩阵、智能错题本等多元功能的搭建，结合大数据和人工智能技术，打通了“课前预习、课上学习、课后复习”的学习全过程，满足了各阶段学生的不同需求，让学员的学习过程变得更加高效、智能和便捷。
近日，海风教育宣布完成数千万元B轮融资，该轮融资由零一创投、涌铧投资、奇成投资、雍时投资等共同投资，由荒合资本担任独家财务顾问。海风教育创始人郑文丞称，本轮融资将主要用于进一步扩大业务规模、提高市场占有率，以及巩固教学平台技术和教研水平国内领先地位。
