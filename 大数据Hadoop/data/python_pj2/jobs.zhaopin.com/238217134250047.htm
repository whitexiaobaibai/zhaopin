举报收藏

职位描述：

1、根据公司文档规范编写相应的技术文档；

2、负责数据的收集、清洗、整理、提取和统计；

3、根据需求进行数据仓库和多维数据集的设计与开发；

4、根据业务需求进行数据分析和数据挖掘工作；

5、负责数据平台搭建与维护。


职位要求：

1、有工作责任心，思维清晰，具备良好的团队合作精神，有一定的承压能力；

2、深入理解数据仓库、数据建模等概念，有商业智能经验；

3、熟悉Microsoft .Net体系架构和MS SQL Server、SSAS、SSIS；

4、熟悉SSAS多维数据库、Cube设计，在现有的OLAP数据库和Cube上开发和维护；

5、熟悉分布式系统概念和架构，熟悉Hadoop/Hive/HBase/Impala等大数据技术的优先。


工作地址：

广州米多网络科技有限公司 该公司其他职位

   米多网络应移动互联网大趋势而生，独创“全场景赋码”理念和“三重空码赋值”技术，专门为传统品牌企业提供基于大数据的防伪、防窜、溯源、产品复购、粉丝互动、社群营销、三级分销、全员开店等一体化、全方位的移动互联网整体解决方案及运营系统，能够有效整合传统品牌的线上线下业务，并通过API接口和企业内部ERP系统以及外部供应链系统有机协同，真正帮助传统品牌企业从（网络技术）时代走向  （数据技术）时代，全面实现“无边界思维、粉丝化驱动、阿米巴经营、大数据生存”，最终让传统品牌企业的经营行为植入互联网基因，使传统品牌企业的运营团队拥有互联网思维。
   米多荣获广东省质量技术监督局“广东名牌产品”防伪鉴证唯一合作商资格，为1800多家广东名牌产品企业提供赋码鉴证服务。截止2016年初，米多的合作代理商超500家；品牌商用户：茅台，西凤酒，九江酒厂，小糊涂仙，伊力特，美的，好迪，拉芳，大益茶，武夷星，雨林古树茶，柏联普洱茶，喜多多，椰牛，福田电工，珠江电缆，家的电工等2万多家传统品牌企业正从米多“全场景赋码”方案中受益。
