举报收藏

岗位职责：
1. 负责基于Hadoop生态系统的海量数据处理分布式平台总体架构设计及海量数据分析研究方案的设计及研发；
2. 制定项目设计及实现规范，指导设计、开发及部署工作；
3. 带领、协助指导工程师解决关键问题；
4. 设计开发关键功能模块。

岗位技能：
1. 计算机、电子信息等相关专业本科或以上学历，具备3年互联网或大数据从业经验者，并具有基于Hadoop生态系统大数据平台建设经验；
2. 熟悉Hadoop家族的多个组件的开发和使用，如Hive、HBase、Spark、Kafka等；
3. 精通基于Hadoop平台等分布式开发，熟悉Linux系统；
4. 具有较强的责任心、执行力，有良好的沟通和团队合作能力；
5. 在大数据处理和分析领域技术规划、架构设计经验者优先；
6. 熟悉统计软件（SAS、R），或有数据分析和数据挖掘项目实施经验者优先；
7. 热爱软件开发，有激情，自我驱动力强，能够独立分析、解决复杂问题，能不断跟踪学习新技术

北京市太极华青信息系统有限公司 该公司其他职位

太极华青信息系统有限公司 是北京新技术产业开发区（中国硅谷）的骨干企业，并被中国信息产业部评为2003年中国软件100强企业。被北京市国家税务局评审为 ‘A级纳税信用企业'

我公司以大型系统集成、行业整体解决方案和独立软件开发提供以及智能应用为主要业务方向。

公司自创立之初即汇聚了一批善经营、懂管理，有专业知识、不断追求卓越的高级人才，并始终坚持“创新即企业生命力，人才即企业之根本”的经营理念。

目前，公司已形成包括软件产品部、软件项目部、系统集成事业部、智能控制系统事业部、市场拓展中心、客户服务中心在内的数个业务部门。近300名高素质工程师组成的强大队伍，在专业系统集成、软件开发、研究及技术支持、工程实施等方面具备了很强的实力。

公司的系统集成和软件开发业务通过了ISO9001国际认证，实现系统集成、软件开发、售后支持的全过程控制。公司目前拥有信息产业部认定的《系统集成一级资质》、建设部颁发的《系统集成甲级专项设计资质》、国家保密局颁发的《涉及国家秘密的计算机信息系统集成》和由中国信息安全产品测评认证中心颁发的《国家信息安全服务资质证书》等多项证书。

公司为各行业用户不断推出高品质的产品与服务，强化市场运作模式，从而赢得客户的广泛认同，市场占有率得到迅速提高。 此外，公司注重加强与国内外著名企业之间技术合作与交流，并结成了长期稳定的战略伙伴关系，以此为基础共同为中国的信息化建设做出积极贡献，公司目前是CISCO金牌认证代理以及HP、IBM、Sybase等公司的行业代理。

太极华青信息系统有限公司抓住历史发展机遇，在激烈的市场竞争中，凭借雄厚的技术实力和灵活的市场运作，取得多个国家重点项目合同。

为满足业务迅猛发展需要，太极华青信息系统有限公司诚邀下列人士加盟，共创美好未来。

以上职位要求具备很强的组织协调能力，团队合作精神和良好的服务意识以及大学公共英语四级以上水平，硕士、博士优先。 凡经择优录用者将享受国家规定的各项福利待遇和富有竞争力的薪金。
