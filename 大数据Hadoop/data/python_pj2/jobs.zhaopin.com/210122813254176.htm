举报收藏

岗位职责：
1. 根据业务方的需求，通过大数据分析给出统计数据。
2. 参与数据、工具平台相关的功能接口、数据接口，实现业务功能。
3. 参与数据平台、工具平台的架构、设计以及实现。
4. 优化数据平台的性能，解决兼容性、功能性Bug，提高稳定性。
职位要求：
1. 全日制本科及以上学历，计算机或相关专业。
2. 精通Java语言、熟悉Spring、Spring MVC、Mybatis等常用Java框架。
3. 熟悉RDBMS（比如MySQL）的开发，熟悉SQL语言。
4. 熟悉Java并发编程，有优良的Trouble Shooting能力，能独立分析解决问题。
5. 熟悉NoSql应用（比如ssdb、Redis等）的开发经验者优先。
6. 熟悉常用大数据中间件，有Zookeeper、Kafka、Elasticsearch等开发经验者优先。
7. 掌握Hadoop、Spark、hbase、impala、hive中至少一种大数据技术，阅读过源代码者优先。
8. 对大数据技术有钻研热情，乐于分享。

工作地址：

关于 爱卡汽车

爱卡汽车是中国第一个以汽车为主题的社会化网络互动媒体，于2002年8月成立。目前注册用户1200万，日均

浏览量1亿，汽车主题交流平台467个。从最新鲜的行业资讯、最专业的试驾评测、最及时的行情信息到最权

威的数据调查。爱卡汽车提供给用户的是全方位的精准服务。从多年网友分享的海量数据中开发出--核心精

准口碑类导购产品“说真的”和指导精明消费的导购产品“养车费用”；开创业内首个SNS平台“My

xcar”和第一个交互式问答系统“爱卡万能”。爱卡汽车还在移动互联网领域中不断开拓。WAP站及手机APP

应用的推出和不断优化，更加强调用户需求和用户体验。爱卡汽车现已成为中国汽车用户的专属精神家园。
