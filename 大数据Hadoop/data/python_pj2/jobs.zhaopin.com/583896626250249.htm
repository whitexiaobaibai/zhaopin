举报收藏

岗位职责：

1. 开展调度自动化软件产品测试，包括软件功能、性能、可靠性测试等；

2. 开展云平台、大数据平台及应用测试；

3. 编写软件检测的相关测试计划和方案，设计和执行测试用例；

4. 开发相应测试工具，编写测试脚本；

5. 撰写检测报告及其他技术文档。

 

任职要求：

1. 计算机、自动化、软件工程、电气工程等相关专业，本科及以上学历；

2. 热爱测试行业，掌握软件测试理论基本知识；

3. 具有2年以上软件测试工作经验，能够独立完成测试工作；

4. 掌握Loadrunner、C++test、Klockwork等主流测试工具；

5. 具有C、C++、Java、.net软件开发能力、熟悉云计算、大数据测试技术、熟悉Linux操作系统者优先。


工作地址：

航天中认软件测评科技(北京)有限责任公司 该公司其他职位

航天软件评测中心隶属于中国航天科工集团二院七0六所,是国内首家从事高安全、高可靠软件评测的专业机构。中心下设评测部、民品部、研发部、质量部和办公室。
航天软件评测中心成立以来，圆满完成了以载人航天工程、探月工程为代表的国家重大科技工程和重点武器装备，以2008北京奥运安保工程、2010广州亚运会安保工程、中国高铁控制系统(列控系统等完成于2013年9月)、国家电网智能调度决策支持系统、公安部信息系统、中广核集团核电安全控制系统以及著名家电企业的洗衣机控制系统等为代表的民用领域中各类关键软件的独立评测工作，为国防现代化和国民经济建设做出了重要贡献。先后获得“全国五一劳动奖”和“全国五一劳动奖章”，国家人事部、解放军总装备部和国防科工委联合授予的“中国载入航天工程突出贡献集体”等荣誉。
航天软件评测中心树立了“国家利益高于一切”的核心理念，秉承了“特别能吃苦、特别能战斗、特别能攻关、特别能奉献”的载人航天精神，建立了“专业、客观、诚信、公正”的行为准则。目前，航天软件评测中心已从技术、人才、管理、环境等方面建设成为国内一流的专业软件评测机构。
