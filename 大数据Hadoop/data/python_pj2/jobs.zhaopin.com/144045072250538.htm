举报收藏

岗位职责：

1.参与系统的需求调研和需求分析，交付项目需求规格说明书；

2.参与系统设计和讨论，交付概要设计、详细设计等设计文档；

3.完成项目关键功能模块的代码实现；

4.系统开发测试、部署和集成，确保研发软件的安全、质量和性能；

5.负责解决开发过程中的技术问题，指导初级开发人员共同完成项目目标；

职位要求：

1.计算机、通信或相关专业本科以上学历，CET-4级以上；

2.能采用SSH进行web开发，并熟知软件开发流程；

3. 熟悉java多线程、socket通讯、IO流等编程，能通过SNMP、Corba、Tr069等协议做设备进行管理者佳；

4. 熟练使用oracle、mysql等常用的数据库系统；

5.有网管开发经验及大数据背景者优先。

上海国响信息技术有限公司 该公司其他职位

    上海国响信息技术有限公司是一家运营级软件系统开发商, 是中国通信行业协会会员单位、国家下一代广电网NGB会员单位、上海市多媒体行业协会会员单位，是上海市著名的软件企业。公司致力于成为运营支撑领域的领导者，为电信和广电运营商提供业界领先的产品和一流的服务。
    公司倡导“尽责、进取、协作、共赢”的企业文化，坚持“科技服务民生”的科技人本理念，注重技术和服务的创新，凝聚了一批具有丰富领域知识及资深软件开发经验的优秀人才。公司建立了以市场为导向、以客户为中心、以技术创新为动力的运营体系，拥有多项具有自主知识产权的产品。
    公司按照ISO9001的要求建立了完整的质量保证体系，各项管理环节标准化、规范化和制度化；同时，作为一家优秀的软件系统开发商，公司还引入了国际通行的CMMI软件产品研发管理框架，面向软件产品生命周期全过程进行精细化管理，确保了产品的质量和研发进度能够有效满足市场的需求，公司已经成为多家国际著名企业的全球合作伙伴。
