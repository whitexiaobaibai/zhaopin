举报收藏

     这不是一般的行政岗位，在这里一年你可以收获同类岗位三年的资本，在这里和我们一起，你将成为：


1. 店面总监的业务助手-- 提供运营核心数据的统计与分析，与运营核心管理者对话，接触公司核心业务领域，提升大数据时代的知识储备；

2. 高效专业的行政经理--门店行政事务管理及公司各项制度的监督管理，从基础工作中扎实管理经验提升自己；

3. 文娱团建活动专家--每一个节日，每一个月都为你准备了表达自己超强组织能力的活动机会，传递公司文化，建设高绩效、有爱有梦想的团队文化；

4. 半个外交官--上传下达，对内准确传递有效信息，对外维护公司形象。

 

我们期待与这样的你相遇：

1. 优秀的你--211学历及以上

2. 高效的你--心细、勤快，善于运用办公软件（word/excel/powerpoint等）；

3. 学习力强的你---对工作充满热忱，有目标，每一天都有进步；

4. 乐于沟通的你---有较好的沟通与协调能力；

5.服务意识强的你---亲和力强，有爱心，喜欢为身边的人服务；

纵向发展：行政助理→行政主管-行政经理—高级行政经理→资深行政经理

管理发展路线：区域行政经理（管理区域助理团队）→高级区域行政经理---资深区域行政经理

 不一样的行政工作,带给你不一样的精彩与收获!
