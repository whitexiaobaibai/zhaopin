举报收藏

职位职责：

1、全面负责大数据分析系统的需求分析和整体架构设计，负责确定软硬件实施方案。

2、负责核心技术问题的攻关，解决项目开发过程中的技术难题；

3、负责大数据分析系统相关具体应用算法的设计、开发和产品化。

4、根据公司产品和业务发展特点，负责研究相关大数据产品和技术发展方向。


任职资格：

1、计算机、信息系统、数学或相近专业本科以上学历，5年以上从事软件开发工作经验，2年以上大数据系统分析、设计和实施经验；

2、精通和全面掌握常用的软件设计方法、计算架构和解决方案，包括面向对象设计、SOA、J2EE、云计算等；

3、拥有2年以上Hadoop开发设计和实施经验，对Hadoop相关的技术和组件（HDFS，MR，Hase，Hive，Spark，Storm等）有全面深入了解，能够熟练安装、配置、部署和优化大型Hadoop集群系统。

2. 有超10TB大数据处理实战经验，熟悉整个大数据的完整处理流程，包括数据的采集、清洗、预处理、存储、分析挖掘和数据可视化。

3. 熟悉互联网常见的业务模式和用户行为分析模型，熟悉常用的数据挖掘技术和算法，如路径分析、关联规则挖掘、分类、聚类分类、协同过滤等。

4. 不怕困难，有关键技术攻关的决心和能力，能够适应和享受高强度高压力的工作。


工作地址：

上海特易信息科技有限公司 该公司其他职位

       上海特易信息科技有限公司（简称特易资讯）成立于2004年，是全球领先的外贸资讯服务提供商，总部位于中国上海，现拥有14家分支机构和遍及亚洲的经销商网络。十余年来，特易资讯始终秉承“化资讯为资本”的理念，聚焦于国际贸易信息的整合和深入研究，依靠高速的信息流动和以人为中心的操作模式完成对海量信息更有效的汇聚、关联以及交互呈现，致力于为中国外贸企业提供全面的外贸资讯信息服务、精准的数据分析服务、可靠的营销策略支持以及大数据金融服务。目前，特易资讯会员已突破50万家，推动会员企业年度交易额突破17亿美元。
      2015年，特易资讯正式启动全新外贸BPO战略，全面布局信息流、物流、资金流“三流合一”的外贸全流程服务平台，特易资讯将集合更多信息、金融、物流等优势资源，于BPO平台上完成资源匹配、交易撮合、贸易融资等外贸流程相关服务，不断推动贸易资讯的应用创新。
      未来，特易资讯将继续奉行“爱家人、爱客户”的企业文化，致力于引领中国外贸资讯服务行业的发展，汇聚人才、追求卓越，以诚信求实之心，担企业责任之行，成为社会尊敬、客户信赖、员工自豪的标杆企业。
