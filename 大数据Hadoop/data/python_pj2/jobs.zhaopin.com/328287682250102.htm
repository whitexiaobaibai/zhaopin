举报收藏

岗位职责：

1. 根据业务需求，进行大数据风控模型设计和开发，包括但不限于用户画像、信用风险模型、欺诈风险模型、行为评分模型等、贷后风险模型；

2. 负责评分模型相关策略制定，跟进模型实施上线； 

3. 负责模型验证，撰写验证报告，结合模型和验证结果，进行模型优化；


任职要求：1. 统招本科以上学历硕士优先，统计学、数据挖掘、计算机科学、数学、金融等相关专业优先；

2. 深入理解数据挖掘、机器学习算法或理念，具备优秀的数据建模能力，能够根据实际问题选择合适的模型和算法；

3. 熟练SAS、SPSS、R、Python等其中一种或多种数据分析和建模工具；

4. 有海量数据处理经验，熟悉Hadoop、Spark等大数据生态技术优先；

5. 能够独立发现业务问题，将业务问题映射为数据问题，制定量化分析目标，设计数据挖掘框架、流程与计划，设计与优化数据挖掘算法。（数据挖掘与建模方向）； 

6. 有较强的适应性及压力承受能力，责任心强，重视团队合作，工作态度积极，勇于进取，有较强自我学习能力。

米袋子信息技术(北京)有限公司 该公司其他职位

米袋子信息技术(北京)有限公司拥有领先的跨金融数据征信、数据建模、互联网技术和运营能力，公司成立于2015年，米米贷是公司旗下的核心品牌。公司核心团队由互联网和金融行业资深人士组成，曾任职于搜狐、宜信、FICO、淡马锡等知名企业，拥有北京大学及海外名校背景。米米贷通过手机客户端，定位于收入有限的蓝领、初级白领客群，基于“移动互联网 + 人工智能 + 大数据”的专业风控体系，帮助符合资质的用户在合作资金机构获得个人消费信贷服务。

    米米贷作为一家网络信用风险管理及服务机构，我们的使命是致力于革命性的金融技术创新，坚持小额分散原则，倾注于传统金融机构没有照顾好的金字塔底部，助力普通民众获取更快捷、规范的普惠金融服务。

    米米贷的愿景是，成为以技术驱动的，信用风险管理领域的标杆企业。
