举报收藏

岗位描述：

1. 负责电力大数据平台的新功能开发、日常bug修复等工作；

2. 负责基于大数据平台的数据建模、海量数据的处理计算工作；

3. 负责多业务数据仓库的设计、构建和 ETL 工作；

4. 负责Hadoop、Hive、HBase、Spark项目开发；

5. 带领大数据组支持日常工作。

岗位要求：

1.熟练掌握 Java 编程语言，并要求熟悉 Shell，Python 等一门以上脚本语言；

2.熟悉 Linux/Unix 环境，有 Hadoop 框架开发经验,熟练大数据数据仓库设计开发；

3.熟练掌握Hive，HBase，Spark，Storm 等一种以上大数据处理工具和技术；

4.逻辑思维能力强，对数据敏感，有较强的学习能力和创新思维；

5.具备较强的沟通能力和文字表达能力，有较强的团队管理能力。

特别说明：

本岗位工作地点为中国电力科学研究院（北京市海淀区清河小营东路15号）

工作稳定，办公环境良好，薪资、福利优厚。

北京振中电子技术有限公司 该公司其他职位

北京振中电子技术有限公司初创于1988年，其前身是中国科学院计算所直属产业集团公司，是中关村科技园的高新技术企业。

公司集研制、生产、销售和服务为一体，专注于电力行业，经营自主研发、拥有自主知识产权的电能量数据采集与分析等产品。主要产品包括：手持抄表终端、负荷管理终端、配变监测计量终端、低压集中抄表系统（集中器、采集器）、无线采集器、配电自动化终端、电压监测仪、PDA、智能电力电容器、在线监测装置等。在国内，抄表领域的市场占有率始终保持绝对领先位置，达到同类产品70%以上份额。

    振中公司凭借多年专注服务于电力行业的的经验积累，执著的专业精神、不断的技术创新和完善的服务体系，公司的系列产品均得到电力用户的广泛认同，并且远销至国际市场，并多次参与行业标准的制定。随着电力市场的持续发展，以及电力系统现代化服务和管理水平的显著提高，振中公司充分发挥自身的技术优势，潜心于用电现场服务与管理系列产品以及配变监测计量终端的研发和生产，经过不懈努力，已开始逐步推广使用并获得用户好评。

    公司产品除电力行业外，在自来水、燃气、图书、医药等公共事业，以及物流、零售、仓储、物资管理等众多领域中广泛使用。公司重视技术性投入，利用自主知识产权的核心产品和高技术人才的优势，开发适销对路的产品，全面满足市场及客户的需求。自成立以来秉承“专业铸就品质，信誉成就振中”的经营理念，致力于企业信息化建设，具有雄厚的产品研发、技术服务、市场营销实力。
