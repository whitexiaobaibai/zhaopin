岗位职责：
负责亚马逊的日常销售运营工作：
1. 包括产品资料编辑上传，挖掘产品卖点和分析产品关键词，维护及优化Listing，追求每个产品效益最大化；
2. 熟悉Amazon平台规则和运用促销推广方法，根据产品销售与排名变化，及时制定和调整产品的销售政策，做好数据的统计分析工作；
3. 定期统计销售数据和库存数据等，根据市场销售情况，合理安排备货，保持较好的库存周转率和库存量，保证店铺产品正常运营；
4. 妥善处理各种争议，及时解决和减少客户投诉，保持店铺的好评率和良好的信用度，规避帐号的各类安全问题；

任职要求：
1.上进心，责任心，有激情。上进心，责任心，有激情。上进心，责任心，有激情。
2.学历无任何要求，只要你觉得有自己有能力就行。
3.无任何语言或其他等级要求，觉得自己的够用就学习。
4.如果你有一年以上的经验，我们欢迎你来带领我们做的更好，如果你只是刚刚毕业，我们可以一起来学习交流。

深圳阳光环球玻璃有限公司 该公司其他职位

深圳市阳光环球玻璃有限公司是一家集建筑玻璃、室内装饰玻璃、特种工艺玻璃等多种建筑玻璃于一体的玻璃深加工企业，现工厂占地近32000平方米，拥有先进的钢化玻璃生产线(8m)、中空玻璃生产线、电脑自动磨边机、中空玻璃、夹胶玻璃、热弯玻璃、以及夹丝、热熔、烤漆、镶嵌等各种工艺装饰玻璃。
公司现有玻璃深加工技术工程师30余人，技术员工200余人。技术工人都经过了严格的专业技术培训，有着丰富的玻璃深加工经验，所以公司生产的玻璃产品以优良的质量而赢得广大用户的好评。
深圳市阳光环球玻璃有限公司的产品获得了国家3C质量的认证，国家玻璃检验中心的认证。产品质量得到了有力的保证。
公司在广州、东莞、深圳、以及全国均有很多优质玻璃工程而深得建筑建设师、建筑装饰公司的好评与依赖。
公司在做好玻璃产品的同时，还积极关注社会慈善事业，关心贫困地区的教育事业，公司多次以国际狮子会成员的名义向社会及贫困地区积极捐赠，捐赠扶贫事业和贫困地区的教育事业，这一远见卓识的义举赢得社会广泛赞誉。
公司将再接再厉，以更好的产品、更好的管理、更好的服务，来满足市场的需求、客户的满意是我们的宗旨，我们将朝这一目标不断努力、不断进取，使我们的产品做得更好！
