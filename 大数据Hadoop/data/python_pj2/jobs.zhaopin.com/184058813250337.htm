举报收藏

岗位职责：
1、根据项目的需求制定数据库架构，配合开发部项目设计，数据库设计，数据字典；
2、数据库存储过程、接口开发； 3、双向同步Oracle数据库和SqlServer数据库； 4、负责对数据库的规划、安装、升级、迁移、复制、备份、转换、恢复和性能优化调整。
5、完成数据库规范文档的编写。
任职要求：
1、计算机相关专业;
2、熟悉数据库设计理论，能建立合理高效的数据库结构；
2、精通Oracle和SqlServer性能优化，服务器优化以及存储过程和函数的优化，具备较强的SQL编写功底和清晰的逻辑思维能力；
3、有OCP/OCM认证者优先
4、有大型系统的设计和开发经验，
5、具备良好的表达和沟通能力；
6、擅长技术文档的书写、擅长沟通;
7、有大数据量、大并发量数据库管理设计经验者优先；
工作地址：

深圳希创旺思电子科技有限公司 该公司其他职位

  希创旺思电子科技有限公司是一家提供企、事业单位信息化建设整体解决方案的系统集成商，致力于信息系统的研发、应用、咨询，产品涵盖系统软件、行业解决方案、移动应用开发、自动化识别技术、物联网应用以及外包服务。

  希创旺思电子科技有限公司成立于1994年，二十多年的励精图治，希创旺思拥有一支以系统分析师、系统架构师为核心成员的人才团队，业务遍布全国各地，公司总部设在广州，在深圳、东莞、中山、香港、厦门、南昌都设有分公司和办事机构。
