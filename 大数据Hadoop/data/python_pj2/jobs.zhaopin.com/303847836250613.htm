举报收藏

【大数据管理】

职位描述：

1、负责项目实施过程中框架搭建，数据解析存储，并能够独立负责数据模型框架设计和ETL开发；

2、根据业务需求设计数据仓库模型和ETL数据流程；

3、负责BI/DW项目的ETL开发、测试与调优，多维数据统计分析；

4、负责数据质量检验和监控。

5、钻研业务，能够更为深层次地理解业务数据。


岗位要求：

1. 熟悉python脚本，Django框架，mangoDB存储过程；

2. 熟悉SQL，有数据库性能优化，海量数据处理经验尤佳；

3. 有ETL工具使用经验，如kettle、DataStage、informatica、ODI等优先；

4. 有很强的学习能力及学习欲望，沟通表达能力强，有较好的服务意识；

5. 对工作有热情，富有团队协作精神、责任心强。
