举报收藏

职位描述：

1. 参与公司内部大数据平台的建设与维护；

2. 参与基于大数据平台的基础服务开发（数据收集、流计算、数据分析等）。

职位要求：

1. 本科或以上学历，计算机相关专业（亦欢迎非计算机专业自学者），能阅读英文技术文档；

2. 熟练掌握任意一门服务端编程语言（erlang/python/java/c++等）；

3. 熟悉常用数据结构与算法，逻辑思维清晰；

4. 热爱开发，敢于创新。

优先条件：

1. 熟练操作Linux/Unix 操作系统, 熟悉Shell编程者优先；

2. 拥有大数据平台（hadoop）开发经验者优先。

2. 个人成长及发展：

1) 发展：专员级-资深/主管级-经理级-总监级；

3) 一对一导师指导、入职培训、岗位技能培训、管理类培训、各种主题的交流培训；

4) 双通道能力发展体系和职级晋升机制；

5) 公司支持内部岗位轮换，个人可根据能力、特长及规划申请岗位调整，确定合适的发展方向。

 

3. 成就奖励：

优秀员工/主管/团队奖；年度最佳进步奖；业绩突破奖；突出贡献奖；长期贡献奖

 

4. 公司福利：
