举报收藏

岗位职责

1、负责Hadoop平台的容量规划、部署、及扩容；

2、负责Hadoop平台各组件服务的监控、故障处理及性能调优；

3、负责Hadoop平台的用户管理、权限分配、资源分配；

4、负责hadoop集群环境的运维和优化工作, 集群性能和资源利用率优化、集群常见问题能迅速定位，并不断优化，提升集群性能，为开发人员提供支持

5、负责整体提升hadoop集群的高可用性、高性能、高扩展特性

6、负责撰写相关技术文档；


任职条件

1、大专以上学历，至少一年以上实际工作经验

2、熟练掌握Linux操作系统，熟悉linux文件系统，内核，网络等性能优化；

3、熟悉HDFS、YARN、Hive、HBase、Spark、Storm等hadoop生态技术的体系结构和运行原理；

4、熟悉一门以上脚本语言（shell、Python、perl）；

5、熟练掌握CDH hadoop的安装部署、性能监控

6、有Hadoop生态系统运维经验

森艾公司简介
北京森艾电子技术有限公司的前身是北京森艾科技有限公司（以下简称：森艾公司）成立于2005年，同年获得北京高新技术企业公司认定。公司位于北京海淀上地信息产业基地，注册资金1000万，员工79人，是一家主要经营IT服务、IT设备采购及IT培训的公司。

森艾公司秉承“诚信为本，服务创新”的经营理念，致力于塑造“IT服务”高端品牌。森艾公司目前拥有普通工程师至高级工程师共三十七位，包括系统、数据库、网络等工程师及技术顾问。
森艾公司主经营IT维护、设备及硬件采购及IT培训的公司。本公司与HP、IBM、SUN、Oracle、CISCO、华为、Juniper、Radware、H3C、中兴等厂商有着长期友好的合作关系，涵括了服务器、网络、存储、系统软件等IT产品,渠道畅通。公司还一直面向社会及客户进行计算机知识、企业管理方面的培训工作，培训教室中安装了Unix，Windows多种网络环境，可对培训人员进行系统而详细的计算机应用知识培训。
