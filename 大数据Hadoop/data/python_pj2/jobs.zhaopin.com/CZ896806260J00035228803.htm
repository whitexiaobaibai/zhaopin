举报收藏

职位描述：
1. 根据设计稿，使用 HTML + CSS + JavaScript 制作网站页面；
2. 配合后台开发工程师，完成页面的交互功能、联调等工作；
3.负责前端团队管理；

任职要求：
1. 专科以上学历，IT软件、计算机，互联网等相关专业；
2. 3年以上的网页前端页面重构工作，熟练使用相关 网页设计制作软件，对交互设计有较深的理解和大量的实践；
3. 精通Javascript，对JavaScript OOP有深刻的认识，熟悉至少一种除 jQuery 外的 JavaScript 框架或库；
4. 精通W3C标准及规范，熟悉应用XHTML+CSS、DIV+CSS构架页面，高效熟练HTML及CSS代码，确保代码对IE、Firefox、Safari、Opera等浏览器的良好兼容性；
5. 精通HTML5和CSS3，有过移动端的开发经验，熟悉响应式页面制作；
6. 善于沟通，具有良好的团队合作精神和高度的责任感；
7. 了解php、ASP.NET等语言及拥有平面设计经验者优先.
