举报收藏

岗位职责：

1.  推荐系统相关的算法研究、应用和优化；

2.  自然语言处理相关工作；

3.  应用机器学习算法模型解决业务场景下的分类、回归、预测等问题。

岗位要求：

1.  熟悉常用机器学习、数据挖掘算法，熟悉算法的适用场景，有海量数据处理经验；

2.  熟悉自然语言处理，包括信息抽取、文本相似计算等；

3.  精通java/scala/python开发语言，scala优先，熟悉linux开发环境；

4.  熟悉Hadoop及其组件，有Spark开发者优先。

同源数智(北京)科技有限公司 该公司其他职位

同源数智公司是一家国内领先的大数据技术咨询公司，专注于提供数据技术和模型应用解决方案。目前面向金融、互联网、电商、零售、汽车、交通、制造、医疗、服务等行业客户，主要提供深度商业场景数据模型构建和应用、商业数据挖掘与分析、金融产品量化设计、金融风险控制、BI系统搭建等服务。

公司拥有大量行业成功经验，凝聚行业最先进理念和技术，已与多家数据及互联网金融拥有合作关系。现承接金融及各行业的数据分析和处理项目。
