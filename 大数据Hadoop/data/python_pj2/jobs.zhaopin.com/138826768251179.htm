岗位描述： 

1. 负责公司大数据分析平台以及数据可视化的产品设计；

2. 用户画像挖掘策略的设定，以及产品商业化模式的探索； 

3. 负责跟诸多行业客户沟通，收集客户反馈，发掘业务用点，思考解决方案，不断改进产品； 

4. 跟踪大数据应用的市场动向，进行竞品分析，探索新的表现方式和方法 

岗位要求： 

1. 三年以上的产品经验，有产品架构思维，并能将设计执行落地； 

2. 具有丰富的需求分析、产品规划设计经验，善于交流沟通，能够敏锐挖掘到客户的痛点，做过2B产品； 

3. 执行力强，能做项目管理，勇于挑战和学习新知识； 

4. 具有较强的文档编写能力和汇报表达能力。

北京世纪高通科技有限公司 该公司其他职位

    北京世纪高通科技有限公司是一家成立于2005年高科技公司，隶属于中国航天科技集团，其控股母公司为北京四维图新科技股份有限公司（A股上市公司，股票代码：002405），注册资本4600万。作为国内首家动态交通信息服务提供商及北京市高新技术企业、软件企业，世纪高通始终坚持自主创新，已经拥有自主知识产权的交通信息RTIC标准，并保持动态交通信息采集、处理、优化、发布等方面的技术优势。凭借对中国道路交通状况及用户需求的深刻理解，不断完善运营保障体系，为公众提供最高品质、最佳用户体验的出行信息和服务。
    世纪高通具有覆盖全国、高可靠的服务运营体系，实现7×24不间断服务和全系统的监控，成功为2008年北京奥运会、2010年上海世博会提供交通运营保障服务。2008年获得第六届中国能源战略高层论坛与中国节能协会共同颁发的“节能中国十大新技术应用奖”及“节能中国贡献奖”，被李克强副总理赞誉为“数字生态工程”。
    世纪高通拥有多项自主知识产权，是中国领先的出行信息服务提供商。作为中国最早实现交通信息服务商用化的企业，其产品服务自2008年正式商用以来，已经覆盖北上广深等全国30多个主要城市，服务于百万汽车用户和千万互联网及智能手机用户，拥有丰田、日产、本田、宝马、奔驰、大众、诺基亚、中国移动、中国电信等数十家商用客户及战略合作伙伴。
交通是永远不会枯竭的事业，相伴而生的交通信息服务必将拥有强大的生命力，在欧美日本等国，交通信息服务已经创造了数百亿美金的市值，而在国内则是一片广阔的蓝海领域，你即将加盟的公司拥有充足的市场空间，同时公司也将为你提供可持续发展的职业环境和机遇。公司的愿景：成为世界一流的交通信息服务企业！
   公司的团队：年轻而富有朝气，其中本科及以上学历的人员占80%以上，国内外一流院校的学生云集于此。作为知识型团队，公司注重员工的成长与发展，为员工提供良好的工作环境、富有市场竞争力的薪酬和完善的福利保障，积极倡导和建设开放型和学习型的企业文化。
   公司的理念：技术服务于市场，时刻以客户体验为念，提供给客户最高质量的交通信息类产品；创造并分享价值，追求产业链各环节的共赢；员工是公司最宝贵的资源，我们将所有员工视为共同的创业伙伴。
