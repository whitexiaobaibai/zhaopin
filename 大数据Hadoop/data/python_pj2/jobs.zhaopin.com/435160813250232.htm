举报收藏

岗位职责：

1. 负责CRM/大数据分析处理项目，组织项目的需求、开发、上线、验收等各环节，对项目的目标、人员、计划、进度、质量、风险等各要素进行管理

2. 项目前期与客户进行沟通，了解客户需求，制定项目解决方案；项目进行中与客户沟通，同步项目状态，维持客户关系；项目后期，组织客户培训，推进项目验收

3. 组织项目开发实施，对项目交付质量进行管理，协调解决项目过程中的重点难点问题，处理项目危机，撰写项目交付文档

4. 总结项目过程问题、产品问题，提出改进意见，优化项目过程和产品


岗位要求：

1. 本科以上学历，计算机/计算数学/软件/管理相关专业

2. 四年以上软件实施项目管理经验，有独立带领团队完成项目的经验和能力

3. 良好的沟通和表达能力，思路清晰，善于合作

4. 具有很强的主动性和自我驱动能力；善于学习新技术，新方法

5. 理解项目管理方法论，熟悉敏捷开发和敏捷项目管理方法，获得项目管理相关证书（PMP）优先

6. 有CRM,互联网及大数据分析处理项目经验者优先，有零售消费品行业相关经验者优先。
