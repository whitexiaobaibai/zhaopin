举报收藏

任职要求：

1、本科及以上学历，数学、统计学、计算机相关专业，5年以上相关经验；

2、深入了解分布式存储和计算系统的架构及运行原理，具备搭建基于Hadoop的高效稳定大数据平台的能力；

3、熟悉Java、Python、R等主流开发语言；

4、深入了解 Hadoop 集群部署及调优；熟悉分布式文件系统HDFS，有MR、Spartk、Storm等分布式编程模型的经验；熟悉主流的Hadoop生态数据采集框架，例如：Sqoop、Kafka、Flume等；精通Spark编程原理和运行结构，Spark生态的各个组件的功能，具有2年以上从事Spark项目的经验。

5、熟练掌握 Linux 基本操作和 Shell 编程；

6、熟悉主流的数据展现工具。

7、较强的自学能力、优秀的逻辑思维能力和良好的沟通表达能力和敬业精神。

8、具备良好的系统分析能力，良好的抽象思维和逻辑思维能力，独立分析问题解决问题的能力； 

9、可承受较大压力，有责任感，较强的沟通协调能力，具有团队合作精神；

10、有电力行业工作经历的优先。


岗位职责：

1、负责大数据基础平台、海量数据存储/处理分布式平台、数据分析系统架构设计和研发； 

2、负责实时计算平台基础架构设计、部署、监控、优化升级； 

3、制定项目/数据仓库设计及实现规范，指导设计研发和部署； 

4、协助策略和算法团队工作，保障数据挖掘建模和工程化； 

5、深入研究大数据相关技术和产品，跟进业界先进技术。

广东益泰达科技发展有限公司 该公司其他职位

广东益泰达科技发展有限公司成立于二零零三年十月，注册资金5001万元。以计算机系统集成、软件开发、电力设备、通信网络供应等为主营业务，集产品的开发、销售、安装服务及工程的设计施工为一体的综合性高科技企业。在长期为南方电网及五省区电网公司的系统集成、软件开发、技术咨询服务中积累了丰富的经验并与一大批国内知名院校、研究院建立良好的合作关系。

公司成立后继续秉承“技术是第一生产力、服务创造价值”的宗旨，积极投身与南方电网的智能电网的建设和服务中，充分利用IT信息技术、电力系统专业人才，积极吸收新技术、探索新课题，与中国电科院﹑南网科研院﹑广东设计院、武汉大学、华南理工、四川大学、华山、华为、联想、IBM﹑HP、南瑞等国内外知名高校、厂商建立了良好的合作关系,优势互补,为电网用户提供优良的产品及服务。

软件产品：电网调度工作评价系统、电网线损理论计算与分析系统、电网调度操作风险评估系统、调度任务管控系统、小水电管理系统、电网运行风险管控系统、EBOS基础开发平台、智能电网调度运行管理操作与指挥系统、安全辅助支持系统、企业绩效考评管理系统、配网通信网络管理系统、易报表应用平台。

公司通过了ISO9001—2000质量管理体系认证，为了更好地为广大客户服务，特别成立了客户管理部、技术服务部及售后服务中心，建立一整套完善的客户服务流程。公司现有制造生产员工100多人，电力及计算机技术员工六十多人，本科以上学历的占90%，拥有高级工程师、高级程序员、硕士等高素质人材十多人，有批获得各种专业资格证书的工程技术人员在技术咨询方案设计、项目管理软件开发等方面具有雄厚的技术实力和丰富的工程经验。

公司十分重视软件开发部、系统集成部的建设，软件开发部现有专业技术人员将近50人，其中包括硕士2人，系统分析员3人、高级程序员15人，程序员27人，公司将继续引进一批高级技术人材。软件开发部下设开发组、测试组、软件实施组。公司具有实力雄厚的研究开发能力。

系统集成部（包括项目管理部）现有专业技术人员12人，其中包括硕士/高级工程师1人，网络工程师3人、安防工程师3人、技术工程师3人、项目经理2人。系统集成部下设硬件组、软件组、测试组。公司拥有技术全面、经验丰富的施工队伍，在大型网络集成项目和复杂集成项目中有很多成功的集成案例。
