举报收藏

岗位职责

1、 负责大数据平台的架构设计、搭建；. 参与下一代实时流式引擎数据平台的设计和开发；
2、 利用sql处理各种复杂的表关联关系；
3、 负责基于hadoop的数据ETL设计与开发；
4、 开发基于http的数据服务层以及数据可视化模块。
5、 理解业务场景、产品逻辑和分析需求，规划各种数据间的关联关系；
6、 运用Hadoop 或者其它数据分析技术，针对数据分析工作，进行包括需求分析、平台选择、技术构架设计等技术支持工作；

7、根据需求使用Spark Streaming和Spark SQL进行数据处理/查询/统计。实现基于Spark框架的分布式数据库的开发； 负责基于Spark技术的海量数据的处理、分析、统计、挖掘工作；
8、 参与公司大数据处理方向的技术拓展。

岗位要求：
1. 熟练使用和配置Hadoop、HBase、Hive、HCatalog、Flume、Impala等大数据开源软件；  熟悉大数据主流框架（如 Hadoop、hive、Spark等）相关技术；熟悉hadoop/spark生态，例如Hive、Hbase、Zookeeper、Sqoop、Yarn、kafka等；熟悉Spark内核原理以及相应生态组件的使用，至少有2年以上Spark开发经验；
2.  熟悉Scala，Python，Java语言，熟悉 MySQL、PostgreSQL、NoSQL（MongoDB、Redis）等主流数据库，以及rabbit MQ等队列技术，熟悉数据仓库逻辑架构，熟悉ETL流程、元数据管理、数据质量监控等数据仓库主要环节；至少3年的开发编程经验；有图计算开发经验者优先，熟悉neo4j使用者优先；
3.  熟悉Spark Streaming和Spark SQL，对Scala原理、底层技术有深入研究；
4.  熟悉kubernetes和docker容器相关技术, 实现spark与kubernetes(含docker容器)的融合研发，性能优化、问题诊断。有kubernetes开发经验者优先，对资源调度&分配策略有深入了解者优先 。

5. 熟悉数据挖掘的流程，相关机器学习算法的原理（分类、回归、协同推荐、聚类、关联分析等）， 有过海量数据系统开发经验者优先；
6.  较强的分析、动手解决技术问题能力。

岗位待遇：月薪20K到50K，五险一金，各种补贴，双休，年度期权和分红奖励；

本公司研发的OA、CRM、ERP等协同软件；微软旗下全系列产品，包括正版办公软件、各种系统软件、私有云软件、企业管理软件、安全防护软件。
