举报收藏

岗位职责：

1、参与大数据应用的技术方案设计、平台搭建等；

2、负责大数据应用平台的详细设计和开发工作；

3、负责大数据应用程序模块的详细设计和开发工作。


任职要求：

1、公立大学本科及以上学历，计算机软件或相关专业毕业；

2、两 年 以 上 J2EE 应 用 开 发 经 验 ， 掌 握 Ajax 技 术 和 jQuery 框 架 等 ，熟 练 使 用 Java/js/jspml/xml/css，对软件开发和测试流程有一定的理解；

3、有在 Linux/Unix 下工作的经验；

4、熟悉 Linux 服务器的编程和调试；

5、熟悉 cloudstack、openstack、Hadoop、hive、hbase、Cassandra、Redis、Zookeeper、Hypertable 等分布式开源项目及工作原理；

6、有分布式、大数据处理相关项目经验，如分布式存储 HDFS、NoSQL，并行运算 Hadoop， 有一定的表结构设计和关系数据库 SQL 优化经验优先；

7、有强烈上进心，积极进取，学习适应能力强，态度乐观，团队意识强；

8、有一定的英文读写水平。

北京神州泰岳信息工程技术有限公司 该公司其他职位

Ultrasoft成立于2003年，注册资本1080万元，座落于北京市中关村科技园区，是北京市高新技术企业、北京市双软认定企业，公司还获得了ISO 9001质量管理体系认证证书。
作为IT服务提供商，Ultrasoft主要面向大中型企业及企业集团，提供全方位信息化建设解决方案及IT服务。公司以“IT服务专家”为企业理想，致力于以持续的应用创新，随需而变，精益求精的经营理念和IT的最新技术服务于企业客户，已经为多个大中型企业及集团提供了解决方案，如金融行业、环境资源行业、能源行业、电信行业、政府等。目前，公司基于J2EE、.NET、SAP和开源技术已经构建了知识管理系统、项目管理系统、风险管理系统、地理信息系统、移动应用业务平台、遥感数据综合显示处理平台等软件产品及解决方案，并为客户提供了卓有成效的IT技术支持与服务。
不断领先的技术水平和强大的服务能力一直被视为公司的生命线，公司为此进行着持续不断的投入。依赖于本地化的需求理解、强大的研发能力、专业的实施服务以及持续的技术支持等全方位的综合实力，通过孜孜以求的辛勤耕耘，公司的技术、产品及服务，取得了众多用户的高度评价。

神州泰岳以工程师文化作为企业文化的核心，把工程师队伍的建设、成长和发展作为公司的发展的源动力，尊重每一位工程师的劳动和个人选择，积极为工程师提供学习成长的空间，也为每位工程师提供优厚的福利和报酬。
