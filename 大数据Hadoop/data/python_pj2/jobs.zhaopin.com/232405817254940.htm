举报收藏

岗位职责：

GIS产品研发。具体内容：

1、负责GIS产品研发的全流程管理；

2、负责GIS产品技术选型，架构设计、功能设计、编码设计以及核心代码的编写工作，GIS新技术研究等；

3、负责基于Hadoop混搭数据库的城市级大数据的GIS应用展现开发，基于城市大数据进行地图+展现，实现大数据+地图的展现效果，和产品经理一起策划设计展现方案并实现；

4、基于大数据的GIS空间分析实现，突破传统GIS思维，实现城市级GIS+BI的展现，把GIS展现构建在Hadoop，HBase大数据库之上，实现GIS分析最大化价值；

任职要求：

1、本科以上，计算机、地理信息、城市管理等专业，开发经验5年以上，有全程参与大型项目经验；

2、精通java开发， 熟悉ARCGIS server开发，熟悉Arcgis Server JS开发优先考虑；

3、熟悉GeoServer，Openlayer等开源组件，至少熟悉天地图，百度地图，高德地图等开发框架之一，了解大数据技术；

4、熟悉OOAD设计分析方法，具有研发思维，具有GIS+大数据展现的新思维，新思路，愿意探索新技术和新工具；

5、具有独立的大型系统的架构能力和复杂系统的核心技术把握能力；

6、诚实正直、学习能力强，沟通协调能力强，承压能力强，与周边团队协作能力强，勇于挑战， 心态积极。

                             十八年自主研发成就大数据领导者
    东方国信，一家专注于大数据软件平台和应用解决方案的软件供应商。
成立于1997年，凭借大数据的专业技术不断开拓客户，成为电信行业大数据市场占有率第一的国内公司，并成功在2011年1月上市，成为国内第一支大数据为题材的上市公司（股票代码：300166，公司简称：东方国信）。
    上市以来，公司密切跟踪和研究国际大数据领域的技术发展趋势，在公司现有技术架构的基础上，加快核心工具平台软件的研发；积极完善现有的营销和服务体系，加大对金融、能源、电力、交通、零售等领域的市场拓展力度和产品支持力度，东方国信已经发展成为国内最具影响力的大数据软件系统提供商。
    未来，首先东方国信继续坚持在大数据领域的自主研发，掌握大数据领域的核心技术，为各个行业提供大数据端到端的完整解决方案，加强并巩固东方国信在国内大数据领域的领先地位，把东方国信打造成为大数据领域的优秀专业公司。其次，积极探索基于大数据运营的新商业模式，依托大数据的核心技术优势和对行业客户的深刻理解，东方国信将会加强在移动互联网流量运营方面投入的同时，探讨在互联网金融和智慧城市等领域的业务创新及业务合作。未来，东方国信将不仅仅提供大数据产品和解决方案，而且还要提供大数据咨询服务和运营服务，利用大数据创造更大的价值。第三，电信和金融将是东方国信未来发展的两个重点行业，积极的并购策略和有效的技术、市场整合将持续加强东方国信在这两个行业中的领导者地位。
