举报收藏

岗位职责：

 1. 根据项目需求从事市场分析研究、精准营销、大数据行业应用等任一方向的管理咨询项目；

2. 通过数据统计与业务分析对大数据商用业务的进行分析、洞察、建议；
3. 独立进行行业研究、案例分析和标杆研究等；
4. 协助客户推进咨询方案的实施。


任职条件：
1、【学历】管理类和营销类等专业，重点院校本科及其以上学历，有PMP证书优先。
2、【经验】三年及以上电信或相关大数据应用工作经验，熟悉电信运营商内部流程机制或大数据产品业务的优先。
3、【技能】具备项目管理经验，具有良好的沟通协调能力以及团队管理能力；有扎实的word、Excel、PPT功底，可根据客户或领导要求统筹完成项目工作，能制定有效可行的工作措施或执行计划，输出高质量的文案报告。
4、【思维】有良好的逻辑思维及沟通能力，通过数据数据及访谈调研对运营商业务或行业应用的进行洞察并提出优化建议。
5、【素养】良好的团队合作精神，能独自承担运营优化工作，及指导项目成员完成；高效的团队与客户沟通，责任心、配合度强，可接受偶发性的加班。

工作地址：

广东省广州天河区体育西路107号天文苑盛雅商务中心B座2楼E室 查看职位地图

广州兆达讯息科技股份有限公司 该公司其他职位

     广州兆达讯息科技有限公司(股票代码：838444）是一家互联网转型综合服务商，是集优秀的技术平台、丰富的销售通路、灵活的商业模式、有效的运营服务于一体的高新技术企业。


     2007年成立以来，公司一直都致力于互联网行业的创新和发展，作为最早投入移动互联网的企业，也一直处于行业领先的地位。

     2009年公司为电信运营商开发了第一款移动客户端，此APP成为全国首个官方掌上营业厅，创下单日活跃用户50万，月度活跃用户350万，安装用户过千万等骄人成绩；同时作为首批开发者与安卓、安智、应用汇等超过20个电子市场建立长期推广合作，此后更自主研发了覆盖iphone、ipad、ophone、android、java等多系统的客户端； 

    2011年公司在数据挖掘方面也得到长足发展，已成功投得号百客户经营（第一期）客户数据建设项目、号百业务潜在用户分析项目、广东电信短信营业厅客户端建设项目、广东电信流量包数据分析项目、电子渠道分流项目、宽带用户存量经营项目等，发展势头迅猛。

    2013年开始整合大数据能力、网络营销能力、互联网传播以及移动互联网推广能力，创下通过微信营销单日增加活跃关注粉丝15万的成绩，为嘉顿、威露士、幸福药业等多家传统行业打通移动互联网销售渠道。

     踏入2015年，公司将以积极的姿态面对4G时代带来的机遇与挑战。在未来的发展规划里，公司将重点聚焦在运营商、铁路和康复医疗这三大客户群。公司的新使命是：为传统行业转型提供最全面的解决方案。凭借丰富经验以及不断的进取心，以及全面的互联网行业产业链条运作经验，公司将以数据挖掘为基础、以移动互联网的应用开发为目标，致力为传统行业提供大数据的运营支撑、协助解决电商化网络营销问题、为向移动互联网化转型企业提供技术支持、同时提供互联网化传播解决方案。

    2016年公司公司将于下半年上市，随着公司的迅猛发展，现竭诚邀请有志青年加盟，公司将为您提供优厚的薪资福利待遇，为您创造更广阔的发展空间，期待与您共创辉煌。

     公司的飞速发展得益于一支年轻、团结、高效的合作团队。公司选址在天河体育中心核心商圈，交通及生活便利，同时高度重视优秀人才的引进和培养，不仅重金广纳良将，唯才是用，更聘请专业的企业教练进行团队管理。公司在汲取了电信行业先进管理经验的基础上，秉承着与员工共同成长的理念，一方面以科学的人才评估和升迁体系为人才创造良好的发展空间，另一方面建立高效率的组织管理机制充分发挥人才的能力，进而形成学习氛围浓厚、思维开阔、良性竞争、积极向上的具有团队精神的企业文化，使公司成为一个协调一致、高效运作的有机整体。
