举报收藏

岗位职责：

1. 负责数据仓库和大数据处理模块的开发工作；

2. 负责基于Spark技术的海量数据的处理、分析、统计、挖掘工作；

3. 基于Spark框架的数据仓库的开发，维护；

4. 根据需求分析模型使用Spark Mllib、Mahout等进行数据挖掘工作。


任职要求：

1、计算机、软件工程等相关专业本科及以上学历；

2、能够熟练地使用Spark MLlib，Mahout等机器学习库,了解spark SQL，spark streaming等相关Spark 组件，至少有2年的Spark开发经验；

3、有基于Hadoop、MongoDB等分布式数据处理经验优先；

4、了解相关数据挖掘算法及应用场景，有应用统计建模、数据挖掘、机器学习方法建立数据模型解决实际问题的项目经验优先；

5、做事严谨，主动、积极，能承担高压力，有强烈的事业心及开拓进取心，良好的人格品质与职业操守。
