举报收藏

基本工作目标：

1、确保公司系统研发方向与公司业务发展和规划保持一致。

2、确保公司技术路线统一、规范。

3、确保项目的技术选型、技术架构设计的合理性。

4、确保技术架构理念传导到系统设计人员与开发人员。

5、协调技术和业务需求方之间的沟通，并能解决技术和需求之间存在的一些矛盾问题。

 

岗位职责：

1、负责理解和管理非功能性系统需求，例如系统的维护、性能、重用、健壮、准确和测试等。

2、负责组织技术研究和攻关工作，对一些技术难题提出一些较为完善的解决方案。

3、协助项目经理制定项目计划和控制项目进度。

4、根据产品所提出的的需求，对开发团队所提出的设计方案进行技术把关。

5、负责对整个软件架构、关键构件、接口的设计和把关。

 

要求：

1、具备 5 年以上软件行业工作经验；具备电商系统软件开发或系统架构者优先考虑；

2、具备 4 年以上 B/S 体系结构软件产品开发及架构和设计经验；

3、具备 4 年以上的代码编写工作经验；

4、具备丰富的大中型开发项目的总体规划、方案设计及技术队伍管理经验；

5、对相关的技术标准有深刻的认识，对软件工程标准规范有良好的把握；

6、对 .Net 技术架构及整个解决方案有深刻的理解及熟练的应用，并且精通；

7、具有面向对象分析、设计、开发能力（OOA\OOD\OOP），精通系统设计工具进行设计开发；

8、精通大型数据库如 Oracle、Sql Server 等的开发经验，熟悉数据库的运作原理，有调优能力者优先；

9、对计算机系统、网络和安全、应用系统架构等有全面和深入的认识，大型项目管理经验；

10、对系统和网络安全方面有深刻的认识和处理经验，能提出较为系统的解决方案；

11、对大数据处理和挖掘方面，有2年以上的经验，能提出较为成熟的处理方案；

12、良好的团队意识和协作精神，有较强的内外沟通能力。


工作地址：

北京市丰台科技园总部基地17区17号楼

翼霖实业集团有限公司 该公司其他职位

    CNCBK国际控股集团成立于2015年，注册资金1个亿，总部设立于中国北京丰台总部基地，集团总资产近10亿人民币，有员工500余人。
    集团旗下拥有多家子公司和分公司，包括北京云联意网络科技有限公司，北京共享科技网络科技有限公司，广州百乐团网络科技有限公司，北京炫酷激情俱乐部有限公司，北京安信易货网络科技有限公司，北京国润环球酒店管理有限公司和近百家代理商公司。
    集团通过对全球优质资源的整合，合理规划，借助领先的创新理念和技术优势，构建了多元化产业发展的生态圈，从而实现经营的集约化，专业化，网络化和品牌化。
    北京云联意网络科技发展有限公司是集团旗下的子公司，公司所运营的CNCBK消费保障网新型互联网创新平台，具有O2O的商务特性，更兼具有F2F的创新性，不紧彻底颠覆了传统的消费观念，而且赋予消费者相应的价值。
    CNCBK消费保障网是家新兴的电子商务平台及生态平台，依托互联网+应用而生，实体经济+互联网共享经济。模式将于创新模式及自身丰富的经验，以及敏锐的市场洞察力于行业凝聚力提供一个共享交流的平台。公司集团的各个板块紧密结合形成一个强大的生态圈，打造出一个独立的闭环。
    CNCBK电子商务平台，以电商和O2O为切入点，基于数据延伸到更多的领域，炫酷出行，旅游酒店，海外地产，大健康，共享出行，农业庄园，债务化解，新媒体，新零售，百乐团购，人工智能，社交，泛娱乐，物流形成全球性的大生态圈。
   CNCBK的电子商务生态平台满足了生活的衣食住行和吃喝玩乐，基于数据的延伸覆盖了所有的服务品类，让消费者爱上消费，喜欢消费，敢于消费。
