举报收藏

岗位职责：

1.负责公司项目ERP的维护与开发工作；

2.负责公司项目ERP和各系统的对接与管理；

3.负责公司ERP系统的功能修正；

4、负责公司O2O商城的开发与维护工作；

5.完成上级领导交办的其他工作


任职要求：

1.NET/C#编程，B/S应用架构，VS2010 以上，1年以上WEB项目开发经验，并具有扎实的编程功底，大、中型电商平台核心代码、主功能模块开发经验； 

2.精通应用MSSQL2008 以上，至少对存储过程、触发器、事务管理有深入了解，擅长性能优化，有实际大数据量数据库的设计经验及性能优化经验； 

3.精通ASP.NET WEB API MVC，Newtonsoft.Json，API Cloud； 

4.熟练使用DIV+CSS布局、JavaScript、JQuery、AJAX、JSON、HTML5、CSS3、有一定的UI框架布局能力；

9:00-18:00，大小周工作制

钱大妈社区生鲜（连锁）品牌全面颠覆了传统农产品流通与销售模式，店面每天19：00开始打折，每隔半个小时再打低一折，直至免费派送。

健康生活，从新鲜餐桌开始-----钱大妈，不卖隔夜肉！

（2）门店规模：门店有600余家，其中在营业的有500多家，100多家在装修待开业；

（3）2017年10月9日公司启动o2o项目，钱大妈即将启动线上商城，线上线下联动；为广大市民带去更便捷的购物体验；
