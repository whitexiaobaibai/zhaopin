举报收藏

岗位职责：

1.负责设计系统整体架构，参与系统架构的技术研究、开发，数据平台、工具平台的架构、设计以及实现； 

2.核心架构原型的实现及验证，对产品的系统安全性设计负责，开发及相关设计文档编写； 

3.基于产品数据，深度分析数据中的规律，挖掘数据价值；

4.参与数据、工具平台相关的功能接口、数据接口，实现业务功能。

任职要求：

1.具有5年以上Java/Java EE方向研发经验，精通面向对象的分析设计技术和方法，包括设计模式、UML建模等，熟练使用PowerDesigner等系统分析设计工具；

2.熟悉Java并发编程，有优良的Trouble Shooting能力，能解决各种性能瓶颈（比如IO、CPU、Memory等）；

3. 拥有丰富的数据库设计经验、精通SQL语言，并精通MS SQL数据库；熟悉数据生命周期设计，针对“热数据”、“冷数据”缓存设计拥有丰富的经验；

4. 有海量数据系统开发工作经验者优先，对分布式原理理解深刻者优先。

薪资范围：5000-12000元
