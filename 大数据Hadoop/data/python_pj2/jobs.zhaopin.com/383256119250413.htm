举报收藏

工作描述：    

1.支持海量实时数据流的清洗、用户行为分析、建模、生产；

2.支持数字电视/智能电视广告等海量数据的整理与建模，针对用户的收视特征将海量数据产品化；

3.支持各类产品和项目的数据统计与分析工作，并能结合分析结果对产品和模型提出优化建议。


综合能力：较好的数据敏感性和逻辑思维能力，善于沟通，自学能力强，对工作富有激情，对机器学习、数据挖掘、文本挖掘等领域有浓厚兴趣。

 

专业能力： 统招本科及以上学历；

熟练运用EXCEL，PPT等办公软件，熟悉Python，R等语言；

有1年以上的大数据处理分析相关工作经验；

具备一定数据分析和编程基础，参与过互联网大数据项目；

熟悉数据挖掘项目的流程和思路，熟悉常用算法和模型；

有互联网/研究机构数据岗位实习经验的应届本科生也可

尼尔森网联媒介数据服务有限公司（Nielsen-CCData）是尼尔森集团在中国从事媒介研究的专业公司，专注于数字环境下媒介和受众研究领域。与“尼尔森市场研究”共同构成了尼尔森在中国的业务版图。

 

我们以数字媒体测量技术为基础，采用全球顶尖的研究和分析方法，致力于为全媒体产业链各方提供权威、优质的解决方案与服务，业务涵盖跨屏收视行为研究、广告投放监测与媒体价值评估等多个领域。

 

尼尔森网联媒介数据服务有限公司，作为媒体研究领域的大数据公司，始终致力于针对全媒体产业链各领域的专业性研究，先后申请取得了国内外专利发明超过50余项，并发布多部行业研究、发展著作。




Nielsen-CCData (NCC) Media Data Services Co., Ltd is a joint venture, the only professional provider of DTV data in China, built by Nielsen Co. Ltd. Nielsen-CCData, adopting the world-leading technology & methods of research,focuses on data monitoring & processing in DTV users’ behaviour. It provides data supports and consulting services for DTV net operators, TV stations, and advertising agencies.
