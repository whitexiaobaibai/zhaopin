举报收藏

ETL 

1.精通数据库，至少有两年Greenplum、Oracle相关的开发经验，能够熟练地编写存储过程，函数，有一定的数据库查询优化基础 

2.精通ETL，至少有一年的DataStage的开发经验 

3.对数据仓库有较深入的了解，并至少有一年的工作经验 

4.熟悉类Linux操作系统，熟识linux常用命令，能够独立编写shell脚本 

5.了解BI相关知识，至少一年以上BI相关开发经验



大数据开发 

1、计算机、数据、统计学相关专业本科以上学历； 

2、2年以上大数据相关工作经验； 

3、熟悉Linux、Unix环境，至少精通Java、Scala一种开发语言 

4、Hadoop/Spark集群数据ETL处理开发经验，有CDH集群开发经验优先； 

5、具备深厚的分布式系统或数据库系统的理论基础，熟悉分布式计算系统的工作机制及调优； 

6、对大数据基础架构和平台有深刻理解，对基于Hadoop的大数据体系有深入认识，具备相关产品（Hadoop、Hive、Impala、HBase、Sqoop、Spark等）项目应用研发经验；



H5 

1.本科及以上学历，3年及以上 Web 前端实际开发经验，有大数据可视化开发经验优选； 

2.熟练掌握前端基础技术，包括 HTML、CSS、JavaScript 等； 

3.熟练使用至少一种流行 JS 框架，如 AngularJs、ReactJs、VueJs 等； 

4.熟练使用至少一种前端工程自动化工具，如 Grunt、Gulp、Webpack 等； 

5.参与过项目的架构选型/设计/优化工作，有项目重要模块负责人的经历优先考虑； 

6.熟练使用 ES6 或 Typescript 优先考虑； 



Java 

1.计算机或者相关专业，3年以上的Java开发应用经验； 

2.熟悉Windows、Linux操作系统； 

3.熟练使用Spring,Spring MVC,MyBits,Tomcat,MySQL,Redis等技术进行开发； 

4.熟练应用HTML、JS、jQuery、CSS等Web前端技术； 

5.熟练使用Eclipse、SVN、Maven等； 

6.熟练使用关系数据库（Oracle/MySQL等）开发，会写存储过程，触发器等； 

7.对设计模式有一定理解，并能够实际应用； 

8.具有良好的编码和自测习惯； 

9.熟练使用ECharts、Datatables、AngularJS等前端技术者优先; 

10、有跨平台、跨系统数据交互可视化系统经验更佳
