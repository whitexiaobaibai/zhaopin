举报收藏

岗位职责

1. 后端系统的功能设计、开发和优化；
2. 了解产品需求和业务更迭，能针对各项业务做出准确的技术决策；
3. 研究和试验新技术，并应用到产品中；

职位要求

1. 有责任心，能为自己的工作和成果负责；
2. 有上进心，学习能力以及沟通能力强，能挑战自我不断追求卓越；
3. 基础扎实，熟悉常用数据结构和算法，能够独立设计系统中功能的架构；
4. 注重工作方法，追求工作效率；
5. 了解常用重构方法，了解设计模式原则；
6. 熟练掌握 Git 版本控制工具，了解 git flow 以及 github flow；
7. 对 MongoDB / MySQL / Redis 有实操经验，了解 DB 设计和优化；
8. 熟悉 TCP、UDP、HTTP 协议以及 Restful API，熟悉前后端的常见知识要点；
9. 有 2 年以上的后端开发经验；
10. 精通 JavaScript & Node.js；
11. 熟悉各种 Node.js 常用模块；
12. 熟悉单元测试，了解常用测试框架 mocha 以及 ava；

加分项
1. 在博客或 Github上 有技术沉淀；
2. 熟悉 Docker 以及集群调度工具，有实际经验；
3. 熟悉 ansible，了解常用模块；
4. 熟悉 TypeScript、Golang 或者 Python；
5. 熟悉 Angular / React / Vue；

你将得到
技术环境
1. 体量：1 亿用户以及千万级日活并发场景下的业务架构挑战；
2. 技术：最新的技术应用，大数据、k8s、ELK、React、Vue 等；
3. 开发：Node.js 全栈以及 DevOps；

公司环境
1. 福利：免费水果零食饮料；节日礼物；每年设有年度长途旅行、不定期 outing；不定期团建
2. 薪酬：行业高薪、绩效奖金、年终奖金等；
3. 办公：开放式办公，MBPR 标配；
4. 自由：带薪假期，灵活调休；

公司简介


我司是移动互联网领域的新锐成员，自主研发及运营各类移动互联网产品，旨在向用户提供最优质的内容及服务，丰富用户的网上娱乐生活。


公司目前运营三款原创移动应用产品，分别是：漫画岛、开卷有益、追书神器


l  漫画岛：你想看的漫画，这里都有


这是一款专为二次元漫迷打造的手机漫画阅读神器，汇聚了日韩、欧美、港台的上万部题材各异的漫画。开放在线、离线双模式阅读，高清、低内存的特点为用户订制别具一格的阅读体验。


l  开卷有益：让每个人爱上阅读


开卷有益是横跨多个平台的小说、漫画PDF阅读软件，2008年上线至今，用户总数已达到4000万，是业内知名的手机阅读器。数万册精品正版书籍，为用户源源不断地提供优质内容。独特清新的界面风格，让用户随心享受阅读的乐趣。


l  追书神器：在小说的世界乘风破浪


追书神器是一款最强的小说导读助手，集合了玄幻、修仙、都市、言情、穿越等超过十种题材的网络小说，极大限度满足网友的阅读需求。更新迅速、书单超全，一键式简单阅读，独有的数据测评，优化你的网文世界！
