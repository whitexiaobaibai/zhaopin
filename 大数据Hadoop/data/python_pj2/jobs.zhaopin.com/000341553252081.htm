举报收藏

岗位职责：


1. 负责应用软件开发，根据需求，可独立负责软件的设计、编码和调试；

2. 负责产品生产过程中问题的分析、跟踪和解决，根据工作需要或安排，对已发布的软件，进行二次开发与维护，可对bug进行分析跟踪和解决；

3. 负责软件测试方案的制定与实施；

4. 与团队技术人员合作，共同完成产品设计，确保软件质量；

5. 编写规范的技术支持和维护文档，为驻点售服工程师提供培训和支持；

6. 完成领导安排的其他任务。

任职要求：

教育背景：本科及以上学历，计算机或相关信息专业；

工作经验：有2年以上工作经验。

知识技能：

1、本科或以上学历,计算机或相关信息专业，英语四级以上；

2、精通面向对象的设计和开发经验，熟悉常用的设计模式；
  3、熟练使用Web相关技术Ajax,HTML5,CSS3,JQuery,   Webservice, XML，JSON等技术；
  4、精通Java， 两年以上SSH、Sprigmvc，Mybatis等多线程项目开发经验，担任过软件项目的需求分析、系统设计和数据库设计等工作；
  5、熟练掌握数据库设计、开发、调优经验，精通SQL、PL/SQL编程，有海量数据处理经验； (例如Oracle, MySQL, SQLServer等) ；

6、熟悉WMS、MES、Oralce ERP（或SAP）相关应用及业务流程；

7、熟悉大数据平台。有电商后台、微信开发等经验，J2EE前端DIV+CSS开发经验者优先。

认真严谨、责任心强、学习能力、动手能力强，沟通素质佳，具良好的团队合作意识，具较强的应变能力，敬业。
