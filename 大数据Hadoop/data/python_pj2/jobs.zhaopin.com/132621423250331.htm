举报收藏

职位描述：

1. 熟悉大规模数据挖掘、机器学习、自然语言处理、分布式计算、统计学、数据挖掘的算法和技术，并具备两年以上相关实际工作经验。

2. 对数据结构和算法设计有较为深刻的理解，有一定编程能力（Python/Java）。

3. 优秀的分析和解决问题的能力，对挑战性问题充满激情，有较强的学习能力，并且能够快速实现数学模型。

4. 良好的团队合作精神，较强的沟通能力。



任职要求：

1.计算机、数学等相关专业本科以上学历

2.熟悉数据挖掘、机器学习、推荐系统的理论（包括但不限于：聚类、回归、决策树、PCA、LSA、LDA、DNN、协同过滤等

4 熟悉图像识别 图计算 语音识别基础理论，熟悉 TensorFlow，Caffe，Deeplearning4j 等深度学习框架

3.扎实的数据结构和算法基础，熟练 Java/Python/scala/R等开发语言之一及其他常用脚本语言。

4.深入理解逻辑回归、决策树、协同过滤、贝叶斯等等常用数据挖掘算法。

5.了解并熟悉使用Hadoop、Spark mllib等相关大数据开源技术。

6.具备优秀的逻辑思维能⼒，对解决挑战性问题充满热情，善于解决问题和分析问题；乐于沟通和技术分享，具有团队合作精神。

北京金谷高科技术股份有限公司 该公司其他职位

    北京金谷高科技术有限公司（以下简称金谷高科），1999年创建，位于北京市海淀区上地信息产业基地，毗邻中关村高科技园区，是国家级高新技术企业。十数年来，金谷高科在农业信息咨询服务、大宗农产品交易研究、涉农数据库建设、期现货投资分析软件开发、电子商务平台搭建、智慧农业和安全溯源交易管理系统研发及应用等领域进行了深入探索和业务拓展，已经成为推动中国农业信息化进程的一支重要力量。
    旗下中国粮油商务网（www.fao.com.cn）是一个为粮油、饲料、食品等涉农行业提供专业资讯服务的大型网络平台，也是国内首批专业提供农产品信息服务的网站之一。如今，中国粮油商务网已经发展成为全国规模最大的农产品信息服务提供商，凭借其专业和优质的服务，赢得了客户的信赖，领跑中国农产品信息服务业。客户群体全面覆盖涉农企事业单位，其中包括农业部、发改委等国家部委；中国土畜进出口商会、中国大豆产业协会等行业协会；中储粮、益海嘉里集团、蒙牛集团、伊利集团和顶新国际集团（康师傅）等粮油食品企业；希望集团、新希望集团、东方希望集团和正大集团等饲料企业；大连商品交易所、郑州商品交易所等期货、证券类企业和经纪公司；中粮集团和中国国投等国际进出口企业。
    作为专业农产品信息服务机构，中国粮油商务网已与全球领先的媒体通讯社—英国路透社 (Reuters)、美国道琼斯社（Dow Jones Newswires）、彭博社（Bloomberg）、德国《油世界》杂志社（Oil World）建立了长期而稳定的合作关系。中国粮油商务网致力于为粮油、饲料、食品等农产品行业提供专业资讯服务，为获取最新国际资讯，长期与美国芝加哥商业交易所集团（CME）、美国大豆协会（ASA）、加拿大油菜籽协会（CCC）、马来西亚棕榈油总署（MPOB）、马来西亚棕榈油贸易促进会（MPOC）、马来西亚交易所（BMD）、印度Godrej国际公司等国际期现货交易企业和各主要农产品出口国在华使馆农业参赞互动、交流。
    经过十余年的积累，中国粮油商务网已经掌握了固定且权威的信息来源和市场信息分析方法，为各类农产品相关企业提供专业化、实用性强的高品质服务，并针对有特殊需求的企业提供纵向深度加工和特色信息服务。网站主要分为粮谷频道、油脂频道和饲料频道，品种涵盖大豆、玉米、小麦、稻米、棉花、菜籽、花生、芝麻、葵籽、食糖、豆油、棕榈油、菜籽油、棉籽油、花生油、玉米油、芝麻油、米糠油、葵花油、牛羊油、豆粕、杂粕、鱼粉、氨基酸、磷酸氢钙和养殖等40余种农产品及其深加工产品。
