举报收藏

岗位职责：
1、负责包含但不限于管理信息系统、决策报表等系统的业务分析，并写出相应的需求报告；
2、负责包含但不限于管理信息系统、决策报表等系统的系统设计,包括系统结构设计、数据库设计以及与其他业务系统的接口设计；
3、外包项目的生命周期管理包括需求分析、系统设计、代码规范制定、变更管理以及项目进度和预算的控制；
4、根据公司业务的需要进行系统开发、测试以及实施；
5、协同业务支持部门制定IT系统的发展策略；
6、协调并指导程序员工作。

任职要求：
1、全日制大学本科及以上学历，并获得学士及以上学位；
2、具备5年以上系统开发经验；
3、精通J2EE,熟悉Spring, Struts, Hibernate等主流开发框架；
4、有大型数据库设计、优化经验，熟悉Oracle, SQL Server等主流数据库；
5、熟悉hadoop、mapreduce、hbase、hive、spark等大数据技术原理并具备开发经验；
6、具备良好的责任和沟通能力；
7、有数据分析、数据挖掘、数据可视化、在线数据系统开发经验优先考虑；
8、有项目管理经验优先考虑。

工作地址：

北京市东城区建国门内大街26号新闻大厦20层

中华联合人寿保险股份有限公司 该公司其他职位

中华人寿，全称中华联合人寿保险股份有限公司，是经中国保监会批准（保监许可〔2015〕1128号），由中华联合保险控股股份有限公司和中华联合财产保险股份有限公司发起设立的人寿保险公司，公司注册资本金为15亿人民币。公司未来主要以创新渠道、互联网业务为主，将竭力为有保险需求的个人和家庭提供简单、便捷、贴心的保险保障和财富管理服务，并将通过创新和差异化经营，有效探索出一条有特色的、稳定健康发展的寿险公司发展新路。
