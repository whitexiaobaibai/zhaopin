举报收藏

岗位职责：

1.负责华南区银行及其他信贷客户（小贷、P2P、消费金融、互联网信贷）的大数据合作，为客户的大数据营销、风险管理及贷后管理等方面提供有效支撑；

2. 完成公司的销售目标，实现公司产品与解决方案与银行客户的合作共赢，推进大数据产品的有效接入应用，提升客户满意；

3. 通过市场活动进行品牌、产品和方案推介。

4. 其他基于大数据的银行客户解决方案的销售。


岗位要求：

1.大学本科及以上学历，3年以上工作经验；

2.责任心强；拥有良好的学习和沟通能力；拥有积极乐观的工作态度；拥有亲和力和协作精神；

3.具有一定的文字功底和较强的宣讲能力；熟练掌握office办公软件。

4.有服务金融行业客户，特别是银行及其他信贷客户经验的优先。


工作地址：

广州市越秀区解放南路123号金汇大厦首层

百融金融信息服务股份有限公司 该公司其他职位

百融（北京）金融信息服务股份有限公司（简称“百融金服”）成立于2014年3月，是一家专业提供大数据金融信息服务的公司。公司依托大数据技术及来自互联网、金融机构、线下零售、社交、媒体、航空、教育、运营商、品牌商等多维数据源，创新性地为信贷，保险，投资理财等行业企业提供获客引流、精准营销、客群分析、风控管理、反欺诈、贷前信审、贷后不良资产管理等服务，提升金融行业整体运营管理水平。

2014年3月，百融金服受邀成为石景山互联网金融中心首批入驻企业之一，也是区政府重点扶持企业，并在2014年12月成功取得企业征信牌照。截至目前，百融金服已经和建设银行、招商银行、光大银行、新华保险、中国人寿、太平洋保险、人人贷、兴业银行、上海大众等250余家金融机构达成合作协议，得到市场普遍认可。

百融金服在业界、学界影响深远，2014年起与清华大学五道口金融学院、人民大学重阳金融学院、北京大学新媒体营销研究中心、成都电子科技大学展开积极合作，并与多家企业结成数据产业战略联盟。

百融金服秉持公正、中立、客观的第三方立场，坚持社会良知和责任，坚持开放、互补的数据联盟战略，致力于运用新技术、新手段，为金融机构搭建营销、风控及贷后不良资产管理体系，立志成为国内金融领域最大的第三方大数据金融信息服务商，并以传播诚信文化理念，开拓诚信生态环境为己任，为全民信用体系建设及普惠金融的实现贡献一己之力。
