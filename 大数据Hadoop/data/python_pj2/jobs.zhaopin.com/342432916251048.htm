举报收藏

此职位隶属卓思数据集团旗下全资数据子公司,请知晓！

岗位职责：

该岗位主要从事汽车行业相关的大数据研究咨询服务，并协助完成研究成果的平台化开发：

1. 主要从事大数据角度的汽车个案研究尤其是消费者研究工作； 

2. 担任中小型项目的项目经理，在项目总监的带领下保证项目顺利推进并高效完成； 

3. 高度参与整个项目过程，从项目设计、进度管理、质量控制到数据分析和报告撰写，并能独立负责部分环节； 

4. 协助完成项目研究成果的网络平台开发；

5. 能够在项目总监陪同下参与计划书及报告的演示和讲解；

6. 参与公司流程/体系的创新与完善，以及新产品及服务的开发；

7. 关注行业动态，撰写行业分析文章。

任职要求：

1. 3年以上汽车行业市场研究咨询经验，有大数据研究经验者优先； 

2. 有外资汽车品牌服务经验者优先；

3. 有网页制作经验，熟练使用相关软件（如Axure）者优先； 

4. 本科及以上学历，出色的中、英文书面及口头表达能力；

5. 良好的沟通及跨部门协作能力，能高效地解决问题； 

6. 学习能力强，愿意接受新事物和新挑战； 

7. 思维敏捷，做事认真，对数据敏感；

8. 积极主动，自信，乐观，勇于承担责任，抗压能力强。

工作地址：

北京朝阳区万红西街2号燕东大厦D座1-2层

北京卓思天成数据咨询股份有限公司 该公司其他职位

    卓思（MaxInsight）成立于2009年，一直专注于汽车市场研究。经过七年的飞速发展，我们目前不仅在传统市场研究领域引领行业，同时在大数据和移动互联网调查上拥有核心优势，已成为业内最领先的市场研究和数据挖掘供应商，位列汽车市场研究领域No.1。2015年卓思销售额达到2.5亿元人民币。2016年卓思已将数据咨询业务拓展到酒店、旅行以及娱乐产业，并依然继续致力于使用我们卓越的技术来为更多客户提供优质咨询与大数据服务。


    同时，卓思已于2016年正式挂牌新三板，我们期待更多新鲜血液加入，共同开创更大发展。


    我们的客户涵盖几乎所有的国内外主流汽车品牌客户，如宝马中国、奔驰中国、一汽大众-奥迪、大众中国、一汽丰田、东风标致、日产中国（Infiniti）等，他们在持续使用卓思（MaxInsight）关于产品定位、定价、品牌策略、渠道服务质量监测及提升、经销商管理、数据挖掘等方面的服务与解决方案。凭借丰富的行业经验和专业知识，我们为客户提供更全面的市场研究服务及更为精准的支持，以帮助客户更好地实现其商业价值。
