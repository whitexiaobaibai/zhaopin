举报收藏


岗位职责：
1. 负责团队大规模机器学习算法的研发和应用；
2. 深入分析现有算法，了解业务需求，给出有效的优化解决方案； 
3. 对业界的分布式机器学习算法和应用有广泛了解并且能够跟踪最新进展； 
4. 制定有效项目进度管理机制，按照项目上线时间表，推进项目研发工作。

任职要求：
1. 拥有统计、计算机科学或相关学科的硕士或者博士学位，或在大数据领域有工作经验的硕士； 
2. 在机器学习、深度学习、大规模分布式机器学习等领域有2～3年研发经验； 
3. Coding能力较强，有C++语言编程经验的优先； 
4. 良好的沟通能力和团队合作精神，有一定的组织协调能力。

  我们致力于通过产品服务、数据技术、生态规则和资源为用户和客户赋能，建设“车媒体、车电商、车金融、车生活”4个圈、从“基于内容的垂直领域公司”转型升级为“基于数据技术的‘汽车’公司”。

  2016年7月中国汽车互联网用户通过PC端浏览汽车信息时，有45%的时间花费在汽车之家（根据iResearch数据）；2016年9月，每天通过移动端访问汽车之家的用户数量超过1800万（根据汽车之家2016年Q3财报）。
