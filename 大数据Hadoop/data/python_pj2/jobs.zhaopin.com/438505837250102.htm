举报收藏

岗位职责：

负责数据仓库模型架构设计和实施；

完成各种面向业务目标的数据分析模型的定义和应用开发；

逐步构建基于用户行为、喜好的标签系统，并应用于相关的个性化系统和业务分析中

任职要求

1、本科及以上学历，计算机、数学及相关专业，5年以上经验,互联网经验3年以上；

2、具有丰富的数据开发经验，对数据处理、数据建模、数据分析等有深刻认识和实战经验；

3、熟悉主流的大数据产品和数据分析技术并具有相关项目经验；

4、有大数据分析与数据仓库设计及开发经验；

5、熟悉主流的开源大数据产品，如spark, Hbase, HIVE 等；

6、熟悉数据仓库相关技术，有互联网数据仓库搭建经验更佳；

7、积极乐观、诚信、有责任心，具备强烈的进取心、求知欲及团队合作精神。

有以下2条件者加分：

有数据挖掘算法实施经验，熟练掌握大规模数据挖掘、机器学习，有实际开发经验者更佳；。

有大型数据项目经验优先，有用户行为分析、用户建模、业务建模经验者优先。

公司简介


 “苏州微浪科技股份有限公司”（简称:Vilang）2010年成立于苏州，在上海及杭州均已成立运营点，目前公司的规模已超过100人，其中90%以上为技术人员。我们吸收并改进跨国公司管理、运营、技术及产品的先进理念，拥有深厚的产品研发专业技能，并在ITSM/BSM领域和服务外包领域具有丰富的咨询、实施和管理经验，是一家以市场为主导、拥有核心研发能力的科技型公司。


Vilang拥有自主知识产权的专业IT运维门户产品NGDP(Next Generation Datacenter Portal)，以最佳实践ITIL为基础，在浙江移动、中国联通、招商银行、上海汽车、红星美凯龙、浙江广电新蓝网、河南地税、内蒙社保等客户做了成功实施。


公司自成立以来也一直为各大知名企业尤其是互联网企业提供优质的人才外包服务，如阿里巴巴集团，平安金融科技咨询、浙江移动及江苏移动等均为公司的长期合作伙伴。


  2016年11月9日，苏州股权交易中心敲钟开板，首批54家企业在苏州股权交易中心鸣锣挂牌，苏州微浪科技股份有限公司（企业代码：990066）作为优选层企业正式进入资本市场扬帆启航。


  苏州微浪科技股份有限公司致力于为客户提供互联网及软件技术开发和服务，以前瞻的视角、专业的技术深耕金融互联网科技。微浪股份将借助资本市场的力量扬帆远航，再创辉煌。

1、提供富有竞争力的薪酬，根据公司绩效管理机制，使员工绩效表现同薪酬回报紧密相连，；

3、保险保障：为员工提供完备的社会保险福利

5、教育培训：提供新员工入职培训、技术培训、第三方机构培训等。
6、职业发展：公司重视员工职业发展，为员工提供多方向的职业发展通道。
7、健康关怀：为员工提供年度健康体检。
8、员工关怀：公司定期举行丰富多彩的团队建设，旅游，健身等活动，丰富员工业余工作时间。
9、节日福利：每年逢重要节日和员工生日等重要日子，公司会为员工送上礼品卡或者礼品一份。
10、年度调薪：每年公司会综合市场同行业数据和公司情况，给符合条件的员工进行薪酬调整！
