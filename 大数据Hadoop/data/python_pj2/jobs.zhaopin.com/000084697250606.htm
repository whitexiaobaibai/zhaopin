举报收藏

企业名称：东莞深证通信息技术有限公司

企业性质：有限公司（国有独资）

企业信息：东莞深证通信息技术有限公司（以下简称：东莞深证通）成立于2013年5月28日，注册资本10亿元，是隶属于深圳证券交易所的深圳证券通信有限公司全资控股的有限责任公司，坐落于东莞市凤岗镇东深二路与凤翔路交汇处，距离深圳龙岗中心城约5公里。公司重点发展机房托管业务，面向深交所、证券公司、基金公司、商业银行、保险、期货等全国各类金融机构。

    东莞深证通正在建设的“中国证券期货业南方信息技术中心项目”（以下简称：南方中心项目），以“成为世界一流的高等级信息技术中心”为建设目标，建设规模约13万平米，总体投资近50亿元，被列为东莞市重大投资项目。南方中心项目作为证券行业核心基础设施之一，紧紧围绕深圳证券交易所的战略部署，立足于服务国家战略和证券市场发展大局，通过合理规划、清晰定位，将建设成为集六大功能于一体的大型数据中心园区，包括：深交所核心数据中心、行业核心机构南方备份中心、行业经营机构南方托管中心、行业云计算与大数据中心、行业研发测试中心、行业金融科技创新实验室。自2016年6月投产以来，南方中心项目获得中国证监会、行业核心机构以及各证券公司普遍认可，已成为行业核心基础设施，在行业内已具有很高的知名度。

    东莞深证通将为你提供优越的发展环境和一流的事业平台，提供有竞争力的薪酬待遇和多渠道的学习培训机会，诚邀责任心强和有能力专业技术人员加入。

 

任职要求：

1.  全日制985或211大学法学本科及以上学历，或政法类院校本科及以上学历;

2.  通过国家司法考试，具有深厚的法学理论根底和法律实务经验；

3.  2年以上法务相关工作经验，具有建筑行业法律经验者优先；

4.  具备良好的组织协调沟通能力，头脑灵活，逻辑清晰，掌握一定的谈判技巧；

5.  具有良好的职业操守和团队合作意识，工作积极主动，自信，责任心强。

岗位职责：

1.  负责公司所有合同、协议等法律文书与新产品、新业务、新流程的法律审查；

2.  日常合同文本及对外文件的审核、标准合同文本的制定；

3.  负责公司业务法律和合规风险管理；

4.  参与法律合规管理制度和流程体系优化；

5.  负责公司日常诉讼及非诉工作，对接外部律师事务所；

6.  完成领导交办的其他工作任务。


特别说明：本岗位为东莞公司岗位，工作地点为东莞市凤岗镇。

深圳证券通信有限公司 该公司其他职位

深圳证券通信有限公司（简称“深证通”）成立于1993年，是深圳证券交易所下属专业技术公司，国家级高新技术企业。经历多年的发展，已成长为中国资本市场最重要的信息处理中心与通信传输枢纽。公司总部位于深圳，下设香港子公司、东莞子公司、北京分公司和上海办事处，致力于服务境内外资本市场的核心机构与大金融行业客户。

 

战略定位：金融信息技术综合服务提供商
使命：致力于为金融机构提供更高效、更安全的信息技术综合服务
愿景：中国金融信息技术服务领域的领导者
价值观：成就客户；以人为本；合作共赢

 

公司为资本市场及金融企业客户提供多方位的IT服 务，包括交易结算技术服务、数据中心服务以及金融云与网络金融服务。其中，交易结算技术服务为面向深圳证券交易所、中国登记结算公司、全国中小企业股份转 让系统公司、中国证券金融股份有限公司等在内的资本市场核心机构，提供行情、交易、结算相关的各类信息系统的研发与运维；数据中心服务为依托建于深圳、北 京、东莞三地的高等级数据中心，面向证券公司、基金公司、期货公司等行业客户，提供主机托管、业务连续性外包、云存储等多种资源与业务外包服务；金融云和 网络金融服务为借助先进的云计算技术、创新的互联网业务模式，以IaaS、PaaS及SaaS方式，为机构客户提供行情云、资管云、泛金融云等各类云服务，以及以互联网、移动互联网方式为机构与个人投资者提供全方位的金融产品信息与交易综合服务。


深圳证券通信公司将和不断创新的中国证券市场共同成长，努力成为中国金融信息技术服务领域的领导者，建设和维护行业IT服务生态，引领技术潮流，创造客户价值，实现合作共赢。


公司为员工提供富有竞争力的薪酬，并为员工提供了晋升、培训机会及优厚的员工福利。公司坚持以人为本、广纳贤才的用人策略，努力为员工提供各种发展机会，实现企业与员工的共同发展和提高。
