举报收藏


职位描述：
1. 负责渠道反作弊业务策略、算法的设计和研发工作，深入理解渠道（各类广告平台）作弊特征，结合用户行为特征加以定量识别，判断渠道质量优劣，指导在线投放工作。
2. 通过海量用户行为数据的分析，选择合理的指标，设计和改进反作弊数据挖掘模型和算法。
3. 通过对数据的敏锐洞察、挖掘作弊特征并解决，开发反作弊工具和平台；
4. 跟踪反作弊技术的最新进展并应用于实践中






任职要求：
1. 计算机、信息类、数学或统计学相关专业本科以上学历
2. 熟练使用至少一门编程语言（Python、C++、Java等）
3. 有大数据处理分析经验，熟悉Hadoop、Hive、Spark等大数据处理平台
4. 熟悉常用的数据挖掘算法，并有相应实践经验
5. 对数据敏感、较强的逻辑分析能力、有强烈的好奇心和求知欲
6. 有反作弊、大规模轨迹数据挖掘、用户行为分析经验者优先
