举报收藏

岗位职责：

1. 基于Spark、hadoop等大数据平台的建设、开发、维护与优化；

2. 协助完成大数据项目需求分析、设计、业务建模；

3. 负责大数据实施过程中相关技术问题解决；

4. 完成与工作相关的技术文档编写工作。

任职要求：

1.本科及以上学历，3年以上编程经验，熟悉Spark、hadoop开发、熟悉大数据集群的搭建、管理及优化。

2.对数字，数据敏感，具备良好的逻辑思维能力，能够从海量数据中发现有价值的规律；

3.熟悉BI流程，参与过大型数据仓库类项目，精通SQL；

4.熟悉JAVA，熟悉linux，至少熟练使用Shell、Python等脚本语言之一；

5.熟悉当前主流MQ, NoSql ，如redis,mongodb，Hbase等；

6.个性乐观开朗，激情、愿意分享，自驱能力强，良好的结果导向和抗压能力；

7.有大型互联网、金融行业经验的优先考虑；

8.对业务有敏锐的洞察力，有较强的业务理解与分析能力。

核心工作内容：基于Spark、hadoop等大数据平台的建设、开发、维护与优化。

公司简介：
   团贷网集团，工商登记主体是派生集团，成立于2011年，因团贷网是派生集团旗下主打品牌，所以暂时也称之为团贷网集团。
经过6年的耕耘和发展，团贷网集团已经成长为一家综合性的金融科技服务集团，主要包括互联网端和资产端两大板块。其中，互联网端（tdw.cn）是一个综合流量入口，为旗下网贷投资、商业保理业务，以及私募基金、保险代理、网络小贷等已持牌业务提供展示和引流等服务。截至2017年6月20日，互联网端已经沉淀了575万多注册投资用户，累计促成交易额超过833亿元。
被外界所熟知的团贷网网贷投资平台（tuandai.com），是团贷网集团旗下众多业务板块之一。该板块于2012年上线，先后成功入选中国互联网金融协会首批会员单位、广东省“互联网+金融”试点项目等，并于今年3月份上线了银行存管系统。
另外，资产端则通过股权投资、内部孵化、战略合作等方式全国布局，基本涵盖了小微企业、供应链金融、小额信贷、消费金融、仓储物流、三农金融等领域，并与团贷网网贷投资平台形成深度合作关系，对应到网贷投资业务的9条业务线。
近日，派生集团获得了新一轮18亿元融资。融资完成后，集团估值超百亿。本次融资除了为集团相关板块未来进入资本市场筹备所需资金之外，将主要用于加大科技创新领域的投入、相关金融牌照的布局，并将增加实业科技板块的投资和布局。未来，派生集团将进一步梳理、整合、发展，除金融板块之外，还将包含科技、实业、投资等四大板块。
