举报收藏

职位诱惑：

高绩效,狼性团队,高成长性,强大资源

职位描述：

A-C轮 AI大数据 FA；

赛道方向：金融科技 、精准医疗、汽车交通和物流、安全科技、教育科技、智能制造、企业服务、芯片半导体


岗位职责：

1. 依据公司发展战略和业务要求，做行业研究，分析出行业地图，能根据自身对行业的丰富经验及深刻理解，给予创业项目专业的评估、判断，筛选优质项目标的，辅助或主导标的谈判；

2. 标的项目融资相关的行研、BP、资料承做，达成内外部共识；

3. 根据一线工作了解到机构客户反馈，向公司提出产品及流程优化建议；

4. 公司发展安排的其他工作。

任职要求：

1. 必须在投资银行、或咨询公司、或风险投资、或FA公司、或其他各类投资机构有1-3年工作经验，有券商分析工作经验优先；

2. 必须灵活运用行业研究、公司基本面研究方法，熟悉wind、choice金融数据库等工具，扎实的ppt、excel功底；

3. 国内外知名高校本科或硕士学位毕业；

4. 具有较强的研究和分析能力，乐于学习，能够以创新精神思考问题；

5. 相关资源，有一定的渠道能够发现标的企业；

6.  具有良好的客户沟通、人际交往及维系客户关系的能力；

7.  待遇：7-12K底薪＋业绩收入＝年薪10-50W；

＊＊月薪备注：7000-12000＝基础工资7000-12000＋配工作手机，公司统一缴纳话费，最高100/月＊＊

北京创新指数科技有限公司 该公司其他职位

北京创新指数科技有限公司专注于成长期优质企业的精品投行服务和项目跟投，发掘最有潜力的细分赛道未来领头羊企业，并提供战略咨询、长期财务顾问和产业链资源整合服务。致力于连接优秀创业者和价值投资者，推动全社会资源和资金最优化股权投资配置，成为最受信赖的企业资本成长伙伴和全方位服务的投资银行。

北京创新指数科技有限公司将通过系统化的挖掘和长期的合作,为投资机构精准推荐匹配机构需求的AI大数据相关科技领域以及文娱、消费相关行业优质投资标的，传递企业价值；为投资机构提供细分赛道行业研究、资料收集、企业对接、尽职调查、价值评估、协议谈判等各项工作的深度支持。

目前北京创新指数科技有限公司已经和国内1000个活跃投资机构（天使/VC/PE/上市公司）建立双向深入合作，并组建了国内最大的人工智能创投俱乐部“蓝海巨浪AI大数据俱乐部”，目前已经有300个已获投资的AI大数据企业创始人、800+机构投资人、300个机构合伙人或上市公司高管、数十位行业顶级专家等加入，致力于构建中国最顶级的人工智能全产业生态网络，汇聚AI大数领域的专业投资机构、产业大公司、行业专家和优秀创业公司，促进产业链上下游企业之间的高层互动合作、创业公司和投资机构之间的投融资对接。
