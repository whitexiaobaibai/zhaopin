举报收藏


 

【主要职责】 

从海量的用户行为数据中，发现和验证用户行为特征，提出评估方法，指导算法和产品改进。建设在线实验平台，统计分析报表。对数据敏感，崇尚从数据发现问题，提出方案并验证上线，从而解决产品问题的端到端模式和综合能力。包括但不限于： 

 

1. 海量数据处理：基于海量用户行为数据，构建高性能，低延迟的处理流程，产出中间数据和统计指标，支撑用户行为分析，建设在线实验系统 

2. 用户行为分析：分析和挖掘用户行为数据，找出有意义的问题，并进行验证和改进，或指导产品，算法的演进方向

 

【能力要求】 

1. TB级以上的大数据处理经验。有大数据基础架构 ，如Hadoop/YARN, Map Reduce, Spark, Storm, Hive, Impala等的使用经验； 

2. 用户行为分析经验。熟悉在线实验，和指标设计方法，或有数据挖掘经验者优先； 

3. 熟悉Linux开发环境，熟悉Python，Scala，数学统计背景者优先； 

4. 优秀的编码能力，对新技术有强烈的学习热情；

5. 有大规模互联网或电信运营商数据仓库、数据平台设计经验者优先；

6. 优秀的沟通理解能力，能快速理解业务，用数据解读业务；

  武汉绿色网络信息服务有限责任公司（武汉绿网）是一家基于DPI（Deep packet inspection）技术研发生产下一代互联网应用系统的软件企业，公司成立于2003年，总部位于中国湖北省武汉市光谷软件园，在北京和武汉两地设有研发中心。公司秉承“因为专注，所以强大”的理念，专注于高性能网络包处理领域。
    我们拥有专业化的高水平研发团队，拥有高素质且经验丰富的技术支持团队、市场营销团队和企业管理团队，其中95％以上的员工具有软件开发、计算机、网络通信、电子信息等专业的大学本科、硕士、博士研究生学历，在从事深度包分析、DMA零拷贝性能优化、数据包高速处理、应用协议分析、应用检测算法、信息安全等方面，积累了丰富的经验，实现了许多重大的技术突破。经过多年的积累与全体员工的不懈努力，我们在业界率先推出了高带宽万兆级别的网络流控系统，基于x86平台的高性能DPI处理平台，充分地满足了电信运营商业务扩展的多元化应用需求。该系统在国内同行业中具备明显的技术领先优势，多项产品具有世界领先水平，已在国内电信运营商成功建立了高带宽万兆级网络流控系统的应用案例，并拥有全球最大的基于x86服务器平台的应用层网络设备串行部署案例，得到Intel的认可，并已加入Intel Network Builder核心伙伴计划。
    北京研发中心专注于SDN/NFV等前沿网络技术、IPv6融合网关以及下一代网络协议识别算法和引擎的研究和开发，诚挚邀请各方人才加盟，我们的宗旨是“一流的人才只愿意与一流的人才工作”，我们将打造一个平台，让员工与企业共成长，在实现企业价值的同时实现个人价值。
