公司简介：

越亮传奇科技股份有限公司（原联想中望系统服务有限公司，2014年7月1日起正式更名）。

我们是一家以电信行业为主要服务对象的it服务企业，以电信运营商相关的移动互联网应用作为主要市场目标和基础，致力于成为服务于电信运营商和其他领域的移动互联网应用专家。

 

我们依托：广州、北京、南京三个研发中心，主要为国内电信、电力、航空、政府等行业客户提供行业应用软件开发业务。公司具有十多年为电信运营商提供软件开发和技术服务的经验积累，为电信运营商开发的业务支撑软件、增值服务软件、移动应用软件均具有较强市场竞争力。公司同时积极开拓其他行业客户，利用电信行业服务积累的软件产品和专业服务为电力、航空、政府等行业客户提供行业应用软件和技术服务。公司与国内主要电信运营商如中国移动、中国电信、中国联通保持密切技术合作关系，业务覆盖到广东、江苏、新疆、广西、福建、贵州、北京、辽宁、天津、吉林、湖南、江西、浙江、上海、宁夏、云南、湖北、西藏等全国二十几个省市，以过硬技术和专业服务赢得电信及其他行业客户的认可。公司自2009年开始涉足电信增值服务，业务品种逐年丰富，正在运营的增值服务包括爱秀、娱乐快报、铃音盒等。全资子公司广州粤亮信息科技有限公司拥有全国电信增值业务运营许可证。

诚邀有志在it互联网行业发展的您，在越亮传奇放飞梦想，创造奇迹。一经录用，我司将为您提供：

（一）、优秀的企业文化

1．公司愿景：一流的移动互联网支撑商。

2．公司宗旨：以电信运营商相关的移动互联网应用作为主要市场目标和基础，成为服务于电信运营商和其他领域的移动互联网应用专家。

3．创新、合作、共赢

越亮传奇秉承“志高行远、创新融合、精准求实、服务共赢”的核心价值观，形成了“创业创新、合作共赢”的文化精髓。

4．温暖 包容

我们是一个温暖的大家庭，更是一个包容的大集体，充满着浓浓的温情与人文关怀的团队。十年来员工伴着公司一起成长，他们把自己最宝贵的青春奉献给了公司的同时，也感受着这个温暖与包容的大家庭对每一个人细心而周到的关怀与照顾。在这里，我们情同手足，互为亲人，并因此不离不弃。

5．亲民 爱员

公司高层一直秉承“亲民 爱员”的团队带领思想，让“领导不像领导、员工不像员工”的工作氛围其乐融融。让每一位越亮传奇员工在尊重与自我的快乐环境中茁壮成长，与公司共同发展

6．感恩 责任

越亮传奇在这十年间经历过变革与创新，能在行业中处于领先地位，我们感恩这个伟大的时代，感恩我们每一位员工，我们也将肩负起社会的责任，为社会上有需要的人提供爱心支持与援助。在每一次危难面前都有我们越亮传奇人的身影和爱心援助行动。

（二）、良好的工作条件

硬件条件：总部与广州平台位于成熟而便利的天河北，另在北京与南京核心地段设立技术与业务平台。与此同时，公司也注重员工的人文关怀：为员工提供读书、休闲活动场所，开展各项娱乐体育活动等丰富员工业余生活。

软件条件：年轻奋发、团结协作、积极进取、快乐向上的工作氛围，优秀的技术与管理队伍，良好的人文关怀。

（三）、完善的培训体系

 公司为员工提供全方位的支持，只要您有工作的兴趣，只要您有成就个人事业的愿望，我们将为您提供一系列培训机会；培训的课程有专业知识培训、技能技巧、心态管理等；同时公司还为优秀的员工出资外送培训。

（四）、 健全的福利体系

 1、薪酬福利：为员工提供富有竞争力的薪酬体系并根据每年的市场薪酬水平和个人业绩表现，同时结合企业实际，对员工进行年度（优才季度）薪酬调整。

 2、上班时间：实行公司规定的五天八小时弹性上下班作息时间，所有假期与政府同，并设人性化奖励假期 如：年假在国家的基础上进行再奖励3-5天/年不等，六一儿童节（十三岁以下父母享有）。

3、保险/公积金：根据国家规定，公司为全体员工在入职当月（当月入职达十五天）缴纳五险一金，并为员工全额出资购买综合商业保险、子女综合险等特别福利，为员工解除后顾之忧。

4、晋升机制：公司设立良好的晋升机制并提供内部招聘、职位轮换机会。

5、节日贺礼：公司在传统节日、六一儿童节（未满十三周岁父母）给予一定礼品以增添节日气氛。

6、生日礼金：员工过生日公司给予祝贺，并定期举行员工生日会或赠送生日礼物。

7、员工锻炼：公司不定期组织体育比赛活动，如篮球、羽毛球、乒乓球及其他锻炼项目等。

8、活动经费：公司每年定期举行户外郊游、拓展等活动，为员工提供一定活动费，以增强团队凝聚力,活跃并丰富员工业余生活,扩大员工休闲视野和与社会接触的机会。

9、其他福利：符合条件的可办理广州市户口和党组织关系转移。
