举报收藏

岗位职责：

进行大数据分析、挖掘与展示的系统架构设计和产品集成；进行行业典型案例、科研分析案例等的集成；进行大数据科研支持平台和大数据教学产品的整合，统筹研发框架

岗位要求：

1、熟悉Hadoop（HDFS、HBase、Hive等）、Spark（Spark SQL、Spark Streaming等）设计原理及应用开发；

2、熟练掌握 Java 及面向对象设计开发，至少一个中大型web项目开发经验；

3、对数据结构、算法有深刻理解，熟悉主流的大数据产品和数据分析技术并具有项目实施经验优先；

4、了解云计算虚拟化和计算集群相关技术；

5、计算机相关专业，3年以上开发工作经验；

6、思维灵活、具有良好的沟通协作能力。

工作地址：

北京市海淀区中关村科学院南路6号

中科天玑数据科技股份有限公司 该公司其他职位

中科天玑数据科技股份有限公司依托于国家大数据分析工程实验室、中科院网络数据科学与技术重点实验室，与中科院计算所网络数据实验室、烟台分所、福州分所等一起构成大天玑团队。依靠多年的核心技术积累与专家团队，在大数据技术与服务领域，坚持以用户为中心，以大数据引擎为驱动，成为我国大数据业界龙头骨干企业。

公司的大数据服务产品、大数据系统产品、大数据安全产品已经广泛应用于政府、国防、安全、科研、教育、互联网等领域，典型客户包括国信办、工信部、公安部、安全部、中国证监会、中国银监会、国家广电总局、中国移动、中国工商银行等国家党政军业务部门及各企事业单位。

天玑团队现人员规模近1000人，普遍为硕士以上学历人员，来自于国内外知名高校及知名企业。团队由程学旗研究员领衔，程学旗研究员曾获国家杰出青年科学基金支持、入选中组部“万人计划”领军人才、国务院特殊津贴专家等荣誉，获得各类科研奖励16项，包括国家科技进步二等奖2项。程学旗研究员主要研究方向网络数据科学、大数据系统、社会计算、Web信息检索与数据挖掘。现任中国计算机学会大数据专家委员会秘书长，中国中文信息学会信息检索专委会主任。 担任国际学术会议ACM WSDM 2015的大会主席、IEEE Big Data Conference 指导委员会共同主席。团队研发实力超群经验丰富，工程项目团队能征善战，市场商务团队积极高效，支撑保障团队专业可靠。天玑团队气氛融洽，充分尊重人才支持人才发展并提供全方位的人性化保障。

 

薪酬待遇：

1、  富有竞争力的薪资，五险一金全基数缴纳，年终奖金、项目奖金、销售提成等机会多多；

2、  规范健全的休假制度，带薪年假全享受，规范的8小时工作制并可弹性；

3、  餐补、节日福利、生日福利、团建、免费体检等福利不断；

4、  温馨舒适的办公环境，地处中关村核心地带交通便利；

5、  快速发展的平台，以人为本的管理方式，和各种大牛一起共事成长的机会。
