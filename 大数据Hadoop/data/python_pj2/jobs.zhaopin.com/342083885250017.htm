举报收藏

岗位职责：

1.参与公司相关项目的开发和基础平台建设工作；

2.能快速理解、消化各方需求，并落实为具体的开发工作；

3.具备良好的架构分析能力与设计能力，积极参与前期设计；

4.具备良好的需求分析能力，业务建模能力和数据库设计能力，参与系统详细设计；

5.相关技术架构文档撰写及进行技术分享培训；

6.具有良好的学习能力，完成分配的调研任务和学习任务；

7.培养和锻炼初级开发人


任职要求：

1.国家统招大学本科以上，学历计算机、软件相关专业；

2.4年及其以上的java的开发经验，并对Java底层有深刻的理解，拥有分布式、大数据量的系统开发经验者优先考虑；

3.精通数据结构和算法实现，熟练掌握设计模式，UML，OOA/OOD，Java EE技术架构；

4.精通Jsp/Servlet/Web Service/JavaScript/CSS等开发技术；了解YUI、JQuery等脚本库的应用；

5.熟练使用Oracle，MySQL等数据库之一，对数据库有较强的设计能力；

6.熟悉SpringMvc/spring boot /Spring /Hibernate/Mybatis 等开源框架，并理解其原理；

7.熟悉分布式系统的设计和应用，熟悉分布式缓存、消息中间件等核心技术；并能合理应用分布式技术，解决问题 ；  

8.有丰富的软件架构设计经验，能够快速搭建、调试或重构大规模软件代码；

9.熟悉Redis、Memcache、MongoDB、搜索引擎、消息队列等；

10.具有良好的沟通能力和协调能力，优秀的学习能力，具备良好的团队精神，能承受工作压力，富有进取心；
