举报收藏

岗位职责：

1. 负责进行产品线规划,制定产品发展路标；

2. 负责完成产品立项、产品需求生成、产品实现策划的具体工作,并负责组建团队进行产品实现；

3. 负责与客户的技术交流、技术方案宣讲、产品演示等工作；

4.  行业/竞争/客户的跟踪和分析，识别重大机会和风险，并输出相关的分析报告，形成对相关业务的建议、推动决策，支撑中长期业务发展。

 

任职要求：

1.通信、计算机或电子类相关专业本科以上学历，两年以上工作经验；

2. 具有扎实的无线通信基础和一定的研究分析能力；

3. 良好的写作和表达能力；

4. 对行业新技术有较强的洞察力和敏感度，学习能力强；

5. 领悟能力强，对客户需求有敏捷的感知能力，能准确理解需求；

6. 具有良好的团队意识和合作精神，责任感强、性格开朗、善于沟通和交流。

工作地址：

北京市海淀区学院路29号 查看职位地图

大唐移动通信设备有限公司 该公司其他职位

    大唐移动，作为我国拥有自主知识产权第三代移动通信国际标准TD-SCDMA及TD-LTE标准的提出者，核心技术的开发者和产业化的推动者，以技术创新为动力，以国际标准为依托，以自主知识产权为纽带，以国内和国际两个市场为发展空间，致力于TDD技术在世界范围的应用与推广，并进而成为中国乃至世界移动通信领域的领先者。
    大唐移动拥有一支优秀、成熟的专业技术队伍，其中包括国内外知名的通信技术专家；公司管理团队经验丰富，其主要成员都具有技术与经营管理两方面的优势，他们不仅对TD-SCDMA及未来移动通信技术走向具有独到的见解，而且对中国移动通信市场具有深刻的把握和成熟的运作经验。
    大唐移动“以先进的技术、优良的服务为客户创造价值”的宗旨，以市场需求为导向，以技术创新为依托，质量与服务并重，致力于提供技术先进、性能稳定、品种多样的移动通信产品，在充分满足客户需求的同时，与客户、合作伙伴一起，为中国通信产业的持久发展和中国通信企业走向世界做出自己的贡献。
    大唐移动引入集成产品开发（IPD）的理念，并吸纳了ISO9000、CMMI以及企业标准化等思想、方法和体系之精髓，通过流程再造，强化全员的质量和客户意识，深化面向市场的组织结构调整，建立和健全了快速响应市场变化的运营体系，通过自身的不断变革，成为一个面向市场、快速成长、科学管理的现代高科技企业。
