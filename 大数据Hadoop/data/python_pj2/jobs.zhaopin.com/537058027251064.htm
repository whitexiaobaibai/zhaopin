艾睦奇企业管理咨询(上海)有限公司

举报收藏

1. 负责大型企业数据分析产品以及大数据产品设计；

2. 负责用户需求的调研和分析、市场分析、同行分析，参与产品规划；

3. 掌握产品发展方向，追求极致的产品易用性，协助开发、测试团队理解和掌握需求；

4. 根据产品实施效果以及业务发展状况，不断改进产品，深度挖掘用户需求。

1. 本科以上学历，三年以上需求分析、产品规划和设计经验，对大型企业大数据分析类软件系统有深刻的理解；

2. 有企业大数据产品相关经验者优先，有推广经验者优先，有大型云计算软件公司或IT系统服务公司或咨询服务公司从业经验者优先；

3. 具备较强逻辑思维能力，良好的沟通、聆听、理解能力，快速的学习能力，能迅速掌握产品知识；

4. 善于捕捉用户需求，具备产品创新意识及业务敏感性；

5. 具备系统的WEB产品设计、交互设计知识和实践经验；

6. 具备物流行业、互联网行业产品设计经验优先；

7.会编写产品远景文档和需求规格说明书，跟踪系统数据和用户反馈，对交互设计过程有深入了解，及时发现问题并做出优化；

8. 强力的推动能力与执行力，能够承受压力迎接挑战。

艾睦奇企业管理咨询(上海)有限公司 该公司其他职位

艾睦奇企业管理咨询（上海）有限公司， 是一家由专业团队组成，为企业提供专业整体招聘解决方案，致力于成为中国人才服务行业领先的人力资源服务提供商。公司总部设立于中国上海，立足于中国最大的经济中心，业务辐射全国。目前已于北京、杭州、大连、青岛、深圳、成都等地设立分支机构（含合作公司），在全国范围内为客户提供包括关键职位猎聘、批量招聘及员工派遣、招聘流程外包（RPO）等服务。公司致力于打造一个开放的合作平台，倡导资源分享和跨界合作，提供可信的、合理的、专业的合作模式，追寻共同发展、协作共赢的最终目标。

艾睦奇人肩负着“让我们的客户成为优秀的企业；让优秀的企业成为我们的客户”的使命和梦想，一直致力于在猎头、委托招聘、人力资源派遣、人力资源外包等专业领域为客户提供优质的服务。

客户构成及行业经验

    公司的30多名全职资深顾问、顾问及30多名兼职顾问，分布于IT，通信，地产，制造，金融，快消等行业。我们服务过众多世界500强在华企业和知名合资，内资大型企业。特别是有着丰富经验的顾问，长年提供专业人才服务给：IBM， Capgemini，Accenture，EMC，AMD,  EBAY，Micrsoft，TATA，INFOSYS，SATYAM，ATOS，Bleum，Abeam，Yahoo, Baidu, 用友软件，万国数据，趋势科技，三星通讯，NEC 中国， 阿尔卡特朗讯、世茂集团，SOHO中国等。

地域分布：北京，上海，广州，深圳，成都，大连，南京，杭州，武汉，昆山，沈阳，西安，天津等；

行业分布：IT，通信、金融，管理咨询，医药，工程制造，汽车等；

职位功能：高级管理职位；研发，咨询，财务，人力资源，采购，市场，销售，IT支持，培训，生产制造等；

职位级别：总经理，VP, 高级总监，总监，经理，技术专家，工程师等； 

专业态度及交付方法论：

    作为专业的人力资源服务供应商，我们提倡结果导向，以快速有效的为客户解决急需职位的招聘需求为目标。公司已经积累了近20万的中高端人才库，人脉范围覆盖全国各主要大中城市。 所提供的服务核心价值在于我们的专注，和专业的做事风格。 

我们的态度：

    永不放弃，态度决定一切。为了1%的希望,我们愿意付出100%的努力,从绝望处找到希望。遵循压强原则,集中力量,持久深入的挖掘某个领域的人才,这是我们专注的态度。 

真正的专家团队：

    我们的招聘顾问在相应的行业里面都是具有深厚行业经验的专家。在各个领域，我们的专、兼职顾问基本都曾经从事过相关工作，他们是质量和效率的保证。 

交付方法论：

       在实践中，我们结合项目管理的要素，整理出完善的交付方法论。流程，角色，交付物，构成了法论的核心组件。不同的阶段，不同的角色来执行，并提交相应交付物。计划，执行、反馈，调整并文档化，这有效的控制过程，对需求，风险，问题，沟通等的高度重视，确保我们的交付效率和质量都做到最好。
