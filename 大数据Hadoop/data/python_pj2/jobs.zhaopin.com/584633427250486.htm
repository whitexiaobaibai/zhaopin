举报收藏

岗位职责:
1、负责互联网金融领域大数据风控类产品或相关解决方案规划及设计，建立大数据风控体系；
2、负责核心风控系统的产品设计，制定产品发展规划及阶段性目标；
3、利用各类业务数据，与风控部门一起优化和迭代风控规则模型；
4、跟踪、分析、挖掘市场需求，并能结合公司的业务规划，进行商业模式评估，分析应用策略和盈利模式。

任职资格:
1、3年及以上的风控产品经验；
2、对互联网金融行业的商业模式及风控痛点有深刻理解；
3、对大数据风控体系、风险评级或评分、反欺诈类产品有深刻认识；
4、熟练使用各种产品策划工具，如Axure、Visio等；
5、具有优秀的表达、沟通与协调能力、团队合作精神、保密意识；
6、有成功落地验证的产品或大数据风控解决方案实践者优先。

和信电子商务有限公司 该公司其他职位

和信贷是和信电子商务有限公司推出的互联网金融服务平台，于2013年8月18日正式上线运营。我们秉承“安全为首，服务至上”的经营理念，为有融资需求的小微企业及有出借需求的个人建立规范、安全、专业的交易撮合平台。

和信贷具备完善的风险控制体系，以金融机构的风险管理标准实施运营管理，全面保障客户的资金与交易安全。我们始终以“安全”为核心，为客户搭建金融级安全系统，构筑可信赖的互联网金融服务平台。

未来，和信贷将继续以更开放的态度，高效的服务理念，积极的技术创新为客户带来更安全、更轻松愉悦的互联网金融体验。
