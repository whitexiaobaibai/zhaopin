举报收藏

岗位职责：

1.    承担人工智能应用系统开发实现工作；

2.    完成人工智能各模块的集成、对接、管理、分析、维护，提升系统质量、稳定性和性能；

3.    WEB应用系统设计与实现 ；

4.    完成相关项目的分析、设计及开发实现，并对现有系统优化。


任职要求：

1.    计算机软件及相关专业，有三年以上Java项目开发经验；对spring、struts、hibernate等基础开发框架有较为深入的理解；

2.    对Hadoop、Spark、Storm、HBase等技术有深入的理解，并至少二种以上实际项目应用经验； 熟悉Redis、Kafka、Flume、Logstash、ES等组件；

3.     具有大数据数据服务（处理）领域的相关工作经验；

4.    熟悉linux系统，具有shell、python等脚本语言开发经验；

5.    熟悉人工智能机器学习算法，熟悉至少一种深度学习训练工具（Caffe/Theano/Keras/ConvNet/TensorFlow/Torch等）；

6.    有互联网、电商行业相关开发经验的优先录用，有人工智能应用开发系统开发者优先录用。


工作地址：

建邺区奥体大街68号新城科技园研发总部园二幢22-24层
