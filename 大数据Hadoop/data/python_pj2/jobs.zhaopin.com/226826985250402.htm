举报收藏

职位描述：

1. hadoop大数据环境开发；

2. 根据业务需求和目标，建立预测模型；

3. 完成数据挖掘及建模项目的设计开发工作，完成模型的落地实施工作；

4. 确保模型开发，部署，实施，以及上线后的监控优化；

5. 推动模型开发团队业务能力建设，组织开展部门内部技术培训工作；

任职资格：

1．数学、统计、计算机、金融工程等相关专业；

2．良好的数据分析能力，有丰富的数据分析、挖掘、清洗和建模的经验；

3. 熟练运用软件，如spss、SAS、R、python等；

4. 出色的学习能力，能快速了解业务并设计相应的数据模型；

5．熟练使用大数据工具，如hive，sqoop，hbase,flume及相关分析工具spark、mahout等；

6．具有艺术品知识及相关行业工作经验有限；
