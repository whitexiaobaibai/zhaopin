举报收藏

岗位职责：

1.负责泛微OA流程设计、优化；

2.负责EAS流程设计；

3.负责门户、帆软报表、BI的设计与维护；

4.负责信息系统集成、接口开发方面的建设、实施及维护；

岗位要求

1.具备2年以上B/S项目开发经验；

2.熟练掌握常用开发工具，有SDK开发经验；

3.熟练掌握Oracle、SQL Server等关系型数据库、熟练使用SQL语句，及SQL语句性能优化，熟练掌握SQL语句，会使用存储过程，索引，游标等，有数据库结构设计经验；；

4.熟练运用HTML5、Javascript、CSS、XML、webservice中一种或以上开发技术；

5.具有优秀的需求理解、系统分析、设计和文档编写能力；

6.具有良好沟通和团队精神、了解三层架构模式；

7.具有较好的沟通能力和良好的团队合作精神；；

8. 熟悉Map API ,高德、百度地图开放平台、 API等互联网地图API调用开发者优先

9.有大数据处理、分析、性能优化，熟悉SSIS、ETL经验者优先；

10.熟悉DLL、JQuery插件、第三方Web Service服务、API接口者优先。

佛山市南海绿电再生能源有限公司 该公司其他职位

佛山市南海绿电再生能源有限公司位于广东省佛山市南海区狮山大学城园区，为上市公司瀚蓝环境股份有限公司（原名：南海发展股份有限公司，股票代码：600323，主要业务有固废、供水、污水、燃气）的全资子公司，是一家主要从事生活垃圾焚烧发电、城乡一体化生活垃圾压缩转运、污泥处理、餐厨垃圾处理等业务的固废处理企业。公司是全国五家“AAA级无害化焚烧厂”之一（AAA为行业最高级别），获批国家住房和城乡建设部“市政公用科技示范工程”、高新技术企业、广东省环境教育基地等。

作为固废处理运营商，公司一直致力于运用国际领先的固废处理技术，引进国内外先进的经营、管理模式，协助南海区政府实现生活垃圾、污泥、餐厨垃圾等固体废弃物的资源化、无害化、减量化处理，并通过垃圾焚烧的余热发电，实现资源的循环利用，形成生活垃圾处理从源头到终端处理的完整产业链，创造卓越的经济效益和社会效益。以环保为企业的生命线，是公司始终坚持的生存观和发展观。公司目前已通过ISO9001\ISO14001\OHSAS18001\SA8000国际质量、环境、职业健康安全和社会责任标准体系认证，也是目前行业内首家引入第三方监管的固废处理企业。公司关键环保指标达到或超过欧盟标准，致力于成为同类项目的国家示范工程、环保教育基地和环保科研基地，以及环保产业示范园。公司运营的南海固废处理环保产业园，作为行业示范项目，吸引了众多国家、省市政府领导团队前来考察学习。国家住建部部长姜伟新、国家环保部科技标准司刘志全司长、广东省住建厅杜挺副厅长等领导前来考察并给予高度评价，号召将“南海模式”向全国推广。

多年来公司在公共事业领域不断延伸产业链，致力打造全方位的环保服务平台。目前拥有日处理生活垃圾3000吨、污泥处理300吨，以及转运生活垃圾4000吨的能力。在建日处理300吨的餐厨垃圾处理项目，中标北京首钢生物质能源项目技术服务正在运行当中。控股公司瀚蓝环境股份有限公司成功并购行业龙头之一的创冠环保（中国）有限公司。

人力资源是企业发展的第一生产力。建立一支专业、高效、敬业的管理和生产运营团队是确保成功的关键。公司通过建立人才培养和内部竞聘管理提拔机制，完善员工职业发展通道。还为员工提供全面的福利保障：5天8小时工作制（电厂运行岗位实行5班3倒）；带薪假期；五险一金和意外保险；拓展旅游；节日礼金、津补贴、年底双薪和月度/年终奖金等。

企业愿景：成为国内有影响力、系统性的固废处理运营商！

企业使命：为社会提供优质服务，为员工创建实现自身价值的平台，为股东创造持久的回报；

企业价值观：合作、尊重、责任、诚信、创新。

企业文化：成为 “城市好管家、行业好典范、社区好邻居”的“三好企业”和“能力分享、价值观分享、财富分享”的“三分享”企业！

我们热诚欢迎有志于固废处理这一循环经济新行业的优秀人才加入到我们的队伍中来，共创美好未来！
