举报收藏

岗位职责：

1. 针对大数据分析设计文档完成数据的开发、调度、部署、上线（ETL工具使用、SQL编写及测试）  

2. 负责对客户人员进行的培训

3. 在项目经理的带领下，根据客户需求开发程序，对项目经理负责，及时汇报任务进度；  

4. 服从项目经理任务安排，负责软件研发任务的详细设计、编码；  

5. 配合项目经理进行项目研发的技术分析工作、参与客户与实施需求调研并进行分析，上报；  


6. 熟悉部门软件，进行测试，完成现场软件发布与程序更新工作；

7. 负责项目过程中相关配套文档编写

任职要求：


1.  有责任心；

2.  计算机及软件工程相关专业、生物医学工程、医学信息学、应用数学，本科（含）以上学历，一本以上优先。

3.  有意向从事软件开发，愿意主动学习，良好的沟通和人际关系技巧，良好团队精神。

4.   较强分析、判断和概括能力， 逻辑思维能力，一定的抗压能力；

此岗位为米健子公司麦健招聘




5.          熟悉大数据分析开发、数据仓库体系架构以及.net、JS等开发语言，熟悉SQL Server、Oracle等数据库，熟悉Linux操作者优先；


公司可以提供良好的学习平台，欢迎优秀应届毕业生加入。


工作地址：

上海市杨浦区隆昌路619号1号A103 查看职位地图

上海米健信息技术有限公司 该公司其他职位

   上海米健信息技术有限公司是隶属于荣科科技（300290）旗下的子公司，公司成立于2012年，坐落于上海杨浦软件产业园。公司在全国拥有逾30多个办事处，上海为全国总部。

     公司是一个拥有领先核心技术的医疗信息化解决方案供应商。创始团队核心成员均来自国内外著名医疗IT公司、中国顶级三甲医院；团队对医学临床信息化有着深刻的理解，目前的产品已经在北大人民医院、中国人民解放军总医院（301医院）等国内众多医院得到了成功的应用。

    米健是一支开放的团队，在这里，你不必察言观色、经营办公室政治，因为我们这里只有快乐、激情和自由，梦想让来自五湖四海的兄弟姐妹亲如一家人。 

       主要业务介绍：
1、急诊急救临床信息系统（ECIS)
2、手术室麻醉临床信息系统（AIMS)
3、重症医学临床信息系统（ICIS)
4、医疗信息集成平台（HUIS)

     我们的使命是服务中国百姓生活和健康，现在只差和我们一起起航的你！
