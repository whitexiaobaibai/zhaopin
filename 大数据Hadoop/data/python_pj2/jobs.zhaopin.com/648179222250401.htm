.NET软件工程师

广州市昊翼信息科技有限公司

绩效奖金带薪年假员工旅游节日福利
  • 职位月薪：6000-10000元/月

举报收藏

岗位职责：
1.ASP.NET/C#项目迭代开发，同时参与相关的需求分析设计；
2.负责项目的核心功能及关键代码的实现，完成单元测试、代码维护、优化工作；
3.书写整理程序开发文档，数据库维护。

任职要求：
1.ASP.NET/C#编程，B/S应用架构，VS2010 以上，1年以上WEB项目开发经验，并具有扎实的编程功底，大、中型电商平台核心代码、主功能模块开发经验；
2.精通应用MSSQL2008 以上，至少对存储过程、触发器、事务管理有深入了解，擅长性能优化，有实际大数据量数据库的设计经验及性能优化经验；
3.精通ASP.NET WEB API MVC，Newtonsoft.Json，API Cloud；
4.熟练使用DIV+CSS布局、JavaScript、JQuery、AJAX、JSON、HTML5、CSS3、有一定的UI框架布局能力；
5.熟悉高性能程序开发及MVC开发模式，具有较强的逻辑代码编写能力，要求有独立设计系统的经验
6.精通融云、极光推送等第三方接口应用；
7.熟悉XML，有Mobile Web制作经验，熟练自适应设计等前端应用，精通更佳；
8.深刻理解Web标准，UTF-8编码格式，前端布局兼容主流浏览器且向下兼容低版本，掌握至少一种调试工具应用；
9.掌握使用至少一种团队开发工具，有规范化、标准化的代码编写习惯，有注释，易维护和良好的技术文档编写习惯；
10.有微信支付、支付宝等支付接口开发经验者优先；
11.有微信公众号开发经验、或小程序开发经验者优先；
12.懂SEO、掌握静态页面生成、精通SiteMap规范者优先。
福利待遇
薪资结构：固定底薪+高提成+丰厚奖金(表现优秀者年底有丰厚奖金)；
工作时间：每天6.5小时工作制，单双休；
员工假期：享有国家法定节假日及劳动法规定的工伤、婚假、带薪年假及带薪病假等；
社会保险：统一为员工购买国家规定的养老、失业、工伤、医疗、生育保险；
节日福利：每年的传统节假日（如中秋节、端午节）等，公司会发放相应的礼金或礼品；
团建活动：公司不定期组织举办各类集体活动，如一年一度的旅游，每周六固定下午茶活动、聚餐等等；
健康保护：关注饮用水、清洁用纸、办公室空气质量等每个细节，追求每位员工的健康保护；
潜能培训：公司提供系统、完整的培训体系以及晋升渠道，能按员工需求提供各类培训：入职培训、在岗培训、素质提升培训等；
工作环境：公司位于广州市天河区商业区甲级写字楼，办公室环境清爽、干净、整洁；交通方便，距离广州东站B1地铁口仅几分钟路程。 
 
工作地址：

天河区林和西路157号保利中汇广场2402
