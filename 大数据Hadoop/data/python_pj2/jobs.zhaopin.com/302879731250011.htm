举报收藏

岗位职责：

1. 主要负责网站和产品移动端页面的创意和设计，制定界面风格以及标准，根据需求设计网站／app的用户界面（包括界面设计、Logo设计、交互设计等）。

2. 配合开发部门，制定网站的整体UI风格，解决遇到的各种问题。

3. 协助市场部门，进行用户调查，优化产品的用户体验。

 

岗位要求：

1. 网页设计、美术设计等相关专业，有实际UI设计工作经验者优先（需要展示相关的作品）；

2. 热爱设计，具有扎实的美术功底和创意设计能力，较好的审美观念及色彩搭配感觉，熟悉界面交互或用户体验；

3. 熟练使用各种图像处理软件、原型设计工具，比如Photoshop、Axure等；

4. 2年以上互联网公司经验，美术或设计相关专业优先考虑；

5. 面试时须准备个人优秀设计作品5个以上。

 

优先考虑：

1. 有门户级网站设计、重大产品界面设计等经验者优先；

2. 有GIS相关网站、智慧城市、智慧农业、农业信息化等相关设计经验者优先；

3. 具有大数据分析与可视化软件交互项目的工作经验者优先；

4. 具有百度地图、高德地图、天地图、Google地图等软件设计交互项目的工作经验者优先。

尚正(北京)信息技术有限公司 该公司其他职位

    尚正（北京）信息技术有限公司（简称“尚正信息”）成立于2015年，位于北京市丰台区总部基地。公司始终坚持“常态化、规模化、高精度、低成本”的农业大数据综合应用体系为目标，是目前国内唯一实现对高分辨率卫星数据进行旬周期计算的大数据处理和应用平台。同时按照用户需求，以领先的多源遥感影像集成处理、地理信息系统、全球定位系统技术等的综合应用，获取地上、地下相关农业数据进行处理、分析，是我国领先的农业领域“数据—交易—金融”完整体系应用服务、解决方案提供商。

         作为一家创新技术服务型企业，秉承“让数据服务于农业”的使命，通过高新技术信息化，服务于农业产业，为提高土地资源有效利用率和农业信息化水平做出贡献。通过可观精准的大数据支持、优势资源的整合为全产业链条上包括：农民、农资供应商、政府主管部门、农业技术专家、农产品采购商、农产品深加工企业、物流企业、金融保险业乃至终端消费者等所有参与者提供服务。
