岗位职责：

负责AI软件将2D图片转换为3D图像。包括应用AI从2D确认3D数据定位及生成3D，以及使用AI滤除来自2D的噪音。

任职要求：

3年在2D影像进行AI开发经验。3-5年3D影像制作、大数据开发、用于Window, Mac, 及 Android应用软件开发经历。

技能：

包括开发AI自身学习模块，AI API数据计算，AI输入数据预计算。基于Window, Mac and Android, Java, Python 软件操作系统的高级语言、在大数据结构中的数据处理，以及问题排查和数据分析。

北京裕天合生物材料科技有限公司 该公司其他职位

北京裕天合生物材料科技有限公司成立于2005年，是专业从事医疗器械的销售及贸易的综合性高科技企业。6年以来，裕天合公司专注在血管外科领域发展，率先开展了大血管封堵术，推动了心血管外科临床与国际接轨。在中国专家的技术支持下，致力于引进新开发的产品，为满足临床的需求而努力奋进！
  
为了顺应市场的变化，公司加大了市场投入，在引进国外先进技术、新产品及融入外资方面都加快了步伐。公司不断向临床提供先进、高效的医疗产品和提高临床诊疗水平作为自己的最终目标。

北京裕天合生物材料科技有限公司的业务主要为代理国内、外优秀产品，在国内进行市场开发、推广、培训等；保证临床的使用质量、高效及安全。

北京裕天合生物材料科技有限公司一直通过不懈的努力，保证提供精益求精的产品及服务质量。同时，通过不断地引进国内、外先进的技术，为提高我国的医疗服务水平贡献自己的一份力量。
