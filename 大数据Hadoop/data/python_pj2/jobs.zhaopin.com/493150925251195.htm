举报收藏

岗位职责：


1. 负责大数据平台开发核心标签算法；

2. 分析设计算法可插拔的计算框架。需关注框架的可扩展性

3. 负责海量（TB级）数据分析、数据挖掘、机器学习等组件的设计和开发工作

4. 保持对于大数据行业的技术敏感度，组织研究行业应用领域最新技术发展方向，负责将新技术应用于实际项目；敢于尝试。

5.积极配合项目成员的工作，发挥自身的能动性，

任职要求：

精通Spark/Scala

熟悉：linux/python/shell

了解Hbase/Hive/ES/Hdfs/Yarn /Storm/kafka/Presto/Kylin/ codis/zeppelin

高数据敏感度
