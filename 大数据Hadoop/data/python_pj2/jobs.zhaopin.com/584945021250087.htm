举报收藏

岗位职责：


1. 负责在线教育平台的微信端/移动端APP设计及优化，提升DAU和用户体验；

2. 分析并及时响应业务需求，提出产品解决方案;

3. 管理项目进度，协调设计、开发、运营等团队资源，协调并推动跨部门的合作，推动产品快速高质量上线;

4. 负责产品原型和交互界面设计，编写产品需求文档(PRD)，研究用户体验的趋势，不断优化用户体验;

5. 跟踪产品研发过程，把控开发质量和进度，对产品版本的最终上线负责;

6. 负责产品的用户行为数据监控与分析，发现产品迭代及产品创新机会，不断改进产品, 根据数据分析结果来持续改进优化产品.



任职资格：

1. 热爱互联网和教育行业，熟悉在线教育行业，有教育类和社交类产品设计经验者优先;

2. 以解决用户问题为己任，能够从用户角度来体验和改进产品;

3. 具有良好的产品感觉，对不同用户群体不同使用场景有生动的想象力，愿意为改善自由行用户的出行体验而投入;

4. 具备数据敏感性和数据分析能力，良好的逻辑分析能力，能从用户行为数据中归纳和发现产品改进方向;

5. 具有交互设计能力，能够输出高质量的产品原型图，完整清晰地表达自己的设计方案，推动团队完成产品迭代;

6. 对互联网、移动互联网、大数据的未来发展有清晰完整的见解，了解产品及竞争对手的最新动态;

7. 具备数据敏感性和数据分析能力，良好的逻辑分析能力，能从用户行为数据中归纳和发现产品改进方向


工作地址：

北京市朝阳区建国路91号金地中心A座38层

北京灵睿博智国际广告有限公司 该公司其他职位

北京灵睿博智国际广告有限公司成立于2008年，是一家为企业和组织提供品牌管理与营销传播等服务的专业机构,长期致力于品牌策划、公关传播、区域营销、服务品牌整合推广。多年来积累了丰富的成功经验，业务涉及地产、快销、旅游、汽车多个领域，与鲁花、远洋地产、富力地产、东风汽车等客户建立了长期稳固的全面合作。目前公司正依托优势资源向整合全国旅游客户方向布局，已占据了90%以上的湖北省著名景区，并在向全国发展。
      公司秉承“求实、平等、创新”的企业文化理念，不断修缮服务体系，提升核心竞争优势，壮大骨干团队，由区域市场向全国市场拓展，实现公司经营规模的跨越式发展。力争在国家“十三五”期间将公司打造成国内最具影响力的整合营销传播服务品牌之一，同时为更多的客户创造品牌传奇。
