公司简介：
深圳旗智奥信网络科技有限公司为大型国有银行业务呼叫中心，是全国最大银行营销方案提供商,承接信用卡生命周期相关业务。总部设在上海浦东，全名上海旗计智能科技有限公司，简称“旗计智能”，是国内首家A股上市的专业金融机构业务流程外包企业。
总部设立大数据运营部，销售运营管理部，供应链集成中心，产品研发设计部等规范运营管理部门，在北京、深圳、合肥、常州、宁波等地设有九家分子公司及办事机构，拥有十多个大型呼叫运营中心，面积超过10000平米，集团员工超过3000余人。
如不方便投递，可直接联系李！
筛选合格的简历我们会在两个工作日内电话邀约！
工作在小旗，你能享受到：
1、无责任底薪2700-4000+高额提成+激励奖金，【综合月薪6000起，100%到手哦！】
2、入职即签订劳动合同，缴纳五险一金；员工生日会、传统节日礼品；不定期集体旅；
3、提供公寓式住宿，家具齐全，还有班车直达公司哦；
4、带薪年假，节假日福利，我们一个不落;
5、入职满6个月的员工还可享受免费学历提升；入职满12个月后，每月享受100元旅游基金可累积无上限；入职满18个月后，可申请享受每月1000元购车补贴，可连续享受12个月；
6、入门培训，轻松上岗
晋升空间：
1、初级服务专员-中级–高级-资深
2、初级专员-中级专员–二线/内勤岗位（质检/培训/企划督导/行政/客服/人力资源等）
3、高级专员–预备主管–团队主管-区域经理；
岗位职责：
1、根据银行提供的优质客户资源，通过银行系统外呼进行回访，解答客

期邮购服务，最大程度减轻客户资金压，容易成交。
任职资格：
1、普通话标准，性格开朗，具有良好的沟通表达能力；
2、工作认真负责有耐心，有良好的客服服务意识；
3、能承受一定的工作压力，可以挑战高薪；
4、工作经验不限，有客服、导购、销售工作经验优先。
