举报收藏

岗位职责：

 1. 根据公司战略，分析市场对大数据需求，制定数据产品运营策略，参与数据产品Roadmap的制定，支持业务团队或第三方调取和使用数据的需求；
2. 关注大数据应用方向的研究，把握行业发展趋势，分析跟踪竞争对手，定期完成分析报告或产品竞争战略和计划；
3. 负责数据产品的需求分析，需求管理，产品设计和产品迭代；挖掘数据的商业价值，丰富产品形态，输出高质量PRD文档；组织需求管理和评审；
4. 负责跨部门协调沟通工作，与UI、UE和研发人员一起合作，确保产品研发进度，直至成功发布；
5. 结合业务反馈以及市场需求，提出产品改进意见，不断完善在线产品、持续提升产品的功能和数据处理能力，支持各类用户对大数据的复杂需求；
岗位要求：
1. 本科及以上学历要求，三年及以上数据分析/产品设计相关经验，能够独产承担产品规划工作。
2. 具备良好的数据敏感度和商业敏感性，能够敏锐的捕获数据价值和产品机会。
3. 对产品有很好的组织规划能力，具有较强的文档撰写能力与经验，精通各种产品原型设计工具。
4. 优秀的表达、沟通与协调能力；良好的团队合作精神、保密意识。
5. 了解底层数据的架构以及应用层数据分析和算法，对大数据分析和数据挖掘有一定认知，具有产品创新精神。
6. 具备数据分析、数据挖掘、数据可视化、数据建模、机器学习等背景或工作经验，有人工智能/数据类/云服务（Paas Saas）/企业级平台类产品工作经验者优先。

工作地址：

北京智慧星光信息技术有限公司 该公司其他职位

公司简介： 

    北京智慧星光信息技术有限公司是全球领先的文本大数据服务商，致力于打造国内最有竞争力的大数据价值挖掘和精准化信息服务平台。智慧星光是国内首家把SaaS服务、大数据技术运用到互联网信息服务、舆情服务、客户大数据应用领域的高新企业。
    智慧星光拥有全球最大的中文文本数据中心，总数据量超过600亿条，日新增数据量超过3亿条，当前已经服务客户超过15000家，是国内最大的舆情服务商和领先的互联网信息及大数据服务应用型企业。
    公司成立于2012年2月，总部位于北京中关村，拥有员工500多人，在23个省市设立办事处。智慧星光全部产品拥有自主知识产权，并自主研发建设了国际领先国内首创的CII云智能信息平台。并基于此平台成功研发运营：舆情秘书、舆情研判、舆情报告、智慧网评、智慧商情、星光数据、舆情OA、KOL监测等多项产品。
    经过多年的研发与实践，智慧星光的产品以其超群的信息挖掘能力，秒级的信息处理能力，近乎人类思维的情感分辨能力以及优秀的用户体验效果受到业界和客户的认可和好评。尤其是“舆情秘书”风险监测产品，因其在舆情监测过程中，所展现的全面性、及时性、精准性等优势，已在政府、公安、宣传、院校、企业等多领域中得以应用，成为各领域公共风险治理的重要工具。
    智慧星光结合为客户提供多年互联网舆情监测与预警的成功经验，利用大数据挖掘技术，成功开拓了企业口碑监测分析、竞争情报分析、精准营销、人物画像、企业画像、行业市场研究、客户满意度分析、风险评估、产品及业务流程优化等等多个场景的成熟应用。从而在大数据应用、数据服务和分析报告等方面为客户提供越来越多的创新业务模式。
