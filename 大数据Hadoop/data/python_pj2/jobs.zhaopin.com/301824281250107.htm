举报收藏

岗位职责：

1. 通过ETL过程，为后续数据分析、用户画像、数据挖掘提供准确及时的数据支持；
2. 根据业务体系，完成数据仓库的建设与推进；
3. OLAP的建立为维护；
4. 为项目提供技术支持，将新技术融合至工作中。

任职要求：

1.大学本科及以上学历；2年以上互联网的数据处理相关工作经验；
2.精通SQL语言以及函数、存储过程的编写；
3.熟练使用Informatica、Kettle、Datastage、SSIS等一种或多种ETL工具；     
4.具有数据仓库开发经验；                                                
5.具有优秀的沟通协调能力和团队合作精神，执行力强。

工作地址：

东华光普大数据技术有限公司 该公司其他职位

光普数据公司拥有一支专门提供大数据应用和服务的团队，业务覆盖行业信息服务、风险监测、风险预警、危机预测和危机处置等领域。

   光普数据公司通过融合最新的网络爬虫、全文检索等技术，实现海量数据实时自动采集、信息过滤、语义分析以及可视化应用，同时结合强大的媒体资源及危机处置实力，为大型国有企业、上市公司及各金融机构提供专业大数据挖掘、分析、处理、应用以及技术支撑、风险管理、危机预警、预测和信息顾问决策等服务。

  公司定位于大数据的观察者、研究者和服务者，通过融合自身创新的大数据挖掘技术以及资讯服务模式，成为客户在数据分析应用领域的最佳顾问和决策参考的支持者
