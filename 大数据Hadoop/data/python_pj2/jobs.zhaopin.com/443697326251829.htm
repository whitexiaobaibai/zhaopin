【关于我们】

湛腾科技是中国值得信赖的科技服务合作伙伴，作为全方位IT服务及行业解决方案提供商，湛腾拥有全球运营能力，通过严格的质量标准和交付流程来保证客户的商业价值。

湛腾在金融服务业、高科技行业、电信业等领域积累了丰富的行业经验，客户重点针对国内顶级金融机构及财富500强企业，具备全面的IT专业服务能力，为客户提供IT咨询、解决方案和外包服务，包括企业解决方案、应用软件开发和维护、质量保证和测试、本地化和全球化、研究及开发、基础设施外包以及业务流程外包/呼叫中心等服务，帮助客户实现投资收益最大化，并使之更专注于自身核心业务。

湛腾充分解析客户需求的重点，采用独有的在岸－离岸服务模式，并融入了在不断为客户提供一流服务过程中形成的与众不同的企业文化。湛腾关注人力资源的发展，拥有一批极富潜力、训练有素的员工，为客户提供涵盖整个应用服务生命周期的服务。

【价值观】

诚实、正直、勤奋、专注服务、赢在细节

【愿景】

成为受到行业及员工普遍尊敬的一流服务外包企业，帮助客户提升核心竞争力，在局部区域形成核心能力，通过帮助员工创造价值提升企业价值

【使命】

为员工：成为发掘和培养优质IT人才的沃土

为客户：成为客户忠实可靠的合作伙伴，帮助客户提升价值

为社会：承担公司应有的社会责任，帮助更多人才提升个人价值，实现可持续发展

湛腾对每位员工的职业发展引导，并非单一的以管理岗位定位职业发展。每个人可能具有多重属性，湛腾根据员工的能力特性针对性的辅助员工不断提升自身能力拓展职业空间和深度，发挥更多的能力，创造更多价值，以下简述一些针对性培养策略：

l  创业型人才：公司帮助员工发展，甚至可以包括帮助员工进行创业，给予其充分的发挥空间和支持，实现个人梦想；

l  管理型人才：公司设有管理人员储训机制，发掘培养有管理能力的人才，并给予岗位进行锻炼；

l  销售型人才：对于服务意识良好，善于沟通及把握客户需求的人员，公司会挑选出来培养锻炼成为客户经理，销售经理，市场总监等职位；

l  技术型人才：技术良好的人员可以成为架构师，技术咨询师，方案策划师等等；

l  培训型人才：拥有有良好的分享意识和信息传述能力，能将知识授予他人在湛腾看来是非常重要的能力，培养发掘更多的人才，本身就在为公司创造更多价值；

l  业务型人才：对行业、客户、产品拥有良好的理解分析能力，能够整合公司资源创建方案服务于客户并获得高度的信任，这是非常重要的能力价值；

l  文案型人才：文档能力是湛腾非常重视的能力特性，几乎所有的输出最终都体现在文档方面，湛腾重视此类人才的培养，通过良好的文档积累和展示创造价值；

l  细节型人才：勤奋扎实、关注细节，是很多人都不具备的特质，同时也是一部分同事非常重要的优势，这份优势用在数字统计，风险分析等方面能创造更大价值；

l  展示型人才：你的口才卓越吗？你善于PPT或者Video制作吗？或者你具备优秀的营销能力吗？你知道客户最想要的是什么吗？如果具备，恭喜你，这是稀缺型人才，大有用武之地；

l  文化型人才：企业要发展，离不开良好企业文化的建立与宣扬，即使你只是团队中的一粒开心果，假使你只善于吃喝玩乐游，我们仍然需要你，有了你，团队更加和谐，我们更加团结拼搏为未来努力！
