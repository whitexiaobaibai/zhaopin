举报收藏

岗位职责：负责构建机器学习和数据挖掘相关系统及平台

1.对现有数据进行数据挖掘和分析，提取关键信息用于产品开发和策略制定；

2.开发机器学习和数据挖掘相关的系统及平台，比如反欺诈平台；

3.应用机器学习等先进建模方法，利用大数据进行研发，保证模型准确率；

4.应用模型成果，部署、跟踪、监控模型表现，根据监控结果进行模型优化，达到模型准确率要求；

5.建立和完善统计分析模型，为运营决策提供支持；

6.组织相关知识培训：编程、统计建模、大数据知识与技能等；

任职要求：

1. 计算机/数据/统计学专业本科及以上学历；

2. 扎实的统计学、数据分析、数据挖掘基础和统计建模能力；

3. 熟悉数据挖掘、机器学习、人工智能技术，尤其是关联分析、分类预测、聚类分析、回归分析、时间序列分析等常用分析方法；

4. 精通R、SAS、SPSS、Matlab等挖掘工具；

5. 有统计、数值计算、模型开发研究基础,风险模型构建经验者优先；

凯泰铭科技(北京)有限公司介绍

KTM(凯泰铭科技)公司是一家的以车险反欺诈服务为核心、结合“汽车维修工时费”与“零配件”报价、供货等反渗漏服务，为您提供车险理赔管理“一站式”解决方案的科技企业。我们结合保险公司对理赔管控的需求，为客户量身打造风险管控引擎；同时我们结合保险公司对内部管理的要求，协助客户对内部查勘、定损环节进行标准化、规范化管理，挤压理赔水分。


我们致力于为保险公司解决以下问题：

1.    打破维修市场的信息不对称，使车辆零配件价格、维修工时价格透明化，使赔付费用“物有所值”；

2.    基于碰撞逻辑、汽车维修等汽车领域专业知识，提供可落实的反欺诈手段，从而应对车险市场不断走高的赔付率；

3.    协助车险公司对查勘、定损环节进行规范化、标准化管理，以提高综合成本率管控能力，挤压理赔水分，更好的应对“费改”带来的车险市场“新常态”。

 

我们目前拥有的：

我们的创始人和运营团队在保险、汽车维修行业及领域拥有平均超过10年以上的丰富经验，并与国内多家财险公司具有良好的业务关系；

我们提供的专家检测结果和工时检测方案受到合作保险公司的普遍认可；

我们与国内各大配件数据供应商有着丰富的合作经验、良好的合作关系，包括广信恒源、浙江联保等；我们现有的两家全国性零配件供应渠道，确保为您提供精准的报价的与实际的落地能力。

我们的优势：

1、富有竞争力的薪酬、福利！

       比同行业高的薪酬、餐费补贴、提供单身宿舍、每年在法定各项休假制度的基础上享受5天带薪病假、男员工陪产假、免费体检、团队建设活动……

2、以产品为导向开展IT工作。

以自主研发产品的形式开展IT工作，非外包，非外派，自主掌握核心技术和业务。

3、敏捷开发流程。

小周期功能迭代开发，小团队合作开发，完善的共同机制。

4、完善的IT管理、自动化环境。

拥有ContinuousDelivery / DevOps IT开发环境，自动化测试、发布流程，消除非必要的重复的工作，专注产品的研发及功能的实现。
