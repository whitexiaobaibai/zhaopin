举报收藏

岗位职责

1. 参与外服云平台功能建设项目的必要性及可行性论证，负责编写项目可行性分析报告和初步技术实现方案；

2. 负责项目计划的制定、跟踪和质量控制，及时发现并跟踪解决项目问题，有效控制项目风险，确保项目按照计划和保证质量完成；

3. 负责项目资源、供应商和相关部门的协调与组织，确保各项目参与人协同工作；

4. 协助制定项目涉及的技术、安全的验收标准，并组织和参与项目验收；

5. 负责项目建设过程中的项目文档资料的保管及归档管理。


任职要求

1. 本科及以上学历，计算机、通信相关专业。5年以上应用系统项目管理的经验；

2. 精通常见的IT业务系统和主流IT技术，熟练掌握信息技术项目管理知识，能独立构建完整的IT产品。

3. 深刻理解移动互联产品发展趋势，熟悉移动互联、大数据、云计算等业界的前沿技术、商业模式。

4. 具有较高水平的沟通技巧，能够快速理解业务需求。具备较好的团队合作精神, 分析解决问题的能力；

5. 工作责任心强, 独立工作能力强, 认真诚实、积极主动, 能适应紧张高强度的工作。

6. 具有大数据、云平台项目管理经验者，拥有PMP证书优先

﻿﻿我们的总部上海市对外服务有限公司（简称“上海外服” www.efesco.com），隶属于国资委，是中国第一大人力资源公司。

上海外服云信息技术有限公司（简称“外服云”）是上海外服围绕新的战略发展目标而注册成立的全资子公司，专做互联网的IT公司。

公司致力于人力资源行业云服务及云计算，公司正自主研发一款基于Saas云平台的企业解决方案，是一家纯正的互联网公司。

公司主要核心人员都是来自于互联网行业，比如一号店，阿里，盛大，携程等知名互联网企业。

“外服云”业务总体目标
1. 业内第一个真正的基于云技术的人力资源服务 O2O 平台（线上线下相结合，传统HR外包服务和HR管理服务相结合）
2. 业内第一个人力资源服务的B2B2C平台
3. 帮助上海外服成为国内乃至区域内人力资源服务相关的技术领跑者，确立在行业竞争中的技术壁垒
4. 帮助上海外服创立全新的人力资源一站式服务模式（管理模式、商业模式），争取建立一个比较完整的人力资源服务的生态圈
