举报收藏

岗位职责：

1.各类型数据的采集、清洗、分析、融合，负责设计数据格式、预处理数据，为产品提供数据支撑；
2.负责项目的数据挖掘分析工作，构建业务数据分析体系，构建用户数据模型；

任职要求：

1、硕士及以上学历，数学、统计学等相关专业，1年工作经验左右或实践经验丰富的应届生；
2、了解统计学原理和常用数据挖掘算法，有文本处理相关经验的优先；
3、熟练使用Python或者R语言中的一种；
4、熟悉Mysql，Sql server 2008等关系型数据库中的一种； 
5、熟悉互联网数据采集，具备较强的大数据数据分析、挖掘经验； 
6、出色的逻辑思维能力、沟通交流能力和团队精神

中润普达（集团）有限公司成立于2010年1月25日，是国内领先的中文大数据公司

集团已先后服务过国家23个部委单位、48家直属央企以及全国各地60多家特大型跨国公司及民营企业。
集团已在北京、武汉、江苏、浙江成立四个战略性的大数据产业基地，并联合湖北、江苏、浙江、北京等省市国资、行业龙头企业及银行共同发起成立和筹备了多家合资公司，其中包括武汉东湖大数据交易中心股份有限公司、九派（武汉）全媒体股份有限公司、华东江苏大数据交易中心股份有限公司、浙江钱塘工业大数据交易中心股份有限公司、九派文化大数据交易中心股份有限公司等。

我们的方向
选择大于努力，一个正确的方向，会引领你到达不同的高度。

中润普达集团秉承大数据为基础，技术为驱动，产品为优势的发展方向，通过整合大数据产业上下游优势资源和平台应用形成全产业链的覆盖。目前集团在非结构化数据分析、分布式矩阵计算和全网中文数据监控三类技术方面具有行业领先的先发优势，拥有3大专利权、6大核心知识产权、100多套大数据应用模型知识产权以及教育、健康等7个大数据平台知识产权。
为商业客户提供敏捷BI、媒体融合、价值挖掘、精准营销等服务，实现企业客户的高速发展要求；
为政务客户提供数据整合、舆情研判、决策辅助、政务公开等服务，实现政务客户的高效管理需求；
为个人客户提供数据支持、高速分析、模块支撑、数据跟踪等服务，实现个人客户的高度应用述求。
