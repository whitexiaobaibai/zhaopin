举报收藏

岗位职责：1.负责大数据采集分析系统需求分析和架构设计，确定软硬件实施方案；
2.参与分布式爬虫和数据采集系统的架构设计和研发；
3.负责日常网络数据抓取需求的实现及采集数据的质量监控；
4.根据公司产品和业务发展特点，研究相关大数据产品和技术发展方向。


任职要求：1.熟悉金融相关业务、互联网业务模式和用户行为分析模型；
2.计算机或数学等相关专业，2年以上大数据系统分析、设计和实施经验；
3.熟悉Hadoop，HDFS，Hive等分布式大数据技术；
4.熟练掌握 Java ， Shell，Python 等一门以上脚本语言，具备反爬虫相关能力；
5.逻辑思维能力强，对数据敏感，有较强的学习能力和创新思维；
6.具备良好的沟通能力和文字表达能力，有较强的团队协作能力；
7.具有大数据项目成功经验者优先。
8.具有验证码破解经验者优先

工作地址：

公司概况
    20余年专业经验，成功助力万千客户成长，香港亚洲联合财务有限公司作为香港个贷行业引领者，于2007年投资成立“深圳亚联财小额贷款有限公司”，积极投身到大陆个贷市场的开拓。目前在全国范围内，已拥有十二间城市公司，包括深圳、沈阳、重庆、天津、成都、云南、大连、北京、武汉、上海、福州、哈尔滨等，员工4200余人，其中深圳公司为内地总部，拥有1000多位业界精英，已开设43间分行，发展潜力巨大。


【亚联财小额贷款】拥有雄厚的资金实力和丰富的风险管理经验，以创新的管理理念、全新的服务模式、杰出的管理团队，为社会大众提供最方便快捷的小额贷款服务。张董事长亦为亚洲联合财务有限公司董事总经理及行政总裁，拥有三十多年私人贷款市场经验，在香港素有【个人信贷之父】的美誉。【亚联财小额贷款】正以领先的经营模式为广大深圳市民及中小企业主提供最方便快捷的小额贷款服务。
 
产品介绍
【亚联财小额贷款】的产品细分贴合大众需求，除了针对个体户、中小企业主的“企业贷”、“生意贷”，还有针对个体一般消费者短期消费的“薪易贷”，针对有房一族的“楼易贷”，针对有车一族的“车主贷”，都是经过香港专业的财务师针对内地市场精心设计、研磨出来的产品。（贷款额高、手续简便、快速批核、即时到款）。
 
公司的使命
【亚联财小额贷款】解决客户财务急需，助力社会经济发展，为股东及员工赚取合理回报。
 
公司的愿景
【亚联财小额贷款】全力支持客户的理财需要，帮助客户实现生活和事业目标，实现与客户共同成功。这个理念是【亚联财小额贷款】始终如一的目标。
 
公司的发展
【亚联财小额贷款】立足深圳，面向全国发展！现有43间分行（营业网点）遍布在深圳各行政区和商业圈；2009年起公司进入全面高速发展的阶段，在深圳、沈阳、重庆、天津、成都、云南、大连、北京、武汉、上海、福州、哈尔滨等城市先后成立了公司，意图在全国布局不断发展。

公正——合乎社会常规下，秉承公平公正之态度判断及处理事情。
诚信——绝不轻言承诺或决断，充分收集信息、多预案比较、审慎评估后决定，言出必行。

员工的发展
【亚联财小额贷款】致力于专业人才的培育与发展，注重对员工的专业引导，提供广泛机会，大力培养职业精英，通过人力资源的不断升值，形成公司的核心竞争力，提升公司在业内的地位。目前公司已建立并完善了人才的选、用、育、留管理体系，为员工设计出合理的晋升通道，公司多数中基层管理人员来自优秀员工的晋升。员工一经录用，待遇从优，并享有以下福利：
1、薪酬福利
根据员工的专业技能、职业素养、绩效等因素调整薪资，鼓励先进、优秀员工获得相对较高的薪资。
（1）薪水：具有行业竞争力薪水，固定工资+餐补+年底双薪+花红
（2）业绩：提成根据完成业绩任务比例享有相应的业绩提成（销售代表）
（3）福利：商业团险、五险一金（养老、医疗、工伤、失业、生育、住房公积金）
（4）培训：提供相应的岗位、专业等培训教育机会
（5）假期：转正员工享有7个工作日带薪年休假、12天有薪病假、国家法定各类假期
2、晋升
提供公平、开放的竞争平台，鼓励员工积极向上，竞争空缺职位，员工享有提升、晋升机会。
3、职业生涯规划
根据员工职业发展意愿，并结合员工知识、技能及岗位要求，合理安置员工，在员工职业生涯方向提供指引，帮助员工更好地规划工作、生活，帮助员工成功，提升员工生活的追求和生活质量。
4、生活福利
(1)提供免费奶品、咖啡、茶饮；
(2)提供年度健康体检、旅游度假活动；
(3)定期组织员工参与各类体育运动、户外活动，现有员工兴趣小组活动（足球队、篮球队、羽毛球队、乒乓球队、瑜伽健身班、区域活动小组）。

5、工作时间
标准工作时，五天八小时。周一至周五，早9：00-晚18：00。
