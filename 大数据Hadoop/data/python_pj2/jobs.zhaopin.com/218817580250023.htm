举报收藏

大数据信用平台产品经理岗位要求：

1. 硕士及以上学历，一年以上IT产品经理从业经验，有金融相关从业经验的优先；

2. 具有较强的团队组织协调能力和团队合作精神，善于与人沟通和交流，为人正直、踏实，有强烈的事业心及责任感，有高尚的职业操守与道德品质；

3. 对网络新产品敏感，有强烈责任心和创新精神，具备良好的职业素养及团队合作精神。

4. 能承受较强的工作压力。

 

大数据信用平台产品经理岗位职责：

1. 负责信用平台产品规划，设计符合市场需求的业务模式，制定平台的发展策略、计划和目标，完成功能规划、产品流程架构、产品形式设计等工作；

2. 研究行业和主要竞争对手的动向，跟踪国内外相关新技术和新产品动态，为公司决策层及时提供前瞻性的建议和方案；

3. 定期与客户和用户保持接触，了解客户和用户对产品的反馈，实施客户/用户调研项目，收集、分析、提炼客户/用户需求，进行产品策划与创新；

4. 根据产品市场需求与技术配合，高质量完成公司产品的设计，并结合用户体验完善界面设计；

5. 配合商务推广、市场宣传等工作。


工作地址：

北京市朝阳区安慧里四区五矿大厦8层

联合信用征信股份有限公司 该公司其他职位

联合信用征信股份有限公司总部设在天津，注册资本5000万元人民币，是联合信用管理有限公司旗下的专业征信机构，是天津市第一家取得中国人民银行《企业征信业务经营备案证》的征信机构。公司主营：企业征信、企业信用信息的评价及利用；社会信用体系建设及相关主体评价；市场信用调查、信用风险咨询；数据库及计算机网络服务。公司依托多年积累的征信技术及信息渠道资源，在企业征信业务上建立了产业链服务、金融服务、监管服务、社会信用体系建设服务等多个产品线，与国内多家大型企业、金融机构、政府部门建立了长期稳定的合作关系。我公司也在积极探索个人征信及互联网大数据的相关服务，并已向中国人民银行申请个人征信牌照。母公司联合信用管理有限公司是经国家工商总局核准，于2000年在北京成立的全国性专业信用服务机构，注册资金1.36亿人民币，在全国设有36家分支机构，是我国最大的信用服务机构，主要业务包括评级、征信及咨询三大板块，在信用服务领域的多个市场中连续多年市场占有率名列前茅。
