举报收藏

工作职责

1.   协助产品经理进行大数据产品规划、需求分析、功能描述及相关原型文档，并撰写相应产品文档以及持续改进和完善等工作。

2.   负责跨部门协调资源，跟进研发和测试进度，确保产品开发周期、进度及质量。

3.   对产品运营数据做收集，整理客户反馈、统计客户使用行为，通过定性和定量的分析，持续改善优化产品，提升用户体验。

4.   关注用户体验，对所有影响用户行为和使用体验，及时将获得的数据和分析结果输出报告，反馈相关部门，积极有效地推动活动和产品改进。

5.   对核心业务指标进行监控分析，并反馈给产品经理、运营人员，并定期输出运营报告。

6.   进行市场调研、客户调研、竞争对手产品的分析和研究，准确把握同类产品及市场发展趋势。

7.   负责产品日常管理与维护，进行产品持续改进。

8.   协助产品运营经理完成其他日常事务。

任职要求

1.   本科及以上学历；有相关互联网产品工作经验一年以上。

2.   熟悉互联网产品特性以及用户行为，对数据敏感,具备优秀的数据统计分析能力，并将分析结果用于产品及运营改进。

3.   熟悉互联网及移动互联网动态及流行趋势，善于独立思考和钻研。

4.   具备较强的沟通协调能力，擅长多部门沟通,快速推动项目。

5.   工作态度负责、严谨、认真，能承受较强的工作压力。

6.   有意愿加入初创公司，适应快速的工作节奏，为团队带来正能量。


工作地址：

上海市黄浦区山东中路311号6楼

信励(上海)信息科技有限公司 该公司其他职位

simplyBrand（信励(上海)信息科技有限公司）成立于2013年，专注于互联网大数据技术服务，提供基于数据驱动的商业洞察解决方案。通过采集分析海量互联网UGC(用户贡献内容)数据，帮助企业挖掘数据的商业价值，更轻松、简单、智能地掌握价值信息，提升在研发、生产、营销等方面的科学决策。

总部上海，设有北京、香港、日本、台北、新加坡业务联络处。核心团队长期致力于互联网消费者行为洞察研究、行业竞争分析咨询及媒体情报监测工作，在软件技术、数字营销、数据分析方面积累了丰富的实战经验。数据产品服务线包括消费市场分析工具、微信公众账号分析平台、数据API接口，及消费者行为洞察、互联网市场研究等解决方案，已为快消、金融、零售等多个行业知名品牌提供服务。

simplyBrand通过服务行业领先用户及特定痛点需求用户，积累大数据算法模型，加强对行业知识及领先品牌主用户的服务经验，提升行业的专业分析能力；同时，也开发标准数据分析平台如simplyKOL及API接口产品，满足一般企业用户的数据使用及分析需求，培育目标用户群的同时，增加企业的营业收入。
