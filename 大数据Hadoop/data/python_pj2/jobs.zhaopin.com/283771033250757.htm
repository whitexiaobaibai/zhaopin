举报收藏

岗位职责：

1. 负责行业项目的客户交流、市场调研及售前支持，牵头组织行业咨询项目实施， 帮助客户清晰的规划实施路线，及提供针对性的可落地的建议方案，包括现状评估、实施设计、规范文档等；

2. 负责行业知识模块提炼及售前技术支撑工作（设定范围、提出解决方案、客户交流、项目投标书编写及讲标）；

3. 跟踪和分析行业动态、客户市场动态数据，负责配合公司制定行业业务市场开拓计划以及研究成果的市场推广；

4. 负责调研行业用户需求，将行业内的业务需求转化为数据及模型需求；

5. 对行业、市场、用户需求等方面定期提出分析报告，为公司的市场方向、市场合作、产品研发等提供建议。


任职要求：

1. 具备智慧城市、智能制造、大数据、机械、汽车等行业背景，具有大型项目IT整体规划、售前咨询经验；

2. 熟悉云计算、大数据、MES、DNC等相关产品和技术，有大数据技术及理论基础；

3. 能独立编写优质的售前技术方案，制作方案演示稿，跟踪制造企业智能制造转型升级的相关需求，进行商机跟踪、客户拜访、维护客户关系等；

4. 跟踪政府及行业智能制造相关政策，跟踪行业信息及产品技术发展动态。为制造企业提供工业化及信息化改造的咨询方案。

工作地址：

北京市海淀区西四环中路16号院7号楼16层

公司简介 


    航天云网科技发展有限责任公司是中国航天科工集团公司联合所属单位共同出资成立的高科技互联网企业。公司拥有9家云制造、大数据、智能制造等新一代信息技术子公司，面向全国实施战略布局，积极拓展德国、俄罗斯、伊朗等海外市场。面向用户提供平台门户、云制造、大数据应用、创新创业、工业品商城及产融结合服务，构建适应互联网经济业态与新型工业体系的航天云网生态系统。 
    航天云网公司作为“互联网+智能制造”领域先行者，获得了广泛市场认可，被评为国家级双创示范基地的支撑平台、国家“互联网+”重大工程、首批国家智能制造试点示范工程、中德智能制造合作试点示范项目，入选中国“互联网+”行动百佳实践案例。
