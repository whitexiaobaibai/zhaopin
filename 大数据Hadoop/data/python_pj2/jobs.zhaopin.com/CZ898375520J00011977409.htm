举报收藏

职责描述：

1、管理技术研发团队，指导系统架构设计、开发等工作；

2、依据业务需要做出技术相关的决策、规划；

3、制定良好的技术开发体系，持续提升公司整体的技术能力及可持续性、延展性；

4、协助公司的产品方向事宜，与公司核心团队共同设计、规划产品，进行业务决策；

5、主导推进技术难题攻关，持续提升核心系统在大数据、高并发请求下的高处理能力；

6、主导推进技术水平提升，保证系统运行安全、数据安全、交易安全；

7、研究行业最新产品、大数据风控技术发展方向，有效与产品结合，制定技术发展战略规划并指导、监督、保障计划落实与效果。


任职要求：

1、毕业于211院校、统招本科及以上学历；

2、8年或以上互联网金融贷端行业技术经历，3年以上互联网行业同等管理层任职经历；

3、有产品或技术团队管理经验；

4、强大的代码编写能力及系统分析能力；

5、具备丰富的开发及产品方向员工核心能力提升经验；

6、具备研发过程管理和控制技能；

7、具备较强的沟通能力及团队合作精神，较强的创造力；

8、负责过大型APP后台的开发优先；

9、负责过大型反欺诈系统开发优先。

北京科创佳投资咨询有限公司 该公司其他职位

北京科创佳投资咨询有限公司创立于2014年，是面向国内中、小、微型企业和社会各类投资主体，基于企业产权的全链条管理，针对各类投资主体的业务整合和资本运作等需求，提供关于投融资咨询等服务的专业机构。公司已形成在产权交易、招商推介、尽职调查、投资估值、并购重组、投资咨询等方面的全要素综合服务能力。通过不断满足客户的市场化、专业化和个性化需求，与客户建立了彼此信任的长期合作关系，业务规模因此获得了长足的发展。通过积极开拓、锐意进取，规范操作，积累沉淀了一套卓有成效的处置服务与风险控制体系，建立了一支实践经验丰富、业务能力复合、执业操守良好的专业化团队。

二、公司业务范围

1、融资财务顾问服务

针对企业发展以及企业项目投资中的资金短缺问题，北京科创佳投资咨询有限公司致力于为中国快速成长的中小企业提供量身定做的全程融资顾问服务，协助企业从银行贷款、私募融资、企业集合债、融资租赁和上市前筹资等多渠道筹措资金，以及引进产业合作伙伴、政府优惠政策等综合资源。

北京科创佳投资咨询有限公司企业融资财务顾问服务的主要服务流程和工作内容包括：

（1）基于融资角度，对企业/项目所在行业前景、企业经营情况、财务状况、技术水平、市场前景、发展战略及管理团队等影响融资成功因素进行详尽的调研，通过专业化的分析及与企业管理层关于发展策略的交流，对企业/项目进行估值，进而对企业/项目价值得出客观、全面的判断，从而完成企业/项目融资评估;

（2）基于具体融资需求及融资评估，以及投资人投资特点、产权交易所与多家金融机构的并购贷款等领域的合作渠道，对比分析融资渠道，准确定位与资金需求相匹配的渠道群体;

（3）基于融资评估及融资渠道选择，协助客户进一步明确企业融资活动需要具备的基本条件、最佳融资渠道及融资活动成功的关键要素，并综合考虑各种融资工具的优劣势，制订最优融资方案;

（4）按照不同融资渠道要求，提供符合国际、国内惯例的企业投资可行性分析报告、商业计划书、行业分析或财务计划等相关资料，全面、客观、清晰地分析和展示融资企业的商业模式、成长潜力、现实估价和融资结构等整体情况;

（5）通过立体多维的推介渠道促进与投资人的实质性接洽、沟通与反馈，策划并安排融资渠道现场参观与考察，引导并参与融资企业演示、谈判等相关工作;

（6）针对不同融资渠道对项目评审过程的差异，协助客户施行有效的项目过程交流与控制，强化后续跟进，直至完成融资活动。

2、商业与财务尽职调查服务

企业融资活动中，尽职调查是一项不可或缺的重要工作，有助于增加对投资人自身战略及被投资企业真实经营状况的准确了解，并在此基础之上对并购决策的合理性作出准确的判断，确保并购能够增加企业的价值。

北京科创佳投资咨询有限公司，以独立的第三方身份，主要提供以下类型的评估商业前景的商业尽职调查和评估实际资产和负债情况的财务尽职调查服务：

（1）投资或并购非上市企业股权的尽职调查;

（2）获取非上市企业经营权的尽职调查;

（3）收购破产重组企业债权时实施的尽职调查;

（4）针对公司新上任的管理层在短时间内迅速了解公司情况的尽职调查;

（5）针对集团公司下属企业进行业务整合时实施的尽职调查。
