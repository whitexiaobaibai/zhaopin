举报收藏

岗位职责：

1、负责从事政府、银行等企事业单位相关产品和大数据项目的产品规划、策划、设计工作；

2、对产品的易用性和可用性进行分析和研究，包括产品调查、市场需求分析、用户研究、可用性测试、行业研究等；

3、对产品进行结构、流程的优化，提出创新功能、界面用户体验、交互做出研究与设计；

4、制作管理产品开发文档，与技术开发团队沟通合作，控制产品开发周期，监督项目执行；

5、负责不断优化产品体验，对研发团队交付的产品进行审核，确保高质量产品按时上线。 

任职要求：

1、本科及以上学历， 计算机相关专业，熟悉互联网行业和大数据分析技术；

2、至少1年以上银行或政府等企事业单位项目管理或者产品设计经验，熟练使用产品经理常用的Axure、Visio和Mindmanager等相关软件；

3、具备独立策划软件系统产品能力，组织过产品立项、设计、研发、推广应用及维护产品周期全过程；

4、具备的较强的逻辑思维能力，以及分析和解决问题的能力， 善于沟通，责任心强；

5、具有较强的沟通能力，对外能和客户顺畅的进行沟通和交流，对内能和产品开发团队无障碍沟通；

6、能准确捕捉客户和用户需求，对市场及用户体验敏感，产品企划能力强。

薪酬福利：

1、薪水：底薪+绩效+全勤奖励+各种补贴，工龄奖、年终奖等；

2、福利：提供养老、医疗、工伤、失业、生育保险和公积金；
3、假期：工作满一年5天带薪年假、周末双休、法定假日；

4、公司年度体检，每季度组织团队活动、每天水果零食、员工生日、过节礼品等。

工作时间：周一至周五 9:00-18:00

面试地址：海淀区苏州街16号神州数码大厦10楼1020-1022室

乘车路线

路线一：地铁十号线苏州街站B口出，往西走20米后往北走300米，红绿灯对面即神州数码大厦。

路线二：公交车运通114、运通118、运通110、26、302、374、528等北京市地震局站即到。

北京一览群智数据科技有限责任公司 该公司其他职位

    一览群智数据科技有限责任公司于2015年成立，是一家具有国内领先技术的大数据分析技术和解决方案提供商。公司的技术团队具有深厚的技术实力，在信息检索、自然语言处理、语义分析、时空数据管理分析方面都具有独到的优势。

一览群智坐落于中关村核心地带，从创始第一天起就具有深厚的互联网基因和技术基因。公司核心团队由文博士领衔，文博士是国家千人计划专家，中科院博士，微软亚洲研究院前高级研究员。在大数据分析、信息检索、人工智能、机器学习等相关领域中具有广泛国际影响力，曾主导开发过诸多大型知名系统和产品。核心团队还包括数名大数据技术方面的博士、具有丰富企业管理经验的资深专家，以及来自于国内外一流大学与技术公司的研发人才。

一览群智是一个兼具梦想和执行力、践行硅谷工程师文化、以实现客户价值为第一原则的高科技企业。

一览群智的愿景是：

•用大数据支持我国的产业升级

•用大数据改善、改变国人的互联网生活方式

•用大数据给客户带来大价值
