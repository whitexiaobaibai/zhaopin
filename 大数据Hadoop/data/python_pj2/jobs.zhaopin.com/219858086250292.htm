举报收藏

岗位职责：

1. 负责开拓目标行业市场，挖掘客户需求，为客户提供方案型销售服务。
2. 负责商务洽谈及合同的签署，达成公司及部门下达的年度销售目标。
3. 协调客户和内部资源，为客户解决售前、售中及部分售后问题，提高客户满意度。
4. 维护良好的客户关系，提升客户对产品的认可度、服务的满意度、公司品牌的美誉度。 

任职要求：

1. 2年—5年左右B2B领域的IT/互联网行业从业经验。
2. 有SaaS、移动互联网产品销售经验优先；开展过社群客户或者协会客户合作，有社群客户方面资源优先。 
3. 具备的核心能力：任务执行能力、市场开拓能力、方案营销能力、客户公关能力、解决问题能力。
4. 工作积极热情，抗压能力强，可接受一定量的出差安排。
5. 形象良好，性格开朗，思维活跃，逻辑性强。

北京慧友云商科技有限公司（大家社区）成立于2014年，是一家致力于为企业服务的新型互联网公司。

我们的产品

北京慧友云商科技有限公司是“企业级移动互联网服务商”，通过核心产品大家社区，为企业提供一站式的互联网转型服务，让企业快速拥有个性化App、微信公众号、移动网站、百度直达号、二维码等，快速实现企业互联网+落地转型，并提供后续运营维护、升级迭代等一整套打包服务，让企业省时、省心、省钱的完成互联网+转型。

大家社区基于SaaS对企业的帮助帮助服务型企业进行改革升级，帮助创业者加快创业速度，同时针对传统企业转型互联网+进行革命性升级，让科技技术与市场挂钩，帮助一切有价值的东西通过短时间的方式完成互联网转型。

我们的团队

来自阿里、思科、用友等知名企业的精英，和曾经多次互联网创业的达人和草根，为了“互联网+”的创客梦想加入团队，这个创客团队，堪称业内最豪华的阵容，正是世界领先的产品方向和强大完善的创业团队，让大家社区在2014年轻松获得千万美元的A轮融资。

我们的客户

大家社区推出以来，客户量迅猛增长，短短半年时间已有近8000家企业正式入驻大家社区，其中不乏各行业知名企业。被称为“保险行业互联网公司”的天安人寿保险、中国亲子阅读第一品牌悠贝、中国百货商业协会、内蒙古中小企业商会、锡林格勒盟中小企业服务平台、百科弗在线教育、中国最大的OA软件提供商致远软件等知名政企事业单位，都是走在“互联网+”潮头的幸运儿和成功者。

 大家社区在创业初期就吸引了大量的同路者共同合作，众多的优秀互联网软件企业成为大家社区的核心伙伴，与大家携手实现互联网+创客的梦想！目前大家社区已在全国发展了渠道伙伴500余家。

我们能提供什么？

● 优秀的职场伙伴：这里有一群来自Google、Zynga、SAP、用友等互联网企业的创业精英。 三人行，必有我师。优秀的职场伙伴将是你职业生涯最重要的财富！

● 充实的现在和充满期待的未来：我们已经获得A轮融资，加入我们的创业团队，除了获得充满竞争力的薪酬福利之外，你还将有机会获得更多挑战，收获属于自己的未来。

● 开放融洽的工作氛围：我们打造了舒适有趣的办公环境，开放式的沟通平台。我们主张开放、融洽、良性竞争的工作氛围。一切不以赢得客户为目的的工作都是耍流氓。

● 工作和生活的平衡：我们主张高效率的工作，保持工作和生活的良好平衡。

● 丰富多彩的员工活动：生日会、旅游、运动会、员工分享会、家庭日等多主题“腐败”活动，加入大家社区，收获家一般的温暖。

 

CEO：文杰，中国最早接触云计算SAAS的高管之一，曾就职于贝尔实验室、SAP、微软亚洲工程院、用友，担任过公司副总裁，BD及研发总负责人；在2004年即加盟SAP最大的企业级云产品Business Bydesign，是最资深的产品设计和负责人之一；随后，在微软和用友又负责了医疗健康服务云产品和企业级移动云产品的相关规划、研发和BD，对企业社区的设计和运营有十分丰富的经验。文杰在美国拥有大数据专利一项，在中国拥有与移动和云相关的专利三项， SAAS业务专著一本。曾有两次互联网创业经验。
