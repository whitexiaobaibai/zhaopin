举报收藏

职位描述：

1. 基于第三方数据和客户数据，深度分析数据中的规律，挖掘数据价值；
2. 参与数据/工具平台相关的功能接口、数据接口，实现业务功能；
3. 参与数据平台/工具平台的架构、设计以及实现；
4. 与数据计算/存储团队合作，优化数据平台的性能，解决兼容性、功能性Bug，提高稳定性。

职位要求:

1. 熟悉Java、Scala、Python等开发语言中的一种，多种为佳。
2. 精通Hadoop、HBase等大数据开发，对hadoop/hive、MapReduce编程或Spark编程有一定的了解；
3. 能够熟练使用Spring/SpringMVC/iBatis等开源框架；熟悉Tomcat（或Jetty）常用的WEB容器；
4. 熟悉MySQL（或Oracle）与SQL语言；
5. 能独立了解数据需求，并可以转化成系统结构设计，并能独立进行代码实现。

北京无限讯奇信息技术有限公司 该公司其他职位

北京无限讯奇信息技术有限公司（以下简称“无限讯奇”）成立于2006年，是中国移动12580业务独家合作伙伴，负责开发和建设中国移动12580业务核心系统、内容采编与发布、广告销售、用户拓展与服务等运营工作。

自成立之初就确立以坚实的技术基础为用户提供全面准确的信息服务。秉承“卓越、优选、精准、尊重”的理念，持续为客户创造关键价值，为用户优质的服务。公司成立10年以来积蓄了当前先进的技术研发与创新能力，广泛采用基于大数据、云计算和移动互联等新兴技术，为高科技、互联网、电信、媒体、旅游、金融、医疗等行业客户提供丰富的解决方案及产品。公司已获得CMMI 3、ISO 9001、ISO 27001，ISO14001，ISO18001，系统集成三级等一系列资质认证。

无限讯奇重视研发和创新能力的提高，通过设立研发中心培养优秀的技术团队，潜心研究搜索引擎前沿技术，引擎技术拥有多项专利。在生活服务领域的同类产品评测中，准确性和性能方面在多个参数上领先于竞争对手。同时公司在数据分析领域也独树一帜，正在研发有关数据分享交易平台。在自然语言处理、机器学习、数据挖掘、推荐系统、位置服务、大数据分析等领域，经验丰富。
