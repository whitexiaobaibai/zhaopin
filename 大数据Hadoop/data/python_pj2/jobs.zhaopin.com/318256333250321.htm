举报收藏

北京凯润婷医疗美容医院


岗位职责：

1、负责医院网电咨询中心商务通组或大数据组顾客的咨询、解答和销售工作；

2、引导顾客到院面诊，做好衔接；

3、准确及时录入客户信息和咨询内容；

4、适时对预约及登记未到诊客人进行跟踪回访；

5、对到诊顾客进行跟踪，并与现场咨询医生保持沟通，协助顾客开发；

6、遇到特殊顾客及时记录客人信息，并向上级主管进行反馈；

7、参与部门每天例会，对一天工作进行总结，工作中遇到问题及时反馈。


任职资格：

1、20-35岁，高中以上学历，口齿清晰，普通话标准；

2、护理、医学美容、临床医学或营销类专业；

3、具有医疗、整形或疾病类行业电话、网络资询、客户回访工作经验；

4、具有良好的沟通销售技巧、电话沟通技巧以及良好的文字和口头表达能力；

5、具有良好的合作意识和团队精神、服务意识，适应能力强；

6、善于学习，爱岗敬业，有良好的工作态度。


工作地址：

北京市朝阳区双井富力城A9-6

北京凯润婷医疗美容医院有限公司 该公司其他职位

企业简介：

    2007年4月26日，一群怀揣梦想的年轻人，带着他们引以为傲的医疗技术扎根于北京，本着“专一于医疗，专心于技术，专注于客户”的理念，成立了凯润婷医疗投资管理有限责任公司，经过八年的不懈的耕耘，这群当年蹒跚学步的坚强少年们，现今已成了医疗美容行业中当之无愧的技术潮流引导者，两项医疗专利技术的拥有者，100余名国内外高级职称医师的合作者，

2010年，凯润婷医疗先后成立了凯润婷生物科技有限公司和欧朗生物科技有限公司

2009年，公司收购有10年历史的宁夏华美医疗美容整形医院

2013年，收购拥有30年发展历史并在中国整形行业具有里程碑意义的北京史三八医疗美容医院。

2014年，公司与中国地产业巨头深圳宝能集团达成协议，完成增资扩股，正式迈向扩张之路。

2014年，凯润婷继续豪情拓步，收购了苏州瑞芙臣医疗美容医院。

2014年，为了更好服务于全球爱美人士，凯润婷收购了北京国际医疗中心整形美容科，与世界医学同步，从预防、干预、治疗、医美、抗衰五步进行由内而外的涉外医疗。

2015年，在与国际先进技术接壤的澳门，成立了以集团为名的美容诊所，继续汲取世界医美精粹。

上海，成都，武汉，天津，沈阳等

凯润婷医疗美容医院将陆续开业，届时公司将完成覆盖中国三分之二人口区域的布点建设。

将统一的理念，更加精湛的技术，以及完善的售后体系，带过国内的爱美人士。

“敬重生命，追求完美”是凯润婷人对专业医疗，专心于技术的核心诠释。

“诚信经营，以义取利，利益共生”是凯润婷人对自己、对客户、对社会庄严的承诺。

 

公司未来规划：

集团将完成覆盖三分之二人口区域的布点建设。
