举报收藏

公司介绍：

 猎你网隶属于领猎网络科技（上海）有限公司，是一家提供专业猎头服务的互联网招聘平台。平台成立于2014年7月，依托于超过3000名专业猎头顾问，利用互联网优势，以用户信用评价体系，通过大数据应用，精准匹配，覆盖了12大行业，为5000家企业提供标准化猎头服务。

岗位职责：

1. 按照公司及部门发展的策略， 积极开拓新的优质客户；
2. 落实项目的具体细节，项目关键节点的把握、控制，以及难点解决；
3. 业务款项的回收；
4. 充分培养下属，招聘合适的团队成员；
6. 按时完成量化任务及考核；
7. 保持部门内及跨部门的团结协作；
8. 积极完成公司布置的全年目标任务。
我们期待你：
1. 大专及以上学历；
2. 具有人力资源行业业务管理岗位1年左右工作经验；
3. 具有培养新人（入行——成熟）全流程的完整经验；
4. 出色的项目操作能力和客户沟通能力；
5. 出色的团队管理能力，富有高度的责任心与职业操守；
6. 能够承受工作压力，具备独立的发现、分析及解决问题的能力；
8. office办公软件操作熟练。

薪酬福利：

1、薪资： 底薪6—10K，根据能力拿底薪，高额提成

2、福利： 团建活动、员工旅游、带薪年假、弹性工作、下午茶、生日福利、结婚礼物

3、文化：“家”文化氛围，给予你温暖和爱的工作环境

我们是一群对人力资源充满热情的团队，公司以90后为主力，公司氛围非常好，提供完整体系化的岗前带薪培训。欢迎优秀的你加入！

“互联网＋”时代，共享经济的商业模式正渗入各行各业。猎你网探索猎头行业共享新模式，整合猎企资源，推动猎头行业转型升级。

提高企业招聘效率，降低企业招聘成本，实现猎头与企业互惠共赢，打造中国专业的一站式猎头服务平台。猎你网，为创业者招聘。

2014年7月，猎你网成立于上海；目前猎你网拥有超过2000名活跃猎头顾问，平均面试率超过40%，覆盖12大热门行业，分布50多个城市，拥有着超过150万的中高端人才报告，为5000家企业提供完整的猎头服务。

猎你网以用户体验为中心，对招聘结果负责，入职再向企业收费7×24小时为您服务。猎你网创始人精英团队，来自国内知名猎企、一线互联网公司，致力于解决企业招人难的痛点。
