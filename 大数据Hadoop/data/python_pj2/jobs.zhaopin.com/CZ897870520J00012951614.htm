举报收藏

1.  负责工程项目的现场组织和管理工作，包括设备调试与验收等；

2. 具体工作内容包含但不限于：编写设计文档、集成图纸设计、现场调试、项目验收、竣工资料整理及归档；

3. 负责与项目中对口的部门及专职人员之间的协调和沟通；

4. 完成对用户的技术培训，包含系统操作培训和系统维护培训；

任职要求：

1、具备电气及控制系统CAD识图和制图能力；

2、良好的团队意识,服务意识以及较强的责任心；

3、能接受较为频繁的出差
