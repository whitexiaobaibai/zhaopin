举报收藏

职位描述

参与和设计开发大型数据仓库和基于MPP数据库（Vertica）和公有云（Azure）的应用。有大数据的处理经验（每日TB数据的处理经验）和善于在数据中挖掘出特定模式，和各类数据之间的联系，在此基础上进行统计和预测分析。了解当前流行的技术趋势，数据流处理，内存管理，并行计算，分布式消息处理，机器学习和性能调优。该职位的职责需基于大数据平台（结构化和非结构数据），分析，设计和开发以数据为中心的应用，满足客户快速和准确访问到数据的需求。灵活应用消息处理，负载均衡，分布式运算等技术开发和部署基于业务需求的算法。需要有良好的团队协作能力，对软件工程有良好的理解，并且适应敏捷开发的流程，快速迭代的方式来提交高质量的软件及应用。

 

日常职责

• 设计和开发数据仓库应用，包括ETL，基于业务需求和算法的数据运算

• 善于从技术角度和业务角度对应用进行调优

• 与数据科学家合作，部署和维护高级算法（预测算法，机器学习）

• 善于学习和利用当前最流行的技术搭建和开发大数据分析平台

• 有良好的独立工作能力，对测试和如何保证代码的功能完整性和准确性有良好的理解

• 能够遵循良好的软件开发流程，从来开发和提交高质量应用程序

 

任职要求

• 本科及以上，在计算机相关行业从事软件工程师，数据仓库工程师，数据库工程师有3年以上工作经验

• 3年以上传统型数据库(sqlserver, oracle)或者nosql数据库工作经验

• 精通脚本语言编程(python, powershell)

• 3年以上大数据平台和分布式环境开发经验

• 对设计模式，性能调优，内存管理和多线程同步有较深的理解

• 对MQ和任务调度有了解

• 有实际的基于公有云的开发经验，比如Azure，AWS，阿里云或腾讯云

• 有较强的分析能力和技术能力，可以以创新的思路来解决现实复杂的需求问题

• 有较强的英语沟通能力

• 有技术管理经验更好

• 能适应适度的加班


POSITION SUMMARY: 

Design and develop large scale data warehouse systems and related applications based on MPP Data Warehouse such as Vertica, as well as the cloud platform such as Azure. A typical application would process terabytes of data on a daily basis to determine meaningful patterns and trends, perform statistical and predictive analysis, detect relationships embedded in the data, and deliver mission-critical insights. Given the massive data volumes and time-sensitive goals of the analysis, applications need to use the latest techniques in data streaming, memory management, parallel execution, distributed messaging and machine learning, and code optimization. The responsibilities of this position include the analysis of requirements for large-scale, data centric software applications; the design and creation of schema and ETL for consuming different variety of data including unstructured data; the design and development of the applications which can provide the fast and efficient access to the data, can implement the business and algorithm logics utilizing the technologies such as messaging, task balancing and coordinating, distributed and elastic computation. Probe and tweak the performance interacting with Database engine. Work with team members delivering high quality applications complying the software engineering process and standard, and work with team in an agile way. 


ESSENTIAL DUTIES AND RESPONSIBILITIES:

• Design and implement data warehouse applications involving ETL, data manipulation and especially analysis logic for realizing business and algorithm logics.

• Collect performance information, Analyze them and optimize the performance

• Work with development management and developers in the team implementing applications that address business needs.

• Work with Data Scientists for implementing and supporting the applications incorporating the advanced algorithms.

• Develop Applications using the latest techniques in massively scalable parallel distributed environments.

• Independently design, code and test functionalities, as well as work jointly with other members of the core development team. 

• Develop high quality applications using object oriented programming and following a standard software lifecycle development process.


QUALIFICATIONS:

• Master’s degree or foreign degree equivalent in Computer Science, Computer Engineering, Mechanical Engineering, Electronic Engineering, Engineering, Mathematics or related plus 3+ years of experience in the occupation of Staff Software Engineer, Staff Data Warehouse Engineer, Senior Data Warehouse Engineer, Senior Database Software Engineer, Database Software Engineer, Software Engineer, or related. 

• 3+ years of practical experience creating analytic and/or numerically intensive applications using massively parallel distributed environments.

• 3+ years of practical experience developing large scale data analysis applications based on an RDBMS or no-SQL engine such as MS SQL, Hadoop based or Columnar database engine.

• In-depth knowledge of software engineering process, design patterns, and the performance tuning and memory management in multi-threaded asynchronous environments.

• Knowledge and experiences in MQ engine, task schedulers

• Practical experiences in environment setup and developing applications especially ML algorithms on Spark eco-system is preferred.

• Practical experiences in cloud computing platforms such as Azure or AWS is a plus.

• Proven track record of delivering high quality products on time. 

• Strong analytical and technical skills and demonstrated ability to creatively solve complex problems.

• Technical leadership skills is a plus


PERSONAL ATTRIBUTES:

• Good written and oral communication skills.

• Strong technical documentation skills.

• Good interpersonal skills.

• Ability to conduct research into algorithmic issues, JEE standards, and products as required.

• Ability to present ideas in user-friendly language.

• Highly self-motivated and directed.

• Keen attention to detail.

• Proven analytical and problem-solving abilities.

• Ability to effectively prioritize and execute tasks in a high-pressure environment.

• Strong customer service orientation.

• Experience working in a team-oriented, collaborative environment.

瑞易RSi是全球零售行业中最大的商业智能和数据分析服务提供商，旨在为CPG（消费品）公司及零售商提供全球化的大数据解决方案。

瑞易服务于快速消费品和零售领域的500多家客户。每天，来自全球上百家大型零售商，超过12亿条数据进入瑞易的数据处理中心。瑞易帮助客户清洗、整合、标准化海量终端数据，并在此基础上提供智能数据分析平台和应用，继续深入挖掘数据价值，找到执行中的异常问题点，包括针对铺货、动销、库存、促销等客户关心的方面给出可执行的行动指南并且跟踪其落地。通过不断的创新与服务，瑞易已成功帮助多家全球和本土客户在中国市场取得成功，包括有：联合利华、金佰利、蒙牛、伊利、好丽友、百事、强生、南孚、可口可乐、立白、亿滋等。

瑞易一直是零供数据协同计划的重要推动力量，希望通过更广泛深入的数据分享将零供协作推向全新的高度。目前，瑞易在中国、北美、欧洲、澳大利亚、中南美与多个零售商及其供应链伙伴正在开展数据协同战略合作。

瑞易连续七年被评为快消品领域最佳客户服务公司，并在福布斯杂志中被评为零售行业“最大的SaaS公司”（SaaS：软件即服务）。瑞易总部位于美国硅谷，全球11个分公司。瑞易中国拥有百人以上的服务团队，我们充分了解本地零售商的商业流程和供应商的商业需求，旨在提供本土化的解决方案。
