举报收藏

工作职责：

1. 负责业务需求的沟通、分析，挖掘潜在业务需求，撰写需求说明书；根据项目计划做好项目各个阶段的组织工作，将工作落实到位；承担业务需求到Hadoop技术体系应用转换的桥梁工作；

2. 管理大数据开发团队，负责执行项目计划并实现项目目标；

3. 负责对项目的全过程管理，并对结果直接负责；

4. 指导项目团队成员的日常开发工作，协调解决开发中的技术难题；

5. 负责调查项目的技术现状和技术需求，编写技术需求书和实施技术方案；

6. 负责大数据项目功能分解、功能质量保障、风险管理、与客户协调沟通；


岗位要求：

1. 本科以上学历，计算机相关专业，5年以上数据类项目相关工作经验；

2.      不少于3年10人以上团队项目管理经验；

3.      熟练掌握hadoop、hive、hbase、spark、kafka、storm等相关大数据技术，并具备基于以上技术的项目管理实施经验；

4.      对数据类项目的架构、流程、数据加工作业、作业调度、作业监控、元数据、数据质量管理有深刻认识，并能运用到大数据项目管理中；

5.      具备大数据采集、清洗、挖掘相关系统实施经验；

6.      有传统数据仓库项目管理实施经验者优先考虑；

7.      有很强的沟通表达能力和团队合作精神，能承受巨大的工作压力者优先。
