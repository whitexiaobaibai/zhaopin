举报收藏

【岗位职责】

1、负责大数据产品线团队的搭建、管理和绩效考核；

2、负责政法大数据产品的设计、建立、数据挖掘、分析的产品架构和运营模式；

3、负责政法大数据产品的研发管理和迭代计划，协调开发进度，推动公司政法大数据平台的建设；

4、组织实现公司内部和外部对大数据服务的需求，和相关产品推广、咨询、支持、培训等工作；

5、跟踪政法大数据应用动向，创新产品应用；

6、根据政法行业特点，提出产品提升建议与发展方向。

 

【任职要求】

1、计算机、数学相关专业，本科或硕士以上学历（硕士、博士）；

2、8年以上分布式系统、大数据专职工作经验；

3、主导过大数据产品/分布式架构的软件产品研发经验，含大数据采集、治理与分析研判；

4、熟悉Hadoop、Spark、Hive、HBase、storm、Kafka等开源项目和社区，至少5年以上产品项目应用研发经验；有Hadoop/Hbase/Cassandra/Storm/Spark/Hive 等多系统的开发经验者优先；

5、熟悉分布式搜索引擎架构，熟悉hadoop/hbase/hive/zookeeper等分布式计算和存储系统，熟悉常见的key-value存储系统(redis,,hbase ,mongodb)；

6、有以下其中之一者，更加优先：

(1)熟悉政法行业大数据特点，参加过政法行业大数据项目建设；

(2)大数据开源项目代码commiter；

(3)基于容器微服务开发经验；

(4)精通或者熟悉数据库设计，熟悉Oracle/MySQL/MySQL 集群等数据库，并具有较好的 SQL 编写及优化能力；熟悉NoSQL 者优先；

(5)具有较强的数据挖掘、优化理论、机器学习的理论基础和实际项目经验，精通数据分析与各种算法与模型，例如分类、聚类、Boosting、SVM、神经网络中的至少一种；

(6) 极强的系统设计和系统架构能力或者项目经验；必要的产品管理意识；

(7) 对分布式系统和云计算尤其是互联网云计算要有深刻的理解，最好有相关技术经验。

广州阳光耐特电子有限公司 该公司其他职位

风云际会，菁英携手，2000年10月，广州阳光耐特电子有限公司应时而生。公司由一批志在中国信息化领域建功立业的创业者组建。成立十余年来，企业凭借公司管理团队丰富的行业经验、一流的技术创新以及“视顾客为上帝”的服务精神，锐意进取，合作共赢，成功发展成为中国信息化领域一家实力雄厚的行业解决方案供应商和服务商。

阳光耐特总部设立于中国信息化建设快速发展的广东省广州市，在省内设有多家分支服务机构。经过多年潜心耕耘，精诚开拓，目前公司各类解决方案和技术服务已经为广东全省二十一个地市、逾百家政府单位的信息化系统提供重要支撑，成为其电子政务不可或缺的合作伙伴。在成功占领广东市场的基础上，阳光耐特近年来还积极开拓建设全国市场服务合作体系，致力于将旗下优秀的行业应用解决方案推向全国市场，迄今已在国内多地获得众多成功案例，打下良好市场口碑。

“十年磨一剑”。自创立以来，在国内纷纭复杂的信息化浪潮中，阳光耐特多年来一直专注于政法行业信息化的研发和实践，在业务上实现纵向贯通和横向整合，成功开发出一系列切合实战应用、强化业务管理的行业解决方案，并在广东省全省政法系统得到大规模的推广应用，为政法行业客户提供了长期、可持续的优质服务。

阳光耐特开发的治安卡口缉查布控系统、治安信息管理综合应用平台等多个核心业务系统，以及智能一体化信息预警和运维管理系统等，凭借其领先的技术、优质的服务以及行业经验，多年来在全省公安等政法行业获得充分认可。除此之外，公司还与国际知名公司、国内专业院校、解决方案提供商、业务服务商等建立了紧密的合作伙伴关系。

阳光耐特是国家高新技术企业，建立了ISO9001质量管理体系，近年来还承担了国家中小企业创新基金项目的研发。在服务现有客户、积极拓展市场的同时，公司坚持不断加大研发投入，紧紧围绕行业发展战略和核心业务发展趋势，提升技术储备和解决方案能力，以更好地满足客户需求。

 海阔凭鱼跃，天高任鸟飞。行业信息化的深入应用正在改变着21世纪人们的生活和工作，阳光耐特愿成为您信息化道路上可资信任的合作伙伴!
