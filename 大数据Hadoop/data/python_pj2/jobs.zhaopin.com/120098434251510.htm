举报收藏

岗位职责：

1. 负责电动汽车充电运营平台核心模块的设计及研发；
2. 参与平台技术选型及核心技术组件的研发；
3. 编写技术规范文档，并对应用开发人员进行代码review；
4. 基于平台技术及思想，对在实施的项目提供方案支持及指导
任职资格:
1. 计算机软件及相关专业，本科及以上学历；
2. 3年以上JAVA项目设计及研发经验；
3. 精通JAVA核心编程技术，精通设计模式，具备大并发服务后台设计思路；
4. 熟悉主流JAVA开源技术，包括：Spring MVC，Mybatis，IBatis等；
5. 熟悉http协议、tcp/ip协议、nio模式，并在开发的熟练使用；
6. 熟悉Linux、Tomcat、Netty、Apache、nginx等基本原理及使用；
7. 熟悉常见的NoSQL产品(memcached/Redis/MongoDB) ；
8. 熟练掌握MySQL、oracle等常用关系数据库使用；
9. 了解web前端开发，主要有javascript、css、jquery、ajax等，有ExtJS开发经验者优先；；
10.熟悉大数据相关经验优先

北京合康新能科技股份有限公司（集团）

北京合康新能科技股份有限公司，曾用名”北京合康亿盛变频科技股份有限公司”。其创建于2003年，隶属于北京中关村高科技园区，是一家专业从事工业自动化控制和新能源装备的高新技术企业。于2010年1月20日在深交所挂牌上市，证券简称“合康新能”,证券代码“300048”。截止2017年6月，合康新能注册资本11.02亿元，净资产达26.47亿元。

目前，合康新能下设4家全资子公司，31家控股子公司，一个重点实验室和一个技术中心。现有员工1800余人，其中核心科研及开发人员占28%，拥有遍布全国的办事处和完善的售后服务网络，产品销往全球六大洲20多个国家和地区。业务领域涵盖了工业自动化、新能源汽车、节能环保等领域，产品广泛应用于电力、冶金、矿山、水泥、石油、市政、电梯、机床、塑胶、核工业、航天风洞、新能源汽车、轨道交通、环保、光伏等行业。

在工业自动化领域，合康新能主要产品包括：高中低压变频器、电梯伺服系统及控制系统、高端装备制造核心部件及大驱动工业自动化整体解决方案。在变频器方面，由合康变频研发团队自主研发生产的水冷型高压变频器与矢量型高压变频器，已先后在多个大型的场所中运用并取得优异成绩，且获得了多项国家相关专利证书；在电梯技术方面，合康新能在德国建立联合研发实验室，引进了德国同步电机（PMSM）驱动技术，推出了业内专业级的HIC-L系列电梯伺服驱动器。经过13年的发展，公司凭借雄厚的技术实力、领先的生产工艺，树立了国内领先的变频节能和控制专家的形象，巩固了自身在行业内的引领地位。

在新能源汽车领域，合康新能提供新能源汽车动力总成、电机控制器、辅助动力系统、电动汽车智能充电桩及新能源汽车租赁平台建设等产品及服务。目前，在新能源汽车动力总成及关键零部件上，合康新能的动力总成产品已与国内多家主流车厂形成配套关系，并联合台湾捷准科技公司与西北高等院校伺服电机研究所共同合作研发永磁同步伺服电机。同时，合康新能集充电网、车联网、互联网的“三网合一”的技术，全方位打造智能充电站及新能源汽车租赁平台的搭建，已逐渐形成了“以车养桩、以桩稳车”的全新业务运营新模式。为更好完善产品研发测试体系，合康新能在武汉基地内成立了“武汉合康工业园综合检测中心”，为旗下各类产品提供强有力的技术支撑，提高产品的核心竞争力。经过不懈努力，合康新能已经成为新能源汽车动力总成核心部件及智能充电站运营的领军企业，旗下的新能源汽车租赁平台在业内也是独树一帜。

在节能环保领域，合康新能紧跟国家发展步伐，专注发展高效节能、先进环保、资源循环再利用等关键技术，采用EPC项目管理、EMC合同能源管理、PPP公私合营等方式致力于工业节能环保、资源综合利用等领域，大力推广光伏发电系统应用等节能环保一揽子完备的系统解决方案，整体解决方案独具一格，以诸多优势成为行业中的领导品牌。

合康新能是国家高新技术企业，掌握了高性能矢量变频技术、PLC技术、伺服技术和永磁同步电机等核心平台技术。截至目前，公司及旗下全资子公司拥有已获得软件著作权31项，及相关专利证书71项。其中发明专利14项，实用新型专利57项。经过多年发展，已在国内外工业自动化领域占据着重要的位置，是国内工业自动化控制领域的先锋军。

为了持续提升产能，促进企业可持续发展，合康新能相继于2010年11月迁址北京亦庄经济技术开发区，总面积为2.6万平方米，具备年产1200套高压变频器生产能力；2011年1月，在武汉东湖新技术开发区组建合康新能孵化器研发生产基地，总面积约为5.6万平方米。2015年1月，于长沙市高新区建立合康新能长沙研发生产基地，并逐步在上海、浙江、深圳等地成立研发中心。

合康新能立志通过技术创新、模式创新、协作创新，帮助客户提升能效水平，主动承担社会责任，争做良好企业公民，推动中国的绿色可持续发展。未来，合康新能愿携手广大客户和合作伙伴，积极探索能源新世界，努力让每个人都可以随时随地享受新技术带来的便利，让人们的生活更加丰富多彩和更加美好的未来。
