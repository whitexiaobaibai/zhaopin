举报收藏

岗位职责：

1. 负责大数据采集分析、系统需求分析和架构设计，以及方案的落地

2. 负责核心技术问题攻关，解决项目开发过程中的技术难题；

3. 负责大数据系统相关具体应用算法的设计、开发和产品化；

4. 能够根据公司项目和业务发展特点，研究相关大数据产品和技术发展方向；

5. 能够持续提升团队的技术氛围和技术能力。


任职资格：

1. 本科及以上学历、计算机或相关专业；

2. 具有至少5年以上软件开发工作经验，3年以上Hadoop实战经验；熟悉Hadoop底层文件系统及分布式计算框架，具备传统数据仓库、MPP、Hadoop等平台架构的设计能力和数据建模能力，对大规模数据并行计算、传输、处理等有丰富的经验；

3. 熟练掌握实时数据处理、离线数据处理、实时计算和离线分析等。

4. 熟练掌握大数据技术，如Hadoop、Spark、Flume、Kafka、HBase等，同时有丰富的Java开发和Shell脚本经验；

5. 卓越的技术领导力，善于沟通，良好的团队合作能力

6. 不怕困难，有关键技术攻关的决心和能力，能够适应和享受高强度高压力的工作；

7. 有互联网、金融行业工作经验者优先。


工作地址：

上海市徐汇区田林路140号越界创意园区28号楼4F-01D

和金在线(北京)科技有限公司 该公司其他职位

和金在线（北京）科技有限公司是由阿里健康前总裁兼CEO王亚卿于2015年9月创立的互联网保险公司。核心团队包括技术VP贺旭(原阿里巴巴高级研发总监-主管阿里健康云医院的产品和研发；原Oracle高级软件开发经理；二十余年IT业-Oracle/IBM/大型国企-专业经验)等、COO李超男（原上海新致软件有限公司副总裁；超10年的保险行业销售及管理经验；客户涵盖了排名前五十的保险公司过半数）、大数据总经理原雷（原阿里健康数据服务平台总经理，搭建数来宝药品电子监管数据服务平台，平台数据量达100TB，可秒级响应4000多家药品生产企业和13000多家药品流通企业的统计分析需求)。
目前公司已完成四轮数千万美元的融资。公司专注深耕于商业健康保险垂直领域，旨在通过APP、微信公众号等理赔服务平台，利用大数据技术为合作伙伴提供在线全流程全自动化健康险理赔服务，为合作伙伴的客户带来极致的保险理赔体验。同时，公司提供保险领域量身定制的大数据服务及信息技术解决方案，能为合作伙伴有效节省运营成本、降低理赔风险积淀理赔大数据、提升运营效率。公司目前已与多家保险公司，以及专业保险服务机构达成商业合作关系，为合作伙伴带来嵌入式、独立APP和SaaS等多样化的产品服务，满足不同商业保险及相关服务公司的不同需求。
