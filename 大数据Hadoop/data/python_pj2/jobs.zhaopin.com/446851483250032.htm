举报收藏

岗位职责：

1、与产品经理和UI设计师一起，研究并持续改善产品的质量、性能、用户体验 ；

2、与后端工程师紧密配合，高效率、高质量地完成页面展现研发工作；
3、负责前端框架设计及组件库开发、前端开发工作统筹；
4、负责对产品页面性能的优化和维护；
5、参与制定公司前端技术规范及流程规范，审核和评估相关方案与设计，以确保其符合架构规划，满足业务需求；
6、指导和培养前端研发团队。

    

任职要求：

1、本科及以上学历，专业不限。3年以上前端开发经验；
2、理解并掌握JavaScript语言核心技术DOM、BOM、Ajax、JSON等，能使用原生 javascript编写大部分功能组件，深入理解javascript的面向对象思想并能够应用到实际项目中；
3、精通HTML5/CSS3、XHTML/XML，熟悉页面架构和布局，对表现与数据分离、Web语义化等有深刻理解；

4、在多浏览器兼容性问题及css/JavaScript性能优化方面有一定的经验；
5、熟练掌握常见前端框架及工具、库，如 jQuery、Vue.js、Webpack、requirejs；
6、熟悉 HTTP 协议，了解服务器端编程；
7、有Echars、Highcahrts、d3js等数据可视化框架开发经验优先；
8、具有良好的逻辑分析能力、沟通能力和协调能力；具备强烈的进取心、求知欲及团队合作精神。工作态度积极主动、细致、有全局观。

集团介绍

    绿欣科技发展（北京）有限公司（简称“绿欣集团”），成立于2016年11月。公司依托自有技术与北京大学技术转化，整合大数据产业资源，为政府及企事业单位提供一站式大数据服务，业务方向涵盖大数据采集、挖掘、分析与应用，大数据安全、云计算与虚拟化，智慧城市等领域。

    绿欣集团是北京大学信息科学技术学院官方授权的科研成果转化平台，是北大科技园战略合作伙伴，三方共同发起成立了“北大-绿欣联合创新实验室”。目前集团布局全国，在北京、成都、天津、西安、深圳、金华、台州、包头、石家庄等地成立了区域子公司及分支机构（含筹建），并秉承“本土化”的服务模式，为全国各地客户提供包括大数据服务在内的各类IT服务与解决方案，多方位满足政府及各类企事业单位的信息化需求。

    旗下绿欣燕信基于大数据技术的“橙子安全”移动安全解决方案可为移动互联网相关用户及企业提供多样化安全保障服务。
