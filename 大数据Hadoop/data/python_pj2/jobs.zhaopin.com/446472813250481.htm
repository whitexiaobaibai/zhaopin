举报收藏

职位工作内容：

1. 负责公司支付、金融和账户产品的统一风控平台的整体设计和迭代发展

2. 整合公司各业务条线的数据资源，统筹架构和设计有效的智能大数据产品

3. 将智能数据及风控平台进行统一规划，打造现在支付自己的智能数据风控产品

4. 领导智能数据风控产品的发展，使之能对外服务好商户、对内有利于公司决策

5. 培养和建设智能数据风控产品团队，形成完整的产品设计、实施、推广体系

 

职位技术要求：

1. 5年以上专业产品经理经验，有知名互联网公司经历者优先

2. 有过完整的互联网产品线的从0到1的建设和运营经历

3. 对产品经理的完整工作流程和规范有行之有效的执行方法

4. 对支付或者金融风控、大数据的一种或者多种有成熟的经验

5. 有较强的分析和逻辑思维能力，有强烈的责任心、沟通、协调能力

6. 对新产品及新领域充满热情，乐于接受新事物的挑战

现在支付简介

 

现在（北京）支付股份有限公司，成立于2011年，致力于支付创新的高科技股份制公司。公司注册资金1000万元，目前在职员工200余人，分别在上海市、天津市、广东省、湖北省、辽宁省、福建省设有分公司。

 

现在支付定位为商户服务，通过自主研发创新，为商户提供综合性的支付服务，互联网运营服务及金融服务。公司已经在全国中小企业股份转让系统（即：新三板）挂牌上市，证券代码：832086，是一家充分市场化的公众公司。

 

现在支付聚焦为移动电子商务提供“安全、快捷、方便”的移动支付解决方案。公司拥有一批原谷歌、微软等高科技公司技术骨干，扩充坚实、可靠的技术力量，同时作为中国银联、支付宝、微信、百度及各大商业银行坚定的合作伙伴，已推出成熟的移动支付平台解决方案，持续不断为合作伙伴提供科学的流程设计，人性化的支付体验和全面的整合营销服务，并在移动支付领域取得突破性进展。自2011年开展业务以来，移动支付合作商户超3000余家，商户月交易额突破20亿元。是银联手机支付业务全国合作伙伴，全国移动支付领域的领军企业。

 

同时，公司亦提供线下支付服务。将银行卡收单与扫码支付相结合，为商户提供软硬结合、核销一体的聚合支付服务方案，自2012年投放市场，线下合作商户品牌数量累计已达12000余家。公司秉承服务为本的理念，为商户与客户之间提供便捷交易支付手段，提供全方位支付服务。

 

现在支付具有明显的移动互联网特质，基于商户移动互联网服务需求，陆续推出公众号、服务窗和直达号建设及运营服务，针对移动端的会员体系、营销功能、在线订购、卡券管理、核销一体等需求实现一站式管理，为商户提供轻型、快捷的移动互联网运营服务。另外，公司正在跟各大商业银行合作研发，筹备于2015年下半年推出金融服务产品，为商户提供更多的服务方案。

 

我们定位自身为一家快公司，公司一直与合作伙伴们保持紧密合作，先后与京东、小米、美团、百度糯米、魅族、携程、美丽说、OPPO、国美、百度、搜狐、陌陌、乐蜂网、豌豆荚、银泰、PPTV、PP助手、蓝港在线、赶集网、酒仙网、尚品网、蚂蜂窝等全国优质企业及互联网公司签订合作协议，为合作伙伴提供支付解决方案，推动公司业务快速发展。

 

现在支付真诚期待与各行业合作伙伴加深合作，共同致力于新一代移动支付的技术发展和市场推动，加速新一代移动商务及支付产业的进程。
