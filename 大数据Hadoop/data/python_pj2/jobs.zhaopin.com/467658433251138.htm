举报收藏

岗位JD：

1. 负责企业客户用餐现场的管理及客情关系维护，包括客户系统调试更新，现场服务，及时处理现场出现的突发问题；

2. 完成公司制定的年度、季度、月度维护额度计划；

3. 进行客户分析，建立并维护客户关系，挖掘客户深度需求，寻求新的合作模式及业务增长点；

4. 内部沟通协作，与市场部、产品部和客服中心紧密合作，确保客户达到百分之百满意；

5. 现场兼职人员管理。

 岗位要求：

1.女不限；

2. 头脑灵活、应变能力强，具备良好的沟通能力；

3. 熟练使用办公软件；

4. 一至两年酒店、餐饮等服务行业工作经验，有餐饮行业实习经验的优秀应届毕业生也可以接受；

5. 大专及以上学历。
