举报收藏

岗位职责：

1.参与过高并发量互联网项目的研发，并熟悉相关的技术；

2.熟悉JUnit单元测试；

3.熟悉设计模式，并且有一定的应用经验；

4.对工作中的问题有自己的思路和见解；

5.对新技术敏感，优秀的学习能力；

6.有一定的文档编写能力，有责任心和团队协作意识；

7.有一定的组织协调能力，能够带领小组完成项目；

8.薪酬面议。


职位要求：

1.扎实的JAVA基础，至少二年以上的JAVAEE项目开发经验；

2.熟悉Spring、Mybtis、Hibernate等开源框架，至少三年以上实际项目开发经验。

3.熟悉MySQL等数据库应用开发；

4.熟悉memcached、redis等cache技术；

5.有丰富的主流开源框架的使用经验，并深入研究过一种以上；

6.对当前广泛使用的tomcat、resin、nginx等HTTP Server 及 Application Server 有较丰富的应用经验；

7.熟悉Javascript前端开发，有AJAX开发经验，使用过主流JS框架如JQUERY等；

8.了解软件设计、开发过程，开发规范以及软件质量控制；

9.性格开朗，沟通协调能力强，具有团队合作精神；

10.计算机软件专业；

11.有微信公众号和APP开发经验者优先；




工作地址：

北京立新盈企大数据技术股份有限公司 该公司其他职位

        北京立新盈企信息技术有限公司成立于2006年3月，公司位于北京市朝阳区安翔北里甲11号北京创业大厦B座，为国家高新技术企业。公司的主要业务为大型网站开发与运营，以及为企业提供信息化综合解决方案服务，并提供咨询服务和技术支持，为客户创造价值。

      公司自主研发的主导产品可视化搜索引擎——图购（www.tusooo.com）是第一中文可视化搜索引擎，被IBM评为“2011年度中国最有创意的创业项目”，是针对宽带互联网时代新需求而提供的新型搜索服务模式。本产品运用了公司自主研发的大量专利技术，在搜索精确度和数据处理能力上取得了重大突破，公测的消息被6月17日的《科技日报》头版报道，市场发展空间十分广阔。
    
      历经数载的发展，今天的立新盈企信息技术有限公司已成长为一个汇聚国内外人才精英、拥有多项独立的知识产权的知名科技企业，拥有一支由博士、软件系统分析师、高级工程师组成的专业研发团队，与多家大型企业形成了稳定的、良好的合作关系。现诚聘各类专业人才，共同发展。
