举报收藏

岗位职责：

1.负责项目交付和管理，善于协调公司各方资源，制定并按期完成计划，保质保时为客户进行部署和配置，顺利完成项目验收和签收；

2.负责处理客户各类故障，提供应急响应和技术支持，熟悉客户网络环境和应用系统架构，协助客户完成问题分析和排查问题；

3.负责与客户的良好沟通和关系维护，并按期完成客户所需的分析报告的编写；

4.负责为客户提供在线、远程或者上门的技术支持服务。

任职要求：

1.本科以上学历，计算机及应用、网络、信息安全技术等相关专业；

2.3年以上系统基础架构实施或服务经验，有APM管理经验优先；

3.熟悉21v的Azure 平台，有相关工作经验；

4.熟悉Windows Server System Center 或者 zabbix监控系统；

5.熟悉Windows Server 和 Linux 操作系统；

6.具备相关资质证书优先，微软工程师，mcitp、mcp、mcse、或者ITIL证书等资质优先；

7.熟悉大数据分析相关技术，如Hadoop、Druid、Kafka、Zookeeper、Suro等优先；

8.工作积极主动，具有较强的敬业精神与工作责任心；

9.具备较强的独立分析问题及解决问题的能力，思维逻辑清晰，语言、书面表达能力强；

10.具备优秀的组织才能及决策能力，良好的沟通、协调、团队管理能力，较强的抗压能力及工作责任心；有项目管理经验者优先。

北京元恒时代科技有限公司 该公司其他职位

北京元恒时代公司（以下简称“元恒时代”）成立于2002年，一班年轻人矢志投身广阔的IT领域，以先进的技术和理念不断创造卓越的产品和创新的企业信息解决方案。今天，它已经发展成为一家集产品销售、技术服务、企业管理软件产品及开发于一身的高科技企业，服务范围涵盖系统规划、系统实施、信息安全构建、业务系统开发等全面的信息化解决方案。
作为世界上最大软件公司的微软公司金牌合作伙伴、微软ISV独立软件开发提供商，解决方案合作伙伴，并拥有微软授权认证的8项资质。同时还是VMWARE企业级合作伙伴、CITRIX银牌合作伙伴、趋势金牌合作伙伴、赛门铁克金牌合作伙伴等国际著名软件公司在国内的战略合作伙伴。
在具有世界知名软件产品供应和支持资源的同时，还拥有较高水平软件技术开发人员和服务队伍，已研发成功并上市Infinite Collaboration 2009、Infinite Flow 2009等协同办公平台（IC）、企业管理门户平台（EIP）和业务流程产品（IF）、项目管理、合同管理,在各行业内具有良好的知名度和信誉度。
北京元恒时代科技有限公司一直致力于为行业100强用户和国、内外500强用户提供优质的IT产品、解决方案和企业管理软件产品、技术。

元恒时代为客户提供以下三方面的服务：
1） 通用软件产品：
微软全线产品代理、VMWARE企业级合作伙伴、CITRIX银牌合作伙伴、趋势金牌合作伙伴、赛门铁克金牌合作伙伴
2）IT服务与咨询顾问：
元恒时代提供包括系统规划、系统实施、信息安全咨询在内的全面IT服务。一方面，我们为客户的系统提供实施和保障，使客户能够全力以赴，主攻核心业务；另一方面，我们帮助客户将数据转化为可以控制和分析的信息，做出快速、敏锐的决定，从而在决策竞争中脱颖而出。

3）  解决方案与技术开发
元恒时代相信，企业管理与信息技术互为促进，相辅相成。依托自己强大的开发队伍，我们从对客户的商业环境和需求分析入手，优化客户的商业策略和企业管理流程，并确保客户的系统规划与其商业策略的吻合。在此基础上，元恒时代为客户提供一系列以价值链管理为核心的解决方案，利用信息技术帮助客户提升企业管理水平和创造更多的商业价值。
