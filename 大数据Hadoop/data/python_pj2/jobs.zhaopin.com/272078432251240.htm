举报收藏

岗位职责:

1.深入理解公司业务发展需求，搭建公司风控体系和平台，参与研究和应用消费金融领域模型构建知识与技术；
2.对风控数据进行统计分析，发现数据中的相关关系，为风控模型创建并筛选变量，为风险管理提供决策支持；
3.对风控数据进行挖掘，利用统计分析、决策树、回归算法、机器学习或时间序列等技术建立信用风险预测模型，包括申请评分卡、行为评分卡、催收评分卡、反欺诈评分卡等；
4.识别消费信贷业务的风险，设计业务风险相关流程，建议风险策略（如反欺诈、审批授信）；
5.风险监控体系，定期产出风险分析报告；
6.外部产品数据采集与测试，外部数据风险分析；
7.参与和跟进与风险管理相关的模型、系统项目开发；
8.参与外部征信机构的数据的测试及应用策略制定。


任职资格:

1.大学本科以上学历，金融、数学、统计、计算机类专业优先；
2.良好的数学、统计学基础，扎实数据挖掘功底；
3.三年以上数据分析、数据建模等相关工作经验；
4.熟练掌握SQL使用， 有数据分析, 数据库管理、MIS报表搭建的相关经验，精通SAS或 Python ，能熟练运用其解决实际问题；
5.较强的分析问题和解决问题的能力；
6.有大数据风控方面工作经验者优先。

工作地址：

翼勋金融简介

上海翼勋互联网金融信息服务有限公司（简称翼勋金融）是一家互联网金融综合服务平台，注册资金1亿元，总部位于上海，公司服务范围已覆盖江苏、浙江、安徽、四川、辽宁等20多个省,70多个城市，16年7月翼勋金融在全国开设门店已达90多家，门店覆盖员工人数逾5000人。

翼勋金融专注于网络信贷撮合服务、消费金融等领域的发展，致力于用自身强大的平台建设，卓越的风控技术、大数据驱动能力为广大用户提供专业、简单、优质、便捷的普惠金融服务。

2016年，凭借良好的行业口碑及领先的金融科技技术， 翼勋金融成为上海互联网金融行业协会理事单位、上海金融信息行业协会副会长单位。

未来，翼勋金融将秉承“普惠金融，服务人人”的理念，为有融资、投资需求的广大用户建立更为高效、透明、安全、便捷的互联网金融服务，为创建一个便捷、健康、多元化的互联网金融生态圈而不懈努力。

二、我们的翼勋

拥有一支高学历、高素质的专业队伍，员工人数逾5000人，干部及专业人才中本科及以上学历占比80%以上，研究生占比达20%。
