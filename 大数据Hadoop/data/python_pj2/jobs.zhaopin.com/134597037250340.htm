举报收藏

岗位描述

1、  运用连锁门店研究的专业知识和经验， 辅助上级完成天会智数项目前期业务开拓，包含项目需求、客户跟进、项目建议书、合同等；

2、  发挥在连锁门店研究中所积累的针对门店不同生命周期的各类研究产品的专业和经验，完善天会智数的连锁门店的商业解决方案，优化公司产品线；

3、  负责项目的执行监督及管理，包括项目以及团队等；

4、  建立各个行业的数据库，并不断改进及优化。

任职要求：

1、  本科及以上学历，专业不限；

2、  五年以上市场研究行业的工作经验，三年以上门店产品研究经验，熟悉门店研究产品的生命周期；

3、  精通各类门店的研究方法，以及具备丰富的相关门店的知识积累

4、  出色的沟通表达能力和创新能力；

5、  具备一定的领导力；

6、  热爱行业，热爱互联网，热爱大数据。

上海天会皓闻信息科技有限公司 该公司其他职位

上海天会皓闻信息科技有限公司是中国发展最快速的网络调查营运者。公司成立于2006年，营业额不断高速递增，并且和世界上排名前列的传统市场研究公司建立了长期稳定的商业合作关系。

 

调研宝是中国领先的免费在线问卷调查平台。它由专业的市场研究人员设计，功能强大，并且具有零门槛操作的特色，是国内首款既满足专业人士又符合大众的在线问卷调查平台。
调研宝一直秉承“让决策更高效”的理念来帮助企业了解自己、了解客户、了解市场，帮助客户更快速、更准确的做出决策。
我们拥有一群充满激情和探索精神的人，使用创新的方法帮助企业搜集用户需求、态度和行为数据，挖掘其背后的商业价值，促进企业健康快速的发展。调研宝广泛使用于电子商务、快消、金融、汽车、IT/电信、人力资源等众多领域。
上海天会皓闻信息科技有限公司成立于2006年，旗下除<调研宝>以外，还包括为国际和中国大型企业提供个案调查业务的<天会调查>。

 

公司为员工提供优厚的薪酬待遇。对有特殊贡献者，公司提供超常规的职业发展机会和各种可能的方式让突出贡献员工享受到公司发展的成果，与公司共同成长。 我们热切地盼望有才华的您与我们同行。
