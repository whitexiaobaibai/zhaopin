举报收藏

岗位描述： 

1、对用户需求进行调研收集，撰写需求分析报告；

2、深入了解客户业务需求，领导数据建模、ETL、数据仓库及BI的设计及建设；

3、根据公司业务特点，构建全面的、合理的、准确的、能反映服务业务线特征的整体指标监控体系，搭建数据分析平台，及时发现与定位业务问题；

4、设计或集成数据可视化界面，结合分析体系，及时发现与定位业务问题；

5、跟客户进行有效沟通，理解客户业务需求，帮助客户定位技术建设方向，构建技术解决方案；

6、有良好的客户沟通能力，能够独立开展和支持项目工作。



任职资格：

1、本科及以上学历，数学、统计学、计算机或相关专业，5年及以上数据仓库相关工作经验；

2、精通SQL语言，精通Oracle、SQL Server、MySQL等市面流行的数据库产品的技术及操作；

3、精通数据仓库技术，熟悉Teradata、Greenplum等至少一种数据仓库产品的技术及操作；

4、深入了解ETL到数据仓库到BI的流程和实践；

5、熟悉Hadoop生态下各种开源工具，能够构建海量、异构数据的处理和分析平台；

6、熟悉建模方法，具有海量数据挖掘、分析相关项目实施经验优先；

7、有SAS、R等统计分析软件使用经验者优先。


工作地址：

威讯柏睿数据科技(北京)有限公司 该公司其他职位

威讯柏睿数据科技（北京）有限公司是一家实时大数据解决方案和产品提供商。公司专注于大数据实时处理的研发，为客户提供基于内存计算的海量数据存储分析平台。

柏睿数据自成立以来，发展极为迅速，创新性的推出了面向实时大数据分析领域的高性能分析应用平台Rapids Data Platform（简称RDP）。RDP采用了“内存计算(In-Memory Computing)”技术，从功能上进可以划分为：Rapids Hadoop（企业版大数据存储管理引擎）、Rapids DB（分布式实时在线处理引擎）、Rapids MGrid（分布式内存分析引擎）、Rapids Manager（统一的管理控制台）等4个模块，实时、迅捷的为客户挖掘隐藏在大数据当中的潜在价值。

在当今的大数据时代，数据基本上每天都在呈现爆炸式增长。大数据行业一些典型的应用场景包括：交通行业、运营商详单账单查询、运营商网络优化、实时视频监控、金融行业、电商行业、医疗卫生行业、工商行业、公安大数据以及税务大数据应用亟需寻找一种新的平台来帮助处理和完善数据。

一直以来，柏睿数据力图建立一个不同的数据公司，与应用软件提供商紧密合作，旨在为客户提供数据管理和使用的最佳体验。

借助Roray RDP平台，用户将会即时获想要的数据；拥有比传统方式快达3600倍的分析能力；更深入的视野、更翔实的数据颗粒度 ；并且易操作，更加节省费用，不仅减轻了IT的复杂度，还能帮助用户管理更多的数据容量。
