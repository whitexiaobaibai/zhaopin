举报收藏

岗位职责：

利用国双自有的一系列监测工具采集、分析客户网站、APP、媒体投放所生成的大数据。

2. 学会并精通国双自有监测工具的操作使用，解答客户数据、系统工具使用疑问。

3. 挖掘用户属性、访问行为、网站内容等数据，提出针对性的解决方案。

4. 分析媒体投放前后端数据，全面识别媒体投放效果，提出投放调整建议。

5. 沟通数据监测需求，协调技术提供代码或定制开发，沟通客户部署代码，确保项目正常进行。

6. 给客户制作并讲解数据分析报告，提供在线业务优化等基于大数据分析的技术解决方案。

任职要求：

1. 本科及以上学历，计算机、数学、信息管理或相关专业；应届生要求硕士学历。

2. 逻辑思维能力突出，熟练使用excel PPT等软件。 

3. 有一定编程基础，懂得SQL的优先。

4. 熟悉市场主流量化工具例如Google Analytics、webtrends、Omniture、Double Click者优先；

5. 做事认真仔细，积极主动，吃苦耐劳，有责任心、耐心，思路敏捷，反应能力强；

6. 有数据挖掘、媒体广告经验者优先。

一、关于国双

  北京国双科技有限公司是领先的基于云计算和大数据技术的商业智能解决方案提供商。
国双擅长构建并行数据仓库、进行实时数据分析及处理非结构化数据，搭建集成式的数据管理和营销平台，探索和认知数据的价值，拥有一套完整的自主研发的数据科学产品系列，囊括网站/移动端数据分析、互联网广告分析及管理、新媒体数据分析、数据集成管理平台、人群画像和营销旅程、网站用户体验优化、数据信息可视化、互联网全媒体信息挖掘等，拥有完善的数据科学产品和解决方案，在数字营销、新媒体、运营商、电子政务、司法等领域协助客户发掘数据价值。
国双服务的商业类客户遍布18个行业600多家国内外知名品牌，包括微软、京东、可口可乐、中国移动、上汽大众、耐克、宝洁等；在新媒体领域，与CCTV、BTV、江苏网络电视台等20多家省级电视台和网络电视台建立合作，为其提供全媒体大数据分析；在电子政务领域，2012年与国家信息中心联合成立网络政府研究中心，迄今已为全国3000多家政府网站提供大数据分析服务。
美国当地时间2016年9月23日，国双成功在美国纳斯达克全球市场挂牌上市，这意味着国双成为中国首家赴美上市的大数据企业，也是2016年以来第五家在美国上市的中国企业。
过去十年，国双专注于数据的收集、检索、分析和高速处理，为客户创造了基于数据的商业（人工）智能。凭借其在大数据领域积累的技术、数据和经验的优势，国双正在迅速切入多个垂直行业进行深耕，数据与专业业务深度结合所产生的爆发力正在逐步显现。
 二、国双产品介绍

1.数据科学产品

国双拥有一套完整的自主研发的数据科学产品系列，囊括网站/移动数据分析、广告数据分析及管理、用户体验追踪、归因引擎与数据整合、互联网舆情监测分析、新媒体数据分析、受众画像分析等七条产品线、十五个以上的数据科学产品，在数字广告、搜索营销、移动应用、电视媒体、运营商、电子政务、司法等七大领域提供基于云计算和大数据大商业智能解决方案。

2.国双技术能力

国双擅长构建海量数据仓库和进行实时数据分析。十年来，国双积累了多项大数据技术，包括分布式处理、分布式文件系统、数据仓库及ETL、流数据处理、数据可视化等技术能力，逐步形成完整的技术架构，即：大数据平台（Big Data Platform）、数据挖掘（Data Mining）、机器学习（Machine Learning）、自然语言处理（Natural Language Processing）和数据可视化（Data Visualization）。
