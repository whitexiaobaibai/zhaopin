举报收藏

岗位职责：

　　1、依据需求文档设计并编写测试用例；

　　2、根据测试计划搭建和维护测试环境；

　　3、完成产品的功能测试、性能测试；

　　4、了解自动化测试;

　　5、执行测试用例，反馈跟踪产品bug；

　　6、编写相关测试报告；

 

任职要求：

    1、计算机相关专业，三年软件测试经验，其中至少一年Web测试经验；

　　2、熟悉软件工程、软件测试理论和方法，熟悉功能测试用例的设计方法； 

　　3、熟悉常用测试工具（Loadrunner，QTP，selenium）；

　　4、熟悉一种性能测试工具，如：loadrunner；

　　5、熟悉一种缺陷管理工具，如：Jira、TD/QC等；

　　6、熟悉linux系统，熟悉java语言，熟悉大数据相关组件优先；

　　7、熟悉面向对象编程思想；

　　8、熟悉自动化测试框架，具有开发自动化测试框架者优先；有单元测试经验者优先

 

 

工作地址：

北京市朝阳区望京东路1号摩托罗拉大厦7层

新智数字科技有限公司 该公司其他职位

新智数字科技研究院（简称新智研究院），是以“从数据→智能，从智能到创新产业价值”为企业使命，致力于通过领先的人工智能和大数据技术，为消费侧和供给侧的传统企业/互联网企业赋能，推动相关产业/业务实现智慧升级的领先型技术研究与孵化机构。

坚持自主创新是新智研究院发展的源动力。目前，新智研究院已在北京、上海和美国硅谷设立了相应的研发机构，在人工智能（深度学习、机器学习）、云计算、大数据、区块链、物联网等相关领域进行了一系列技术布局。并与公安部交通管理科学研究所、上海交通大学、同济大学、北京航空航天大学、麻省理工人工智能实验室、英特尔公司等知名研究所、高校、企业在人工智能和大数据领域成立了联合实验室，共同探寻人工智能、大数据和云计算等技术在相关领域技术及应用的新突破。同时，新智研究院也进行相关产业与技术的前瞻研究，并以此为牵引，以投资方式对前沿技术和模式进行孵化。

新智研究院隶属新奥集团。新奥集团是国内领先的创新民营企业，截止2016年底，集团总资产超过1200亿，年收入超过1000亿元，员工总数近5万人。新奥集团涉及的行业主要涵盖能源、环保、健康、旅游、地产等领域，拥有三家国内主板/香港主板上市公司，一个国家重点实验室，两个国家工程实验室，经营地域涵盖国内160多个城市，并在北美、欧洲、澳洲、东南亚等地均有控股子公司独立运作。新奥集团为新智研究院提供了丰富的行业资源积累、应用场景和数据支撑。

新智研究院的核心团队均为原IBM、Google、微软研究院、Intel等公司的技术领军人才，有多年创业经验；同时新智研究院有多位能源、公安、交通等行业领袖、专家供职或作为顾问。
