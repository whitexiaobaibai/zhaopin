  • 职位月薪：30001-50000元/月 
  • 工作地点：深圳
  • 发布日期：2017-10-10 11:13:09
  • 工作性质：全职
  • 工作经验：10年以上

  • 职位类别：部门/事业部管理
  • 职位描述
  • 公司介绍

举报收藏

岗位职责：

1、根据公司整体经营策略，制定团队销售计划及市场拓展计划；

2、与行业纵队紧密配合，挖掘销售机会及商务谈判，最终签署项目，带领团队完成公司既定的销售目标；

 3、负责与客户商务谈判、合同签订、客户资料及款项的回收。

任职要求：

1、本科及以上学历，八年以上大数据相关工作经验；

2、五年拥有10个以上行业大数据相关项目案例并成功实施；

3、熟悉大数据相关领域，拥有丰富的人脉资源；

4、能够与政府客户进行商务谈判、关系维护；

5、擅于学习，了解大数据解决相关问题；

6、有较强的责任感，较强的理解能力及沟通能力，并能承受较大的工作压力；

7、能接受出差。

工作地址：

北京市海淀区王庄路1号清华同方科技广场D座西楼16层
